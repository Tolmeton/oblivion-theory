# 力としての忘却 (Force is Oblivion)

[English](README.md)

> 構造は保持されたものからではなく、失われたものから生まれる。

**忘却論** は、統計多様体上の忘却関手という単一の操作から、力・相補性・知覚・存在・エントロピー・時空を導出する 14 本の論文シリーズです。

## 核心の主張

**方向性定理** (Paper I, 定理 5.1):

$$F_{ij} \neq 0 \iff d(\Phi T) \neq 0$$

力は忘却が方向的に不均一なときに、そしてそのときに限り生じる。均一に忘れれば何も起きない。不均一に忘れれば曲率が現れ — その曲率が力として作用する。

- $\Phi$: 忘却場 (統計多様体上の KL ダイバージェンス)
- $T$: Chebyshev 1-形式 (Amari-Chentsov テンソルの Fisher 計量縮約)
- $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$: 忘却曲率

これは比喩ではない。情報幾何学における証明済みの数学的定理である。

## 数学的基盤

- **情報幾何学** (Amari, Chentsov): 統計多様体、α-接続、Fisher 計量
- **圏論**: 忘却関手、随伴 (F⊣G)、ガロア接続、Grothendieck トポス
- **自由エネルギー原理** (Friston): FEP は忘却論の α > 0 セクターへの制限 (Paper I §6.5.1)
