## 忘却論とはなにか

忘却論の核となる主張は一つである——「忘却」こそが、数学的構造や物理法則を成立させる先駆的な操作である。忘却論はこの操作を、圏論の随伴 自由 ⊣ 忘却(free ⊣ forgetful)として形式化し、力・知覚・存在・エントロピー・時空が、いずれもこの一つの操作の異なる射影であることを示そうとする理論である。

本理論では、熱力学が「外部へ仕事として取り出せる余地」を Helmholtz 自由エネルギーと呼ぶのと同じ意味の余地を、認知と情報の場へ一般化したものを自由として定義した上で、それと対を成す概念として忘却を定義する。

本理論において自由とは、ある観測の枠のもとで、差異・選択肢・解釈・行為可能性がまだ一つの構造へ収束しきらず、仕事・行為・解釈へ変換可能な余地として保たれている状態である。

それに対し忘却とは、その自由を減少させる向きの運動——状態の変動——である。状態間にある差異や差分を同値化し、識別可能性を失わせ、より変動の余地の少ない状態へ収束させる操作である。本理論は、その操作によって到達した状態も含めて、忘却と呼ぶ。圏論の言葉を借りれば、忘却関手は対象の存在を消さず、かつて別物だった射を同一視する。すなわち、情報自体は消えず、情報間の区別だけが薄れ消える。

言わば、忘却は自由を消費して収束し、自由は忘却されうる差異を開いて拡散する。圏論的には、自由が左随伴(自由関手、新しい区別を生成する側)、忘却が右随伴(忘却関手、区別を畳む側)をなす。自由と忘却、このどちらが欠けても系は動かない——自由だけなら状態は定まらず、忘却だけなら新しい状態の可能性を開けない。
