# 力としての忘却 (Force is Oblivion)

[English](README.md)

> 構造は保持されたものからではなく、失われたものから生まれる。

この repository は **忘却論** の公開読者版です。論文とエッセイを外部読者のために並べたアーカイブであり、内部の研究台帳や制作正本ではありません。

## 状態注記

この repository にある論文は、すべて **たたき台** です。未完成であり、
preprint より前、投稿準備稿より前、査読より前の構想段階にあります。
読解、批判、今後の発展のために公開しているもので、完成済みの学術論文として
置いているものではありません。

## 公開契約

この repository が担うもの:

- 読む、引用する、議論するための公開アーカイブ
- シリーズ論文、伴走ドラフト、番外編の curated snapshot
- 初見の読者の入口

この repository が担わないもの:

- 企画、出版順、裏側の改稿運用のための内部正本
- private ledger や作業用 meta 文書の置き場
- preprint archive や完成済み論文の記録
- 各たたき台が同じ成熟度に達しているという保証

現時点でこの公開版の中で最も強く素描されている形式的核は **Paper I** です。ただし Paper I もなお、完成稿ではなくたたき台です。後半の架橋論文にはさらに探索的・予想的な面が残るので、どの主張も確定済みとして読まないでください。

## 主張ラベル

主張は次のラベルで読まれることを想定しています。

| ラベル | 意味 |
|:--|:--|
| [定理] | 既存の形式枠組みの中で導出可能 |
| [予想] | 明確に定式化されているが未証明 |
| [構造的対応] | 強い類比や共通構造はあるが完全同値の証明は未了 |
| [哲学的示唆] | 概念提案であり形式化に未充足面が残る |

これらのラベルは、たたき台内部での主張の役割を示すためのものです。
周囲の論文が完成稿または投稿準備稿であることを意味しません。
