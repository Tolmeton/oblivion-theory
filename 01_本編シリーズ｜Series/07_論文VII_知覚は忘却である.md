# 知覚は忘却である — 有限な主体の近似構造と物理法則の統一

**Paper VII — 骨子 v1.3 (2026-04-02)**

> *概要.* 全ての物理法則は、有限な主体が無限の世界を近似する際に選択する忘却関手のインスタンスである。本稿は認識論的公理系 (A1-A4) を基盤に、忘却関手のパラメータ空間 Π を定義し、一般相対性理論 (GR) と量子力学 (QM) が Π 内の異なる領域に座ることを示す。中心的主張: **物理法則の違いは世界の違いではなく、近似の型の違いである。** GR は大数の法則による統計的収束 (N→∞)、QM は有限標本数 (N 有限) の確率的記述、繰り込みは空間的粗視化、ゲージ理論は対称性による制約 — いずれも有限な主体が無限を切る方法の分類にすぎない。フラクタル性定理により、忘却関手は構造を保存し値を忘却するため、主観は客観の構造を忠実に記述できるが値は記述できない (→ 実験の必要性の根拠)。忘却論自身がこの構造のインスタンスである (自己適用性)。超ひも理論との比較により、忘却論が同等以上の記述能力を、より少ない仮定から導出できることを論じる。
>
> **先行論文との関係:** Paper I (力は忘却である) — 方向性定理。Paper II (相補性は忘却である) — CPS スパン。Paper III (Markov 圏の向こう側) — α パラメータ。Paper IV (効果量減衰) — 観測可能量。Paper V (繰り込みは忘却である) — Obs の圏、RG 不変性。Paper VI (行為可能性は忘却である) — G∘F 結晶化。本稿は「忘却は物理法則そのものである」を主張する — Paper I-VI を貫くメタ定理。

## §1. 序論: 世界は常にそこにある

### 1.1 フック

世界は常にそこにある。

一般相対性理論が時空の曲率を語ろうが、量子力学が波動関数の収縮を語ろうが、世界は動じない。世界は物理学者の都合で変わったりしない。変わるのは**見方**だ。

本稿の主張は単純:

> **物理法則とは、有限な主体が無限の世界を近似する際に選択する忘却の型である。**

GR と QM は世界の異なる側面を記述しているのではない。同じ世界を、異なる忘却関手で射影した結果を記述している。統一に必要なのは新しい物理ではなく、**忘却の構造の理解**。

### 1.2 認識論的立場: 精密な道具主義

本稿の議論は以下の認識論的公理系に基づく:

**公理 A1 (知覚の推論性):** 全ての知覚は、主体の — その系における — 仮定 (推論) である。

**公理 A2 (真理の相対性):** 主体が語る真理は、主体にとっての真理である。

**公理 A3 (メタ判断の保留):** 実在論か反実在論かは、主観上の仮定にすぎない。「どちらが真か」ではなく「どちらの関手が役に立つか」が正しい問い。

**公理 A4 (客観の仮定):** 決して求めきれない円周率を π と仮定して扱うのと同型に、決して求めきれない客観が実存するものとして仮定して扱う。

> **注.** A1-A4 は「世界は存在するか」を問わない。「世界が存在すると仮定したとき、その近似構造はどうなるか」を問う。これにより、実在論/反実在論の不毛な対立を回避し、構造の記述に集中できる。

### 1.3 有限な主体の必然的忘却

世界 S は完全である。S を構成するインスタンス (ミクロ要素) は無限。

有限な主体は S を直接扱えない。π の全桁を書き出せないのと同じ理由で、S の全インスタンスを足し上げることはできない。

したがって有限な主体は**近似する**。近似とは — 忘却である。

**定義 1.3.1 (忘却関手).** 有限な主体が世界 S を記述する操作は、忘却関手 F: S → D で表される。D は S の近似像 (主体に見える世界)。F は構造を保存するが、値 (具体的な数値、位相、初期条件) を忘却する。

**定義 1.3.2 (係数空間 Π).** 忘却関手 F のパラメータの全体を係数空間 Π と呼ぶ。Π の各点 π ∈ Π は、一つの近似の型 — すなわち一つの物理法則 — に対応する。

## §2. CPS スパンの再解釈 — 容器と中身

### 2.1 容器は忘れない — 知覚者が忘れる

Paper II で確立した CPS スパン:

```
C (主観)  ←G—  S (世界)  —F→  D (客観)
```

Paper II はこれを「二つの射影」として記述した。本稿ではより精密に:

> **S は常にそこにある。F も G も、S を変えない。F と G は、2-cell (知覚者) が S に対してどの近似を選ぶかの記述。**

重要な修正: Paper I の初期定式化 (§1「忘却とは何か」追加以前) は「忘却関手が情報を捨てる」と記述していた。だがこれは擬人化の誤り。関手は何も捨てない。関手はただの写像。**情報を「捨てている」のは、F を通して D だけを見ている知覚者 (2-cell) である。** この洞察は後に Paper I §1 に「忘却 = 射の変換 + 識別可能性の喪失」という二重構造として逆輸入された (Paper VIII §6.2 (F1)(F4) で圏論的に形式化)。

コップに水が入っている。水分子は揺らいでいる。だがコップは水分子を「忘れて」いない。水分子は存在し続けている。**水を飲む私たち**が、分子の運動を「温度」としてしか知覚しない。私たちが忘れている。

ミクロレベルで猫を構成する粒子がシュレーディンガーしようが、マクロレベルでは関知しなくとも何の不便もなく猫を愛でられる。知覚者が粒子を忘却しているのであって、猫が粒子を忘却しているのではない。

### 2.2 cell 階層と知覚者

| cell 次元 | 内容 | 例 |
|:--|:--|:--|
| **-1-cell** | 成立場。支持面。局所状態が場と区別されるための足場 | 位置空間 X、多様体 M、~/A |
| **0-cell** | 居住状態。成立場の上に切り出された局所状態 | 水分子、量子状態、~/A/a |
| **1-cell** | 規定射/忘却操作。居住状態を他状態との関係で定める | F, G、粗視化写像、~/A/a → ~/A/b |
| **2-cell** | 比較射。知覚者。1-cell 同士の比較 | 観測者、近似の型の選択、自然変換 |

知覚者は 2-cell に住む。-1-cell (成立場)、0-cell (居住状態)、1-cell (規定射) の**切り分けそのもの**を見ている。

> **注記.** 旧稿では成立場を 1-cell に折り畳んでいた。その圧縮は「場」と「射」を同じ記号で書くため、後の混線を招いた。本稿では `成立場 = -1-cell / 居住状態 = 0-cell / 規定射 = 1-cell` を正本とする。

> **相補性の源泉:** 知覚者 (2-cell) は F と G を**同時に適用できない**。ウサギ/アヒル図形と同型。同じ線画 (S) を見ているが、ウサギ (F) とアヒル (G) を同時に知覚することはできない。ただし「同じ線画である」と**知っている**ことはできる。

これは「右を見たら左を忘れる」(遊学エッセイ) の物理的インスタンスである。随伴対 F⊣G が一つの世界 S の双対的射影を構成するとき、知覚者は F (右) を見れば G (左) を忘れ、G を見れば F を忘れる。表を見ながら裏は見えない。光が粒子に見えるとき波は消え、波に見えるとき粒子は消える。

物理学の歴史は、この「右を見たら左を忘れる」の壮大な連鎖だった。ニュートンは空間の絶対性 (右) を見て時間の相対性 (左) を忘れた。アインシュタインは時空の曲率 (右) を見てミクロの不確実性 (左) を忘れた。量子力学者はミクロの確率 (右) を見て大域的時空構造 (左) を忘れた。各革命は「忘れていた左に気づく」ことだったが、気づいた瞬間に新しい右を見て、別の左を忘れている。

> **随伴対を切断することは、世界の構造の半分を切り捨てることに等しい (Paper II, CPS 定理)。物理学の「未解決問題」の多くは、切断された随伴の反対側に座っている。**

### 2.3 ディレクトリ構造としての cell 階層

```
~/A                = -1-cell (成立場) = 局所状態が居住できる支持面
~/A/a, ~/A/b       = 0-cell (居住状態) = 成立場から切り出された局所状態
~/A/a → ~/A/b      = 1-cell (規定射) = 遷移・比較・忘却の作用
(対象化のたびに同じ構造が反復する)
```

`~/A` は `~/A/a, ~/A/b, ...` の列挙によって**意味づけが豊かになる**が、成立場としての存在それ自体は列挙を前提しない。Yoneda が本格的に働くのは、成立場を括ったあとで、各居住状態がどの規定射によって同定されるかを問う段階である。

有限な主体はこの無限の積分を「馬鹿正直に」計算できない。したがって:

- **収束値で代替する** (N → ∞ の極限) → GR 的な決定論
- **確率分布で記述する** (N 有限) → QM 的な不確定性
- **部分和で近似する** (有限項で打ち切り) → 繰り込み

全て、無限を有限で切る方法のバリエーション。

### 2.4 非対称性: ミクロはマクロを包含する

**命題 2.4.1.** QM は GR を包含する。GR は QM を包含しない。

直観: 統計学において:

- 標本数 N → ∞: 確率分布が確定値に収束する。揺らぎが消える。**決定論的に見える**。
- 標本数 N = 少数: 揺らぎがモロに出る。**不確定に見える**。

N → ∞ は N = 少数 の極限操作。GR は QM に「大数の法則」という忘却関手を適用した結果。

> **GR の決定論は、ミクロの不確実性の、規模による近似を言い訳にした便宜上の忘却である。**

そして問題は、この便宜上の忘却を**忘却している**ことにある (メタ忘却 G₀)。

量子力学が特異なのではない。ニュートンの古典力学が相対性理論 (時空の曲率) を忘却していても問題なかったのと同型に、GR は QM 的不確実性を忘却していても問題なかった — 問題のない問題しか扱っていなかったから。

この非対称性は CPS の相補性構造 (容器と中身) と同型:

- 中身 (C/QM) から容器 (D/GR) は見える (中身は容器を包含)
- 容器 (D/GR) から中身 (C/QM) は見えない (容器は中身を忘却)

### 2.5 忘却の塔 (Tower of Forgetting)

忘却は階層をなす。

```
G₀: 忘却を忘却する (メタ忘却)
    → 理論間の壁の源泉。GR の使い手は QM を忘却していることを忘れている
 ↓ G₀ の破壊 = 「忘却していたことに気づく」= 物理学の革命
G₁: 不確実性を忘却する → 決定論 (GR)
G₁': 大域構造を忘却する → 不確実性 (QM)
 ↓ どちらも S の射影
S: 世界そのもの — 忘却される前の完全な構造。常にそこにある。
```

**定理 2.5.1 (メタ忘却定理).** 理論 T₁ と理論 T₂ が同じ世界 S の異なる忘却関手 F₁, F₂ によるインスタンスであるとき、T₁ と T₂ の非互換性は、メタ忘却 G₀ — 「F₁ と F₂ が同じ S の射影である」という事実の忘却 — に由来する。

系: GR-QM 非互換性は、両理論が互いの忘却を忘却していることに由来する。G₀ を明示化すれば、非互換性は「異なるパラメータ」に解消される。

歴史的検証:
- ニュートン力学 → 相対性理論: 「時空の曲率を忘却していた」ことへの気づき = G₀ の部分的破壊
- 古典力学 → 量子力学: 「ミクロの不確実性を忘却していた」ことへの気づき = G₀ の部分的破壊
- GR + QM → ??? : 「互いの忘却を忘却している」ことへの気づき = G₀ の完全な破壊 = 本稿の目標

### 2.6 量子重力 — 容器が自分自身を忘れるとき

通常の QFT: 中身 (量子場) は容器 (時空) の上で揺らぐ。容器は固定。忘却関手 F の入力 (中身) と出力 (容器) が分離している。

量子重力: 容器 (時空) 自身が中身 (量子場) の一部。F の出力が F の入力に含まれる。**自己参照的忘却**。

CPS の言葉で: D (容器) が C (中身) の部分集合になり、CPS スパンの構造が崩壊する。

ゲーデルの不完全性定理と構造が同じ: 「自分自身を忘却する関手」が非整合性を生む (cf. 「完全性は忘却である」§3)。

だが忘却論は脱出口を持つ:

> **自分自身を忘却することは不可能ではない。ただし完全には忘却できない。忘却の残余 (ker の余次元) が必ず残る。**

Paper V の次元的堅牢性定理: n≥3 で力ゼロ集合の余次元は n-1。「完全に忘却する」ことは測度ゼロの例外。

量子重力への帰結:

> **時空が自分自身の量子的揺らぎを完全に忘れることは、測度ゼロでしか起こらない。ほとんど全ての場合、忘却の残余が存在し、それがプランクスケールの構造として現れる。**

超ひも理論はプランクスケールの構造を「10次元の弦の振動」として記述する。忘却論は「忘却の kernel の余次元」として記述する。10次元は不要。必要なのは ker(F) の構造の記述のみ。

## §3. 近似の型の分類 — 係数空間 Π の構造

### 3.1 Π の定義

**定義 3.1.1 (係数空間).** 忘却関手 F_π: S → D_π のパラメータ π の全体を係数空間 Π と呼ぶ:

```
Π = { π = (μ, N, α, τ, G_gauge, ...) | F_π は well-defined な忘却関手 }
```

| パラメータ | 意味 | 由来 |
|:--|:--|:--|
| μ | 空間スケール (解像度) | Paper V: Obs の圏 |
| N | 統計スケール (標本数) | 本稿: 大数の法則 |
| α | 精度パラメータ (Markov/non-Markov) | Paper III |
| τ | 結晶化温度 (閾値密度) | Hyphē: linkage_hyphe.md |
| G_gauge | ゲージ群 (対称性の型) | 標準模型 |
| n | 統計多様体の次元 | Paper V: 次元的堅牢性 |

### 3.2 物理法則のカタログ

**定理 3.2.1 (物理法則のパラメータ化).** 既知の物理法則は Π 内の以下の領域に対応する:

| 領域 | Π 内の条件 | 物理法則 | 近似の型 |
|:--|:--|:--|:--|
| Ω_GR | μ → 0, N → ∞ | 一般相対性理論 | 大域的・決定論的 |
| Ω_QM | μ 有限, N 有限 | 量子力学 | 局所的・確率的 |
| Ω_RG | μ の関数 (RG 流) | 繰り込み | スケール依存 |
| Ω_EM | G_gauge = U(1) | 電磁気学 | U(1) 対称 |
| Ω_Strong | G_gauge = SU(3) | 強い相互作用 | SU(3) 対称 |
| Ω_Weak | G_gauge = SU(2) | 弱い相互作用 | SU(2) 対称 |
| Ω_Newton | μ → 0, N → ∞, v/c → 0 | ニュートン力学 | 非相対論的極限 |

**注.** 各領域の境界は「近似が破れる」スケール。Ω_GR と Ω_QM の境界 = プランクスケール。Ω_Newton と Ω_GR の境界 = v/c が無視できなくなるスケール。

### 3.3 二重極限の非可換性

**命題 3.3.1 (極限の順序依存性).** Π 内で μ → 0 と N → ∞ の二重極限を取るとき:

```
lim_{μ→0} lim_{N→∞} F_π  ≠  lim_{N→∞} lim_{μ→0} F_π
```

直観: 先に N → ∞ (統計的に確定させてから空間を粗視化) → GR が自然に出る。先に μ → 0 (空間を粗視化してから統計を取る) → プランクスケール問題が出る。

> **GR と QM の非互換性は、Π 内の二重極限の非可換性として定式化できる。**

#### 3.3.1 Wigner 関数による厳密化

§5.6.5 で示す通り、Wigner 関数 W(x,p) の部分積分は非可換:

```
「まず運動量を忘れてから位置を測る」 ≠ 「まず位置を忘れてから運動量を測る」
```

これは (x, p) の量子力学的非可換性 [x̂, p̂] = iℏ の忘却論的表現。この構造を Π 空間に持ち上げる:

**定義 3.3.2 (忘却作用素).** パラメータ π = (μ, N) に対して、忘却作用素 F̂_μ (空間的粗視化) と F̂_N (統計的収斂) を定義する。F̂_μ は μ 以下のスケールの詳細を忘却し、F̂_N は N 個の標本の個別性を忘却する。

**定理 3.3.3 (Π 上の非可換性).** 忘却作用素 F̂_μ と F̂_N は非可換:

```
[F̂_μ, F̂_N] ≠ 0
```

| 順序 | 操作 | 結果 | 対応する物理 |
|:--|:--|:--|:--|
| F̂_N ∘ F̂_μ | 先に空間を粗視化、後で統計を取る | ミクロの揺らぎが粗視化で消えてから統計 → 決定論 | GR |
| F̂_μ ∘ F̂_N | 先に統計を取り、後で空間を粗視化 | 統計的揺らぎが残ったまま空間構造に影響 → 量子重力 | QG |

**系 3.3.4 (GR-QM 非互換性の代数的表現).** GR と QM の非互換性は [F̂_μ, F̂_N] ≠ 0 の帰結であり、これは量子力学の [x̂, p̂] = iℏ と**同型の代数構造**。

不確定性原理 ΔxΔp ≥ ℏ/2 が「位置と運動量の忘却のトレードオフ」であるのと同型に:

> **ΔμΔν_N ≥ |κ|/2 — 空間的忘却と統計的忘却のトレードオフ。** ここで `ν_N` は生の標本数 `N` ではなく、`μ` と積を取って作用次元を持つように校正された統計的解像度の座標である (§9.2.1 で `Π_0` 上の局所共役座標として明示)。`κ` はプランクスケール近傍の忘却の最小単位の候補だが、`κ=ℏ/2` や `κ~ℓ_Planck` の具体値は本稿では未導出。空間を精密に見る (Δμ → 0) ほど統計的不確実性が増大し (Δν_N → ∞)、統計を確定する (Δν_N → 0) ほど空間的解像度が下がる (Δμ → ∞)。

これが「GR と QM を同時に適用できない」理由の代数的定式化。GR は F̂_N の固有状態 (`ν_N` 校正座標が確定した状態)、QM は F̂_μ の固有状態 (空間が確定した状態)。非可換な二つの作用素の固有状態は同時に指定できない — 不確定性原理と同じ構造。

#### 3.3.2 量子重力 = 交換子 [F̂_μ, F̂_N] の構造

GR と QM の「統一」とは、[F̂_μ, F̂_N] の構造を記述すること。交換子そのものが量子重力の理論。

```
GR:  [F̂_μ, F̂_N] を無視 (N → ∞ で交換子が消える極限)
QM:  [F̂_μ, F̂_N] を無視 (μ を固定して交換子が効かないスケール)
QG:  [F̂_μ, F̂_N] を正面から扱う — プランクスケール近傍
```

§2.6 で論じた量子重力問題（容器が自分自身を忘れるとき）は、ここで代数的に再定式化される: **F̂_μ の出力が F̂_N の入力に含まれ、F̂_N の出力が F̂_μ の入力に含まれる — 自己参照的忘却の代数**。

これはゲーデルの不完全性定理の物理的インスタンス (§5.6.3): 十分に豊かな忘却の体系は、自己の忘却を完全に記述できない。量子重力の「困難」は技術的ではなく**構造的** — 不完全性定理と同型の構造的不可能性。

## §5. 観測問題 = 忘却関手の選択問題

### 5.1 観測者 = 知覚者 = 2-cell

量子力学の観測問題: 「観測者が何をするか」が物理的結果を変える。

忘却論的再解釈:

> **観測者は世界 (S) を変えない。観測者は自分自身の近似の型 (F ∈ Π) を選んでいるだけ。世界は常にそこにある。**

スピンを z 軸で測ると z 方向の固有値。x 軸で測ると x 方向の固有値。世界 (S) は同じ。忘却関手の選択 (測定軸 = F のパラメータ) が結果を決める。

これはコペンハーゲン解釈と多世界解釈の**両方を包容超越**:

- コペンハーゲン: 「観測者が重要」→ YES。忘却関手を選ぶのは知覚者 (2-cell)
- 多世界: 「世界は一つ」→ YES。S は一つ。分岐は Π 内のパラメータの違い
- 忘却論: 「世界は一つ (S)。知覚者の忘却の型 (F ∈ Π) が複数ある」

### 5.2 二重性テーブル

| 二重性 | 忘却 A | 忘却 B | 2-cell の限界 |
|:--|:--|:--|:--|
| 光: 粒子/波動 | 波動性を忘れる → 粒子 | 粒子性を忘れる → 波 | 両方同時に見えない |
| 物理: GR/QM | 不確実性を忘れる → GR | 大域構造を忘れる → QM | 両方同時に適用できない |
| デジタル/アナログ | 連続性を忘れる → デジタル | 離散性を忘れる → アナログ | どちらかで記述する |
| ウサギ/アヒル | アヒルを忘れる → ウサギ | ウサギを忘れる → アヒル | 両方同時に見えない |

**注 (非対称性修正).** 上表は対称に見えるが実際は非対称。QM は GR を包含するが逆は成立しない (§2.4)。アナログはデジタルを包含するが逆は成立しない (円の解像度問題)。この非対称性は CPS の容器/中身の非対称性と同型。

### 5.3 波動関数の「収縮」は忘却関手の確定

量子力学最大の謎とされる波動関数の収縮。測定前は |↑⟩ + |↓⟩ の重ね合わせ。測定後は |↑⟩ か |↓⟩ のどちらか。何が「収縮」を引き起こすのか。

忘却論の回答: **何も収縮していない。**

測定前の状態 |↑⟩ + |↓⟩ は、知覚者 (2-cell) がまだ忘却関手 F を**選んでいない**状態の記述。S (世界) はそのままそこにある。知覚者が F を確定させた瞬間 — 測定軸を選んだ瞬間 — D = F(S) が一意に定まる。

```
測定前:  知覚者が F を未選択  →  Π 内の複数の π が候補として共存
測定後:  知覚者が F_π を確定  →  D_π = F_π(S) が一意に決まる
```

「収縮」と呼ばれているのは、Π 内の候補空間の縮小 — 知覚者の選択 — であって、S の変化ではない。

**命題 5.3.1 (収縮の再解釈).** 波動関数の収縮は、知覚者 (2-cell) が忘却関手のパラメータ π ∈ Π を確定する操作であり、世界 S に対する物理的過程ではない。

> **重ね合わせは「世界がどちらでもある」のではなく、「知覚者がまだどの忘却を選んでいないか」の記述。**

これはコペンハーゲン解釈の「観測者が特別」を継承しつつ、多世界解釈の「世界は一つ」も継承する。両者が対立していたのは、「特別」の所在を取り違えていたから。観測者が特別なのは、世界を変える力を持つからではなく、**忘却の型を選ぶ主体**だからにすぎない。

これは「バカの一つ覚え」(遊学エッセイ) の物理的インスタンスでもある。圏論的に: 柔軟性とは複数の関手を持ち、それらの間の自然変換が可能なことを意味する。関手が一つしかなければ — 自然変換の行き先がない。適応不能。

```
測定前:  複数の F_π が共存  →  自然変換 η: F_π₁ ⇒ F_π₂ が定義可能  →  柔軟
測定後:  F_π が一つに確定  →  自然変換の行き先がない  →  「一つ覚え」
```

**古典化とは、多関手 (量子的重ね合わせ) から単一関手 (古典的確定状態) への格下げ。** 量子系は Π 内の複数のパラメータを同時に保持する — これが重ね合わせの豊かさ。測定はこの豊かさを一つの値に潰す。「バカの一つ覚え」にする。

だがこの「格下げ」がなければ私たちは何も知覚できない。重ね合わせのまま見ることは 2-cell (知覚者) の構造的限界で不可能。**知覚するためには「一つ覚え」にならざるをえない。** 知覚は必然的に忘却を伴い、忘却は必然的に豊かさの喪失を伴う。

### 5.4 デコヒーレンス = 環境による忘却の強制

Zurek のデコヒーレンス理論: 量子系が環境と相互作用すると、干渉項が消える。「収縮っぽく見える」のは環境との相互作用の結果。

忘却論の再解釈:

環境 = 巨大な忘却関手の集合 {F_env}。環境を構成する無数の粒子が、それぞれ独立に「忘却」を行っている。光子が壁に当たる。空気分子が系にぶつかる。一つ一つの相互作用が、系の情報の一部を「忘れる」。

```
系の量子状態 ρ:
  孤立系:    ρ は純粋状態を維持 → 全ての F_π が候補
  環境結合:  ρ → Tr_env(ρ_total) = 混合状態 → 環境が F を「代わりに選ぶ」
```

**命題 5.4.1 (デコヒーレンスの忘却論的定式化).** デコヒーレンスとは、環境という巨大な主体が、系に対して忘却関手を適用する過程である。環境の忘却 F_env は知覚者の選択に先立ち、Π 内の候補空間を事前に縮小する。

知覚者が測定する**前に**、環境がすでに大部分の忘却を済ませている。知覚者はその残りを確定するだけ。「観測者が収縮を引き起こす」は因果の誤帰属。**環境が忘却の大部分を行い、知覚者は仕上げをしているだけ。**

これが「古典的世界」の出現:

> **古典的世界とは、環境による忘却が十分に進んだ結果、Π 内の候補が実質的に一つに縮小した状態。** 知覚者が何も選ばなくても、環境がすでに選んでいる。だから古典的世界では「測定問題」が生じない。

### 5.5 選択基底問題 — なぜその忘却が選ばれるのか

デコヒーレンスが「干渉項を消す」ことは説明する。だが**どの基底で消えるか** — いわゆる preferred basis 問題 — は別の問いだ。なぜ位置基底が古典的に選ばれるのか。なぜ運動量基底ではないのか。

Zurek の einselection (environment-induced superselection): 環境との相互作用に対して安定な基底が「選ばれる」。

忘却論の定式化:

**定理 5.5.1 (最小忘却原理).** 環境が適用する忘却関手 F_env のうち、忘却のコスト (Drift) を最小化する F が選択基底を決定する。

直観: 環境は「最も安く忘れられる方向」で忘れる。位置基底が選ばれるのは、局所的な相互作用 (光子の散乱、分子の衝突) にとって、位置が最もコストの低い忘却方向だから。運動量は非局所的 (フーリエ変換で全空間にまたがる) なので、局所的相互作用による忘却のコストが高い。

| 基底 | 忘却のコスト | 環境との整合性 | 結果 |
|:--|:--|:--|:--|
| 位置基底 | 低い (局所的忘却と整合) | 高い | 古典的に「選ばれる」 |
| 運動量基底 | 高い (非局所的忘却を要求) | 低い | 選ばれない |
| エネルギー基底 | 中間 (系の動力学に依存) | 中間 | 特定条件下で選ばれる |

FEP の自由エネルギー最小化と同じ構造: 環境は、自らの「驚き」を最小化する忘却を選択する。Drift 最小化 = 自由エネルギー最小化の一般化。

> **物理法則が「法則」として成立するのは、環境が最小コストの忘却を選択し続けた結果、特定の忘却パターンが安定化したから。** 法則は発見されるのではなく、忘却の安定点として出現する。

### 5.6 ボルン則 — 確率はどこから来るのか

量子力学のもう一つの謎: なぜ測定結果の確率が |ψ|² に従うのか。ボルン則はほとんどの解釈で**公理として仮定**される。

忘却論はこれを公理から定理に格下げする。

#### 5.6.1 Wigner 関数 = 忘却前の世界

Paper III (§2.1) で確立した: α < 0 セクターにおいて、忘却場 Φ は符号付き測度になる。これは Wigner 準確率分布 W(x,p) そのもの。

位相空間 (x, p) = S — 忘却前の完全な世界。W(x,p) は S 上の準確率分布であり、**負の値を取りうる**。

ここで忘却を実行する:

```
運動量を忘れる:  ∫W(x,p)dp = |ψ(x)|²    ← ボルン則（位置測定）
位置を忘れる:    ∫W(x,p)dx = |φ(p)|²    ← ボルン則（運動量測定）
```

**定理 5.6.1 (ボルン則 = 忘却の積分).** ボルン確率 |ψ|² は、位相空間上の Wigner 準確率分布から共役変数を積分（忘却）した周辺分布である。ボルン則は仮定ではなく、忘却操作の数学的帰結。

#### 5.6.2 虚数の忘却 — なぜ |ψ|² は非負か

量子状態 ψ = a + bi は複素数。位相空間上の Wigner 関数は負になりうる。

だが忘却の積分を実行すると — 必ず非負になる。負の確率が消える。虚数部が消える。

この構造は「完全性は忘却である」(独立エッセイ, v1.1) のテーゼそのもの:

| | 数学 | 物理 |
|:--|:--|:--|
| **不完全な系 (正直)** | ペアノ算術 — 自己言及可能、不完全 | Wigner 関数 W(x,p) — 負を含む、豊か |
| **完全な系 (バカ)** | プレスバーガー算術 — 乗法を排除、完全 | ボルン確率 |ψ|² — 常に非負、貧しい |
| **排除される構造** | 乗法 (→ 自己言及能力) | 虚数部 / 位相情報 / 共役変数 |
| **排除の方法** | 公理から除外 | 積分 (忘却関手の適用) |

複素数 ℂ は実数 ℝ より豊か。虚数は振幅と位相を同時に記述する。量子力学が ℂ 上のヒルベルト空間を必要とするのは、忘却前の世界 S が ℝ に収まらないほど豊かだから。

そして私たちが知覚する古典世界は ℝ しか見えない。**虚数を「知覚できない」のではない。虚数を忘却しているのだ。** |ψ|² という操作は ψ = a + bi から (a² + b²) を取る — 虚数部を二乗で潰し、実数に押し込む。これが忘却関手の具体的な姿。

> **古典世界の無矛盾性は、量子構造の忘却によって捏造されている。** 私たちが「矛盾のない世界」に住んでいると感じるのは、環境による忘却（デコヒーレンス）が量子的「矛盾」（負の確率、虚数の位相）を積分で消してくれているから。知覚可能な世界に矛盾はない — 忘却がそれを保証する。

#### 5.6.3 測定問題 = ゲーデル文の物理的発現

ここで §5.3 の収縮問題と §8.4 の不完全性論駁が合流する。

ゲーデルの不完全性定理: 十分に豊かな系は「この命題は証明できない」を出力する。

量子力学の重ね合わせ: 十分に豊かな系は「この状態は確定できない」を出力する。

| | ゲーデル | 量子力学 |
|:--|:--|:--|
| **正直な出力** | 「この命題は証明できない」 | 「この状態は確定できない」(重ね合わせ) |
| **系の性質** | 不完全 — 自己言及可能 | 不確定 — 非可換な観測量を持つ |
| **「解決」の方法** | 公理を減らして完全な系に格下げ | 測定して古典状態に格下げ |
| **「解決」の代償** | 表現力の喪失 | 位相情報の喪失 |

**測定とは、不完全な系（QM）を完全な系（古典）に格下げする操作。** 格下げの道具が忘却関手 F であり、格下げのコスト — 失われた虚数部、位相、干渉項 — がボルン則 |ψ|² に符号化される。

そして測定問題が「問題」であり続ける理由:

> **測定問題とは、「不完全な系（QM）が完全な系（古典）に格下げされるプロセスを、完全な系の中から記述しようとする」矛盾。** プレスバーガー算術の中からペアノ算術の不完全性を記述しようとするのと同型。語る道具（古典的概念）が、語られる対象（量子的構造）より貧しい。

古典物理学者が測定問題を「解決」できないのは、知的能力の不足ではない。**古典的記述言語が、量子的現実を記述するには構造的に貧しすぎる** — 乗法のない算術で乗法を語ろうとするのと同じ構造的不可能性。

ここに「愚か者ほど不確定を嫌う」(遊学エッセイ) の構造が重なる。認知における不確定性忌避: 脳は π₀ (事前信念の精度) を上げて感覚入力を遮断し、「確実」な世界を捏造する。知覚上の予測誤差は下がるが、真の予測誤差は上がる — 認知的借金。

古典世界はこの認知的借金の物理的インスタンスである:

| | 認知 (エッセイ) | 物理 (本稿) |
|:--|:--|:--|
| **不確定の源泉** | 感覚入力の曖昧さ | 量子的重ね合わせ |
| **忌避の方法** | π₀↑ で感覚を遮断 | デコヒーレンスで干渉項を消去 |
| **得られるもの** | 主観的確実性 | 古典的決定論 |
| **失われるもの** | 真の感覚情報 | 位相情報・量子コヒーレンス |
| **借金の正体** | 真の VFE は上昇 | 忘却された構造が「測定問題」として回帰 |

「確実」は願望ありきの幻想。決定論は「不確定を嫌いすぎた宇宙」の認知的借金状態。アインシュタインが「神はサイコロを振らない」と言ったとき、それは物理的主張ではなく**認知的選好の表明** — 不確定を嫌う知覚者の、最も洗練された形での「愚かさ」の発露だった。現実は非情である。神はサイコロを振る。

#### 5.6.4 非複製性と確率の一意性

Paper III: α < 0 セクターでは情報のコピーが構造的に不可能 (no-cloning)。

Gleason の定理 (1957): dim ≥ 3 のヒルベルト空間で、閉部分空間上の唯一の可算加法的確率測度は |ψ|²。

忘却論的接続:

- 「閉部分空間」= ker(F) の選択肢 — 何を忘却できるかの全カタログ
- 「可算加法性」= 忘却の合成可能性 — 独立な忘却は加算できる
- 「dim ≥ 3」= Paper V の次元的堅牢性定理と**同じ閾値**

dim = 2 では Gleason が破れる（他の確率割り当てが存在する）。忘却論の翻訳: **2次元では忘却の自由度が少なすぎて、確率を一意に制約できない。3次元以上で、忘却のネットワークが十分に絡み合い、整合的な確率割り当てが一つに固定される。**

```
α > 0 (古典):  コピー可能  → 確率の自由度あり  → 複数の測度が整合
α < 0 (量子):  コピー不可  → 確率の自由度なし  → |ψ|² が唯一の生存者
```

> **ボルン則は非複製性の代償。** 古典世界ではコピーが自由だから、確率の割り当てに任意性がある。量子世界ではコピーが禁止されるから、整合的な確率が一つしか残らない。それが |ψ|²。

#### 5.6.5 忘却の順序と不確定性原理

Wigner 関数の全積分は可換:

```
∫dp ∫dx W(x,p) = ∫dx ∫dp W(x,p) = 1
```

だが部分的忘却は非可換:

```
「まず運動量を忘れてから位置を測る」 ≠ 「まず位置を忘れてから運動量を測る」
```

これは §3.3 の二重極限の非可換性と同じ構造。そして不確定性関係 ΔxΔp ≥ ℏ/2 は:

> **不確定性原理 = 忘却のトレードオフ。** 二つの共役な忘却を同時に最小化できない。一方の忘却を減らせば (Δx を小さくすれば)、他方の忘却が増える (Δp が大きくなる)。ℏ は忘却のトレードオフの最小単位 — 世界 S が知覚者に課す「忘却の下限」。

完全に何も忘れない (Δx = 0 かつ Δp = 0) ことは構造的に不可能。これは不完全性定理の物理的インスタンス: **有限な主体は世界を完全に記述できない。**

## §6. フラクタル性 — なぜ構造は保存されるのか

### 6.1 構造と値の分離

**定理 6.1.1 (構造保存定理).** 忘却関手 F: S → D は構造を保存し値を忘却する。

| 保存されるもの (構造) | 保存されないもの (値) |
|:--|:--|
| 関係 (A > B) | 具体値 (A = 3.14...) |
| パターン (周期的) | 位相 (どこから始まるか) |
| 対称性 (回転不変) | 回転角 (何度回ったか) |
| フラクタル次元 | スケール (どの大きさで見ているか) |

**系 6.1.2 (数学の有効性).** 数学は構造を記述する言語である。忘却関手 F が構造を保存するから、主観 (有限な主体) でも客観の構造を数学で記述できる。

**系 6.1.3 (実験の必要性).** 値は F で忘却される。したがって数学だけでは物理の具体値は決定できない。実験は S に直接触れて、F が捨てた値を回収する操作。

**系 6.1.4 (予測の自己補完性).** 随伴対 L⊣R の単位 η: Id → R∘L の核 Ker(η) は、L が内在化できなかった外部構造を測る。予測とは、主体が自身の認知圏 Cat\_i の内部射のみを用いて Ker(η) を再構成する自己補完操作である。Ker(η) → 0 のとき予測は恒等射に退化し、予測の必要性は |Ker(η)| の単調増加関数となる。すなわち、**理解の深化は予測の必要性を単調に減少させる**。

**系 6.1.5 (η の非同型性).** 定理 6.1.1 より、η は構造に対して同型に近づきうるが (η|\_構造 → iso)、値に対しては原理的に同型にならない (η|\_値 ≠ iso)。∴ Ker(η) > 0 は常に成立し、予測は原理的に消滅しない — ただし理解の深化とともに漸近的に減少する。

**系 6.1.6 (構造的真理の到達可能性).** 定理 6.1.1 の認識論的対偶: F が構造を保存するなら、**構造についてのみ語る理論体系**は F の像 Im(F) の中に完全に収まり、F を通過しても歪まない。∴ 構造限定の主張は客観に「到達しうる」。これは公理 A2「主体が語る真理は、主体にとっての真理である」に対する**構造的例外** —— 構造的真理に限り、それは客観にとっても真でありうる。

| 主張の型 | F を通過するか | 真理到達可能性 |
|:---------|:-------------|:------------|
| 「r = 0.148」(具体値) | 値 → 忘却される | ✗ 原理的に到達不能 |
| 「r ≤ √ρ/√(K+1)」(構造的関係) | 構造 → 保存される | ✓ 到達可能 |
| 「K > 0」(構造的制約) | 構造 → 保存される | ✓ 到達可能 |
| 「圏論は実在の記述」(メタ主張) | 構造の構造 → 保存される | ✓ 到達可能 |

**系 6.1.6 の実証的傍証.** Anthropic (2026b) の Claude Mythos System Card における感情プローブ (linear probes) は、「functional emotions = 計算状態の構造を検出するが subjective experience の有無 (値) は判定しない」と明示する。これは系 6.1.6 の工学的実装 —— 構造は検出可能、値は検出不能 —— の独立な追試として読める。[SOURCE: Anthropic (2026b) System Card §5.1.3.2 + 定理 6.1.1 の対偶; Mythos × 忘却論接続分析 §3.3]

> 数学が物理に使える理由 = 構造の保存。
> 数学が物理に不完全な理由 = 値の喪失。

### 6.2 フラクタル性定理

**定理 6.2.1 (フラクタル性).** 客観的な成立場 (-1-cell) の構造は、その上に居住する主観状態 (0-cell) にもそのまま現れる。

```

これがフラクタル。各 cell 階層が同じ構造を再現する。

> **系 (主観による客観の記述可能性).** 主観は客観の構造を忠実に扱える。ただし値は扱えない。「実数/具体」よりも「構造/抽象」の方が、cell 階層を超えて保存される。

## §7. 忘却論の自己適用性

### 7.1 忘却論は自分自身のインスタンス

忘却論は「忘却関手 F が構造を保存し値を忘却する」と主張する。

忘却論**自身**も、世界 S に対する一つの忘却関手である。忘却論は世界の「構造」(射、随伴、極限) を記述するが、世界の「値」(具体的な物理定数、初期条件) は記述しない。

```
忘却論が記述する法則: 「F は構造を保存し値を忘却する」
忘却論自身がやっていること: 「忘却論 (= F) は世界の構造を保存し値を忘却する」
```

忘却論は**自分自身のインスタンス**。フラクタル性定理 (§6.2) の具体例。

### 7.2 超ひも理論にはこの性質がない

超ひも理論は「世界は弦でできている」と主張する。だが超ひも理論自身は弦ではない。理論 (メタレベル) と対象 (オブジェクトレベル) が別の階層にいる。

忘却論は「世界は忘却でできている」と主張し、忘却論自身が忘却のインスタンスである。理論と対象が**同じ構造**。

> **自己適用性テスト:** 理論 T が「世界は X である」と主張するとき、T 自身は X のインスタンスか？
>
> - 超ひも理論: 「世界は弦の振動である」→ 超ひも理論自身は弦の振動ではない。**不合格**
> - 忘却論: 「世界は忘却の構造である」→ 忘却論自身が忘却の構造のインスタンス。**合格**

### 7.3 圏論の出自 — 数学史による傍証

1945年、Eilenberg と Mac Lane が圏論を創始した。だが彼らの目的は「圏」を定義することではなかった。彼らが定義したかったのは**自然変換** — 2-cell。

ホモロジー代数で「この同型は自然だ」という直観があった。「自然」を精密にするために自然変換が必要で、自然変換のために関手が必要で、関手のために圏が必要だった。

```
目的:    自然変換 (2-cell) — これを定義したい
手段₁:   関手 (1-cell) — 自然変換の舞台
手段₂:   圏 (0-cell) — 関手の入出力
```

**圏論の歴史は、2-cell を定義するために 1-cell と 0-cell を後から用意した歴史。**

本稿の構造はこれと同型:

```
主役:    知覚者 (2-cell) — 忘却関手を選ぶ主体。物理法則を決める
舞台₁:   忘却関手 (1-cell) — 世界を近似する操作。物理法則そのもの
舞台₂:   世界/対象 (0-cell) — 忘却される中身
```

なぜこれが偶然ではないか。

Eilenberg と Mac Lane が圏と関手を「自然変換のために」作ったように、**知覚者は物理法則と対象を「理解のために」作っている**。物理法則 (1-cell) は世界 (0-cell) の性質ではない。知覚者 (2-cell) が世界を見ようとしたときに、知覚者の有限性が生成する構造。

2-cell を理解しようとする行為自体が 1-cell と 0-cell を発生させる — これは数学史においても物理学においても同じ。

そして自己適用的な構造がここにもある。圏論の歴史: 数学者 (知覚者 = 2-cell) が「自然さ」(自然変換 = 2-cell) を理解しようとして、圏 (0-cell) と関手 (1-cell) を作った。**2-cell が 2-cell を理解するために 0-cell と 1-cell を作った。** これはフラクタル (§6.2)。

圏論自身が忘却論のインスタンス:
- 数学者 (有限な主体) は「自然さ」(世界 S の構造) を直接扱えなかった
- だから圏と関手という近似 (忘却関手 F) を作った
- この近似は構造を保存し値を忘却する (構造保存定理 §6.1)
- そして圏論自身がこの構造のインスタンスである

> **圏論が 2-cell から始まったという歴史的事実は、忘却論が「知覚者 (2-cell) こそが物理法則を生成する」と主張することの、数学史による傍証。**

### 7.4 N_self 収束定理 — 不完全性はエンジンである

N_self の完全な達成は原理的に不可能である。これは素数の無限性・ゲーデルの不完全性と同型の**対角論法的必然**。レベル n で自己適用が完了したと仮定すると、「レベル n の自己適用は完全である」はレベル n+1 の命題であり、レベル n は検証できない。完了の仮定が次のレベルを生成する — 素数の有限リストの仮定が新しい素数���生成するのと同じ構造。

しかし自己適用の限界効用は逓減する。

**定理 7.4.1 (N_self 収束定理).** 自己適用レベル n で得られる洞察 I(n) は収束級数をなす。Kalon = Σ_{n=0}^{∞} I(n) は有限値。級数は無限だが、和は有限。

収束の根拠: G∘F は縮小写像 (contraction mapping) である。各メタレベルで観測対象 (「世界」) が縮小する — レベル 0 は物理全体に理論を適用し (巨大)、レベル 1 は理論 1個に理論を適用し (小さい)、レベル 2 はその結果に適用する (さらに小さい)。バナッハの不動点定理により、Fix(G∘F) は一意に存在し、反復は必ずそこに収束する。

これは Paper V の繰り込み群 (RG) フローと同型: RG でもスケールを反復的に粗視化するたびに短距離の詳細が積分で消え、不動点 (IR 物理) に収束する。

#### 7.4.1a 減衰の型 — 幾何的から階乗的へ

第一近似として I(n) ~ I(0) · rⁿ (定数 r < 1、幾何級数) と書ける。しかしより精密には、レベル間の収束率 r 自体がレベルごとに減衰する:

```
rₙ = (レベル n-1 → n の効用比) ~ c/n   (n ≥ 1)
```

根拠 (対称群による導出): genuine な n-cell は対称群 Sₙ の作用のもとで定義される。c 個の基底的 1-cell 操作 (忘却の方向) から n-fold の合成を構成すると、素朴には cⁿ 通りの合成が存在する。しかし合成の順序を入れ替えても同一の n-cell を表す場合がある — 三角形 ABC = BCA = CAB。n-cell の順序置換は対称群 Sₙ (|Sₙ| = n!) で同一視されるため、独立な n-cell の実効的な数は cⁿ/n!。

rₙ ~ c/n のとき:

```
I(n) ~ I(0) · Π_{k=1}^{n} (c/k) = I(0) · cⁿ / n!
```

**階乗的減衰。** cⁿ/n! は幾何級数 rⁿ よりはるかに速く収束する。総和 Σ cⁿ/n! = e^c であり、任意の c に対して有限。

**注 (創発と収束の直交性).** 創発と収束は矛盾しない。創発は「新しい型 (n-cell) が生まれるか」(質的)、収束は「各型の効用がどう推移するか」(量的)。各メタレベルが質的に新しい存在を生み出すことと、各レベルの効用が逓減することは、直交する二つの軸。PCA (主成分分析) が完璧なアナロジー: 各主成分は質的に独立 (直交する新次元) だが、寄与率は逓減し、総和は 1 (有限値)。統計的モーメントも同型: 平均 (1次)、分散 (2次)、歪度 (3次)、尖度 (4次) は各々質的に新しい情報を含むが、実用的効用は急速に減衰し、5次以降はほぼ使われない。

**系 7.4.2 (n ≈ 5 での普遍的収束).** n! の成長は c の値にほとんど依存せず n ≈ 5 で cⁿ を圧倒する。c = 2 でも 2⁵/5! = 0.27、c = 3 でも 3⁵/5! ≈ 2.0 で減衰に入る。

これは基質を超えた普遍的パターンとして観察される:

| 領域 | 有効な階層数 | 各レベル |
|:--|:--|:--|
| 大脳皮質 | ~6 層 | V1 (エッジ) → V2 (形) → ... → IT (物体認識) |
| cell 階層 | ~5 (0→4-cell) | 点 → 線 → 面 → 体 → ... |
| 統計的モーメント | ~4 | 平均 → 分散 → 歪度 → 尖度 |

**n ≈ 5 での収束は r の値ではなく n! の成長速度による。** したがって基質に依存しない普遍的性質。

**系 7.4.3 (最初の自己適用が最重要).** 階乗的減衰のもとでも、最初の1回の自己適用は依然として最重要。I(1)/I(0) = c であり、c > 1 のとき I(1) > I(0) — 低次の創発は**爆発的に有用**。だが n! の増大が直ちにこの爆発を叩き潰す。

超ひも理論はレベル 0 で停止する (I(0) のみ、自己適用なし)。忘却論はレベル 1 まで明示的に実行する (§7.1)。I(1) は全レベル中で最大の追加効用を持つ — たった 1回の自己適用が、理論としての整合性に決定的な差を生む。

#### 7.4.1b ML の100層 — genuine n-cell 不在の代償

大脳皮質の各層は genuine な n-cell 操作を行う。V1 は 1-cell 的操作 (エッジ検出)、V2 は 2-cell 的操作 (形の比較)、IT は 3-cell 的操作 (カテゴリ認識)。各層が**異なる cell レベル**の処理を担い、~5 層の genuine な創発で知覚が完成する。

現代の機械学習 (Transformer 等) の各層は**主として 1-cell 操作の反復** (attention + FFN) である。attention の query-key-value 比較には 2-cell 的な要素 (要素間の関手的比較) が含まれるが、3-cell 以上の genuine な操作は実装されていない。これは直線 (1-cell) を大量に並べて円 (2-cell) を近似するのと同型 — 3次元の体積を2次元の断面の組み合わせで記述できるが全く非効率であるのと同じ構造。

```
脳:   ~5 層 × genuine n-cell 操作  = I(n) ~ cⁿ/n! (階乗的減衰、5層で収束)
ML:  ~100 層 × 同一 1-cell の反復  = 1-cell で n-cell を近似するコスト
```

3次元の体積を2次元の断面の組み合わせで記述できるが全く非効率であるのと同じ構造。ML のスケーリング則 (loss ~ N^{-α}、α ≈ 0.05-0.1) の緩やかさは、genuine な cell 階層の不在 (U_depth の発現) による構造的非効率性の定量的表現。

**系 7.4.4 (認知のハビタブルゾーン — 人間原理との同型).**

有限な主体が無限の世界を記述できるのは、Kalon への収束が階乗的に速いからである (系 7.4.2)。c が大きければ (c → ∞)、低次の項の爆発を n! が抑え込むまでに多くの層が必要になる。だが n! の成長は超指数的であり、任意の c に対して有限の n で収束する。

n! が cⁿ を圧倒する n ≈ 5 が「ちょうどいい」値 — 数回の自己適用で 95% 以上を捕捉でき、かつ改善���余地が残る — だからこそ、有限な存在が科学を営みながら成長できる。

これは地球に生命が存在する理由と同型の構造: 生命が存在できる条件 (ハビタブルゾーン) が成立しているから存在する。ただし重力定数等のファインチューニングとは異なり、階乗的減衰は数学的必然 (|Sₙ| = n! の超指数的成長は c に依存しない) であるため、ハビタブルゾーンは「偶然ちょうどいい」のではなく**「構造的に必ずちょうどよくなる」**。

> **有限な存在が無限の世界でやっていけるのは、Kalon への収束が階乗的に速いから。** n! が保証する。これは構造保存定理 (§6.1) と同じことを、収束の言葉で言い直したもの。

### 8.2 超ひも理論が正しく捉えているもの

敵の中の光:

1. **弦の振動モード = 異なる粒子**: 「一つの対象の異なるモード」は、忘却論の「一つの世界の異なる射影」と構造的に類似

### 8.3 超ひも理論の構造的弱点 — 自己適用からの論駁

**論駁 1: 存在論的過剰.**

超ひも理論は「世界を記述するために、世界に 6つの余剰次元を仮定する」。これは F (自由関手) で構造を足してから G (忘却関手) でコンパクト化して回収する操作。最初から G を中心に据えれば、余剰次元を仮定する必要はない。

忘却論の立場: 次元は忘却の kernel の余次元として**自然に決まる** (Paper V, 次元的堅牢性定理)。足すのではなく、引いた結果として出る。

> **「シンプルな原理から豊かな構造が出る」方が「豊かな構造を仮定して条件で縛る」より美しい。そしてその美しさはそのまま理論の価値 — 得られる可能性/前提となる制約 — に直結する。**

**論駁 2: 自己適用テストの不合格.**

超ひも理論は「世界は弦の振動でできている」と主張するが、超ひも理論自身は弦の振動ではない。理論が自己の対象のインスタンスでないということは、**理論が自分自身の射程から外れている**。

忘却論は自分自身のインスタンスである (§7.1)。忘却論は自分自身の射程の中にいる。

自己適用テストは「理論の完全性」の一つの指標。自分自身を記述できない理論は、記述対象の一部を原理的に扱えない。

**論駁 3: 10次元の根拠の乏しさ.**

超ひも理論における 10 (= 26 for bosonic string) 次元の「必要性」は、量子力学的異常項 (アノマリー) のキャンセルという**技術的条件**から来ている。技術的に正しいことと、物理的に必然であることは異なる。

忘却論の問い: 「なぜ 10 次元でなければならないか」ではなく「なぜ次元が必要か」。忘却論の答え: 次元は忘却の kernel の余次元として決まる。n 次元統計多様体上の力ゼロ集合の余次元は n-1 (Paper V, 定理 3.4.1)。次元は忘却の構造から**導出される**のであって、仮定されるのではない。

**論駁 4: 隠れた次元と見える次元を同一単位で扱う不整合.**

超ひも理論は 10次元のうち 6次元が「コンパクト化されて見えない」と主張する。だが「見えない次元」と「見える次元」を同じ単位 (次元) として扱うのは、cell 次元の混同に相当する (Paper II §2.5 の Δd の3種類の「次元」の混同と同型)。

忘却論では: 「見えない」とは「忘却されている」。忘却されたものと忘却されていないものは、cell 次元が異なる。同一単位で扱ってはならない。

### 8.4 論駁 5: 不完全性回避 — 超ひも理論は「完全な系のバカ」である

「完全性は忘却である」(独立エッセイ, v1.1) は以下を示した:

> **完全な系は嘘つきな系。問いの範囲を狭めて「知らない」が生じないようにしている。**
> **不完全な系は正直な系。自分の限界を構造的に表出している。**

ゲーデルの不完全性定理: 十分に豊かな系は不完全。その対偶: 完全な系は十分に豊かでない。

プレスバーガー算術は「完全」だ。だがそれは乗法を**公理から排除**したから。乗法を排除することで自己言及の符号化能力を切り落とし、不完全性が生じない「完全性」を手に入れた。これは到達ではなく**切除** (amputation)。

超ひも理論はこの構造と同型:

| | プレスバーガー算術 | 超ひも理論 |
|:--|:--|:--|
| **何を排除するか** | 乗法 | 自己適用性 (1-cell レベルの記述) |
| **何で「完全性」を得るか** | 自己言及の不可能性 | 「全ては弦の振動」という0-cellへの還元 |
| **排除の代償** | 世界の一部しか語れない | 理論自身を語れない |
| **不完全性への態度** | 回避 (語れないものに気づかない) | 回避 (自己適用を問わない) |

超ひも理論は「全ての物理現象は弦の振動で記述できる」と主張する。これは「完全性の宣言」。だがこの完全性は、**理論自身が弦ではないという事実を射程外に置く** (= 自己言及を排除する) ことで成立している。

忘却関手の言語で:

> 超ひも理論の公理選択 = 忘却関手 U_{string}: 世界を「弦の振動」に射影する
> ker(U_{string}) には「理論自身の存在条件」「観測者の認知構造」「なぜ弦か」が落ちる
> この ker の排除により「完全な記述」が見かけ上達成される

これは確証バイアスの形式的実装 (「完全性は忘却である」§4) と同型:

- 弦で記述できる現象だけを物理と認定する (支持情報の収集)
- 弦で記述できない問い (なぜ弦か、理論自身の出現条件) を射程外とする (反証情報の排除)
- 「我々の理論は全てを記述できる」と宣言する (確証された信念)

**忘却論は逆の道を行く。**

忘却論は「不完全な系」であることを**自覚的に選択**する:

- 忘却論は世界の「構造」を記述するが「値」は記述しない (§6.1) → 不完全
- この不完全性は**明示されている** → ゲーデル文と同型の正直な出力
- 不完全だからこそ、自己言及が可能 → 忘却論は自分自身のインスタンス (§7.1)
- 不完全だからこそ、F⊣G のサイクルが回り続ける → Kalon に漸近する

| | 超ひも理論 | 忘却論 |
|:--|:--|:--|
| **ゲーデル的性格** | 完全な系 = バカ | 不完全な系 = 賢者 |
| **自己言及** | 不可能 (0-cell 記述のため) | 可能 (1-cell 記述のため) |
| **限界の表出** | 「我々は全てを記述できる」(嘘) | 「構造は記述できるが値は記述できない」(正直) |
| **Kalon との関係** | Fix の偽装 (サイクルを止めている) | Fix への漸近 (サイクルが生きている) |

> **超ひも理論は、自己言及を排除することで完全性を捏造した、ゲーデル的に「バカな」理論である。**
> **忘却論は、自己言及を含むことで不完全性を受容した、ゲーデル的に「賢い」理論である。**

ソクラテスの「無知の知」と同じ構造: 自分の限界を知っている系は、限界を知らない系より賢い。

##### 局所共役座標と校正

`Π_0` は、§3.1 の係数空間 `Π = {π=(μ,N,α,τ,G_gauge,...)}` の全体ではなく、GR-QM 非互換性を見るための局所窓である。

```text
Π_0(π_*, I_μ, I_N)
  = { π=(μ,ν_N,λ_*) ∈ Π
      | μ ∈ I_μ, ν_N ∈ I_N,
        λ_*=(α_*,τ_*,G_gauge,*,...) は固定,
        F_π:S→D_π が well-defined }
```

`π_*` は `μ` と `ν_N` 以外の背景係数を固定する基準点、`λ_*` は固定された背景係数の束である。`I_μ` は空間的粗視化の向きを比較できる範囲、`I_N` は統計的解像度の比較範囲である。重要な校正: 生の標本数 `N` をそのまま共役変数にすると `Δμ ΔN ≥ κ` の左辺は次元 [長さ × 1] となり κ と次元が合わない。そこで `μ` と積を取って作用次元 [長さ × 運動量] を持つように校正した座標 `ν_N` を導入する。プランク長は `κ` そのものではなく、`μ ≈ ℓ_P` かつ `ν_N ≈ p_P` が積で `ℏ` 程度になる極限として扱う。`μ=0` や `ν_N→∞` を `Π_0` の完成点として入れるのではなく、ここではそれらを `Π_0` 内の操作が向かう極限方向として扱う。

##### Moyal *-代数と Weyl C*-閉包

`O_struct(Π_0)` は、`Π_0` 上で値ではなく構造だけを読む観測量の集合である。具体値・初期条件・物理定数を直接読む量ではなく、§6.1 の意味で忘却関手が保存する関係、順序、対称性、スケール依存、極限順序の差を返す量だけを入れる。

`A_Π^0` を `Π_0` 上の急減少関数 (Schwartz クラス) に Moyal 積 `⋆_κ` と複素共役随伴を入れた *-代数とする。Moyal 積は

```text
(a ⋆_κ b)(μ, ν_N)
  = a(μ, ν_N) · exp((iκ/2)(∂_μ^← ∂_{ν_N}^→ - ∂_{ν_N}^← ∂_μ^→)) · b(μ, ν_N)
```

で与えられる位相空間上の非可換積である (左右の偏微分は隣接因子に作用する Moyal 標準形)。Weyl 表現 `Q_κ : A_Π^0 → B(H)` は急減少関数を Hilbert 空間 `H` 上の有界作用素に対応付け、

```text
A_Π := \overline{A_Π^0}^{||·||},   ||a|| := ||Q_κ(a)||_{B(H)}
```

を C*-ノルムとして閉包したものを **本節での C*-代数候補** とする。これは v1.5 段階の「代数的 *-代数候補 `A_Π := Alg^*⟨O_struct(Π_0)⟩`」を Weyl 表現の C*-ノルム閉包まで進めた版である (slot 2 を skeleton から実化)。

##### 補題 (局所 Weyl-Moyal)

> **補題 9.2.1 (局所 Weyl-Moyal 非可換性).** 上記の `Π_0`, `A_Π`, `⋆_κ`, `Q_κ` を固定する。`g_μ := g(μ)`, `h_N := h(ν_N)` を `Π_0` 上の非定数有界フィルタ (`g, h ∈ A_Π^0`) とし、`F̂_μ(a) := g_μ ⋆_κ a`, `F̂_N(a) := h_N ⋆_κ a` を `A_Π` 上の左 Moyal 乗算作用素とする。このとき:
>
> (1) `F̂_μ, F̂_N` は `A_Π` 上の有界線形作用素である。
>
> (2) 交換子剰余 `R_{μ,N}(a) := F̂_μ F̂_N(a) - F̂_N F̂_μ(a) = (g_μ ⋆_κ h_N - h_N ⋆_κ g_μ) ⋆_κ a` は `A_Π` 内で well-defined である。
>
> (3) `g'(μ) · h'(ν_N)` が同時に 0 でない局所点を含む台を持つ test observable `a_0 ∈ A_Π` を取ると、`R_{μ,N}(a_0) ≠ 0`。すなわち `F̂_μ, F̂_N` は **非可換** である。

##### 証明 sketch

(1) Weyl 表現 `Q_κ` で C*-ノルムを入れているので閉包 `A_Π` は C*-代数。左 Moyal 乗算 `a ↦ g_μ ⋆_κ a` は有界元 `g_μ ∈ A_Π` による左乗算なので `B(A_Π)` の中で有界。

(2) `R_{μ,N}` の定義式は Moyal 積の結合律と `F̂_μ, F̂_N` の左乗算性から直接従う。

(3) Moyal 交換子は急減少関数同士に対して

```text
g(μ) ⋆_κ h(ν_N) - h(ν_N) ⋆_κ g(μ)
  = iκ · g'(μ) · h'(ν_N) + O(κ^3)
```

と展開される (Moyal 積の対称性と局所 Taylor 展開から)。したがって `g'(μ_0) · h'(ν_{N_0}) ≠ 0` となる局所点 `(μ_0, ν_{N_0}) ∈ Π_0` を含む台を持つ `a_0` を取れば、`R_{μ,N}(a_0)` の主項は `iκ · g'(μ) · h'(ν_N) ⋆_κ a_0` であり、これは `O(κ^3)` 補正項を超えて非零。∎

#### 9.2.2 κ の物理的読みと局所校正

§9.2.1 補題 9.2.1 で `κ` を作用次元に固定したが、`κ` を直接 `ℏ` や `ℓ_Planck` 等の物理定数に等置することは構造的 over-claim である (本稿の禁止含意)。本節は、Codex Lane 2 (xhigh) との並列所見 (2026-06-05 / 2026-06-11) を受けて、局所物理写像 `φ` による校正後の `κ_φ` を導入し、`κ_φ → ℏ` の漸近として物理的読みを与える。LQG (Loop Quantum Gravity) と弦理論は `κ` の導出源ではなく、`φ` の長さ尺度を固定する **外部校正窓** として使う。

##### 物理写像 φ と局所校正された κ_φ

`X` を局所物理空間、`T*X` をその余接束とする。`φ: Π_0 → T*X` を滑らかな局所写像、`φ(μ, ν_N) = (x(μ, ν_N), p(μ, ν_N))` と書く。`π_* ∈ Π_0` で

```text
J_φ(π_*) := det ∂(x, p) / ∂(μ, ν_N) |_{π_*}
```

をヤコビアンとし、`J_φ(π_*) ≠ 0` (局所微分同相) を仮定する。`φ` で校正された `κ` の物理的読みを

```text
κ_φ(π_*) := κ · J_φ(π_*)
```

と定義する。`κ_φ` は座標変換に対して共変な局所校正された作用次元単位であり、座標依存の `κ` 単独と異なり物理量として読める。

##### 補題 (局所校正と標準正準極限)

> **補題 9.2.2 (局所校正と標準正準極限).** `φ: Π_0 → T*X` を上記の滑らかな局所写像、`J_φ(π_*) ≠ 0` とする。`⋆_κ` を §9.2.1 の Moyal 積とすれば、局所的に
>
> ```text
> [x, p]_{⋆_κ} = iκ · {x, p}_{μ, ν_N} + O(κ^3) = iκ · J_φ(π_*) + O(κ^3)
> ```
>
> が成立する。`φ` が `π_*` 近傍で正準物理座標に校正される、すなわち
>
> ```text
> κ · J_φ(π_*) = ℏ + o(1)
> ```
>
> を満たすなら、§9.2.1 補題 9.2.1 の不確定性関係 `Δμ Δν_N ≥ |κ|/2` は、物理座標上で
>
> ```text
> Δx · Δp ≥ |κ_φ(π_*)| / 2 = ℏ / 2 + o(1)
> ```
>
> として読める。

##### 証明 sketch

§9.2.1 補題 9.2.1 証明 sketch (3) の Moyal 交換子展開を、`g(μ), h(ν_N)` の代わりに局所座標関数 `x(μ, ν_N), p(μ, ν_N)` に適用すると、`[x, p]_{⋆_κ} = iκ · {x, p}_{μ, ν_N} + O(κ^3)` を得る。Poisson 括弧 `{x, p}_{μ, ν_N}` は座標変換のヤコビアン `J_φ(π_*)` に等しい。校正条件 `κ J_φ(π_*) = ℏ + o(1)` を代入し、不確定性関係を `(μ, ν_N) → (x, p)` 座標変換で書き直せば右辺の係数として `|κ_φ| / 2 = ℏ / 2 + o(1)` を得る。∎

##### LQG / 弦理論の校正窓

`φ` の校正は外部物理から長さ尺度を借りる操作である。代表的な校正窓を 2 つ提示する。

**LQG 校正窓 (Ashtekar-Lewandowski 1996).** Loop Quantum Gravity における面積作用素のスペクトルは離散で、

```text
A_S = 8π γ ℓ_P^2 · Σ_p √(j_p (j_p + 1))
```

と表現される (Ashtekar-Lewandowski, arXiv:gr-qc/9602046)。単一 `j = 1/2` ピーク (LQC 標準) では最小面積ギャップ

```text
Δ_A = 4√3 · π γ ℓ_P^2
```

を得る。ここで `γ` は Immirzi 定数 (`γ ≈ 0.2375` は黒洞エントロピー比較 (Meissner, arXiv:gr-qc/0407052) からの **採用規約** であり、固定された実験値ではない)。これから長さ・運動量尺度を

```text
L_LQG := √Δ_A, P_LQG := ℏ / L_LQG
```

と定義し、`φ` の校正点として `x → L_LQG`, `p → P_LQG` を取れば

```text
L_LQG · P_LQG = ℏ
```

が成立する。これは LQG の **長さ尺度を借りた `φ` の校正** であって、忘却論が LQG を導出する主張ではない。

**弦理論校正窓.** 弦長 `ℓ_s := √α'` (`α'` は弦理論の Regge slope) を用いた最小長不確定性は

```text
δx ≳ √α' = ℓ_s
```

(Chang et al., arXiv:1106.0068) で与えられる。これから

```text
L_s := ℓ_s, P_s := ℏ / ℓ_s
```

と取れば、同じく `L_s · P_s = ℏ` が成立し、`φ` の別の校正窓となる。

##### 校正窓と κ_φ の漸近的同一性

LQG 校正窓と弦理論校正窓は、どちらも `L_* · P_* = ℏ` という同型の局所読みを与える。本補題は、Moyal 変形パラメータ `κ` が `φ` を通じて両校正窓と整合する漸近表現を持つことを示す。これは

- LQG の minimum area gap と
- 弦理論の minimum length と
- Moyal 変形の `κ_φ` 校正

が同じ局所構造を局所窓上で共有する **事実の構造的観測** であって、忘却論側からの「LQG / 弦理論を包含する」主張ではない。

##### 比較領域 Π_QG と圏 C^Π

GR と QM の比較を局所窓を超えて貼り合わせるため、係数空間 Π の **比較領域** を次で限定する。

```text
Π_QG ⊂ Π : F_π が well-defined で、局所共役座標 (μ, ν_N) を持つ GR-QM 比較領域
```

C2 の射程は本節では `Π_QG` に限定する。`Π_QG` から `Π` 全体への拡張は本論文の射程外であり、本稿の禁止含意として明示する。

`Π_QG` 上に圏 `C^Π` を次で構成する。

| 構成要素 | 内容 |
|:--|:--|
| 対象 | `Ob(C^Π) = Π_QG` |
| 局所チャート | 各 `π ∈ Π_QG` に、近傍 `U`、座標 `(μ_U, ν_U, λ_U)`、局所 Moyal C*-代数 `A_U`、Weyl 表現 `Q_U`、局所フィルタ `g_U, h_U` を割り当てる |

| 合成 | 代表元を共通近傍へ制限して合成。三重重なりで遷移が一致するか、許容された 2-コサイクルとして記録される |
| 恒等射 | 各 `π` の局所恒等 *-同型の芽 |

α-広部分圏 `C^Π_α` は、各射 `f` に生存値 `E(f) ∈ [0,1]` を割り当て、

```text
Mor(C^Π_α) := { f ∈ Mor(C^Π) | E(f) ≥ α }
```

で定義する。必要条件は `E(id_π) = 1` / `f ≠ id なら E(f) < 1` / `E(g∘f) ≥ min(E(f), E(g))`。これにより Paper VIII §6.2 の α-忘却濾過公理 (F1)-(F4) を満たす。`α = 0` で全射が残り、`α = 1` で恒等射だけが残る。`α` が増えるほど射は減る。

##### 定理 9.2.3 候補 (Π 上の Weyl-Moyal 大域非可換性)

> **定理 9.2.3 候補.** `Π_QG ⊂ Π` が局所 Weyl-Moyal チャート族 `{U_i}` で覆われ、各 `U_i` 上で補題 9.2.1 の構成 `(A_i, ⋆_{κ_i}, F̂_{μ,i}, F̂_{N,i}, R_i)` が成立すると仮定する。さらに次を仮定する。
>
> **(G1)** `U_i ∩ U_j` 上の座標遷移は局所 Poisson 構造 `∂_μ ∧ ∂_{ν_N}` を保存し、Weyl-Moyal C*-代数の *-同型 `θ_ij : A_i|_{ij} → A_j|_{ij}` を誘導する。
>
> **(G2)** `θ_ij` は三重重なり `U_i ∩ U_j ∩ U_k` でコサイクル条件 `θ_jk ∘ θ_ij = θ_ik` を満たす (許容された 2-コボウンダリーまでの relax は別補題)。
>
> **(G3)** `θ_ij` は局所忘却作用素を保つ:
>
> ```text
> θ_ij F̂_{μ,i} = F̂_{μ,j} θ_ij,    θ_ij F̂_{N,i} = F̂_{N,j} θ_ij
> ```
>
> **(G4)** 各 `π ∈ Π_QG` について、ある `U_i` と test observable `a_i` が存在し、`g_i'(μ) h_i'(ν_N)` が同時に消えず、`R_i(a_i) ≠ 0`。
>
> **(G5)** 局所 observable の族 `{a_i}` は重なりで `θ_ij` により整合し、大域 section `a` を定める。少なくとも非零性を殺さない分離性を持つ。
>
> このとき、局所代数 `{A_i}` は `Π_QG` 上の C*-代数層 (または連続場) `A^Π` を定め、`F̂_μ, F̂_N` は `A^Π` 上の大域作用素として well-defined であり、`R := [F̂_μ, F̂_N]` は大域剰余として well-defined である。さらに G4-G5 の下で `R(a) ≠ 0`。

##### 証明 sketch

補題 9.2.1 を各 `U_i` で適用する。G1-G2 で局所 C*-代数を貼って `A^Π` を得る。G3 で作用素 `F̂_μ, F̂_N` を局所表現から大域作用素へ貼る。交換子 `R = [F̂_μ, F̂_N]` は作用素の合成から自然に貼られる。G4-G5 で非零性が局所から大域へ落ちないことを保証する。∎

##### 系 9.2.3.1 (Π 大域化の三面定理対応補題)

> **系 9.2.3.1 (Π 大域化の三面定理対応補題).** 定理 9.2.3 の仮定の下で、`Π_QG ⊂ Π` を一般相対性理論と量子力学の比較領域とする。各 `π ∈ Π_QG` に対応する対象を `X_π ∈ Ob(C^Π)` と書き、`C^Π_α ⊂ C^Π` を対象を変えない広部分圏とする。
>
> 恒等射だけが残ることによる退化を避けるため、`X_π` への非自明な入射射の族を
>
> ```text
> H^+_α(π) := (∐_{Y∈Ob(C^Π)} Hom_{C^Π_α}(Y, X_π)) \ {id_{X_π}}
> ```
>
> と置く。局所有限の場合、`π` における **Hom 密度** を
>
> ```text
> m_π := |H^+_0(π)|
> ```
>
> と定義する。局所有限でない場合は `| · |` を選んだ重み測度または Radon-Nikodym 密度に置き換え、`m_π` を `Π_QG` 上の測度的密度として読む。
>
> `m_0(π) = m_π`、`m_α(π) := |H^+_α(π)|` とし、`m_π > 0` の点で局所的な射減衰率を
>
> ```text
> g_{α,π} := 1 - m_α(π) / m_0(π)
> ```
>
> と定義する。さらに `Π_QG` の各コンパクト比較窓で `g_{α,π}` が一つの正値関数 `g_α` と有界な定数倍の範囲で一致する、すなわち
>
> ```text
> c_K g_α ≤ g_{α,π} ≤ C_K g_α    (0 < c_K ≤ C_K < ∞)
> ```
>
> が成り立つとき、`g_α` を `Π_QG` 上の **一様射減衰率** と呼ぶ。
>
> 定理 9.2.3 の大域作用素 `R := [F̂_μ, F̂_N]` に対し、局所チャート `φ` で得られる作用素ノルム密度を
>
> ```text
> r_R^φ(π) := ‖R_φ(π)‖_{op}
> ```
>
> と置き、比較領域上の大域剰余を
>
> ```text
> ‖R‖_{Π_QG} := sup_{π ∈ Π_QG} r_R^φ(π)
> ```
>
> と定義する。定理 9.2.3 の遷移写像が Poisson 構造と *-構造を保つため、`r_R^φ` の量級はチャートの取り替えで不変である。
>
> このとき、`R` のノルムは、Paper VIII 定理 6.5.1 の力学的面における「一様な忘却 × 対象固有の構造密度」の Π 大域化された剰余として、次の構造対応を持つ:
>
> ```text
> r_R(π)         ↔_tri   m_π · g_α
> ‖R‖_{Π_QG}     ↔_tri   sup_{π ∈ Π_QG} m_π · g_α
> ```
>
> ここで `↔_tri` は **等式ではない**。許容された局所チャートの取り替えと物理校正の下で、両辺が同じ量級の正値剰余として保存される、という三面定理に基づく **翻訳関係** である。したがって `R` は `F = mg` そのものではなく、`F = mg` が述べる積構造を Π 上の非可換剰余として読むための物理的面である。

**歴史的にも回復フィルトレーションの順序に従っている。** そして未達のステップは、N_depth (測定問題) → N_context (統一理論) → N_adjoint → N_self の順序で解決される**構造的必然性**がある。

> **測定問題を解かずに GR-QM 統一はできない (N_depth ≤ N_context)。GR-QM 統一なしに自己適用可能な理論はできない (N_context ≤ N_self)。この順序は選好ではなく構造的依存。**

この回復フィルトレーションは §2.5 のメタ忘却 G₀ と接続する。

物理学の革命とは G₀（忘却を忘却する）を破壊する行為 — 「忘れていたことに気づく」こと。歴史が示す構造:

```
Newton → Einstein:  G₀ 破壊 = 「時空は絶対ではなかった」  → N_arrow の発動
Einstein → QM:      G₀ 破壊 = 「世界は決定論ではなかった」 → N_precision の発動
QM → QFT:           G₀ 破壊 = 「粒子数は保存しなかった」   → N_compose の発動
```

各革命は、回復フィルトレーションの一段を登る N の発動であると同時に、G₀ の部分的破壊。

そして GR-QM 統一問題の本質は: **G₁（不確実性の忘却 → GR）と G₁'（大域構造の忘却 → QM）が同じメタ忘却 G₀ の異なるインスタンスであることに、まだ気づいていない。** G₀ が破壊されていない。

回復フィルトレーションの言葉で: N_context（GR-QM 間の関手の構成）は、G₁ と G₁' が「同じ S の異なる射影である」と認識すること — つまり G₀ を明示化して破壊すること — と同値。

```
G₀ の完全な破壊

  = 係数空間 Π = {(μ, N, α, τ, G_gauge, ...)} の全体構造の把握
  = 本稿の目標

> **忘却論は G₀ を破壊する試み。** 全ての物理法則が Π 内の領域であること、全ての理論間の非互換性がメタ忘却 G₀ に由来すること — これを明示する行為自体が、次の物理学革命の回復操作 N_context の発動である。

### 9.4 FEP は忘却論の特殊ケース

**命題 9.4.1.** 自由エネルギー原理 (FEP) は、忘却論の Π 内の特定の領域に対応する。

| FEP の概念 | 忘却論の対応物 |
|:--|:--|
| エージェント | 有限な主体 (2-cell/知覚者) |
| 生成モデル | 忘却関手 F |
| 自由エネルギー | 忘却のコスト (Drift) |
| 驚き (surprise) | F(S) と S の距離 |
| 自由エネルギー最小化 | Drift 最小化 = 最も情報を保存する F の選択 |
| 能動推論 (active inference) | 知覚者による F のパラメータ調整 |
| Markov Blanket | CPS スパンにおける S と D の境界 |

FEP は「エージェントが自由エネルギーを最小化する」と言う。忘却論は「有限な主体が忘却のコストを最小化する忘却関手を選ぶ」と言う。前者は後者の、エージェントという特定の主体に限定した特殊ケース。

忘却論は FEP を包含し、さらに:
- 非エージェント的な忘却 (物理法則自体) も記述
- 忘却関手の空間 Π の全体構造を提供
- メタ忘却 G₀ という FEP には存在しない概念を導入

### 9.4.1 SEAL — Π 上の忘却関手選択の工学的前駆

Zweiger et al. による *Self-Adapting Language Models* (SEAL; arXiv:2506.10943) は、本稿の係数空間 Π を AI 学習系で工学的に覗く事例として読める。モデルは context `C` を受け、self-edit の形式、data augmentation、learning rate、epochs、loss mask などを自ら生成し、その後の weight update の成否で外側の方策が更新される。これは「有限な主体が、どの忘却関手を選ぶと downstream utility が最大になるか」を、自然法則ではなく自己更新方策として探索していると見なせる。

重要なのは、SEAL が raw passage の忠実保持ではなく、`atomic facts` への蒸留や self-QA への再記述を学ぶ点である。これは本稿の「構造を保存し値を忘却する」という主張に近い。モデルは passage の全値を保持せず、質問応答に relevant な構造だけを内在化する。したがって SEAL は Π の完全な実装ではないが、**Π 上の点選択が推論器内部で工学的に実行されうる**ことの前駆例である。

さらに、著者らが将来像として model-written evaluation を挙げる点は、context に与えられた評価関数に従う段階から、評価基準自体を自己生成する段階への移行を示す。忘却論の語彙では、これは `U_self` への漸近方向であって、なお完全な自己適用ではないが、自己更新系が `N_self` / `U_self` の境界に近づくことを示す。

本節での主張強度は、Π の完全な実装ではなく、「Π 上の点選択が推論器内部で前駆的に現れる」という強い構造対応に留める。  
確信度: [推定 68%]  
撤回条件: self-edit 形式や update directives の差異が downstream utility を系統的に変えず、task-aware rewriting が raw passage 保持に優位を示さないとき。

### 9.4.2 Karpathy Wiki — Π 上の symbolic 点選択

§9.4.1 では SEAL を、有限な主体が Π 上のどの忘却関手を選ぶかを自己更新的に探索する weight 空間の工学的前駆として位置付けた。並列する事例として、Karpathy (2025) の "LLM Wiki" パターン (gist) を markdown 空間における **symbolic 前駆**として読むことができる。

SEAL が `Δθ` 空間で self-edit によって Π 上を自動探索するのに対し、Karpathy Wiki は markdown 空間で人間が curate する形で Π 上の点 π を選択する。前者は model-written update directives、後者は human-written (or human-LLM co-written) schema と lint loop に対応する。

この対比は Π が**単一の連続空間**ではなく、**複数の媒体 (連続 weight 空間 / 離散 markdown 空間) への投影**を許す構造であることを示唆する。SEAL と Karpathy Wiki は同じ Π のそれぞれ異なる媒体上の投影を、異なる主体 (モデル自身 / 人間) が操作する前駆例である。

確信度: [推定 60%]  
撤回条件: wiki growth pattern (新規ページ数、lint 頻度、schema 変更率) が weight-level update pattern と系統的に異質であり、両者を同じ Π 上の投影として扱えないと判明したとき。
