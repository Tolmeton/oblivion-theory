# Face Lemma のホモロジー厳密化

**作成日**: 2026-04-14  
**役割**: Paper II §3.4 の再基礎づけ。2026-03-27 版の「Kan 仮定除去」は維持しつつ、`dim Ξ` の定義を `Star complex の最高次元` から `normalized chain complex` 上の照合面 `B_1^{ver}` へ差し替える。

---

## 0. 何がまだ弱かったか

2026-03-27 版の patch は重要だった。  
`Kan complex` の暗黙仮定を外し、Segal/Grothendieck の `2-coskeletal` で高次 simplex の冗長性を説明したからである。

しかし、その版でもなお次の弱点が残る。

1. `dim Ξ = St(x) の最高非退化 simplex 次元` は循環定義ではないが、**照合可能性そのもの**ではない  
2. `(S2) 非退化 2-simplex がある` と `dim Ξ ≥ 2` が実質的に同じ内容を別名で言っており、定理の心臓がまだ definition-heavy  
3. 精読レポートが提案した `dim Ξ = rank H_1` 案は、そのままだと危うい  

最大の理由は 3 である。  
**埋まった三角形は `H_1` を生むのではなく、むしろ 1-cycle を境界として埋める側**だからだ。  
Face Lemma が欲しいのは「穴」ではなく「検査面」である。したがって主役は `H_1` そのものではなく、`∂_2` が作る **照合面** でなければならない。

---

## 1. 新しい中心量

対象 `x ∈ Ob(C)` に対し、`x` を頂点にもつ star complex `St(x)` の 2-切断を考える。  
係数体は `k = ℝ` とし、正規化単体チェイン複体

```text
C_2^nd(St(x);k) --∂_2--> C_1^nd(St(x);k) --∂_1--> C_0^nd(St(x);k)
```

を用いる。ここで `nd` は退化 simplex を除いた正規化を表す。

### 定義 1.1 照合面

`x` の **照合面空間** を

```text
B_1^ver(x) := im(∂_2 : C_2^nd(St(x);k) → C_1^nd(St(x);k))
```

と定める。

意味は単純である。

- `C_1^nd ≠ 0`: 射はある  
- `B_1^ver = 0`: だが、それらを外部から照合する 2-simplex がない  
- `B_1^ver ≠ 0`: 2-simplex の境界が 1-chain として立ち、合成を外から突き合わせる「照合面」がある  

非退化 2-simplex `σ = (f, g)` に対し、

```text
∂_2 σ = [g] - [g∘f] + [f]
```

である。  
これが Face Lemma の最小 syndrome、すなわち「照合面が本当に立っている」ことの最小記号である。

---

## 2. 忘却スペクトラム Ξ の再定義

Face Lemma が欲しいのは「最高次元の simplex」ではなく、「照合面を最小コストで担う複体」である。

### 定義 2.1 照合支持複体

`Ξ(x)` を次の inclusion-minimal 非退化部分複体として定める。

1. `B_1^ver(x) ≠ 0` のとき  
   `im(∂_2^K) ≠ 0` を満たす `K ⊂ St(x)` のうち最小のもの
2. `B_1^ver(x) = 0` かつ `C_1^nd(St(x);k) ≠ 0` のとき  
   `C_1^nd(K) ≠ 0` を満たす `K ⊂ St(x)` のうち最小のもの
3. `C_1^nd(St(x);k) = 0` のとき  
   `Ξ(x) = {x}`

そして

```text
dim Ξ(x) := dim_simp Ξ(x)
```

とする。

この定義だと

- `dim Ξ = 0`: 点しかない
- `dim Ξ = 1`: 射はあるが検査面はない
- `dim Ξ = 2`: 照合面を担う最小面が立っている

となる。

---

## 3. 新しい Face Lemma

### 定理 3.1

以下は同値である。

1. `dim Ξ(x) = 2`
2. `B_1^ver(x) ≠ 0`
3. `x` を頂点とする非退化 2-simplex が存在する
4. `x` は CPS-安定である

さらに、`dim Ξ(x) > 2` は起こらない。  
高次 simplex は照合面を直接には増やさず、2-face を通じてのみ寄与する。

### 証明の骨格

- `2 ⇒ 3`: `B_1^ver(x) ≠ 0` なら照合面を与える 2-chain `c` がある。2-chain の support は有限だから、その中のある非退化 2-simplex `σ` で `∂_2 σ ≠ 0` が成り立つ
- `3 ⇒ 2`: 非退化 2-simplex の境界 `∂_2 σ = [g] - [g∘f] + [f]` は零でない
- `2 ⇒ 4`: `B_1^ver ≠ 0` なら 2-simplex があるので `(S2)` が立つ。辺は非恒等だから `(S1)` も立つ
- `4 ⇒ 2`: `CPS-安定` の定義を `B_1^ver ≠ 0` に取れば直ちに戻る
- 飽和: `B_1^ver` は `im ∂_2` として次数 2 の情報だけで決まる。したがって高次 simplex は 2-face の束としてしか効かない

ここで重要なのは、飽和の証明がもはや `2-coskeletal` を**必要条件**としないことである。  
`2-coskeletal` は互換性確認としては有用だが、Face Lemma の最小性自体は chain degree で閉じる。

---

## 4. Segal/Grothendieck の位置づけ変更

2026-03-27 版では `2-coskeletal` が証明の心臓だった。  
2026-04-14 版では、位置づけは次のように変わる。

- **旧**: 高次冗長性の主証拠
- **新**: nerve の通常理解と新定義が矛盾しないことを示す整合チェック

つまり Segal/Grothendieck は不要になったのではない。  
**主証明の土台から、整合性の裏書きへと降格**したのである。

## 5. 下流定理への影響

### 5.1 Blanket 生成定理

`dim Ξ ≥ 2` を使う箇所は、より正確には

```text
B_1^ver(X) ≠ 0
```

を使っている。  
Markov blanket の生成に必要なのは「三角形が見えること」ではなく、「合成の検査境界が立つこと」だからである。

### 5.2 FEP 包含

`FEP = CPS|_{α>0, dim Ξ≥2}` という表現は残してよい。  
ただし意味は

```text
FEP = CPS|_{α>0, B_1^ver ≠ 0}
```

と読むべきである。

### 5.3 意識・創発

`1-skeleton → 2-skeleton` という直感は残る。  
ただし、その数学的中身は

```text
C_1^nd ≠ 0 だが B_1^ver = 0
```

から

```text
B_1^ver ≠ 0
```

への遷移である。

---

## 6. Kalon

この差し替えで Face Lemma は

- 「3射があると定義する」話ではなく
- 「`∂_2` が照合面を初めて立てる最小次数が 2 である」話

になる。

これが Kalon である。  
Paper II の基盤を、「図形の見た目」から「三角形が本当に照合面として立っているか」へ移すこと。  
これで Blanket・FEP・意識診断は、definition の延長ではなく theorem の延長として読める。
