# 忘却の元素論 — CCL 特徴量空間と化学の構造的同型

**計算日**: 2026-04-04
**前提**: ROM `rom_2026-03-31_49d_purification_chemistry` (49d 浄化実験), Aletheia §2.1 (9 原子パターン) + §4 (テンソル積), EXPERIMENTS §18 (49d 設計), code_ingest.py v4
**目標**: ROM の「49d は分子式であって構造式ではない」を精密化し、化学 ↔ 忘却関手の同型対応を**4層すべて**にわたって構成する

---

## 0. 主張

> CCL 特徴量空間と化学的分子記述の間に **4 層の構造的同型** が存在する。
> この同型は偶然の類似 (analogy) ではなく、両者が共有する**圏論的構造** (忘却関手の階層) から帰結する。

## 2. 第 1 層: 元素 ≅ U パターン

### 2.1 周期表の構成

化学の元素は原子番号 Z (= 陽子数) で順序づけられ、電子配置で**族** (縦) と**周期** (横) に整理される。

Aletheia の U パターンは n-cell レベルで順序づけられ、忘却する構造の種類で分類される。

| 化学 | Aletheia | 対応の根拠 |
|:-----|:---------|:----------|
| Z=1 (H, 最軽元素) | n=1 U_arrow (最低次の忘却) | 最も基本的、他の全てが依存 |
| Z=6 (C, 四価) | n=2 U_depth (自然変換の忘却) | 最も多くの結合を形成。認知でも最多の複合パターンに参加 |
| Z=8 (O, 高電気陰性度) | n=3 U_precision (精度の忘却) | 他の原子から電子を引き寄せる = 他の忘却を増幅する |
| Z=2 (He, 不活性) | n=ω U_self (自己関手の忘却) | 完全閉殻 = 他と結合しない。U_self も最も孤立したパターン |

### 2.2 9 元素 = 9 基底パターン

| # | U パターン | n | 化学的対応物 | 根拠 |
|:--|:----------|:--|:-----------|:-----|
| 1 | U_sensory | Basis | **光子 (γ)** | 他の全てに先立つ。元素ではなく**媒介粒子**。入力 = 光 |
| 2 | U_arrow | 1 | **水素 (H)** | 最も単純な原子。全ての化合物の基底 |
| 3 | U_compose | 1.5 | **炭素 (C)** | 鎖を作る能力。合成律 = 炭素鎖 |
| 4 | U_depth | 2 | **窒素 (N)** | 三重結合可能 = 深い比較能力。アミノ酸の骨格 |
| 5 | U_precision | 3 | **酸素 (O)** | 電気陰性度が高い = 他の精度を決定する |
| 6 | U_causal | 4 | **リン (P)** | ATP のエネルギー伝達 = 因果連鎖の媒介 |
| 7 | U_context | ∞-1 | **鉄 (Fe)** | 遷移元素 = 文脈で性質が変わる。多価 |
| 8 | U_adjoint | ∞ | **硫黄 (S)** | ジスルフィド結合 = 双方向の橋かけ |
| 9 | U_self | ω | **希ガス (He/Ne)** | 完全殻 = 自己完結。他と結合しない |

**注意**: この対応は **motivated choice** であり一意ではない。ただし以下の構造的制約は保存される:
- **Tower 半順序**: Z が大きいほど n が大きい (周期表の周期 = フィルトレーション)
- **結合価**: 多くの複合パターンに参加する U パターンほど、化学的に多価の元素に対応する
- **安定性**: 単独で安定なパターン (U_self) は不活性元素に対応する

## 3. 第 2 層: 分子式 ≅ テンソル積 (カウント)

### 3.1 分子式の限界

化学で分子式 C₂H₆O は**エタノール** (CH₃CH₂OH) と**ジメチルエーテル** (CH₃OCH₃) を区別できない。

同様に、テンソル積 U_sensory ⊗ U_depth は「2 つの U が同時に発動している」ことしか言わず、**どのように結合しているか**を記述しない。

### 3.2 49d ベクトルが分子式である証拠

ROM の浄化実験:
- 49d (分子式) baseline: R@1 = 6.0%
- 253d (2 次テンソル積 = 分子式の精緻化): R@1 = 2.0% ← **悪化**
- テンソル積は分子式の次元を増やしただけ。構造異性体を区別する力がない

### 3.3 VFE の加法性: 分子量の類似物

Smithe Thm 46 転用:

$$F[U_a \otimes U_b(q)] \approx F[U_a(q)] + F[U_b(q)] - F[q]$$

これは**分子量の加法性**に同型:

$$M(AB) = M(A) + M(B)$$

テンソル積の VFE は各成分 VFE のおおよその和。分子量が原子量の和であるのと同じ構造。

## 4. 第 3 層: 構造式 ≅ CCL テキスト (結合パターン)

### 4.1 化学結合 ↔ CCL 演算子

ここが核心。化学で分子の性質を決めるのは原子の種類ではなく**結合の仕方**。CCL でコードの構造を決めるのはトークンの種類ではなく**演算子の結合パターン**。

| 化学結合 | 性質 | CCL 演算子 | 性質 | 同型の根拠 |
|:---------|:-----|:----------|:-----|:----------|
| **σ 結合** (共有, 単結合) | 軸対称、自由回転 | **>>** (射の合成) | 順序的、方向あり | 最も基本的な結合。鎖構造を形成 |
| **π 結合** (二重結合の一部) | 平面に拘束、回転不可 | **~** (随伴/双方向) | 双方向的、対称性あり | 自由度の制限。剛性を追加 |
| **イオン結合** | 電子の完全移動 | **!** (副作用, 例外) | 制御フローの切断 | 局所性の破壊。遠距離相互作用 |
| **水素結合** | 弱い、方向性あり | **{}** (スコープ) | 弱い束縛、局所的 | 分子間力に類似。グローバルではなくローカル |
| **金属結合** | 電子の非局在化 | **F:[]{}** (関手適用) | コレクション全体への適用 | 電子が自由に動く = 関数が全要素に適用される |
| **ファンデルワールス力** | 最も弱い | **暗黙の隣接** | トークン間の物理的近接 | 構造的に最弱の関係 |

### 4.2 構造異性体の CCL 版

**エタノール vs ジメチルエーテル (化学)**:
```
CH₃CH₂OH    (エタノール: -OH 基 = 水素結合能 → 水溶性、高沸点)
CH₃OCH₃     (ジメチルエーテル: -O- 結合 = 弱い極性 → 揮発性、低沸点)
```

**CCL 構造異性体 (同一 49d, 異なる構造)**:
```
CCL-A: input >> V:{pred} >> F:[each]{fn} >> output
       (フィルタ → マップ: バリデーションが先行)

CCL-B: input >> F:[each]{fn} >> V:{pred} >> output
       (マップ → フィルタ: 変換が先行)
```

49d ベクトル: 両方とも V:{} 1個、F:[] 1個、>> 3個 → **同一の分子式**。
だが構造は異なり、計算の意味が違う (フィルタ-ファーストは効率的、マップ-ファーストは全要素を変換)。

### 4.3 結合次数 = 射のレベル

| 化学の結合次数 | 意味 | CCL 対応 | Aletheia レベル |
|:-------------|:-----|:---------|:--------------|
| 単結合 (σ) | 1 対の電子共有 | >> (1-cell 合成) | n=1 |
| 二重結合 (σ+π) | 2 対の電子共有 | >> + ~ (合成 + 随伴性) | n=2 |
| 三重結合 (σ+2π) | 3 対の電子共有 | >> + ~ + V:{} (合成 + 随伴 + 精度) | n=3 |
| 配位結合 | 一方からの電子対供与 | << (逆方向合成) | 反対圏 |

**洞察**: 結合次数が上がると自由度が減る (回転が制限される)。CCL でも随伴や精度の制約が加わるほど、構造が硬くなる (許容される変換が減る)。

## 5. 第 4 層: 示性式 ≅ CCL 部分パイプライン (官能基)

### 5.1 官能基 = 特性を決める部分構造

化学で有機化合物の性質を決めるのは炭素骨格ではなく**官能基** (-OH, -COOH, -NH₂, ...)。
CCL でコードの「やっていること」を決めるのは全体構造ではなく**部分パイプライン** (官能基)。

| 化学の官能基 | 特性 | CCL 部分パイプライン | 特性 |
|:-----------|:-----|:------------------|:-----|
| **-OH** (ヒドロキシ基) | 水素結合、親水性 | **>> V:{pred} >>** (バリデーション) | 入力を検査し品質を保証 |
| **-COOH** (カルボキシ基) | 酸性、プロトン供与 | **>> F:[each]{fn} >> agg** (マップ-リデュース) | コレクションを変換して集約 |
| **-NH₂** (アミノ基) | 塩基性、プロトン受容 | **C:{...} E:{...}** (try-except) | エラーを吸収して回復 |
| **-SH** (チオール基) | ジスルフィド結合形成 | **>> ~ <<** (双方向パイプ) | 随伴対、双方向変換 |
| **-CHO** (アルデヒド基) | 還元性、反応性高 | **>> !** (副作用発行) | 外部に影響を与える（IO, raise, yield） |
| **=O** (カルボニル基) | 極性、求核攻撃の標的 | **>> I:[cond]{...} E:{...}** (分岐) | 条件で流れが変わる |
| **-O-** (エーテル結合) | 安定、不活性 | **scope{...}** (コンテキストマネージャ) | リソースを管理し解放 |

### 5.2 官能基による分類 = コードのパターン分類

有機化学でアルコール、カルボン酸、アミンという分類は官能基による。
CCL で「バリデーション関数」「マップリデュース関数」「エラーハンドラ」という分類は部分パイプラインによる。

```
エタノール: CH₃-CH₂-OH      → アルコールに分類 (OH 基が支配的)
酢酸:       CH₃-COOH         → カルボン酸に分類 (COOH 基が支配的)

CCL-A: _ >> ¥ >> V:{pred} >> fn >> output → バリデーション関数 (V:{} が支配的)
CCL-B: _ >> ¥ >> F:[each]{fn} >> agg      → 集約関数 (F:[] >> agg が支配的)
```

## 6. 同型が偶然でない理由: 共有圏論的構造

### 6.1 忘却関手の普遍性

化学も CCL も、同じ抽象的操作を具体化している:

$$U: \mathbf{Rich} \to \mathbf{Poor}$$

- 化学: $U_{chem}: \mathbf{Structural} \to \mathbf{Molecular}$ (結合情報の忘却)
- CCL: $U_{code}: \mathbf{Code} \to \mathbf{49d}$ (演算子結合の忘却)

両者が共通の問題 (構造異性体の区別不能) に直面するのは、同じ忘却関手の像だから。

### 6.2 回復関手の非可換性

化学反応の順序は一般に重要 (A を B に加えるのと B を A に加えるのは異なる)。
Aletheia §5.5.4 が証明する通り、N パターンの合成も非可換:

$$N_a \circ N_b \neq N_b \circ N_a$$

これは化学反応の順序依存性 (反応経路、触媒、中間体) と構造的に同型。
一方、忘却 (U) のテンソル積は**可換** — 忘れる順序は問わない。分子の分解は順序不問。

### 6.3 予測: 化学的推論の CCL への転写

同型が構造的であれば、化学で成立する法則が CCL に転写できる:

| 化学の法則 | 予測される CCL 対応 | 検証方法 |
|:---------|:------------------|:---------|
| **定比例の法則** (一定の原子比で化合) | 安定な CCL パターンは特定の演算子比率を持つ | パターン分布の峰が離散的か |
| **倍数比例の法則** (整数比) | CCL 演算子の共起は整数的比率 | 共起行列の固有値構造 |
| **質量保存** | U + N で情報量が保存 | VFE の出入りの収支 |
| **ヘスの法則** (反応エンタルピーの加法性) | N 操作の VFE 減少は経路非依存 | 異なる Phase 順序で同じ VFE 到達 |
| **ルシャトリエの原理** | 忘却に対してシステムが抵抗する方向 | Nomoi の BC スコアの恒常性 |

## 7. 49d → 構造式: Phase C が解く問題

### 7.1 化学の歴史的教訓

化学は以下の順序で分子を理解した:
1. **分子式** (Dalton, 1808) — 原子の種類と数
2. **構造式** (Kekulé, 1858) — 結合パターンの明示
3. **立体化学** (van't Hoff, 1874) — 3次元配置
4. **量子化学** (Schrödinger, 1926) — 電子雲の確率分布

Lethe プロジェクトはこの歴史を圧縮して辿っている:
1. **49d ベクトル** = 分子式の段階 (2026-03: Phase B まで)
2. **CCL テキスト直接入力** = 構造式の段階 (Phase C: これから)
3. **Attention パターン** = 立体化学の段階 (Phase C 以降: Structural Attention)
4. **LLM 内部表現** = 量子化学の段階 (Phase B2 Attentive Probing の延長)

### 7.2 分子式の限界が Phase C を必然にする

49d (分子式) で R@1=6% は、化学で分子式だけから融点を予測するようなもの — **原理的に精度が限られる**。

Phase C が CCL テキスト (構造式) を transformer に直接読ませることは、化学における Kekulé の構造式の発明に対応する。構造異性体の区別が可能になり、予測精度が質的に向上する。
