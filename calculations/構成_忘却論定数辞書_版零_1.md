## 1. 核原則

この辞書の核は 1 つである。

> **同じ glyph でも、型が違えば別定数として扱う。**

したがって本稿では、`1` や `-1` や `i` を「同じ字だから同じもの」とは扱わない。  
むしろ `どの担体に属するか`、`どの射で生成されるか`、`どの法則に参加するか` を優先する。

---

## 2. 前提となる担体

本稿では、少なくとも次の 0-cell を前提に置く。

| 記号 | 役割 |
|:--|:--|
| `A_abs` | 絶対量の担体。加法 chart |
| `A_rel` | 相対量の担体。乗法 chart |
| `ΔA` | 変化の担体 |
| `Θ` | 角度の担体。位相の additive chart |
| `A_phase` | 位相そのものの担体 |
| `RotGen` | 回転生成子の記号層 |
| `A_cplx` | 加法・位相が合流する複素担体 |

補助射は次を採る。

- `add : A_abs × A_abs → A_abs`
- `neg : A_abs → A_abs`
- `mul : A_rel × A_rel → A_rel`
- `inv : A_rel^× → A_rel^×`
- `d : A_abs → ΔA`
- `∫_a : ΔA → A_abs`
- `log : A_rel → A_abs`
- `exp : A_abs → A_rel`
- `add_Θ : Θ × Θ → Θ`
- `cis : Θ → A_phase`
- `Exp_c : RotGen × Θ → A_cplx`
- `ι_abs : A_abs → A_cplx`
- `ι_phase : A_phase → A_cplx`

## 3. 定数辞書

### 3.1 基本表

| glyph | typed token | 本籍 | 型 | 正準生成 | 役割 |
|:--|:--|:--|:--|:--|:--|
| `0` | `0_abs` | `A_abs` | `1 → A_abs` | primitive | 加法単位元。絶対量ゼロ |
| `0` | `0_Δ` | `ΔA` | `1 → ΔA` | `d(const)=0_Δ` | 変化ゼロ |
| `0` | `0_Θ` | `Θ` | `1 → Θ` | primitive | 回転ゼロ |
| `1` | `1_abs` | `A_abs` | `1 → A_abs` | primitive | 絶対量 1 |
| `1` | `1_rel` | `A_rel` | `1 → A_rel` | primitive | 乗法単位元 |
| `1` | `1_phase` | `A_phase` | `1 → A_phase` | `cis(0_Θ)` | 位相単位元 |
| `-1` | `(-1)_abs` | `A_abs` | `1 → A_abs` | `neg(1_abs)` | 数直線上の反転点 |
| `-1` | `(-1)_phase` | `A_phase` | `1 → A_phase` | `cis(π_Θ)` | 半回転の到達点 |
| `e` | `e_rel` | `A_rel` | `1 → A_rel` | `exp(1_abs)` | 指数移送の自然基底 |
| `i` | `i_phase` | `A_phase` | `1 → A_phase` | `cis((π/2)_Θ)` | 四分の一回転の到達点 |
| `i` | `J_rot` | `RotGen` | generator | primitive symbol | 指数の中で回転を生成する作用素 |
| `π` | `π_Θ` | `Θ` | `1 → Θ` | primitive | 半回転の角度座標 |

### 3.2 最小コメント

- `0_abs` と `0_Δ` と `0_Θ` は、glyph は同じでも役割が違う。  
  `0_abs` は加法単位元、`0_Δ` は変化なし、`0_Θ` は回転なし。
- `1_abs` と `1_rel` と `1_phase` も同様である。  
  絶対量としての 1、乗法の単位としての 1、位相の単位としての 1 は同一視しない。
- `-1` は **二重国籍** を持つ。  
  `(-1)_abs` は符号反転点、`(-1)_phase` は半回転の到達点である。
- `i` も **二重化** する。  
  `i_phase` は到達点、`J_rot` は回転生成子である。
- `π` は位相点ではない。  
  あくまで角度 chart `Θ` の座標である。

## 4. 正準移送表

定数どうしの移送は、次を最小セットとして採る。

| 射 | 読み |
|:--|:--|
| `exp(0_abs) = 1_rel` | 加法ゼロは乗法単位へ移る |
| `log(1_rel) = 0_abs` | 乗法単位は加法ゼロへ戻る |
| `exp(1_abs) = e_rel` | 絶対量 1 が自然底を生む |
| `d(c) = 0_Δ` | 定数は微分で変化ゼロへ落ちる |
| `cis(0_Θ) = 1_phase` | 回転ゼロは位相単位 |
| `cis(π_Θ) = (-1)_phase` | 半回転は位相 `-1` |
| `cis((π/2)_Θ) = i_phase` | 四分の一回転は位相 `i` |

この表から直ちに読めることは 2 つある。

1. `e` の本籍は `A_abs` ではなく `A_rel` である。  
   `e` は `exp(1_abs)` によって生まれるからである。
2. `π` の本籍は `A_phase` ではなく `Θ` である。  
   `π` は位相そのものではなく、位相を指定する角度だからである。

## 5. 禁則

本稿では次を禁則とする。

### 5.1 `1` の潰し込み

- `1_abs = 1_rel` と書かない。
- `1_rel = 1_phase` と書かない。

理由: glyph の一致は型の一致を保証しない。

### 5.2 `-1` の潰し込み

- `(-1)_abs = (-1)_phase` と書かない。

理由: 前者は数直線の反転点、後者は unit phase の半回転到達点である。

### 5.3 `i` の潰し込み

- `i_phase = J_rot` と書かない。

理由: `i_phase` は回転後の点、`J_rot` は回転を生む生成子である。

### 5.4 `π` の誤配置

- `π` を `A_phase` の点にしない。
- `π` を `A_abs` の普通の定数として処理しない。

理由: 本稿での `π` はまず `Θ` の角度座標である。

### 5.5 `e^{iπ}` の即時実数化

- `e^{iπ}` を最初から `A_abs` の式だとみなさない。

理由: これは角度 chart、位相担体、複素担体をまたぐ複合表現である。

## 6. 最小読解規則

この辞書を使うと、各 glyph の自然な読解は次のように固定される。

| glyph | 最小読解 |
|:--|:--|
| `0` | 加法閉鎖の終点、または変化なし、または回転なし |
| `1` | 乗法不変の基準点、または絶対量 1、または位相単位 |
| `-1` | 数直線上の反転点、または半回転の到達点 |
| `e` | 指数移送の自然基底 |
| `i` | 四分の一回転の位相点、または回転生成子 |
| `π` | 半回転を指定する角度座標 |

この規則の核心は、

> `0` は加法側で自然、`1` は乗法側で自然、`e` は指数移送で自然、`π` は角度側で自然、`i` と `-1` は位相側で自然

という非対称性にある。

## 7. オイラー等式への最小接続

この辞書の目的は、`e^{i\pi} + 1 = 0` をただちに散文解釈することではなく、その前に **型崩れを止めること** にある。  
したがって現段階では、オイラー等式を次の 3 段に分けて読む。

1. `π_Θ` は角度 chart `Θ` にある  
2. `cis(π_Θ) = (-1)_phase` により、半回転が位相 `-1` に到達する  
3. それを複素担体 `A_cplx` に埋め、`ι_abs(1_abs)` と加えると `ι_abs(0_abs)` に閉じる

このため、表面形 `e^{i\pi}` は厳密型ではなく **短縮形** とみなす。  
より安全な正規形は、

```text
u_π := cis(π_Θ) = (-1)_phase
ι_phase(u_π) + ι_abs(1_abs) = ι_abs(0_abs)
```

または、生成子を露出するなら、

```text
Exp_c(J_rot · π_Θ) = ι_phase((-1)_phase)
Exp_c(J_rot · π_Θ) + ι_abs(1_abs) = ι_abs(0_abs)
```

である。
