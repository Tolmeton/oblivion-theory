## 2. Paper I (v0.10) — 詳細分析

### 2.1 強み

**方向性定理は本物の定理。** $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$ から $F = 0 \iff d(\Phi T) = 0$ は Leibniz 規則の直接的帰結であり、証明に穴はない。系 5.1.1（指数型分布族で $dT = 0$ → $F = 0 \iff d\Phi \wedge T = 0$）も正しい。ガウス族上の具体計算 $F_{12} = 6\mu/\sigma$ は数値的に機械精度で検証されている。

**忘却接続の構成が一意的に動機づけられている。** §3.2 の議論は的確: $\Gamma^{(\alpha)}_i$ は 1-形式でないため直接結合できず、テンソル的部分 $T_i$ を抽出するしかない。この「選択の一意性」の論証は v0.3 にはなかった改善。

**Appendix C の変分的安定性が正直。** §6.8 の曲率選択則が当初「正曲率 → 自発的忘却」と主張していたのを、Appendix C で「安定性の一次的決定因子は sign(λ) であり K ではない」と修正した。有効臨界値 $\lambda_c^{\text{eff}} \approx -0.301$ の数値計算も含め、主張の過剰さを自ら削った。

**命題 6.6.1（F の α-線形性）は小さいが重要な結果。** $A_i$ が α の1次式 → $F_{ij} = \partial_i A_j - \partial_j A_i$ も α の1次式。証明は自明だが、FEP 対応が「曲率レベルでは近似ではなく正確」という主張を支える。

### 2.2 問題点

**[P1-F1] 「力」の定義が曖昧。** §5.3 で「力」は $F_{ij} \neq 0$ の慣用的表現であり質点に作用する力 $f^i$ と直接同一視しないと注記しているが、タイトルが「力としての忘却」である以上、読者は「何に対する力か」を期待する。$F_{ij}$ がパラメータ推定の漸近的振る舞いに及ぼす影響は §6 で定量化すると言っているが、実際には α-遷移層力（§6.3）がガウス族上の1つの具体例を示すのみ。一般の統計多様体上で $F_{ij}$ がどのような「運動方程式」に結合するかは未定式化。

**改善案**: §6 に、Amari の自然勾配法との関係で $F_{ij}$ が推定量のバイアスに寄与する具体的メカニズムを書く。もしくは Fisher-Rao 測地線方程式に $F_{ij}$ が「力の項」として入る形式を導出する。

**[P1-F2] 選択的忘却実験（§5.5）の Ξ と定理の Ξ の乖離。** 方向性定理の Ξ は統計多様体上の忘却スペクトラム（パラメータ間の保持率分散）であり、§5.5 の実験の $\Xi_{\text{Var}}$ は「保持されたターン群と棄却されたターン群の間の情報量分散」。これらが同一の量であるという橋渡しが不在。r = +0.28 は有意だが、これが方向性定理の検証であるという主張には「LLM のコンテキスト圧縮 = 統計多様体上の粗視化」という非自明な同定が暗黙に使われている。

**改善案**: §5.5 冒頭に「操作化」の明示的な定義を追加。LLM のコンテキスト → 統計多様体のパラメータの対応を形式的に書く。

**[P1-F3] Appendix C.5 のスケーリング則に飛躍。** 命題 C.5.2 は「$V''(s_0) \to \alpha^2 n^2/2$ as $n \to \infty$」を主張するが、座標変換因子 $(d\theta/ds)^2 = 1/g_{11}^* \approx n$ の「$\approx n$」の根拠が曖昧。均等分布 $p_i = 1/(n+1)$ での $g_{11}^* = p_1(1-p_1) = n/((n+1)^2) \approx 1/n$ だから $1/g_{11}^* \approx n$ は正しいが、摂動モードの重心が均等分布からずれた場合のスケーリングは未検証。

## 3. Paper II (v1.1) — 詳細分析

### 3.1 強み

**FTC フックは優秀。** §1.1 の「微分 = 忘却、積分 = 構成、積分定数 = Drift」は読者を引き込む入口として機能する。Zhang-Guo-Keigher の Rota-Baxter 代数との接続は先行研究としても適切。

**§2.5 の CPS インスタンスの圏論的構成は最も価値ある節。** 各インスタンスに対し (1) 圏 $\mathcal{C}_D$, (2) 忘却関手 $U_A, U_B$, (3) 射の型, (4) $\Delta d$ の根拠, (5) Layer 1 四条件の検証, (6) Face Lemma の具体化 を6項目で定義している。これにより「CPS は何でも説明できるトートロジーではないか」という批判に対する具体的な回答を提供。

**§3.7 Blanket 生成定理は最も野心的で最も重要な主張。** FEP が blanket を前提するのに対し、CPS が Face Lemma から blanket の生成条件を導出するという構造は、もし成立すれば FEP に対する真の拡張。

### 3.2 問題点（致命的 3件 + 重要 5件）

**[P2-F1] Face Lemma の証明に本質的な穴がある（致命的）。**

Face Lemma は「安定性に必要な独立な生成射の最低数は 3」を主張する。証明の核心は dim Ξ = |独立な生成射| - 1 という定義に依存するが、この定義自体が循環的:

- 「独立な生成射」= 他の射の合成や恒等射として得られない射
- n = 2 のとき合成 g∘f は「独立でない」と主張するが、g∘f が f, g とは独立な情報を持つか否かは圏の構造に依存する
- n = 4 の冗長性の議論「horn-filling 条件により 2-simplex の合成で完全に決定される」は **nerveが Kan complex であることを暗黙に仮定している**。一般の圏の nerve は Kan complex ではない

**改善案**: (a) 圏の nerve が 2-coskeletal であるという条件を明示的に仮定に追加する。(b) dim Ξ の定義を nerve の homology 次元で厳密化する: $\dim \Xi = \text{rank}(H_1(N(\mathcal{C}); \mathbb{R}))$ とすれば、2-simplex の出現が $H_1$ の非自明性と対応し、3射の最小性が Hurewicz の定理から従う可能性がある。

**[P2-F2] Blanket 生成定理の Step 3 に隠れた仮定（致命的）。**

§3.7.2 の証明 (i) ⟹ (ii) の Step 3a で「合成制約 h = g∘f は以下を強制する: ext から int への任意の射 k は B を経由する」と述べるが、これは CPS 圏が **factorization system** を持つことを暗黙に仮定している。一般の圏では、g∘f = h が成立しても ext → int の射が B を経由するとは限らない（B を迂回する射が存在しうる）。

**改善案**: CPS 圏に orthogonal factorization system (OFS) の構造を仮定し、B が OFS の middle object であることを明示する。これは Layer 1 の条件に第5の条件として追加すべき。

**[P2-F3] α と Δd の層化（命題 3.3.1）は motivated definition であり証明ではない（致命的）。**

命題 3.3.1 は α(θ) ∈ Fiber(Δd) を主張するが、「証明」は (1)-(4) の手順を述べるのみで、α(θ_∞) の整数部分が Δd に対応するという主張に数学的な根拠がない。特に「α → +1 (m-接続) と α → -1 (e-接続) の非対称性が Δd = 1 に、より高次の cell 差には α の漸近値がより大きくなる」は **定義**であり**定理**ではない。

**改善案**: 命題ではなく定義（Definition）として再記述する。「α(θ_∞) の整数部分 = Δd」が数学的に導出される条件を明示し、それを公理として CPS に追加する。

**[P2-C1] Δd の割り当てが ad hoc（重要）。**

§2.3 で CPS3（時空/質量）の Δd = 2 は「位相 M = 0-cell, 計量 g ∈ Sym²T* = 2-cell」から来ると主張する。しかし位相多様体は 0-cell ではない — n 次元多様体は n-cell を含む。「0-cell」は「点の集合としての」位相を指すが、「計量が 2-cell」は「Sym² の 2」を cell 次元と同一視しており、基準が不統一。CPS4（心身問題）の Δd = 2 も同様の問題を抱える: 「状態空間 = 0-cell」は状態の**点**を指し、「曲率 = 2-cell」は曲率テンソルの**階数 2**を指す。テンソルの階数と cell 次元は一般には異なる概念。

**[P2-C2] §6.2 ゾンビの不可能性の論証が不完全（重要）。**

「十分に複雑な推論系では h（直接体験）の欠如は F = 0 を強制し、メタ認知 g の存在と矛盾する」とあるが、F = 0 がメタ認知 g を不可能にするメカニズムが不明。F = 0 は曲率ゼロ（忘却の方向性なし）を意味するが、曲率ゼロでもメタ認知の「計算」は可能（平坦多様体上でも推論は行える）。不可能になるのは「安定な」メタ認知だと §6.2 は言っているが、安定性条件と F ≠ 0 の関係が曖昧。

**[P2-C3] §7.1.4 の反証可能性の議論が自己言及に陥っている（重要）。**

(b) で「反証/確証対は CPS のインスタンスである」と述べ、(c) で「CPS の健全性は反証可能性ではなく自身の忘却パターンの認識で判定すべき」と述べる。これは論理的には整合的だが、外部の読者にとっては「反証できないことを正当化する枠組み」にしか見えない。少なくとも Layer 1 の具体的な反証手続き（「この実験でこの結果が出たら CPS は棄却される」）を§7.1.2 に明示すべき。

**[P2-C4] §3.7.3 FEP 包含定理の (b) VFE の導出が粗い（重要）。**

作用汎関数をエージェント軌跡上で評価して VFE を「回復する」と主張するが、$F_{ij}F^{ij}$ → Accuracy, $\lambda\Phi^2$ → Complexity の同定が形式的対応の域を出ない。VFE の Accuracy は $\mathbb{E}_q[\log p(o|s)]$ であり、$F_{ij}F^{ij}$ がこれに対応するには $F_{ij}$ が prediction error に比例することを示す必要がある。

**[P2-C5] 6インスタンスの中で「力の統一」(CPS5) が最も弱い（重要）。**

§2.3 の6インスタンスのうち CPS5（EM/重力/...）は C_D も U_A, U_B も具体的に定義されていない。Δd の根拠が「ゲージ群の cell 差」とのみ記述。Layer 1 の4条件の検証もなし。これは「予測」ではなく「希望」。

## 4. Paper III (v0.1) — 詳細分析

### 4.1 強み

**着想は刺激的。** Markov 圏の copy/del が α ≤ 0 で崩壊し、反可換構造（anti-copy）に置き換わるという主張は、もし厳密に構成できれば Fritz の枠組みに対する本質的な拡張。ボソン/フェルミオンの統計との対応も自然。

**§3.2 の反-Markov 圏の公理系は具体的。** (aM-1)–(aM-4) の4公理が明示され、Markov 圏との対比テーブルが読みやすい。

**§4.2 の (λ, α) 相図は Papers I–III を統合する視覚的ツール。** 有効質量 $\lambda_{\text{eff}}(\alpha) = \lambda + \alpha^2\langle|T|^2\rangle/4$ による臨界線は、Paper I の Appendix C と Paper III の Z₂-次数付け構造を単一の相図に収める。

### 4.2 問題点

**[P3-F1] 構造定理（§3.3）の証明 (ii) に本質的な問題。**

Step 3 で anti-copy を「$\text{anti-copy}_X(x) = \frac{1}{\|\Phi\|}\sum_{y \neq x}\Phi(y)(e_x \wedge e_y)$」と定義するが、この写像が CPS 圏の**射**として well-defined であることの検証がない。外積空間 $X \wedge X$ が CPS 圏の対象として存在するかどうかも未検証。Markov 圏では copy: X → X⊗X のコドメイン X⊗X は**テンソル積**（モノイダル構造から保証される）だが、反-Markov 圏で X∧X が同等のモノイダル構造から来ることを示す必要がある。

**[P3-F2] aM-3 の検証が Φ の具体的性質に依存しすぎる。** 「f がα構造を保存する射であること（$\Phi_Y \circ f \propto \Phi_X$）」は CPS 圏の一般的射に対しては成立しない。比例関係 $\Phi_Y \circ f = c \cdot \Phi_X$ は非常に強い条件であり、一般の忘却関手の射に対して自動的に成立するかどうか疑問。

**[P3-F3] TODO が多すぎる。** §2.4 (CD 圏の一般化), §5.1 (スピン-統計定理の再導出), §5.3 (意識のコピー不能性の検証), §4.3 (LLM での α 符号変化の実験) がすべて TODO。骨格としては理解できるが、証明の核心部分に TODO があるため、現段階では主張の妥当性を判定できない。

## 5. 文書間の整合性と矛盾

### 5.1 Paper I §8.3 と Paper II §3.7.4 の重複

両方が Champion et al. の EFE 統一問題を議論している。Paper I は Remark レベル（「この方向は Paper II で圏論的に展開する」）、Paper II は定理レベル（§3.7.4 で CPS による超克を主張）。重複自体は問題ないが、Paper I の Remark 8.3.1 が「可能性がある」と書く内容を Paper II §3.7.4 が「解消される」と書いている。トーンの不整合を修正すべき。

### 5.2 SST の行方

v0.3 の §8 SST（Stability Simplex Theorem）は Paper I の統一定理だった。v0.10 では Paper II に移動し、§3.4 Face Lemma と §3.5 Stability Simplex Theorem に分割された。移動自体は正しい判断だが、移動の過程で**証明の厳密性が下がった**: v0.3 では読み方関手 $R_A, R_B, R_C$ の構成と循環 A→B→C→A の証明スケッチがあったが、v1.1 の §3.5 では符号付き忘却量の証明（R_≥0 vs R の比較）にフォーカスが移り、読み方関手の構成が消えている。

### 5.3 Paper I の数値計算と Paper II の理論の乖離

Paper I §6.3–6.4 の α-遷移層力はガウス族 $\mathcal{H}^2$ 上での具体計算に基づく。Phase 2 の数値計算はカテゴリカルシンプレックス $\Delta^2$ 上で行われ、$F_2/F_1 \approx 0.15$–$1.38$（収束解）という結果を得ている。Paper II §3.3 の層化定理は $\alpha(\theta)$ と $\Delta d$ の対応を主張するが、数値計算で得られた α のプロファイルがこの層化構造と整合するかどうかは未検証。

### 5.4 Paper III の α < 0 と Paper II §4.3 の α < 0 の整合性

Paper II §4.3 は α < 0 を「脱構造化」として定性的に記述し、4つのインスタンス（熱力学、認知、生命、宇宙論）を列挙する。Paper III はこれを Z₂-次数付き圏として形式化する。両者の α < 0 の意味は一致するが、Paper II が「Markov category 構造の崩壊」として否定的に記述するもの（§3.7.1 の (i)–(iv)）を、Paper III は「反-Markov 圏として再構成可能」と肯定的に記述する。この視点の反転は整合的だが、Paper II の読者には Paper III の存在が見えないため、Paper II §4.3 の記述が「α < 0 は扱えない」という誤った印象を与える。Paper II に「α < 0 の代数的構造は Paper III で定式化する」と前方参照を追加すべき。

## 6. 「圏論的衣装」の検出

### 6.1 本質的に機能している圏論

- **Paper I の方向性定理**: 外微分 d, 外積 ∧, Leibniz 規則は微分幾何の道具であり圏論ではない。定理の内容は「$d(\Phi T) \neq 0 \iff F \neq 0$」であり、圏論なしで完全に述べられる。ただし Chebyshev 1-形式 $T_i$ の構成には Amari-Chentsov テンソルが必要であり、これは情報幾何の固有の道具。**圏論は不要だが情報幾何は本質的。**

- **Paper II §2.5 の CPS スパン**: 忘却関手 $U_A, U_B$ の Layer 1 条件（faithful, 左随伴, 結合生成, 非同型）は圏論固有の概念であり、集合論や代数では表現できない。CPS の構造はスパンの圏論的概念に本質的に依存している。**圏論は本質的。**

- **Paper II §3.7 の Fritz Markov category**: 条件付き独立性を合成的に定式化する Fritz の枠組みは圏論の威力が遺憾なく発揮される箇所。**圏論は本質的。**

### 6.2 装飾的な圏論

- **Paper II §3.4 Face Lemma の「nerve の 2-skeleton」**: 本質的な主張は「3つの独立な関係があれば検証可能性が生まれる」であり、nerve の言語は必要ない。3体問題の制約（2体では自由度不足、3体で充足）として述べれば圏論なしで伝わる。
  追記: この再記述はさらに符号理論の語彙へ押し出せる。Face Lemma を「圏論版 syndrome 条件」と読み、stable / detectable / recoverable を分離して定義すれば、nerve のオーバーヘッドを下げつつ操作的意味を保持できる。参照: `drafts/infra/FaceLemma_技術設計.md`

- **Paper II §6.1–6.2 の「創発 = 1-skeleton → 2-skeleton」**: 創発の定義としては興味深いが、実際に使われる場面（意識のハードプロブレム）では圏論的定義が具体的な予測を生まない。「3射が必要」は直感的にもわかる主張であり、nerve の形式化はオーバーヘッド。

- **Paper III §3.1 の Z₂-次数付き CPS 圏**: 構成は厳密だが、Koszul 符号則の導入が CPS 固有の予測を生むかどうかが不明。ボソン/フェルミオンの区別を CPS で「再解釈」しても、新しい予測が出なければ言い換えにすぎない。

## 7. 最も致命的な弱点（体系を根底から崩しうる1点）

**Paper II の Face Lemma が証明として成立していない。**

Face Lemma は Paper II 全体の基盤。Blanket 生成定理（§3.7.2）、FEP 包含定理（§3.7.3）、EFE 統一（§3.7.4）、創発の定義（§6.1）、意識のハードプロブレム（§6.2）、ゾンビの不可能性——これらすべてが Face Lemma の「3射で安定性が生まれる」に依存する。

現在の「証明」は dim Ξ = |独立な生成射| - 1 という定義に依存し、n = 4 の冗長性が nerve の Kan complex 性を暗黙に仮定している。この仮定が明示されなければ、Face Lemma は定理ではなく**定義**——「3射で安定性が生まれると定義する」——に退化し、その上に積み上げられた全ての帰結がdefinitional rather than theoremicになる。

**これは修正可能。** dim Ξ を nerve の simplicial homology で再定義し、3射の最小性を Euler 標数や Betti 数の議論で示す方向が有望。あるいは、CPS 圏に明示的な条件（例: 圏が finite group-like、または nerve が 2-coskeletal）を追加して、その条件下で Face Lemma を証明する。

## 10. 全体的所見

3本の論文は異なる成熟度にある。

**Paper I は出版に近い。** 方向性定理は正しく、数値検証も堅実で、FEP 対応の議論も誠実に限界を認めている。残る課題は「力」の具体的メカニズム（§6 の拡充）と実験の操作化の明示化。v0.3 → v0.10 の改善は着実で、方向は正しい。

**Paper II は構造的に野心的だが、基盤（Face Lemma）に穴がある。** Face Lemma が定理として確立されれば、Blanket 生成定理 → FEP 包含定理の流れは強力。しかし現状では Face Lemma が定義と定理の境界にあり、その上の構築物は砂上の楼閣のリスクを抱えている。最優先課題は Face Lemma の証明の厳密化。

**Paper III は着想段階。** Z₂-次数付き CPS 圏という概念は興味深いが、anti-copy の構成の well-definedness や反-Markov 公理の検証に本質的な作業が残る。現段階では「こういう構造があり得る」という提案であり、定理の体系ではない。

**3本の関係**: Paper I が定理（方向性定理）→ Paper II が構造（CPS スパン）→ Paper III が拡張（Z₂-次数）という流れは自然。Paper I の堅実さを Paper II/III が引き継げるかどうかが、体系全体の命運を握る。
