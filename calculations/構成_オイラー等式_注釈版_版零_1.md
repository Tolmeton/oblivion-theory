## 1. この面の核

本稿の核は単純である。

> **オイラー等式は 1 本の裸の式ではなく、角度 chart、位相担体、加法担体が合流した圧縮表記である。**

したがって本稿では、まず

```text
e^{i\pi} + 1 = 0
```

をそのまま「全部が同じ種類の数の式」とは読まない。  
先に「どこで何が起きているか」を分け、その後で最後の等式を読む。

---

## 2. 表面形と正規形

### 2.1 表面形

人間が日常的に読む形はこれである。

```text
e^{i\pi} + 1 = 0
```

これは短く、美しい。  
だが短いぶん、少なくとも 3 種の層を潰している。

- `π` は角度 chart の座標
- `i` は回転生成子と回転到達点を兼記している
- `+1=0` は複素担体への埋め込み後の加法閉鎖を圧縮している

### 2.2 第一正規形

位相到達点を露出すると、まず次が現れる。

```text
u_π := cis(π_Θ) = (-1)_phase
```

これは、

- `π_Θ` が角度 chart `Θ` にある
- 半回転を与えると
- unit phase 上の到達点 `(-1)_phase` に着く

という 3 段を明示している。

### 2.3 第二正規形

加法閉鎖まで露出すると、より安全な形は次である。

```text
ι_phase(u_π) + ι_abs(1_abs) = ι_abs(0_abs)
```

ここで初めて「足し算」が起きる。  
つまり `+1=0` は、最初から実数直線で起きているのではなく、位相側の到達点を複素担体へ埋めたあとで起きる。

### 2.4 生成子を露出した形

`i` の潰し込みを避けたいなら、さらに次の形が安全である。

```text
Exp_c(J_rot · π_Θ) = ι_phase((-1)_phase)
Exp_c(J_rot · π_Θ) + ι_abs(1_abs) = ι_abs(0_abs)
```

ここでは

- `J_rot` は回転を生む生成子
- `i_phase` は四分の一回転の到達点

として分離される。  
表面形 `e^{i\pi}` は、この二つを 1 字に潰した短縮形として扱う。

## 3. 項別注釈

### 3.1 `e`

`e` は「ただ特別な数」ではない。  
本稿では、`e` は **指数移送の自然基底** であり、忘却が連続的に進むときに現れる基準核として読む。

直感的には、

- 足し算の世界で 1 だけ進む
- それを指数移送で乗法の世界へ送る
- その最初の到達点が `e`

である。

したがって `e` の本籍は `A_abs` ではなく `A_rel` である。

### 3.2 `i`

`i` は最も危険な glyph である。  
この字は、実際には 2 つの役を同時に背負っている。

| 記号 | 役割 |
|:--|:--|
| `J_rot` | 回転を生む生成子 |
| `i_phase` | 四分の一回転で到達する位相点 |

したがって、表面式の `i` をそのまま「一点」だと思うと崩れる。  
指数の中では `J_rot` が働き、到達点としては `i_phase` が現れる。

### 3.3 `π`

`π` はまず角度である。  
円周率という数値面をいったん脇へ置くと、本稿での `π` は **半回転を指定する角度座標** として読むのが自然である。

したがって `π` の本籍は `A_phase` ではなく `Θ` である。

### 3.4 `1`

`1` は「単位」であっても、どの世界の単位かを区別しなければならない。  
オイラー等式の右半分で効いている `1` は、最終的には加法側へ埋め込まれた `1_abs` である。

ここでの読みは、

- 反転しなかった元の基準が `1`
- 半回転で到達した `-1` と
- 同じ加法面に載せると
- 相殺して `0` に閉じる

である。

### 3.5 `0`

`0` も単なる「無」ではない。  
ここでの `0` は **加法閉鎖の終点** である。

重要なのは、

- 対象が蒸発した

のではなく、

- 正方向の単位と反転後の位相が
- 同じ加法面で相殺された

という点である。

### 3.6 隠れた `-1`

オイラー等式の表面には `-1` は直接書かれていない。  
だが実際には、この式の背骨は

```text
cis(π_Θ) = (-1)_phase
```

である。  
`e^{i\pi}` は、半回転が `-1` に到達するという事実を、指数表示で巻いた短縮形だと読める。

## 4. 3 段読解

### 4.1 角度指定

最初にあるのは `π_Θ` である。  
これは「どれだけ回すか」を指定する角度 chart 上の座標であり、まだ加法の `-1` ではない。

### 4.2 位相到達

次に起きるのは

```text
cis(π_Θ) = (-1)_phase
```

である。  
半回転をかけると、unit phase 上の反対側の点に着く。  
ここで出る `-1` は、まず位相の `-1` であって、数直線上の `-1` とは本籍が違う。

### 4.3 加法閉鎖

最後にその位相点を複素担体に埋め、

```text
ι_phase((-1)_phase) + ι_abs(1_abs) = ι_abs(0_abs)
```

が起きる。  
ここで初めて、私たちが慣れ親しんでいる「足してゼロになる」という読解が可能になる。

## 5. 忘却論としての読み

この式を忘却論として読むと、五つの定数が一度に「同じもの」だと言っているのではない。  
むしろ逆で、**別々の層に属していたものが、ある特定の経路で合流できる** と言っている。

### 5.1 `e`

`e` は忘却の連続核である。  
忘却が一歩ずつではなく連続に進むとき、その変化率を自分自身で保つ最小基底が `e` になる。

### 5.2 `i`

`i` は「虚数」というより、まず回転の印である。  
忘却方向が単なる減衰だけでなく位相回転を伴うとき、その直交回転を担う最小印が `i` になる。

### 5.3 `π`

`π` は「半周期」である。  
回転がちょうど反対側へ達する最初の節目として、半回転の座標が `π` になる。

### 5.4 `+1=0`

これは「元の保存」と「反転後の到達」が、同じ加法面で相殺されることを表す。  
したがってオイラー等式は、消滅の式というより **反転と保存が同じ面で閉じる式** と読んだほうが強い。

## 6. 禁則

本稿では次を禁則とする。

### 6.1 全部を同じ数だと思うこと

- `π` を最初から普通の実数定数として扱わない
- `i` を常に一点だと思わない
- `-1` を数直線の `-1` だけだと思わない

### 6.2 表面式をそのまま正規形だと思うこと

- `e^{i\pi}` をそのまま最終的な型付き表現だと思わない

理由: この書き方は、角度、位相、生成子、加法閉鎖を 1 行に折り畳んだ圧縮記法だからである。

### 6.3 `0` を無内容な消失だと思うこと

- `0` を「何もなくなった」とだけ読まない

理由: ここでの `0` は、埋め込み後の加法面で相殺が閉じた endpoint である。

## 8. この面の仕事

この注釈面の仕事は 3 つに限られる。

1. オイラー等式の圧縮をほどく  
2. 各定数の本籍を固定する  
3. 後続で忘却論の散文へ展開するときの足場を与える

したがって本稿の到達点は、「オイラー等式を証明した」ではない。  
到達点は、

> **`e^{i\pi} + 1 = 0` を、どこで何が起きている式なのかを崩さずに読める状態にした**

という一点にある。
