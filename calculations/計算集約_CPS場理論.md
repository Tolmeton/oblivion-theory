## 1. 設定と基本方程式

**結合 Euler-Lagrange 方程式** (v2 §8.1):

$$\text{方程式 I}: \nabla^2 \alpha - V'(\alpha)\phi(\Theta) = 0$$
$$\text{方程式 II}: \kappa\nabla^2\Theta - V(\alpha)e^{-\Theta} + \mu^2 - \nu^2\Theta = 0$$

- **φ(Θ) = 1 - e^{-Θ}** (スクリーニング関数)  
- **V(α) = (λγ/3)α³**  
- **αc = (3ν²/λγ)^{1/3}** (臨界結合定数)

**二つの力**:

$$F_1 = |\nabla\alpha| \cdot \phi(\Theta) \quad \text{(第一の力 — 非対称性の不均一)}$$
$$F_2 = \kappa|\nabla\Theta| \quad \text{(第二の力 — マスクの不均一)}$$

## 2. Δ² 上の幾何

**Δ² の Fisher 計量**: 曲率 K = 1/4 (正曲率)  
**Θ の閉形式** (α非結合時):

$$\Theta(p_1, p_2) = (1-p_1)\left[\ln 2 - H\!\left(\frac{p_2}{1-p_1}\right)\right]$$

**Θ の拘束**: Θ = 0 が Δ² 上で不安定 (正曲率のため)。均一 Θ₀ が安定基底状態。

### BC と解の分類

| BC | 解の種類 |
|:---|:---|
| Neumann (∂Θ/∂n=0) | 均一 Θ₀ が唯一の安定解 |
| Dirichlet (α 固定) | Dome 解: 単体重心 (1/3,1/3,1/3) に極大 |

**2D 臨界**: m²_eff(2D) ≈ 11 (1D の約 4 倍)。正曲率が閉じ込め効果を生む。

## 3. Higgs 型相転移

**ポテンシャル** (v2 §8.1):

$$U(\Theta) = -\mu^2\Theta + \frac{\nu^2}{2}\Theta^2$$

- **固定点 FP1**: (α=0, Θ=0.5) — MB 相の平衡
- **固定点 FP2**: (α=1.0, Θ=0) — QM 相の臨界点

**Higgs 型** (Θ 単独計算): 臨界 m²_eff ≈ 2.75 で対称性の破れ。Bump 解が自発的に生成。

## 4. Coleman-Weinberg による U(Θ) の導出

**CW ポテンシャル**: 1-ループ有効ポテンシャルから U(Θ) を導出。

$$B = \frac{3g^4}{64\pi^2}, \quad U_{CW}(\Theta) = B\Theta^4\!\left[2\ln\Theta - \frac{1}{2}\right]$$

**自由パラメータの消去**: μ², ν² → g, g_c から完全決定。残る自由パラメータは **κ のみ**。

$$\boxed{\text{κ は Allen-Cahn 型の界面幅パラメータ。物理的自由度はκ1つ}}$$

**v2 §9 SU(2) との接続**: κ = 1/3 が Zamolodchikov 計量から導出 (§3 参照)。

## 6. Zamolodchikov 計量から κ の導出

**前提**: κ = SU(2) Yang-Mills の Zamolodchikov 計量の成分  

$$g_{αα}^{Zamolodchikov} = \frac{1}{3} \quad \Rightarrow \quad \boxed{\kappa = \frac{1}{3}}$$

これにより CPS の全パラメータが消去され **自由パラメータゼロ** に到達。

## 7. 収束改善の結果

**手法**: Continuation method (λγ = 0 → 3) + Active-Set Newton  
**残差改善**: Picard 前回 10⁰–10¹ → Active-Set Newton **10⁻¹³** (λγ ≤ 1.0)

| λγ | 残差 | Active 点 (%) | αc |
|:---|:---|:---|:---|
| 0.60 | 10⁻¹¹ | 0 | 2.15 |
| 1.00 | 10⁻¹³ | 3.3% | 1.82 |
| **3.00** | **10⁻¹³** | **21.8%** | **1.26** |

**警告**: λγ = 1.5–2.5 は continuation gap で未収束。λγ = 3.0 の解枝に疑問あり。

**λγ = 0.60 の確実な結果** (F2/F1):

$$\boxed{F_2/F_1 = 0.15 \sim 1.38 \quad (p_1 \approx 0.42 \text{ で } F_1 = F_2)}$$

**前回の過大評価修正**: 旧 F2/F1 ≈ 10–12 は Picard の不収束による誤り。実際は同程度のオーダー。

## 8. α_c 界面の構造 (最重要発見)

### 力の反転

| 相 | F2/F1 | 物理的意味 |
|:---|:---|:---|
| **QM** (α < αc) | **6–12** (F2 支配) | Θ 急変動 → ∇Θ 大 → マスクの不均一が力を生む |
| **界面** (α ≈ αc) | **≈ 1** (均衡) | 量子的力と古典的力の切り替え帯 |
| **MB** (α > αc) | **0.1** (F1 支配) | Θ 飽和 (≈ 4.1) → ∇Θ ≈ 0 → 非対称性の不均一のみ |

### 勾配のアラインメント

| 領域 | cos(∇α, ∇Θ) |
|:---|:---|
| QM (深部) | +0.50 ± 0.45 (部分的平行) |
| 界面 | +0.19 ± 0.49 (**ほぼ直交**) |
| MB | +0.06 ± 0.55 (**ほぼ直交**) |

「∇α ∥ ∇Θ (cos ≈ 0.999)」は **QM 相内部に限定された現象** (前回計算を修正)。
