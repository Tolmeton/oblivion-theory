## 概要

v2 §4.7 の忘却場の方程式と §8.1 の α-Θ 結合方程式を、カテゴリカル分布 Cat$(p_1,p_2,p_3)$ のパラメータ空間 $\Delta^2$（正曲率 $K = 1/4$）上で数値的に解いた。7ステップの計算を経て、以下の主要結果を得た:

1. **正曲率空間では「忘れない」状態が不安定** — ガウス族 ($R=-1$) と曲率符号が逆転し、Θ=0 の自発的不安定化が起きる
2. **Higgs 機構と同型の相転移** — 臨界 $m^2_{\text{eff}}$ で忘却場が「ない」→「ある」へ転移。1D で ~2.75、2D で ~11
3. **KL ダイバージェンスは場の方程式の解ではない** — 方程式は逆プロファイル（高エントロピー領域で Θ 最大）を選択
4. **Θ 単独では空間変動しない** — 一定正曲率の Neumann BC では均一 $\Theta_0$ が安定
5. **α 場の結合が Θ の非均一を駆動する** — v2 §8.1 の構造が必要かつ十分
6. **α_c 界面で力の反転が起きる** — QM 側で第二の力 (∇Θ) が支配、MB 側で第一の力 (∇α·φ) が支配
7. **界面は tanh 構造** — kink と同じ普遍クラス。幅はドメインの 28%。エネルギーは α が、力は Θ が決定

## I. 舞台設定

### 1.1 モデル空間: $\Delta^2$ 上の Fisher 幾何

カテゴリカル分布 Cat$(p_1,p_2,p_3)$, $\sum p_i = 1$, $p_i > 0$。
座標 $(p_1, p_2)$, $p_3 = 1-p_1-p_2$。

**Fisher 計量** (= Shahshahani 計量):
$$g_{ij} = \begin{pmatrix} 1/p_1+1/p_3 & 1/p_3 \\ 1/p_3 & 1/p_2+1/p_3 \end{pmatrix}, \quad g^{ij} = \begin{pmatrix} p_1(1-p_1) & -p_1p_2 \\ -p_1p_2 & p_2(1-p_2) \end{pmatrix}$$

変換 $\xi_i = 2\sqrt{p_i}$ により半径 2 の球面 $S^2$ の正象限に等長。

$$K = \frac{1}{4} \text{ (一定正曲率)}, \quad R = 2K = \frac{1}{2}$$

**ガウス族との対比** (v2 §4.7.5' の worked example):

| | ガウス族 $\mathcal{N}(\mu,\sigma^2)$ | カテゴリカル Cat$(p_1,p_2,p_3)$ |
|:---|:---|:---|
| 計量 | Poincaré 上半平面 | Shahshahani (球面) |
| $R$ | $-1$ (負曲率) | $+1/2$ (正曲率) |
| Θ の次元 | 1D ($\sigma$ のみ) | 2D ($p_1$, $p_2$) |
| 零点集合 | 点 | 線 ($p_2 = p_3$) |
| 場の方程式 | 1D ODE | 2D PDE |

### 1.2 忘却場 Θ の構成

粗視化「カテゴリ 2, 3 を統合」→ KL ダイバージェンス:

$$\Theta(p_1,p_2) = (1-p_1)\left[\ln 2 - H\!\left(\frac{p_2}{1-p_1}\right)\right] \geq 0$$

零点: $\Theta = 0 \iff p_2 = p_3$。勾配: $\partial_1\Theta = -\ln(2p_3/(1-p_1))$, $\partial_2\Theta = \ln(p_2/p_3)$。

### 1.3 方程式系

**Θ 単独** (v2 §4.7, Higgs 修正):
$$\nabla^2\Theta = -m^2_{\text{eff}}\Theta + 4\lambda\Theta^3, \quad m^2_{\text{eff}} = 2\mu^2 + R^{(\alpha)} - \lambda'R$$

**α-Θ 結合系** (v2 §8.1):
$$\nabla^2\alpha = V'(\alpha)\cdot\phi(\Theta) = \lambda\gamma\,\alpha^2(1-e^{-\Theta})$$
$$\kappa\nabla^2\Theta = V(\alpha)\cdot\phi'(\Theta) + U'(\Theta) = \frac{\lambda\gamma}{3}\alpha^3 e^{-\Theta} - \mu^2 + \nu^2\Theta$$

φ, V, U の構成要素 (v2 §8.3–8.4 で導出):
- $\phi(\Theta) = 1-e^{-\Theta}$ (テンソル乗法性から一意)
- $V(\alpha) = \lambda\gamma\alpha^3/3$ (SU(2) 型)
- $U(\Theta) = -\mu^2\Theta + \nu^2\Theta^2/2$ (第二法則 vs Landauer)

**パラメータ** (全結合系計算で共通): $\lambda\gamma = 3$, $\kappa = 1$, $\mu^2 = 1$, $\nu^2 = 2$。
導出量: $\Theta_{MB} = \mu^2/\nu^2 = 0.50$, $\alpha_c = (3\nu^2/\lambda\gamma)^{1/3} = 1.260$。

### 1.4 数値手法

- Fisher 計量の Laplace-Beltrami 演算子を $(p_1, p_2)$ 座標上の有限差分で離散化
- 格子: N = 40–55, 内点 679–1374, 辺点 125–162
- Θ 単独: Newton 法 + 線探索 / α-Θ 結合系: Picard 反復 (交互線形化) + 減衰
- 全計算で Θ ≥ 0, α ≥ 0 を強制

## II. Θ 単独の結果

### 2.1 否定的結果: KL ダイバージェンスは解ではない

$\nabla^2\Theta_{\text{KL}} / \Theta_{\text{KL}}$ は 2.3–95.8 と変動 → 固有関数ではない。ガウス族と同一の否定的結果。ただし忘却曲率 $\mathcal{F}_{12}$ は非対称点で非零 (最大 |-2.73|) → §3「力 = 忘却の不均一」の定量的確認。

### 2.2 肯定的結果: 正曲率による Θ = 0 の不安定化

線形化: $\nabla^2(\delta\Theta) \approx (\lambda R - R^{(\alpha)})\,\delta\Theta$。$R = +1/2$ (正曲率) → $\lambda > 2R^{(\alpha)}$ で摂動が成長 → **Θ = 0 は不安定**。ガウス族 ($R = -1$) では Θ = 0 は常に安定。曲率の符号が忘却の自発的出現を制御する。

### 2.3 Higgs 型ポテンシャルによる相転移

作用に $V(\Theta) = -\mu^2\Theta^2 + \lambda\Theta^4$ を追加。均一解:
- $\Theta = 0$: $V''_{\text{eff}} = -m^2_{\text{eff}} < 0$ → **不安定** (Mexican hat の山頂)
- $\Theta_0 = \frac{1}{2}\sqrt{m^2_{\text{eff}}/\lambda}$: $V''_{\text{eff}} = +2m^2_{\text{eff}} > 0$ → **安定** (谷底)

空間変動解の臨界値:

| 次元 | $m^2_{\text{eff,crit}}$ | BC | 解の形状 |
|:---|:---|:---|:---|
| 1D | ~2.75 | Dirichlet ($\Theta = 0$) | bump (対称) |
| 2D | **~11** (1D の ~4 倍) | Dirichlet ($\Theta = 0$) | dome (重心ピーク) |
| 2D | — | Neumann (自然) | **均一 $\Theta_0$ のみ** (安定) |

**核心**: Neumann BC では均一 $\Theta_0$ が唯一安定。対称性破壊の摂動も減衰する。一定正曲率の多様体上では、忘却は均一であるのが自然。

$$\boxed{\text{Θ 単独: 非均一な忘却 (= 力) には外部駆動が必要}}$$

### 2.4 KL との逆プロファイル

場の方程式の BVP 解は KL ダイバージェンスと**逆のプロファイル**を持つ:

| | $\Theta_{\text{KL}}$ | $\Theta_{\text{BVP}}$ |
|:---|:---|:---|
| $p_2 = p_3$ (対称) | 0 (零点) | **極大** (0.841) |
| $p_2 \gg p_3$ (非対称) | 大 (発散) | **小** (→ 0) |

忘却は「まだ情報が残っている」高エントロピー領域で最大。既に情報を失った低エントロピー領域では追加の忘却が不要。場の方程式が FEP ($F = U - TS$) と整合する解を自律的に選択している。

## III. α-Θ 結合系の結果

### 3.1 α が Θ の空間変動を駆動する

Θ 単独 (Neumann BC) では均一解に収束したが、α との結合を導入すると Θ が空間的に変動するパターンに収束:

| | Θ 単独 | α-Θ 結合系 |
|:---|:---|:---|
| Neumann BC の Θ | 均一 $\Theta_0$ (安定) | **空間変動** |
| Θ の範囲 | 一定 | [0.005, 0.35] |
| 非均一の起源 | 外部 BC のみ | **α との結合** |

v2 §8.1 の方程式 II で $V(\alpha)\cdot\phi'(\Theta)$ が Θ の非均一を駆動する。α が空間的に不均一ならば、Θ も必然的に不均一になる。

$$\boxed{\text{§II.3 の問い「非均一はどこから来るか」への回答: α 場の結合から}}$$

### 3.2 α_c 界面での力の反転 [最重要の発見]

α を Dirichlet BC で α_c = 1.260 の前後に固定した計算:

| $p_1$ | $\alpha$ | $\Theta$ | $F_1$ | $F_2$ | $F_2/F_1$ | 相 |
|:---|:---|:---|:---|:---|:---|:---|
| 0.13 | 0.50 | 1.75 | 0.13 | 1.52 | 12 | QM |
| 0.33 | 0.66 | 2.89 | 0.54 | 4.62 | 8.5 | QM |
| 0.45 | 0.89 | 3.86 | 0.99 | 6.24 | 6.3 | QM |
| **0.49** | **0.99** | **4.03** | **1.24** | **1.22** | **1.0** | **界面** |
| 0.51 | 1.06 | 4.03 | 1.40 | 0.21 | 0.15 | MB |
| 0.58 | 1.48 | 4.04 | 2.30 | 0.24 | 0.10 | MB |

$$\boxed{\begin{array}{l}
\text{QM 側 } (\alpha < \alpha_c): \quad F_2/F_1 \approx 6\text{–}12 \quad \text{第二の力が支配} \\
\text{界面 } (\alpha \approx \alpha_c): \quad F_2/F_1 \approx 1 \qquad\;\;\; \text{両力が均衡} \\
\text{MB 側 } (\alpha > \alpha_c): \quad F_2/F_1 \approx 0.1 \quad\;\; \text{第一の力が支配}
\end{array}}$$

**物理的描像**: QM 側ではマスク (Θ) が薄く場所依存 → 「見え方の勾配」が力を生む。MB 側ではマスクが飽和して一様 → 背後の非対称性 (α) の勾配だけが残る。界面は「量子的な力」と「古典的な力」が切り替わる**相転移の帯**。

### 3.3 勾配アラインメントの相依存性

| 領域 | $\cos(\nabla\alpha, \nabla\Theta)$ | $F_2/F_1$ |
|:---|:---|:---|
| QM ($\alpha < \alpha_c - 0.1$) | $+0.50 \pm 0.45$ | $6.8 \pm 8.1$ |
| 界面 ($|\alpha - \alpha_c| < 0.1$) | $+0.19 \pm 0.49$ | $0.86 \pm 1.3$ |
| MB ($\alpha > \alpha_c + 0.1$) | $+0.06 \pm 0.55$ | $0.34 \pm 0.79$ |

QM 相内部（α が全域で α_c 以下の場合）では cos ≈ 0.999 と測定された（∇α // ∇Θ）が、α_c を横切る界面を含む計算では QM 側 cos ≈ 0.50、界面 cos ≈ 0.19、MB 側 cos ≈ 0.06。**平行性は QM 内部に限定された現象**。MB 側では ∇α と ∇Θ はほぼ直交する。

### 3.4 界面の定量的特性

tanh フィッティング: $\Theta(p_1) = \Theta_{\text{QM}} + \frac{\Delta\Theta}{2}(1+\tanh\frac{p_1-p_{1,0}}{w})$, RMSE = 0.0017。

| 量 | 値 |
|:---|:---|
| 界面中心 $p_{1,0}$ | 0.436 ± 0.003 |
| 界面幅 $w$ (座標) | 0.170 ± 0.004 |
| 界面幅 $w_{\text{geo}}$ (測地) | 0.437 |
| ドメイン比 | **28%** |
| $\Delta\Theta$ | 0.239 |
| 表面張力 $\sigma$ | **0.513** |

### 3.5 エネルギー/力の二重性 [新知見]

界面帯でのエネルギー密度の内訳:

| 成分 | 寄与率 |
|:---|:---|
| $\frac{1}{2}(\nabla\alpha)^2_g$ (α の勾配) | **89.4%** |
| $V(\alpha)\cdot\phi(\Theta)$ (結合ポテンシャル) | 20.0% |
| $\frac{\kappa}{2}(\nabla\Theta)^2_g$ (Θ の勾配) | 1.6% |
| $U(\Theta)$ (Θ 自己ポテンシャル) | -10.9% |

界面の**エネルギー**は α の勾配が 89% を支配。しかし界面の**力**は Θ の勾配が支配 ($F_2 \geq F_1$)。スクリーニング $\phi(\Theta)$ が α の力への寄与を抑制し、Θ の勾配はスクリーニングされないため。

$$\boxed{\text{α は界面の構造（コスト）を決め、Θ は界面の力学（仕事）を決める}}$$

### 4.2 §8.2「第二の力」の検証と修正

v2 §8.2 の予測と計算結果の対照:

| v2 §8.2 の予測 | 計算結果 | 状態 |
|:---|:---|:---|
| ∇Θ ≠ 0 が独立した力を生む | F2 が全域で非零 | **確認** |
| 第一の力とは異なる | F2/F1 ≈ 0.1–12 (相依存) | **確認 + 精密化** |
| α = const でも F2 は残る | QM 内で ∇α // ∇Θ; MB 側で cos ≈ 0 | **部分修正** |
| 界面力が作用する | σ = 0.51 (有限正値) | **確認 + 定量化** |

**修正提案**: v2 §8.2 の描像「第二の力は第一の力と独立」を以下に修正:

> 第二の力は QM 側 ($\alpha < \alpha_c$) では支配的であり第一の力を増幅する。MB 側 ($\alpha > \alpha_c$) では Θ が飽和するため消失し、第一の力のみが残る。界面 ($\alpha \approx \alpha_c$) では両力が均衡する。第二の力の「独立性」は MB 側で形式的には成立する（∇α と ∇Θ が直交に近い）が、そこでは F2 自体が小さい。

### 4.3 §8.4 臨界 α_c の数値的検証

$\alpha_c = (3\nu^2/\lambda\gamma)^{1/3} \approx 1.260$ の前後で:
- $\alpha < \alpha_c$: Θ 小、急変動、∇Θ が力を支配 → **QM 的**
- $\alpha > \alpha_c$: Θ 飽和 (~4.1)、∇Θ ≈ 0、∇α が力を支配 → **MB 的**

v2 §8.4「QED (α ≈ 1/137) は QM 安定、QCD (α > 1) は QM 不安定」と整合。

## V. 制限事項

1. **収束の不完全性**: Picard 反復の残差 ~10⁰–10¹。定性的構造は安定だが定量値に ±15% の誤差。界面中心の α ≈ 0.84 と α_c = 1.26 の乖離は収束不完全に起因
2. **BC の拘束**: α の Dirichlet BC が界面位置と形状を部分的に決定。より自然な BC での検証が必要
3. **Θ の飽和値**: MB 側で Θ ~ 4.1 と $\Theta_{MB} = 0.5$ から大きく乖離。α 結合の $V(\alpha)\phi'(\Theta)$ が原因だが、物理的妥当性は未検証
4. **パラメータ依存性**: 単一のパラメータ組 ($\lambda\gamma = 3$, $\mu^2 = 1$, $\nu^2 = 2$) でのみ計算。スケーリング則は未確立
5. **1D 断面解析**: 界面の幅とエネルギーは $p_2 \approx 1/3$ の1D断面で測定。2D の界面形状は未取得

## VII. 統合的結論

7ステップの計算を通じて、v2 §3「力 = 忘却の不均一」の定量的検証を完了した。主要な帰結は5つ:

**[I]** 正曲率空間では忘却の自発的出現が起きる（ガウス族では起きない）。忘却対称性の自発的破れが Higgs 機構と同型の相転移として定式化される。

**[II]** 場の方程式は KL ダイバージェンスを棄却し、逆プロファイル（高エントロピー領域で Θ 最大）を選択する。忘却は「まだ情報が残っているところ」で最も強い。

**[III]** Θ 単独では空間変動しない。非均一な忘却（= 力）は α 場の結合から内在的に発生する。v2 §8.1 の結合方程式の構造が必要かつ十分である。

**[IV]** α_c 界面を境に支配的な力が反転する。QM 側では第二の力 (∇Θ) が支配し、MB 側では第一の力 (∇α·φ) が支配する。界面は両力が均衡する相転移帯であり、tanh 構造を持ち、幅はドメインの 28%、表面張力 σ = 0.51。

**[V]** 界面のエネルギーは α が決め、界面の力学は Θ が決める。スクリーニング関数 φ(Θ) がこの分離を生む。

これらの結果は Phase 2 姉妹版 C（場の方程式論文）の核心素材であり、Phase 3（CPS1'' → pointer basis）への直接的な橋渡しとなる。
