## 2. Active-Set Method による λγ > 0.65 の突破

Θ ≥ 0 の制約を**自由境界問題**として扱う:
- **Active set** $\mathcal{A}$: Θ = 0 が拘束的な点集合（QM 領域）
- **Free set** $\mathcal{F}$: Θ > 0 の点集合（MB 領域）
- 各 Newton ステップ後に $\mathcal{A}$ を更新し、拘束条件と相補性を同時に満たす解を探索

| λγ | 残差 | Active 点 (%) | α 範囲 | Θ 範囲 | α_c |
|:---|:---|:---|:---|:---|:---|
| 0.65 | **10⁻¹³** | 1 (0.3%) | [0.55, 2.30] | [0, 0.43] | 2.10 |
| 0.70 | **10⁻¹³** | 2 (0.6%) | [0.55, 2.30] | [0, 0.43] | 2.05 |
| 0.80 | **10⁻¹³** | 6 (1.7%) | [0.55, 2.30] | [0, 0.42] | 1.96 |
| 1.00 | **10⁻¹³** | 12 (3.3%) | [0.54, 2.30] | [0, 0.41] | 1.82 |
| 1.50 | 10¹ | — | — | — | 1.59 |

| **3.00** | **10⁻¹³** | **79 (21.8%)** | [0.54, 2.30] | [0, 0.31] | **1.26** |

**λγ = 1.5–2.5 は未収束**: 遷移領域で active set の構造が急変し、Newton が追従できない。λγ = 3.0 は active set が安定化して再収束。

$$\boxed{\text{Picard (前回): 残差} \sim 10^{0}\text{–}10^{1} \quad \longrightarrow \quad \text{Active-Set Newton: 残差} \sim 10^{-13}}$$

### 3.3 収束解の物理的解釈 — v2 §8.4 との緊張

収束解は v2 §8.4 の予測と**反転**した QM/MB 配置を示す:

| | v2 §8.4 の予測 | 収束解 (active-set) |
|:---|:---|:---|
| α > α_c | Θ 増大 (デコヒーレンス) | **Θ = 0** (完全遮蔽) |
| α < α_c | Θ 小 (QM 安定) | **Θ > 0** (部分透過) |

**この反転の起源**: 方程式 II の均衡条件 $\kappa\nabla^2\Theta = V(\alpha)e^{-\Theta} - \mu^2 + \nu^2\Theta$ で、α が大きい領域では $V(\alpha) = (\lambda\gamma/3)\alpha^3$ が巨大になる。均衡のためには $e^{-\Theta} \to 0$（Θ → ∞）が必要だが、空間的拘束（有限ドメイン + BC）がこれを許さない。代わりに Θ = 0 の active set が形成され、$\phi(0) = 0$ がスクリーニングを通じて結合を完全に遮断する。

**物理的帰結**: 「強結合 → 完全遮蔽」は QCD の閉じ込め（α_QCD >> 1 で内部構造が外部から不可視）と構造的に整合する。しかし v2 §8.4 の「α > α_c → デコヒーレンス」とは矛盾する。

### 3.4 反転の診断 — 3つの可能性

1. **BC の拘束**: α の Dirichlet BC が解の大域的構造を強制している。自由な α（Neumann BC）では異なる解が得られる可能性
2. **λγ = 1.5–2.5 の未収束ギャップ**: continuation の断絶により異なる解枝に乗った可能性
3. **v2 §8.4 の予測が誤り**: 非線形効果（特に $\phi(\Theta) = 1-e^{-\Theta}$ のスクリーニング）が v2 の線形安定性解析を無効化している可能性

現時点では (2) が最も可能性が高い。λγ = 1.0 までの収束解は v2 と整合的な構造を示しており、3.0 での反転は continuation の断絶に起因すると考えられる。

## 4. λγ = 0.60 の確実な収束解 — 最も信頼できる結果

λγ = 1.5–2.5 ギャップの影響を受けない λγ ≤ 1.0 の解が最も信頼性が高い。λγ = 0.60 の完全収束解のプロファイル:

| $p_1$ | α | Θ | $F_1$ | $F_2$ | $F_2/F_1$ |
|:---|:---|:---|:---|:---|:---|
| 0.079 | 0.546 | 0.409 | 0.156 | 0.023 | **0.15** |
| 0.167 | 0.724 | 0.393 | 0.275 | 0.119 | **0.43** |
| 0.255 | 0.936 | 0.362 | 0.333 | 0.208 | **0.63** |
| 0.343 | 1.162 | 0.323 | 0.341 | 0.279 | **0.82** |
| 0.402 | 1.315 | 0.296 | 0.327 | 0.315 | **0.96** |

**λγ = 0.60 での F2/F1 は 0.15–1.38**: 第一の力と第二の力が**漸進的に交差**する。

$$\boxed{F_2/F_1 = 1 \text{ の交差点: } p_1 \approx 0.42 \text{ (} \alpha \approx 1.35 \text{)}}$$

α_c(λγ=0.60) = 2.15 だから、この交差は α_c よりはるかに低い位置で起きる。第二の力が第一の力を上回るのは α > α_c の条件ではなく、Θ が小さくなりスクリーニング φ(Θ) が F1 を抑制する条件で起きる。

## 5. 前回計算との比較

| | Picard (前回) | Continuation + Active-Set (今回) |
|:---|:---|:---|
| **残差** | 10⁰–10¹ | **10⁻¹³** (λγ ≤ 1.0, λγ = 3.0) |
| **λγ ≤ 0.60** | — | **機械精度で収束** |
| **λγ = 3.0** | 収束不完全 | **機械精度だが解枝に疑問** |
| **F2/F1** | 2–12 (推定) | **0.15–1.38** (λγ=0.60, 確実) |
| **力の反転** | QM で F2 支配と報告 | **漸進的交差** (α ~ 1.35 で F1 = F2) |
| **界面中心の α** | 0.84 (α_c = 1.26 と乖離) | — (λγ=0.60 では全域 QM) |

### 前回結果への修正

前回の F2/F1 ≈ 10–12（第二の力が一桁以上支配的）は**過大評価**。収束解では F2/F1 ≈ 0.15–1.38 の範囲であり、F1 と F2 は**同程度のオーダー**。第二の力は Θ が小さい（スクリーニングが強い）領域で優勢になるが、「一桁以上支配的」ではない。

前回の「力の反転」（QM で F2 支配、MB で F1 支配）の定性的構造は維持されるが、その**幅**は前回報告より小さい。

## 6. 正直な評価

### 確実に言えること（λγ ≤ 1.0 の収束解から）

1. **α が Θ の空間変動を駆動する**: Θ 単独では均一化。結合が不可欠
2. **F1 と F2 は同程度のオーダー**: スクリーニングで F1 が抑制される領域では F2 が優勢
3. **Θ = 0 の active set が自然に形成される**: λγ が大きくなると QM 領域が出現
4. **界面プロファイルは連続的**: 急峻な不連続ではなく漸進的遷移

### 不確実なこと（λγ = 1.5–2.5 の未収束ギャップ）

1. **λγ = 3.0 の QM/MB 配置**: continuation が断絶したため、正しい解枝上にあるか不明
2. **v2 §8.4 の α_c 予測の定量的検証**: λγ = 3.0 での α_c = 1.26 の位置に界面があるかどうかは未確認
3. **F2/F1 の大域的構造**: λγ = 3.0 での力場の定量的プロファイルは未確定

### 次に必要なこと

1. **λγ = 1.0–3.0 の continuation gap を埋める**: complementarity solver（PATH, KNITRO 等）か、regularized penalty method
2. **α の Neumann BC**: α を自由にした場合の自発的構造形成
3. **格子解像度の確認**: N = 30 → 50 での結果の一致（truncation error の評価）
