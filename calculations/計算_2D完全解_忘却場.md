# Step 2.1b': 2D 完全解 — カテゴリカル分布シンプレックス上の忘却場
## Higgs 型ポテンシャル付き場の方程式の Δ² 上での数値解

**計算日**: 2026-03-25
**前提**: Step 2.1b (1D slice 解析) の結果
**舞台**: カテゴリカル分布 Cat$(p_1, p_2, p_3)$ のパラメータ空間 Δ²
**方程式**: $\nabla^2\Theta = -m^2_{\text{eff}}\,\Theta + 4\lambda\,\Theta^3$ (Higgs 修正場の方程式)

---

## 1. 計算手法

Fisher 計量 (Shahshahani 計量) の Laplace-Beltrami 演算子を有限差分で離散化。
座標 $(p_1, p_2)$, $p_3 = 1 - p_1 - p_2$ 上の三角形格子。
$N = 50$, 辺からのマージン 0.04, 内点 ~1100, 境界点 ~150。
Newton 法 + 線探索で非線形 PDE を求解。

## 2. 結果 A: Neumann BC（零法線微分）

Θ の法線微分がゼロ（自然境界条件）。

| 初期推定 | 収束先 | 安定性 |
|:---|:---|:---|
| $\Theta = 0.8\,\Theta_0$ 均一 | $\Theta = \Theta_0 = 1.118$ 均一 | **安定** |
| $\Theta_0 + 0.3\,\Theta_0(p_2 - p_3)$ (対称性破壊) | $\Theta = \Theta_0$ 均一に回復 | 摂動は**減衰** |

$$\boxed{\text{Neumann BC: 均一忘却 } \Theta_0 \text{ が唯一の安定解}}$$

**解釈**: 自然境界条件下では、シンプレックスの一定正曲率 ($K = 1/4$) が全方向に等しく作用するため、空間的に均一な忘却が最安定。対称性破壊の摂動を加えても、均一解に戻る。

## 5. 核心的帰結: なぜ非均一解には外部駆動が必要か

### 5.1 定理的帰結 (一定曲率多様体上)

$$\boxed{\text{一定正曲率の多様体上では、均一忘却 } \Theta_0 \text{ が自然な基底状態}}$$

非均一解 (ドーム、バンプ) が存在するのは Dirichlet BC ($\Theta = 0$ の外部強制) がある場合のみ。Neumann BC では均一解が安定で、摂動は減衰する。

**物理的意味**: 一定曲率 = 対称性がある。対称な空間では忘却も対称 (均一) であるのが自然。非均一な忘却 (= 力) を生むには:

1. **外部境界条件**: 忘却がゼロに固定される領域が存在する (ドーム解の起源)
2. **α 場との結合**: v2 §8.1 の結合方程式で、α(x) の不均一が Θ(x) の不均一を駆動する
3. **曲率の不均一**: 一定でない曲率を持つモデル空間 (n > 3 のシンプレックスの部分多様体)

### 5.2 v2 §8.1 との接続

v2 の結合方程式:

$$\nabla^2\alpha = V'(\alpha)\cdot\phi(\Theta) \qquad \kappa\nabla^2\Theta = V(\alpha)\cdot\phi'(\Theta) + U'(\Theta)$$

Step 2.1b' の結果は、**Θ 方程式を α なしで (切り離して) 解いた場合**の挙動。α なしでは Θ は均一化する。

v2 §8.1 の予測: α が空間的に不均一なとき、$V(\alpha)\cdot\phi'(\Theta)$ が Θ の非均一を駆動する。**α 場が「外部ソース」の役割を果たし、ドーム解の物理的起源を与える。**

### 5.3 §3 テーゼの2段階精密化

**Step 2.1b の帰結** (1D):
> 力 = 忘却の勾配 $\nabla\Theta$。忘却は情報量の高い領域で最大。

**Step 2.1b' の帰結** (2D):
> 均一空間では忘却は均一。力 (= 忘却の不均一) は**外部駆動**が必要。

**統合**:
> **力は忘却の不均一から生まれるが、不均一自体は自発的に生まれない。不均一を駆動するのは (a) 境界条件 (外部からの抑制)、(b) α 場の結合 (精度の不均一)、(c) 曲率の不均一 (空間の複雑性)。v2 §8.1 の α-Θ 結合方程式は (b) を定式化したもの。**
