## なぜ representative ではないか

[SOURCE: 考察_自動数学_ケーキュー定義_認識.md L24, L34-38]

> `K_q` の本体は defect 2-cell でも defect 2-cocycle でもなく、その両者を representative として持つ **first obstruction class** である。

class-valued な対象は coboundary / gauge 変更で representative が動く。rich vector を `K_q` の representative として固定すると、gauge 不変性が立たない可能性がある。

したがって rich vector は **diagnostic** として独立に保持し、representative は別の構成 (例: Walsh primitive `λ:=1` の Iteration 2 採用候補) に委ねる。Problem 10 (a) の本筋は別 annex で扱う。

## H1/H2/H3 三投影との対応

[SOURCE: 構想_自動数学_第一障害可視化閾値.md §3 予想 H*, L137-143]

> 予想 H*: `H1/H2/H3` は独立な 3 仮説ではなく、同一の obstruction の 3 つの投影である。
>
> | 投影 | 見えるもの |
> | スペクトル的投影 | `C_q ↔ X_q` の優勢交代 |
> | 算術的投影 | `E_q` と `X_q` の共鳴 |
> | 圏論的投影 | `K_q = [κ_q]` として現れる first obstruction class |

これに対し rich vector の coefficient 対応:

| 投影 | hub での表現 | rich vector での coefficient 表現 |
|:---|:---|:---|
| スペクトル的 | `(C_q, X_q)` の優勢交代 | `C_q mod 2`, `X_q mod 2` の parity |
| 算術的 | `E_q` excess | `E_q mod 2` の parity |
| 圏論的 | `K_q = [κ_q]` | (representative ではなく class なので rich vector では直接表現不可) |

つまり rich vector は **スペクトル + 算術** の coefficient 表現であり、圏論的 class `K_q` 自身は別の構成を要する。

## 観察

### O1: q=4 の sector parity (0, 1, 1)

BF excess `E_4 = 0` で flat だが、Sym²/Λ² parity は既に `(1, 1)` と立ち上がっている。これは:

- 第一障害可視化閾値 `q*=5` が単純な on/off ではなく、**段階的な前駆現象**を持つことを示唆
- AlyciaBHZ Issue #25 が観測した「Bowen-Franks Fibonacci subsequence break が q=5 で起きる」直前の q=4 で、すでに sector レベルでは何かが起き始めている [SOURCE: AlyciaBHZ Issue #25 comment 2026-04-13]
- hub §1.3 の `q* := min { q | X_q > 0, E_q > 0, k_q = 1 }` 定義は **3 条件の同時立ち上がり**を要求するが、`X_q > 0` (= Λ² 優勢) の前駆として `X_q mod 2 = 1` が q=4 で既に立つことが見える

### O2: q=5 の (1, 1, 1) — 三投影同時可視化

これは hub §3 予想 H* の **直接的 coefficient witness**:
> H1/H2/H3 は独立な 3 仮説ではなく、同一の obstruction の 3 つの投影である。

q=5 で 3 sector parity が同時に `1` に立ち上がるのは「同一 obstruction の 3 投影」読みと整合する coefficient 観察。

ただし圏論的投影 `K_q` 自身は class-valued であり、parity だけで表現できない。rich vector は **スペクトル + 算術 の同時可視化**を捉えるが、圏論的 class の独立確認は別途必要。

### O3: q=7 の (0, 0, 0) — sector parity reset (novel observation)

これは Iteration 2 で初めて見えた pattern:

- q=5 で `(1, 1, 1)` 立ち上がり、q=6 でも `(1, 1, 1)` 維持
- q=7 で `(0, 0, 0)` reset

proxy 由来なので official `collisionKernel7` 待ちだが、もし confirmed なら:

> **q=5 boundary resonance + q=7 sector parity reset の 2 段構造**

を示唆する。これは hub §3 予想 H*1 「最初の共鳴点が 5 に現れたこと」の延長として、**「2 番目の構造変化点が q=7 に現れる可能性」**を提示する。

ただし現段階では proxy artifact の可能性も同等。official `collisionKernel7` の構築待ち。

## open questions

1. **q=4 の `(0, 1, 1)` の意味**: BF flat と sector parity 非ゼロの両立は、`第一障害可視化閾値` の境界条件として何を意味するか。前駆現象として hub に組み込めるか
2. **q=7 の `(0, 0, 0)` reset の origin**: proxy artifact か、official kernel でも保たれる構造か (`collisionKernel7` 待ち)
3. **rich vector を `K_q` の shadow として読む可能性**: representative ではないが、`K_q` class が決定する **coboundary 不変な 3-tuple shadow** として再定式化できないか — gauge 変換で sector parity が動かない条件を特定すれば、rich vector を圏論的 class に近づけられる
4. **AlyciaBHZ スペクトル自己双対性との接続**: `χ(t,λ)+χ(t,-λ)=4(1-λ⁴)` が偶/奇 spectral moments を分離する [SOURCE: AlyciaBHZ Issue #25] — これは rich vector の Sym²/Λ² parity 分離と直接対応する可能性
