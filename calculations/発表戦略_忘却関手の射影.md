# 遊学エッセイ 発表戦略 — 忘却関手の射影

> 2026-03-16
> 原理: **エッセイ自体に忘却関手 $U$ を適用し、読者のレベルに合わせた射影を作る**

---

## §1 原理

$$U_{\text{pub}}: \mathcal{C}_{\text{rich}} \to \mathcal{C}_{\text{reader}}$$

エッセイの完全版 ($\mathcal{C}_{\text{rich}}$) から読者のレベルに応じた構造を忘却し、
到達可能な射影 ($\mathcal{C}_{\text{reader}}$) を作る。

**これ自体がシリーズの主張を実演している** — 知は受信者の MB に到達して初めて意味を持つ。

---

## §2 三層の射影

| 層 | 忘却する構造 | 残す構造 | 媒体 | 目的 |
|:--|:--|:--|:--|:--|
| **L0 フック** | 数式・圏論・FEP | タイトル + 1行洞察 + アナロジー | X/Twitter スレッド | 初速。接触面積の最大化 |
| **L1 読み物** | 数式 | FEP の言語的説明 + 日常例 + アナロジー | note.com / Zenn | 理解。共感と「なるほど」 |
| **L2 完全版** | なし | 全構造 (FEP + 圏論 + 数式) | PDF / GitHub / 自サイト | 権威。学術的信頼性 |

### 各層の変換規則

**L2 → L1 (忘却 $U_{\text{math}}$)**:
- 数式を自然言語に翻訳
- 圈論の用語を日常語に置換
- 数式は折りたたみ (details タグ) で残す
- アナロジーを増量 (化学・料理・建築等)

**L1 → L0 (忘却 $U_{\text{detail}}$)**:
- 1エッセイ → 1スレッド (5-8ツイート)
- 核心の洞察を1行に圧縮
- 挑発的な問いかけで終わる
- note.com へのリンクで誘導

---

## §3 初手の選定

### Phase 1: ①② の L0/L1 版

**① 構造を見ろ** — 最も「あるある」で刺さる
- L0: 「バカと賢い人の違いは知識量ではなく構造の見え方」
- L1: 信号機の例 → MB の概念 → 構造が見える=予測できる

**② 自信をもつ** — Dunning-Kruger が広く認知済み
- L0: 「バカほど自信を持つ理由を、自由エネルギー原理で数学的に証明した」
- L1: 山登りの比喩 → precision の概念 → 確信=精度の誤認

### Phase 2: ⑦⑬ → ⑧③

読者層を少しずつ広げながら、圏論の概念を段階的に導入。

### Phase 3: HGK 論文の前座

L2 完全版の一部を arXiv / 査読誌に提出。
遊学エッセイの読者層が HGK 論文の初期読者になる。

## §4 バズの構造

$$\text{初速} = \text{タイトルの引力} \times \text{到達可能な読者の MB 数}$$

- タイトルの引力: 「バカ」は普遍的フック。標準偏差ゼロ
- 到達可能な MB: L0 → L1 → L2 の漏斗構造
- 目標: L0 で 10,000 リーチ → L1 で 500 読了 → L2 で 50 深読 → HGK 論文の初期読者

---

## §5 HGK 論文への橋渡し

遊学エッセイで確立する概念:
1. FEP = 認知の統一原理 (← ①②⑩)
2. 忘却関手 = 構造を失うこと (← 全エッセイ)
3. 随伴対 = 世界の二面性 (← ⑮)
4. 不動点 = 美・真理 (← ②'⑯)

→ これらが HGK 論文の前提知識として機能する。
読者が「なるほど FEP か」と既に理解していれば、HGK の Markov blanket 議論はスムーズに入る。
