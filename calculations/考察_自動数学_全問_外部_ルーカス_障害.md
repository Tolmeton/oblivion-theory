## 3. 分解すべき二つの obstruction

Theorem 5 は二つの失敗を同時に扱っているが、all-`q` 化では別問題として分ける。

| 面 | 等式 | 必要な SOURCE | 現在の見込み |
|---|---|---|---|
| exterior-Lucas | `c_{q,2}=L_q` | `c_{q,2}` の q 方向の構造、閉形式、漸化式、または上界 | 先に攻める |
| determinant-excess | `ε_q=L_q` | `sum_i c_{q,i}` または signed determinant excess の構造 | 別ルート。値が不規則で、同じ補題では済まない |

したがって「all-`q` obstruction theorem」を外に提案するなら、まず弱い形:

```text
For all q >= 6, c_{q,2} != L_q.
```

を第一目標にする。triple coincidence の全域否定は第二段階。

## 4. 補題候補

### Lemma A: gap positivity

目標:

```text
For all q >= 6, g_q = L_q - c_{q,2} > 0.
```

十分条件:

```text
c_{q,2} < L_q = phi^q + psi^q.
```

使える可能性がある形:

- `c_{q,2}` の q 方向の閉形式が取れるなら、主成長項を Lucas の `phi^q` と比較する。
- `c_{q,2}` に線形漸化式が取れるなら、`g_q` の漸化式を導いて正値保存を示す。
- automaton / transfer-matrix 由来なら、`c_{q,2}` をある重み付き path count の係数として解釈し、Lucas path count への injection failure / missing sector を固定する。

注意: Draft D' にあった「Binet-style asymptotic comparison」は方向としては有望だが、現時点で `c_{q,2}` の閉形式が SOURCE になっていない。外向きにはまだ出さない。

### Lemma B: monotone gap propagation

bounded 表では `q>=6` で `g_q` が増加している。もし次を証明できれば、有限確認から全域化できる。

```text
g_{q+1} >= g_q + g_{q-1} - R_q
```

またはより強く

```text
g_{q+1} >= g_q
```

ただし、これは `c_{q,2}` の q 方向 recurrence が必要。現時点の upstream SOURCE は各 `q` に対する moment recurrence rows であり、`q -> c_{q,2}` の recurrence ではない。

### Lemma C: classified bad-example mechanism

all-`q` 定理が重い場合の代替:

```text
The continuation fails after q=5 because the q-direction coefficient c_{q,2}
tracks a lower-growth sector than Lucas L_q, while q=3,4,5 are a finite overlap window.
```

この形なら「なぜ q=5 で見えたか」を説明できる。ただし `lower-growth sector` の実体を SOURCE 化する必要がある。
