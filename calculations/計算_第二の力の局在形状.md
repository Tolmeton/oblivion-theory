## 2. 勾配アラインメント [最重要の発見]

$$\cos(\nabla\alpha, \nabla\Theta) = 0.999 \pm 0.003$$

- 99.9% の内部格子点で $\cos > 0.8$ (平行)
- 0% の点で反平行 ($\cos < -0.8$)

$$\boxed{\nabla\alpha \text{ と } \nabla\Theta \text{ はほぼ完全に平行}}$$

**物理的意味**: 第二の力は第一の力と**同じ方向に作用**する。∇α が指す方向（非対称性が変化する方向）と ∇Θ が指す方向（マスクが変化する方向）が一致している。

これは v2 §8.1 の結合方程式の構造から理解できる:
- 方程式 II: $\kappa\nabla^2\Theta = V(\alpha)\cdot\phi'(\Theta) + U'(\Theta)$
- α が大きい場所で $V(\alpha)$ が大きい → Θ が駆動される
- → α が大きい方向に Θ も大きくなる → **勾配が平行になる**

**帰結**: 第二の力は第一の力を**増幅する**。非対称性の不均一（第一の力）があるところに、マスクの不均一（第二の力）が重畳し、合計力が単独の ~11 倍になる。

## 5. v2 との接続と物理的解釈

### 5.1 §8.2「第二の力」の定量的検証結果

| v2 §8.2 の予測 | 計算結果 | 状態 |
|:---|:---|:---|
| ∇Θ ≠ 0 が独立した力を生む | $F_2$ が全域で非零 | **確認** |
| 第二の力は第一の力と異なる | $F_2/F_1 \approx 11$ (一桁上) | **確認** (予想以上に強い) |
| 量子-古典界面に局在する力 | シンプレックス辺 (低エントロピー) に集中 | **部分的確認** |
| α = const でも F2 は残りうる | ∇α // ∇Θ が平行 → α が変化する所で F2 も変化 | **否定的**: α 一定なら F2 もゼロ |

### 5.2 予想外の発見: ∇α // ∇Θ の平行性

v2 §8.2 は第一の力と第二の力を**独立な**力として提示した:
> 「完全に対称 (α → 1) な系であっても、マスクの不均一 (∇Θ ≠ 0) による力は残る」

計算結果はこの予測を**修正する**: ∇α と ∇Θ が 99.9% 平行であるため、∇Θ は ∇α の「影」であって独立ではない。

**修正後の描像**:

$$F_{\text{total}} = F_1 + F_2 = |\nabla\alpha|\cdot\phi(\Theta) + \kappa|\nabla\Theta| \approx (1 + R) \cdot F_1$$

ここで $R = F_2/F_1 \approx 11$ は「増幅比」。

> **第二の力は独立した力ではなく、第一の力の増幅器である。**
> α の不均一が Θ の不均一を駆動し (方程式 II)、
> ∇Θ が ∇α と平行に発生するため、合計力は ~12 倍に増幅される。
> 増幅率はスクリーニング関数 φ の値で制御される。

この修正は v2 §8.2 の[推定 60%]を支持する方向に動く（第二の力は実在する）が、その性質を修正する（独立ではなく増幅的）。

### 5.3 力の局在形状 — シンプレックス辺への集中

$F_2$ (そして $F_1$ も) は**シンプレックスの辺に近い領域**に集中する。これは:
- 辺に近い = 1つのカテゴリが支配的 = 低エントロピー
- 辺から離れた中心 = バランスした分布 = 高エントロピー

力が辺に集中する物理的意味: **分布が「極端になりかけている」遷移帯で力が最大**。中心（バランスした状態）でも辺（完全に偏った状態）でもなく、その間の**インターフェース**に力が局在する。

これは v2 §8.2 の「量子-古典界面の力」と構造的に整合する。
