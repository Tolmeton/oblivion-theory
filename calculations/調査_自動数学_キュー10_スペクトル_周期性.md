**結論**: ordinary と signed convention は **mathematical artifact ではなく Galois-theoretically 分離した 2 つの spectral face**。統一表 v0.2 donor D4 §7.1「q=5 sign-flip の source 掘り」への spectral 面での**確定的回答**: 単なる convention 規約選択ではない。

### 3.2 ordinary_exact_moment_proxy (副 convention)

[SOURCE: /tmp/codex-q10-phase2-result.md L80-83]

- **A arg residue**: yes (trivially: 全 q で Perron real root 支配、arg=0)
- **B trace FFT**: no
- **C Perron recovery**: no (q=10 で real mode、ただし q=5 でも real mode で complex boundary 未発動)
- **Overall: 両義 (yes_count=1/3)**

ただし ordinary proxy では **q=5 で複素対に移る現象自体が欠落** (全 q Perron 実根)。これは Tolmetes probe §3.1 の「q=5 で支配モードが複素共役対」と矛盾する。ordinary convention の companion matrix は boundary premise を成立させず、判定材料として信頼性低い。

### 3.3 主 convention の採用

signed convention が Lean official (q=5) と一致する唯一の規約であり、Tolmetes probe §3.1 の spectral 移行も再現する。従って **signed convention での棄却が主結果**、ordinary の両義は artifact と看做す。

### 3.4 総合判定

**棄却**: q=5 → q=10 の 5-step φ-周期性は、signed companion proxy 面で成立しない。
