# Force is Oblivion

[日本語版](README.ja.md)

> Structure emerges from what is lost, not from what is kept.

**Force is Oblivion** is a series of 14 papers deriving force, complementarity, perception, existence, entropy, and spacetime from a single operation: the **forgetful functor** on statistical manifolds.

## The Claim

The **directionality theorem** (Paper I, Theorem 5.1):

$$F_{ij} \neq 0 \iff d(\Phi T) \neq 0$$

Force arises if and only if oblivion is directionally inhomogeneous. Forget uniformly and nothing happens. Forget unevenly and curvature appears — curvature that acts as force.

- $\Phi$: oblivion field (KL divergence on a statistical manifold)
- $T$: Chebyshev 1-form (contraction of Amari-Chentsov tensor with Fisher metric)
- $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$: oblivion curvature

This is not a metaphor. It is a proved theorem in information geometry.
