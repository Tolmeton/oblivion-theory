#### §1. 導入: Wittgenstein の境界は引けるか (3–4頁)

**問題提起**: Wittgenstein は「語りえぬものについては沈黙せねばならない」(TLP 7) と述べたが、
何が語りえないかの判定基準を与えなかった。本稿は、その境界を圏論的に定式化する。

**核心命題** (1文):
> 語ること = 選択的忘却。語りえぬもの = いかなる選択的忘却 (CPS 適合分解) も許容しない対象。

**先行研究の位置づけ** (簡潔に):
- Bohrification (Heunen-Landsman-Spitters 2009): 量子論の圏論的基礎づけ。語り可能性一般を論じない
- Coecke-Duncan (2011): Frobenius 代数による相補性。構造は精密だが Wittgenstein に到達しない
- Smithe (2020–2024): Bayesian レンズによる FEP の圏論化。認知科学内に閉じる
- Goguen (1999): 代数的記号論。Peirce 的で Saussure と接続しない
- Mazzola (2020): 関手的記号論。音楽理論に特化
- Zafiris-Karakostas (2019): 量子相補性の圏論的随伴。CPS と最も近い先行研究

**水準ラベルの約束**: [定理] / [予想] / [構造的対応] / [哲学的示唆] の4段階

**母体の対応箇所**: §1 (結論先出し) + §4.6c 冒頭 + 先行研究比較レポートから抽出

#### §4. 排除力逆転 + 安定度スペクトラム (7–8頁)

**母体の対応箇所**: §4.6j (排除力逆転) + §4.6f' (素数/手筋/定石)

**内容 (前半: 排除力逆転)**:
- 形式化度 φ の情報理論的定義: $\varphi(D) = I(\text{op}; \text{outcome}) / H(\text{outcome})$
- 排除力 $E(D) = (1 - \varphi) \cdot \lambda$ の定式化
- 3ドメイン × 2ケース (成立 + 破綻) の体系的検証:
  - 知覚: 犬 ✅ (CPS 成立, 水準 ii) / 痛み ❌ (水準 i: 完全排除)
  - 言語: 正義 ✅ (CPS 成立, 水準 ii) / 美 ❌ (水準 i: 完全排除)
  - 社会: 市場 ✅ (CPS 成立, 水準 ii) / 友情 ❌ (水準 i: 完全排除)
- 命題: 「数学 = CPS が自明に成立するドメイン」
- §3 との再帰: $E = (1-\varphi) \cdot \lambda$ ≅ 力 = 忘却の強度 × 不均一度

**内容 (後半: 安定度スペクトラムの圏論的定式化)**:
- VFE 最小化 = 安定度最大化 [定理]
- 3層構造を nerve の skeleta で特徴づける [定理]:
  - 素数 = 1-skeleton (射のみ、面なし) — 合成の検証が不可能 → 分解不能
  - 手筋 = 2-skeleton (composable triple の存在) — Face Lemma: $d \geq 3$ で初めて到達。2-simplex = 面積 = 安定な組み合わせ
  - 定石 = 完全 nerve — 究極の安定 = Fix$(G \circ F)$ = Kalon
- **Face Lemma = 手筋の到達条件**: 射が3本未満では面積がなく、合成の検証が不可能で、安定な操作が成立しない
- **符号理論的再読**: Face Lemma は圏論版 syndrome 条件、n-cell tower の隣接同時忘却禁止は検査対象と検査規則の同時消失禁止に対応する。安定 / 検出可能 / 修復可能の三分法と修復可能性への次段設計は `drafts/infra/FaceLemma_技術設計.md`
- 化学的同型: 原子 (1-skeleton) → 分子 (2-skeleton) → 結晶 (complete) ≅ 音素 → 単語 → 文法 [構造的対応]
- CPS 排除力との接続: E が高い ↔ composable triple 不在 ↔ 1-skeleton ↔ 不安定。E が低い ↔ 2-simplex 以上 ↔ 安定

#### §7. 統一構造: SST と姉妹論文への接続 (5–6頁)

**母体の対応箇所**: §4.6f'' (Stability Simplex Theorem) + §4.6h' (同型と比喩) + §4.6h'' (米田の認識論) + §10.4 (対称性 = 特殊解) + §4.7 (場の方程式)

**内容 (§7.1: CPS 構造は $\Delta_F$ を内蔵する)**:
- CPS の A (公理間整合性 = 合成の検証) = 随伴三角恒等式 $\Delta_F$ の読み方 A
- CPS の B (Ξ による力の定量化) = $\Delta_F$ の読み方 B (忘却スペクトラム)
- CPS の C (語ること = 選択的忘却 → 本稿自身の自己参照) = $\Delta_F$ の読み方 C (Kalon)
- **Paper I §8.1.2 との接続**: 姉妹論文 [Paper I] は忘却曲率 $F_{ij}$ の分解 $F_{ij} = (\alpha/2)[\Omega_{ij} + \Phi(dT)_{ij}]$ と $\Delta_F$ の具体的対応を構成している:

  | 三角恒等式 $\Delta_F$ | 曲率の分解 | CPS の読み方 |
  |:--:|:--:|:--:|
  | $F(\eta)$: 展開 | $\Omega_{ij} = (d\Phi \wedge T)_{ij}$ | A: 合成 (辺) |
  | $\varepsilon_F$: 収縮 | $\Phi(dT)_{ij}$ | B: スペクトラム (面積) |
  | $\text{id}_F$: 相殺 | $F_{ij} = 0$ | C: Kalon (頂点) |

  - nerve の 2-simplex: $d\Phi \wedge T$ の非零性 = 方向の独立性 = 非退化な 2-simplex の存在
  - CPS 排除力との接続: 排除力 E が高い ↔ nerve の 2-simplex が退化 ↔ 安定性の崩壊

**内容 (§7.2: Stability Simplex Theorem)**:

- **ターゲット圏の定義** (Paper I §8.2 と同一):
  - $\mathbf{Simp}_2$: 対象は 2-simplex $(f, g, h ; g \circ f = h)$、射は辺の保存写像
  - $\mathbf{Spec}_3$: 対象は符号付き3つ組 $(\lambda_1, \lambda_2, \lambda_3)$ with $\sum \lambda_i = 0$
  - $\mathbf{Fix}$: 対象は不動点 $x = GFx$ を持つ自己関手のペア $(F, G)$

- **読み方関手の構成**:
  - $R_A = N_2 \circ (\text{graph})$: $\Delta_F$ の 3 射から nerve の 2-simplex を構成
  - $R_B = \Theta_\bullet \circ R_A$: 各射に符号付き忘却量を適用しスペクトラムを得る
  - $R_C = \text{Fix}_{G \circ F}$: 随伴の不動点を構成する

- 箱入り定理: $\boxed{\mathcal{F} = \Delta_F}$ — **姉妹論文 [Paper I] §8.2 と同一の定理**
- 読者が Paper I を読んでいれば: 「これは Paper I の §8 で見た」と気づく
- 読者が本稿だけ読んでいても: 独立に理解可能
- 辺 (合成 = A) ・面積 (スペクトラム = B) ・頂点 (不動点 = C) の対応

**内容 (§7.3: 「語りうるものの三角形」)**:
- 本稿が主題とする「語り可能性」は $\Delta_F$ の読み方 C (自己参照性)
- 姉妹論文 Paper I の方向性定理は $\Delta_F$ の読み方 A (合成 = 力の方向性)
- 両者は同じ三角形の辺と頂点
- **閉じ**: 「語りうるものと語りえぬものの境界」自体が $\Delta_F$ の 2-simplex の境界
- 米田の認識論: CPS-schema は presheaf の1射に過ぎないが、SST はその射が不動点を持つ条件を与える

## V. Phase 2–4 技術ロードマップ

### Phase 2: 忘却場の方程式の数学的完成

**出発点**: §4.7 の否定的結果 (KL ダイバージェンスがガウス族上の場の方程式の解にならない)

**Step 2.1**: 正曲率モデル空間での Worked Example
- カテゴリカル分布のシンプレックス $\Delta^{n-1}$ 上で Fisher 計量・曲率を計算

## VIII. Phase 2–4 技術的詳細 (2026-03-25 追記)

### Phase 2 詳細: 忘却場の方程式

#### 2.1 否定的結果の精確な診断

§4.7.5' の否定的結果の原因は3つに分解される:

**原因 1 — 負曲率** ($R_{\mathcal{M}} = -1$):
場の方程式の右辺 $\lambda\Theta R$ が $\Theta = 0$ を安定にする方向に作用。
自発的忘却が起きない。物理的に正しい予測でもある — 単純な分布空間では「忘れる動機」がない。

**原因 2 — 粗視化の ad hoc 性**:
「分散を忘れる」は天下り的選択。場の方程式が解を選別するなら、粗視化自体が方程式から決定されるべき。

**原因 3 — 自己相互作用の不在**:
2次作用のみ。忘却の自己増幅/抑制には4次項 (Higgs 型) が必要。

#### 2.2 カテゴリカル分布のシンプレックスでの計算

Cat$(p_1, \ldots, p_n)$ のパラメータ空間 = $(n-1)$-次元シンプレックス $\Delta^{n-1}$。
Fisher 計量 = Shahshahani 計量: $g_{ij} = \delta_{ij}/p_i - 1$

**決定的事実**: シンプレックス上の Shahshahani 計量は**正の断面曲率**を持つ。
$n = 3$ (3カテゴリ) → ガウス曲率 $K = +1/4$。

正曲率 → $\Theta = 0$ が不安定になりうる → 自発的忘却の出現。

**具体的計算課題**:
1. $\Delta^2$ 上で $\Theta$ を構成 (粗視化: 3カテゴリ → 2カテゴリ統合 = カテゴリの「忘却」)
2. 忘却接続 $\mathcal{A}_i$ と曲率 $\mathcal{F}_{ij}$ の具体計算
3. 場の方程式に代入、非自明解の存在条件
4. $\Theta = 0$ の Hessian 計算 → 正曲率 ($R > 0$) での不安定性確認

#### 2.3 Higgs 型ポテンシャルの動機と構成

$V(\Theta) = -\mu^2\Theta^2 + \lambda\Theta^4$ ($\mu^2 > 0$) を作用に追加:
- $\Theta = 0$ は極大 (不安定)
- $\Theta_0 = \mu / \sqrt{2\lambda}$ が極小 (安定な非自明解)
- 忘却対称性の自発的破れが「力の獲得」を生む (Higgs 機構との構造的同型)

修正された作用:
$$S[\Theta] = \int_{\mathcal{M}} \left( \frac{1}{4} \mathcal{F}_{ij} \mathcal{F}^{ij} + V(\Theta) + \frac{\lambda'}{2} \Theta^2 R_{\mathcal{M}} \right) \sqrt{g}\, d^n\theta$$

#### 2.4 非 Abel 拡張

スカラー $\Theta$ → 行列場 $\Theta^a_b$。忘却のリー代数 $\mathfrak{g}$:

| $\mathfrak{g}$ | 忘却の干渉 | CPS 対応 |
|:---|:---|:---|
| $\mathfrak{u}(1)$ | Abel 忘却 (現行 §4.7) | $f = 0$, 勾配の不均一のみ |
| $\mathfrak{su}(2)$ | 2方向の忘却干渉 | QM の $[x, p] = i\hbar$ |
| $\mathfrak{su}(3)$ | 3方向の忘却干渉 | 知覚の形/色/動き? |

CPS3 ($\sigma > 0$) の構造定数 $f^a_{bc}$ を通じた発現: $\sigma \sim |f|$

**困難**: 統計多様体上の非 Abel ゲージ理論の先行研究がほぼ存在しない。
**好機**: Phase 2 の最もオリジナルな貢献になりうる。

#### 2.5 成果物と判定基準

| 判定 | 条件 | 出力 |
|:---|:---|:---|
| **成功** | カテゴリカル分布上で場の方程式の非自明解が存在 | 存在定理を含む数理物理論文 |
| **部分的成功** | 非自明解の存在条件が定量的に与えられる (数値的) | 構造的制約を示す論文 |
| **否定的結果** | いかなる修正でも非自明解が存在しない | 「忘却場の構造的制約」として publication |

### Phase 3 詳細: CPS1'' → デコヒーレンス → 位置選好

#### 3.1 Zurek の einselection の構造

$H_{int} = \sum_n |x_n\rangle\langle x_n| \otimes B_n$ (位置に対角的)。
Zurek: 「$H_{int}$ がこの形だから」(**How**)。
CPS1'': 「容器 (位置) > 内容 (運動量) だから」(**Why**)。

#### 3.2 CPS1'' の Lindblad 演算子への翻訳

$$\dot{\rho} = -i[H, \rho] + \sum_k \left(L_k \rho L_k^\dagger - \frac{1}{2}\{L_k^\dagger L_k, \rho\}\right)$$

- $L_x = \sqrt{\gamma_x}\, |x\rangle\langle x|$: 位置への射影 (reductive forgetting)
- $L_p = \sqrt{\gamma_p}\, |p\rangle\langle p|$: 運動量への射影 (destructive forgetting)
- **CPS1'' は $\gamma_x > \gamma_p$ を含意**: 容器への射影が自然に起きやすい

#### 3.3 Θ とデコヒーレンス率の接続

$$\Theta(t) \approx \gamma\, t \quad (\gamma = \text{デコヒーレンス率})$$

- $\Theta = 0$ → Fourier マスク完全 → 位置/運動量が対称に見える
- $\Theta > 0$ → マスク不完全 → 非対称性 (容器 > 内容) が露出

#### 3.4 Pointer basis 導出の方針

**主張**: CPS1'' + CPS5 ($\Theta > 0$) → pointer basis = $U_{ctr}$ 側 = 位置

**証明方針**:
1. CPS1'' → $U_{pos}$ は $U_{mom}$ に非依存 (独立に well-defined)
2. $\Theta > 0$ → Fourier マスク不完全
3. 不完全マスク下では、独立に well-defined な射影 ($U_{pos}$) のみが安定
4. ∴ pointer basis = 位置基底

**困難**: Step 3 が鍵。Zurek の "predictability sieve" との接続が未証明。

**代替戦略** (weaker but defensible):
CPS1'' が pointer basis の**必要条件**であることを示す:
「CPS1'' が破れている系では pointer basis が存在しない or 非一意」

#### 3.5 Zurek との差異

|:---|:---|:---|
| 説明対象 | ある系での pointer basis の特定 | なぜ一般に位置が選ばれるか |
| 出発点 | $H_{int}$ の形 | CPS1'' (存在論的非対称性) |
| 問いの種類 | How | Why |
| 反証条件 | $H_{int}$ が運動量対角な系 | CPS1'' 破れの系での pointer 非存在 |

### Phase 4 詳細: Ξ の計算論的検証

#### 4.1 Rate-distortion theory との精密な接続

Shannon の $R(D)$: 歪み $D$ 以下で送信する最小レート。

**Ξ の rate-distortion 的再定義**:
$$\Xi = \text{Var}\left(\frac{\partial R}{\partial D_i}\right)_{i=1}^{n}$$

- $\Xi = 0$: 全方向に等しく歪む (均一忘却)
- $\Xi > 0$: 特定方向だけ歪みが小さい (選択的忘却)
- 「ゲージ」の語を避け「選択的忘却定理」と呼称

#### 4.2 間接測定プロトコル3種

**Protocol A — Attention Entropy Variance**:
各 layer $l$ のアテンション分布エントロピー $H_l$。圧縮前後の変化の分散:
$$\Xi_{\text{proxy}} = \text{Var}(\Delta H_1, \ldots, \Delta H_L)$$

**Protocol B — Task-Specific Ablation**:

| 条件 | タスク関連保存率 | 非関連保存率 | 予測 $P$ |
|:---|:---|:---|:---|
| 均一 ($\Xi = 0$) | 1% | 1% | 低 |
| 選択的 ($\Xi > 0$) | 50% | 0.1% | 高 |
| 逆選択的 ($\Xi > 0$, 逆) | 0.1% | 50% | 最低 |

反証条件: 「均一」が「選択的」を上回れば $\partial P / \partial \Xi > 0$ は棄却

**Protocol C — C. elegans の Θ 推定**:
1. S (感覚) → I (介在) → A (運動) パーティションを Markov blanket として定義
2. $I(\eta; s)$ (外部刺激 → 感覚応答の相互情報量) を計算
3. $I_{cross} = I(\eta; \mu)$ (外部 → 内部の相互情報量) を計算
4. $\Theta = -\log(I_{cross} / I_{max})$ で blanket 厚を推定
5. 行動状態 (摂食/探索/回避) 間での $\Theta$ 変動を測定
6. 予測: 探索時 $\Theta$ 小 (外部に開放) / 回避時 $\Theta$ 大 (内部に閉鎖)

### 2026-03-25: Phase 2 Step 2.1b Higgs 型ポテンシャル付き解析完了

   $\Theta_{\text{KL}}$: 非対称で大 → $\Theta_{\text{BVP}}$: バランスした所で大。
   **忘却は情報量の高い領域で最強。既に情報を失った領域では忘却不要。**

4. **力 = 忘却の勾配**: $|\nabla\Theta|_g$ は bump の斜面で最大。
   力は忘却の量ではなく変化率。§3 テーゼの精密な確認。
