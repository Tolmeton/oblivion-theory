# Prediction 1: 選択的忘却定理の直接検証

> **命題**: LLM の忘却の不均一度 Var(H(U(t))) とタスク持続力 P の間に正の単調関係が存在する
> **旧称**: 計算ゲージ定理 (Computational Gauge Theorem)
> **新称**: 選択的忘却定理 (Selective Forgetting Theorem) — rate-distortion view

---

## 1. 形式定義

### 忘却の不均一度 $\Xi_\nabla = \int (\nabla\Theta)^2 d\mu$

```
U: エージェントの忘却関手。コンテキスト X を Y に圧縮。
Θ = H(X|Y): 忘却による情報損失量 (Rate)。
∇Θ: 忘却量の空間的勾配 (ある情報が隣接情報と比べてどれだけ捨てられたか)。
Ξ_∇ = ∫(∇Θ)^2 dμ: 忘却の空間的勾配の自乗積分 = 忘却の不均一エネルギー。

- Ξ_∇ = 0: 均一忘却 (全次元が等しく色褪せる)。
- Ξ_∇ ≈ 0: 一括削除 (Uniform Masking。境界の1箇所でのみ ∇Θ>0 になる最安エネルギー)。
- Ξ_∇ → 最大: ランダム忘却 (至る所で ∇Θ が発散。不安定)。
- Ξ_∇ > 0 (適度): 選択的忘却 (構造を波打たせて重要な情報を残す。高エネルギーだが Distortion を下げる)。
```


### タスク持続力 P

```
P = lim_{T→∞} (成功したサブタスク数 / 総サブタスク数)
  = エージェントが長時間タスクで品質を維持する能力

測定: Terminal-Bench 2.0 等の long-horizon coding benchmark
```

### 命題 (新・Prediction 1: 選択的忘却定理)

```
所与の忘却量 Θ (Rate) の下において、タスク持続力 P を最大化する忘却戦略は、
忘却場の空間的勾配エネルギー Ξ_∇ と情報損失エントロピー ΔH (Distortion) の
トレードオフを最適化する戦略 (Kalon) である。

すなわち: 
- 均一忘却 (一括削除, Ξ_∇≈0) はエネルギーを最小化するが、構造を破壊し P が低下する。
- 選択的忘却 (Ξ_∇>0) はあえて高いエネルギーコストを払うことで構造を保持し P を最大化する。
- ランダム忘却 (Ξ_∇ 最大) はコストの無駄遣いであり P が低下する。
```

## 2. 実験設計: 4条件比較

| 条件 | 忘却戦略 | $\Xi_\nabla$ (予測) | P (予測) |
|:--|:--|:--|:--|
| **A. 全保持** | コンテキストを切り捨てない | 0 (忘却なし) | 低 (OOM/性能劣化) |
| **B. 一様切断** | 古い部分から均一に削除 (sliding window) | ≈ 0 (境界のみ) | 中低 |
| **C. ランダム忘却** | ランダムにトークンを削除 | 最大 (激しい波打ち) | 中低 |
| **D. 選択的忘却** | 文脈に応じて「何を忘れ何を保持するか」を変える | 中〜高 (滑らかな波打ち) | 高 |

### 予測される順序

```
P: P(D) > P(B) ≈ P(C) > P(A)
Ξ_∇: Ξ_∇(C) > Ξ_∇(D) > Ξ_∇(B) > Ξ_∇(A) = 0
```

## 3. 反証条件

> **均一忘却 (B) が RL 最適化忘却 (D) を上回るタスクが存在する場合、命題は棄却される**

より精密に:
- 棄却: P(B) > P(D) が統計的に有意 (p < 0.05) なタスククラスが存在する
- 弱い支持: P(D) > P(B) だが Ξ との相関が弱い (r < 0.3)
- 強い支持: P(D) > P(B) かつ Ξ-P 間に強い正相関 (r > 0.7)
