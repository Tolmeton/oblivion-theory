## 3 Phase

### Phase α: 公開データの Ξ 計算 ($0, 1日)

**データ**: JetBrains HuggingFace (5モデル×3条件×500問 trajectory)

**測定**:

1. Per-trajectory の Ξ_trajectory = Var(retention_ratio per turn)
   - masking: 保持=最新M turn, 破棄=残り → retention pattern は均一 → Ξ≈0
   - LLM-Summary: 要約が保持する情報は不均一 → Ξ>0
2. Ξ(masking) vs Ξ(summary) の差を定量化

**期待**: Ξ(summary) > Ξ(masking) が確認される → Ξ が測定可能な量であることの実証

### Phase β': 公開データの Ξ-P 相関検定 ($0, 1日)

**データ**: Phase α と同じ + Phase B probing データ

**測定**:

1. `xi_from_trajectory.py` の改修 (完了済みの準備段階):
   - 現状の `Var` ベースの計算から、空間的勾配 $\Xi_\nabla$ とエントロピー $\Delta H$ (KL) に基づく新定義へ移行。
2. 3条件の $\Xi_\nabla$ 特性分析:
   - raw (全保持): $\Xi_\nabla = 0$, $\Delta H = 0$
   - summary (要約): $\Xi_\nabla > 0$ (滑らかな波打ち), $\Delta H > 0$
   - masking (均一切断): $\Xi_\nabla \approx 0$ (境界のみ), $\Delta H \gg 0$
3. 2層の Ξ が異なる相関を示すことの統計検定

**反証条件**: Agent レベルの Ξ-P 相関が有意に負 → Prediction 1 棄却

### Phase γ'B: 構造的 masking (新規実験, $200, 1-2週)

**目的**: 公開データでは答えられない唯一の問い — **Ξ を操作したとき P は変わるか？**

**実装**: JetBrains のコード (SWE-agent fork) を改造

```
# 均一 masking (JetBrains 実装, Ξ≈0):
全 observation を一律マスク (oldest first)

# 構造的 masking (新規, 0<Ξ<Ξ_summary):
type-aware な差分的 masking:
  - file_read: 保持 (構造情報が豊富)
  - test_output: 破棄 (冗長)
  - error_trace: 保持 (診断に必要)
  - directory_listing: 破棄 (冗長)
```

**4条件の Ξ スペクトル**:

| 条件                     | Ξ (予測)                | データ源         |
|:---------------------- |:--------------------- |:------------ |
| Raw (忘却なし)             | N/A                   | JetBrains HF |
| Uniform masking        | ≈ 0                   | JetBrains HF |
| **Structured masking** | **0 < Ξ < Ξ_summary** | **新規実験**     |
| LLM-Summary            | > 0                   | JetBrains HF |

**反証条件**: P(structured) ≤ P(uniform) → 忘却の構造 (不均一度) は性能に寄与しない
