# 力としての忘却 — 統計多様体上の場の方程式

**Paper I — v1.5 (2026-04-11)**

> *概要.* 力はどこから来るのか。本稿は、統計多様体上の忘却——確率分布のパラメータを粗視化する操作——から力が創発される機構を提示する。核心は**方向性定理**である: 忘却場 Φ と Chebyshev 1-形式 T_i から忘却接続 $A_i := \partial_i\Phi + (\alpha/2)\Phi T_i$ を定義し、その曲率 $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$ を構成すると [1]、忘却曲率がゼロであることと忘却で重み付けされた Chebyshev 1-形式 ΦT が閉であることは同値となる。忘却の*大きさ*には一切依存しない——均一に大量の情報を忘れても力はゼロだが、方向的に不均一に忘れれば曲率が生じ、力として作用する。指数型分布族上では d(ΦT)=0 は dΦ∧T=0 に帰着し（系 5.1.1）、条件は忘却勾配と Chebyshev 形式の整列性に単純化される。α-パラメータを動力学場 α(θ) に昇格させると、精度の遷移領域に力が局在するという測定可能な予測が得られる。驚くべきことに、忘却曲率 F_{ij} は α に対して正確に線形であり（命題 6.6.1）、得られた精度場方程式は線形化レベルで自由エネルギー原理 [2,3] の精度最適化と構造的に同型となる——この対応は曲率レベルでは近似ではなく正確である。

## §1. 序論

あらゆる力は忘却から生まれる——これが本論文の主張である。

**忘却とは何か。** 圏論的に、忘却関手 U: C → D は対象と射を写像する変換である。存在のレベルでは何も消滅しない——U は全てを像に写す。しかし識別可能性のレベルでは情報が失われる——U が非忠実であれば異なる射が同一視され、区別が潰れる。従って忘却は「変換」と「喪失」の合成である: **忘却 = 射の変換 + 識別可能性の喪失**。本稿の忘却場 Φ(θ) = D_KL(p_θ ‖ q) は、この識別可能性の喪失を測定する。D_KL = 0 は p_θ と q が識別不能であることを意味し、D_KL が大きいほど識別可能性が保持されている。この二重構造は本シリーズ内で独立に定式化されている: Paper VII §1.3.1・§2.1 は認識論的観点から (「F は構造を保存するが値を忘却する」「関手は何も捨てない、知覚者が捨てる」)、Paper VIII §6.2 の α-忘却濾過公理 (F1)(F4) は圏論的観点から (Ob(C_α) = Ob(C) 不変 + Mor(C_α) 単調減少) それぞれ rigorous に形式化する。

この主張は比喩ではない。統計多様体上に忘却のスカラー場 Φ(θ) = D_KL(p_θ ‖ q_θ) を定義し、忘却で重み付けされた Chebyshev 1-形式 ΦT の外微分 d(ΦT) が忘却曲率に比例すること——そして d(ΦT) ≠ 0 が力を生む必要十分条件であること——を証明する。これが**方向性定理**（定理 5.1）の内容だ。指数型分布族では dT = 0 により条件は dΦ ∧ T ≠ 0 に単純化される（系 5.1.1）。

ここで「力」とは何か。古典力学において力は先験的に与えられる基本量であり、なぜ存在するかは問わない。本稿はこれを逆転する。力とは忘却の方向的不均一から*創発*される量であり、均一な忘却から消え、不均一な忘却から立ち上がる。パラメータ空間のあらゆる点で、知覚者が忘却関手 U を通じて同じ割合の情報を失えばゲージ場の曲率はゼロだが、ある領域で精密に、別の領域で粗い観測を行えば——その境界に力が現れる。

この機構は3つの段階で構成される:

1. **場と曲率**（§3-4）: 忘却場 Φ(θ) と忘却接続 A_i を統計多様体上に構成し、ガウス族 Toy Model で具体計算する。等方的忘却（μ にも σ にも同じ割合で忘れる）では F_{12} = 0、異方的忘却（μ を忘れるが σ を保持する）では F_{12} = 6μ/σ——鋭い二分法が示される。

2. **定理と予測**（§5-6）: 方向性定理を証明し、α-パラメータを動力学場 α(θ) に昇格させて精度の遷移層に力が局在することを予測する。α-遷移層力は Yang-Mills ゲージ理論には類似物を持たない現象であり、情報幾何学に固有の結果である。

3. **対応と線形性**（§6.5-6.6）: α 場の方程式が自由エネルギー原理（FEP）の精度最適化と構造的に同型であることを示す。さらに、忘却曲率 F_{ij} が α に対して正確に線形であることを証明し、この対応が曲率レベルでは近似ではなく構造的に正確であることを確立する。

本稿は力の4つの側面に貢献する:

| 貢献 | 節 | 内容 |

## §2. 数学的準備

### 2.1 統計多様体

S = {p_θ : θ ∈ Θ ⊂ ℝⁿ} を可測空間 (X, F) 上のパラメトリック分布族とする。Fisher 情報計量

$$g_{ij}(\theta) = E_{p_\theta}[\partial_i \ell \cdot \partial_j \ell], \quad \ell = \log p_\theta$$

により Θ にリーマン構造が入る。三つ組 (M, g, C)（C_{ijk} = E[∂_iℓ ∂_jℓ ∂_kℓ] は Amari-Chentsov（歪度）テンソル）が統計多様体を定義する。

### 2.2 α-接続族

各 α ∈ ℝ に対し、α-接続を次で定義する:

$$\Gamma^{(\alpha)k}_{ij} = \Gamma^{(0)k}_{ij} - \frac{\alpha}{2} C^k_{ij}$$

ここで Γ^(0) は Levi-Civita 接続、C^k_{ij} = g^{kl}C_{ijl}。

**注意.** 縮約トレース Γ^(α)_i := Γ^(α)k_{ik} は 1-形式として変換*しない*。座標変換 θ → θ̃（ヤコビアン J）の下で:

$$\tilde{\Gamma}^{(\alpha)}_i = \frac{\partial\theta^j}{\partial\tilde{\theta}^i} \Gamma^{(\alpha)}_j + \partial_{\tilde{\theta}^i} \log|J|$$

非同次項 ∂log|J| により Γ^(α)_i は座標依存となる。

**記号規約.** 本稿の α は Amari の接続パラメータ α_III を指し、Paper VIII 以降で用いる忘却強度 α_VIII ∈ [0,1] とは区別する。また本稿では $F = \nabla \Phi$ を忘却の力、$F_{ij}$ を忘却曲率テンソルとして使い分ける。Paper XI の prompt 関手は P₁/P₂/P₃ と表記し、この F 記号とは衝突させない。CPS0'[^cps0i] はこの α_III = 0 臨界セクターを系列内で先行参照するための作業名である。

[^cps0i]: 本系列では CPS0' を本稿段階では作業公理として用い、Paper VIII [in preparation] §6 で改めて導出する。

> **前方参照について.** 本稿は Paper VII, VIII（いずれも in preparation）への展望的参照を含むが、本稿の全定理と主張はこれらなしで自己完結する。前方参照は系列全体の構造的位置づけを示す目的で置かれており、証明の依存関係には含まれない。

### 2.3 Chebyshev 1-形式

**定義.** Chebyshev 1-形式は Amari-Chentsov テンソルのトレースである:

$$T_i := g^{jk} C_{ijk}$$

**命題 2.1.** T_i は M 上の真の余ベクトル（1-形式）である。

*証明.* C_{ijk} は階数3の共変テンソル（Čencov の定理）。g^{jk}C_{ijk} の縮約は標準的なテンソル縮約であり、共変に変換する。□

**命題 2.2.** T_i は接続間の α-依存性を符号化する:

$$\Gamma^{(\alpha)}_i - \Gamma^{(0)}_i = -\frac{\alpha}{2} T_i + (\text{座標アーティファクト})$$

ここで負号は §2.2 の定義 Γ^(α) = Γ^(0) - (α/2)C から従う。Chebyshev 形式はこの差分からテンソル的な内容を抽出する。

## §3. 忘却の場の方程式

### 3.1 忘却場

q を固定された参照分布（最大限に粗視化された状態）とする。**忘却場**を定義する:

$$\Phi(\theta) = D_{\text{KL}}(p_\theta \| q)$$

Φ は M 上の独立な動力学スカラー場として扱う。Φ ≥ 0 であり、Φ = 0 は p_θ = q（エージェントのモデルが粗視化参照に一致した状態 — 忘却が完了し、もはや q に対して失うべき構造が残っていない）を意味する。Φ が大きいほど p_θ は q から遠い — すなわちエージェントのモデルはまだ精密な構造を保持しており、忘却の余地（忘却ポテンシャル）が大きい。

**リマーク（Φ の意味論）.** Φ(θ) = D_KL(p_θ ‖ q) は「忘却の量」ではなく忘却の**ポテンシャル** — 「まだどれだけの構造が忘却されずに残っているか」 — を測る。忘却の進行に伴い Φ は減少し、完全忘却で Φ = 0 に到達する。∇Φ は忘却ポテンシャルが最も急に減少する方向を指し、方向性定理（定理 5.1）は ∇Φ のみに依存する。したがって力の生成に本質的なのは Φ の勾配構造であり、Φ の絶対値ではない。

### 3.2 忘却接続

$$A_i := \partial_i\Phi + \frac{\alpha}{2} \Phi \, T_i$$

命題 2.1 により A_i は真の余ベクトルである: ∂_iΦ は勾配として、Φ T_i はスカラー×余ベクトルとして変換する。

**結合形式の動機.** なぜ Chebyshev 1-形式 T_i による結合か。最小結合の自然な候補は $A_i = \partial_i\Phi + (\alpha/2)\Phi\Gamma^{(\alpha)}_i$ であるが、§2.2 で示したように $\Gamma^{(\alpha)}_i$ は真の 1-形式ではない（座標変換で非同次項 $\partial\log|J|$ が入る）。テンソル的に well-defined な結合を構成するには、$\Gamma^{(\alpha)}_i$ から座標不変な部分を抽出する必要がある。命題 2.2 より $\Gamma^{(\alpha)}_i - \Gamma^{(0)}_i = -(\alpha/2)T_i + (\text{座標アーティファクト})$ であるから、T_i はこの差分のテンソル的内容そのものであり、構成の選択は**一意**である。この結合は、忘却場 Φ が α-接続の逸脱方向（Chebyshev 方向）と幾何学的に相互作用することを表現している。

### 3.3 忘却曲率

$$F_{ij} := \partial_i A_j - \partial_j A_i = \frac{\alpha}{2}\left[(\partial_i\Phi)T_j - (\partial_j\Phi)T_i + \Phi(\partial_iT_j - \partial_jT_i)\right]$$

滑らかな関数上で ∂_i∂_jΦ - ∂_j∂_iΦ = 0 なので（ここでは α は M 上の定数とする; α(θ) への昇格は §6 で扱う）、スカラー×1-形式の外微分に対する Leibniz 規則 d(ΦT) = dΦ∧T + Φ·dT を適用して:

$$\boxed{F_{ij} = \frac{\alpha}{2}\left[d(\Phi T)\right]_{ij} = \frac{\alpha}{2}\left[(\partial_i\Phi)T_j - (\partial_j\Phi)T_i + \Phi(dT)_{ij}\right]}$$

> **Boxed Note (一般形と指数族簡約の関係).** 一般の統計多様体では $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij} = (\alpha/2)(d\Phi \wedge T + \Phi\,dT)_{ij}$ が基準形である。指数型分布族では $dT = 0$ が成立するため、ここから $F_{ij} = (\alpha/2)(d\Phi \wedge T)_{ij}$ へ簡約される。以下で「整列条件」と呼ぶものは、この簡約形を特殊ケースとして含む。

**物理的解釈:** F_{ij} は忘却で重み付けされた Chebyshev 1-形式 ΦT の外微分に比例する。F_{ij} ≠ 0 は ΦT が閉でないこと — すなわち忘却が*方向的に非均一*であること — と同値である。

### 3.4 作用と場の方程式

$$S[\Phi] = \int_M \left( \frac{1}{4} F_{ij} F^{ij} + \frac{\lambda}{2} \Phi^2 \right) \sqrt{g} \, d^n\theta$$

**作用の一意性（命題 3.4.1）.** 以下の3条件を課すと作用汎関数は $(1/4)F_{ij}F^{ij} + (\lambda/2)\Phi^2$ に一意に確定する（$\lambda$ の符号のみが自由度として残る）:

(a) **$F_{ij}$ の2次式**: $F_{ij}$ に関する最低次（2次）の動力学項。
(b) **微分同相不変性**: $M$ の座標変換に対してスカラー不変。
(c) **最低階微分演算子**: $\Phi$ の高次導関数項を排除。

*証明の概略.* 反対称2テンソル $F_{ij}$ から構成できる微分同相不変なスカラー2次形式は Hodge 内積
$$F_{ij}F^{ij} = g^{ik}g^{jl}F_{ij}F_{kl}$$
に限られる。これは余接束の $O(n)$ 表現が反対称2テンソル部分において既約であることから従う（Bleecker [11], §3.3; Schur の補題による）。$\Phi$ の0次項で唯一の2次不変量は $\Phi^2$。したがって許容される最一般の作用は $\int_M [c_1 F_{ij}F^{ij} + c_2\Phi^2]\sqrt{g}\,d^n\theta$ であり、規格化 $c_1 = 1/4$ を採用すると $(1/4)F_{ij}F^{ij} + (\lambda/2)\Phi^2$ に確定する。$\square$

**λ の起源と役割.** 質量項 λ は VFE の二次展開から自然に出現する。VFE[Φ] を Φ=0 の周りで展開すると:

$$\lambda = \frac{\partial^2(\text{Complexity})}{\partial\Phi^2}\bigg|_{\Phi=0} - \frac{\partial^2(\text{Accuracy})}{\partial\Phi^2}\bigg|_{\Phi=0}$$

- **∂²Accuracy/∂Φ²**: 忘却によるデータ適合度の低下率。N·ḡ_eff (サンプルサイズ × 有効 Fisher 情報量) に比例し、常に正
- **∂²Complexity/∂Φ²**: 忘却によるモデル単純化の利得率。d_eff/L² (有効次元数 / 特性長²) にスケールし、常に負

したがって λ の符号はデータ精度とモデル複雑度の競合で決まる。λ < 0 (Φ=0 不安定 → 自発的忘却) は、各パラメータが十分な固有情報を持つとき (N·ḡ_eff > d_eff/L²) に成立する。これは **Bayesian Occam's Razor のスカラー場理論的表現**である: MDL/BIC がモデル選択で行う Accuracy vs Complexity の競合を、パラメータ空間全体に連続的に拡張したものと見なせる。Φ=0 の変分的安定性に関する厳密な解析は Appendix C を参照。

**ゲージ構造について.** A_i はゲージ場*ではない*: Φ と背景幾何 (g, T) から完全に決まり、残余のゲージ自由度 A_i → A_i + ∂_iχ は存在しない。本理論は統計多様体上の**非最小結合スカラー場理論**に分類される。

## §4. ガウス族 Toy Model

### 4.1 設定

1変量ガウス族 p(x|μ,σ) = (1/√(2π)σ) exp(-(x-μ)²/(2σ²))。パラメータ空間 M = {(μ,σ): σ > 0} ≅ ℋ²（ポアンカレ上半平面）。

Fisher 計量: g = (1/σ²) diag(1, 2), g⁻¹ = σ² diag(1, 1/2)

### 4.2 Amari-Chentsov テンソルと Chebyshev 形式

非零成分: C₁₁₂ = 2/σ³, C₂₂₂ = 8/σ³

**T = (T₁, T₂) = (0, 6/σ)**

*観察:* T₁ = 0。Chebyshev 形式は ∂/∂μ で張られる**1次元の核**を持つ。この消滅が方向性定理の鍵となる。

### 4.3 2つの忘却場

**ケース A: 等方的忘却。** 参照 q = N(μ, σ₀²)（平均は保存、分散のみ忘却）:

$$\Phi_A(\mu,\sigma) = D_{\text{KL}}(N(\mu,\sigma^2) \| N(\mu,\sigma_0^2)) = -\log(\sigma/\sigma_0) + \frac{\sigma^2 - \sigma_0^2}{2\sigma_0^2}$$

σ のみに依存。∂_μΦ_A = 0。

**ケース B: 異方的忘却。** 参照 q = N(0,1)（平均も分散も忘却されうる）:

$$\Phi_B(\mu,\sigma) = -\log\sigma + \frac{\sigma^2 + \mu^2}{2} - \frac{1}{2}$$

μ と σ の両方に依存。∂_μΦ_B = μ ≠ 0（一般に）。

### 4.4 忘却曲率: 二分法

**ケース A:** F₁₂ = (α/2)(∂_μΦ_A)T₂ = (α/2)(0)(6/σ) = **0**

**ケース B:** F₁₂ = (α/2)(∂_μΦ_B)T₂ = (α/2)(μ)(6/σ) = **3αμ/σ**

α = 2（e-接続）のとき: F₁₂ = **6μ/σ**

*二分法は絶対的:* 等方的忘却はどれだけ情報を失っても曲率をちょうどゼロにする。異方的忘却 — Φ の勾配が ker(T) の外に成分を持つ — のみが力を生む。

### 4.5 数値検証 (算術的健全性)

解析導出 F₁₂ = 6μ/σ と、忘却接続 A_i からの独立した有限差分近似の一致を確認する:
- A₁, A₂ の解析式から記号微分
- F₁₂ の有限差分近似 (中心差分 dμ = dσ = 10⁻⁶)
- 一致: 相対誤差 1.4 × 10⁻¹⁰

**範囲の明示.** これは**解析導出の算術的健全性**を浮動小数点レベルで確認するものであって、理論そのものの検証ではない。本稿の主張 — すなわち「F = 0 ⟺ d(ΦT) = 0」の同値性 (§5.1) と「指数型分布族で dT = 0」の成立 (§5.4) — の検証は、それぞれ §5.2 の証明と App B のカテゴリカル計算に委ねる。

## §5. 方向性定理

### 5.1 定理の主張

**定理 5.1（方向性定理）.** (M, g, C) を Chebyshev 1-形式 T_i = g^{jk}C_{ijk} を持つ統計多様体、Φ を M 上の滑らかな忘却場とする。忘却曲率 F_{ij} = (α/2)[d(ΦT)]_{ij} を定義する（§3.3）。

このとき以下は同値:

(i) すべての i, j に対し F_{ij} = 0

(ii) 1-形式 ΦT は閉である: d(ΦT) = 0

**対偶（力の条件）:** F_{ij} ≠ 0 ⟺ d(ΦT) ≠ 0 — 忘却で重み付けされた Chebyshev 1-形式が閉でない。

**系 5.1.1（指数型分布族）.** dT = 0 のとき、d(ΦT) = dΦ ∧ T であるから:

$$F_{ij} = 0 \iff d\Phi \wedge T = 0$$

すなわち、力は忘却の勾配が Chebyshev 形式と非整列であるときに、そしてそのときに限り生じる。

**系 5.1.2（十分条件）.** dΦ ∧ T = 0 かつ Φ · dT = 0 ならば d(ΦT) = 0 であるから F = 0。

### 5.2 証明

§3.3 の Leibniz 恒等式 F_{ij} = (α/2)[d(ΦT)]_{ij} から直ちに従う。α ≠ 0 のもと:

$$F_{ij} = 0 \quad \forall\, i,j \iff [d(\Phi T)]_{ij} = 0 \quad \forall\, i,j \iff d(\Phi T) = 0$$

**系 5.1.1 の証明.** dT = 0 のとき Leibniz 規則 d(ΦT) = dΦ ∧ T + Φ · dT は d(ΦT) = dΦ ∧ T に帰着する。よって F = 0 ⟺ dΦ ∧ T = 0。□

**系 5.1.2 の証明.** dΦ ∧ T = 0 かつ Φ · dT = 0 ならば d(ΦT) = dΦ ∧ T + Φ · dT = 0。□

**注.** 指数型分布族では Chebyshev 1-形式が閉 (dT = 0) であるため (§5.4)、系 5.1.1 により条件は dΦ ∧ T ≠ 0 に単純化される。一般の統計多様体では dT ≠ 0 が許容され、Leibniz の2項 (dΦ∧T と Φ·dT) の相殺を含む完全な条件 d(ΦT) ≠ 0 が力の必要十分条件となる。

### 5.3 解釈: なぜ方向が重要か

この定理は印象的な物理的解釈を持つ:

**忘却の大きさ（どれだけの情報が失われるか）は力を決めない。忘却の方向（Chebyshev 構造に対して、どのパラメータが情報を失うか）が力を決める。**

ガウス族上では:
- T = (0, 6/σ) は σ 方向を指す
- σ 方向にのみ変化する忘却: dΦ ∝ dσ ∝ T → dΦ ∧ T = 0 → **力なし**
- μ 方向に変化する忘却: dΦ に dμ 成分あり、T に dμ 成分なし → dΦ ∧ T ≠ 0 → **力あり**

Chebyshev 形式 T_i は多様体の「曲率の自然方向」を符号化する（α-接続が Levi-Civita から最も逸脱する方向）。この方向に沿った忘却は曲率にとって「見えない」。Chebyshev 方向に*直交する*忘却のみ — いわば知覚者が U を通じて「間違った方向の情報」を失うこと — が力を生む。

**用語上の注意.** 本稿において「力」は曲率 $F_{ij} \neq 0$ の幾何学的帰結を指す慣用的表現であり、ニュートン力学における質点に作用する力 $f^i$ と直接同一視するものではない。統計多様体上の曲率が推論の運動方程式に具体的に結合する機構は §6（α-動力学）で定量化される。この用語法は場の理論における「場の強さ」（field strength）に類似する: $F_{ij} \neq 0$ は情報幾何学的な力の*源泉*が存在することを示し、その源泉がパラメータ推定の漸近的振る舞いに及ぼす影響が観測可能な「力」となる。

**最適化における独立な検証.** 方向性定理の操作的含意は、深層学習の最適化において独立に確認されている。Sharpness-Aware Minimization (SAM) [7] はパラメータ空間のユークリッド球 $\{\epsilon : \|\epsilon\|_2 \leq \rho\}$ で損失面の鋭さを制御するが、Kim et al. [8] はこの近傍を Fisher 情報行列による楕円体 $\{\epsilon : \epsilon^T F(\theta) \epsilon \leq \rho\}$ に置き換えることで、汎化性能が向上することを示した (Fisher SAM)。本理論の言語では、ユークリッド球による等方的擾乱は忘却の不均一性 $\Xi = 0$ の退化ケース — 全パラメータ方向に同じ量の情報を捨てる — に対応し、Fisher 楕円体による異方的擾乱は $\Xi > 0$ — 統計多様体の曲率に応じて方向ごとに異なる量の情報を捨てる — に対応する。SAM → Fisher SAM への修正が汎化を改善するという事実は、「正しい距離」が Fisher 計量であること、したがって **忘却の方向が力を決める** という方向性定理の主張の最適化的な傍証を提供する。

### 5.4 一般化: 力の Leibniz 分解

定理 5.1 の条件 d(ΦT) ≠ 0 は、Leibniz 規則により2つの独立な寄与に分解される:

| 寄与 | 式 | 物理的意味 |
|:---|:---|:---|
| **方向的不整合** | dΦ ∧ T | 忘却が Chebyshev 形式と非整列の方向に変化 |
| **Chebyshev ねじれ** | Φ · dT | Chebyshev 形式が閉でない（多様体が「ねじれ的」構造を持つ） |

力の必要十分条件は d(ΦT) = dΦ∧T + Φ·dT ≠ 0 であり、これは個別の項がゼロでなくても相殺する可能性を含む。ガウス族上では dT = 0（T は閉）なので第1の寄与のみが残り、系 5.1.1 に帰着する。**注: カテゴリカルシンプレックス Δⁿ を含む全ての指数型分布族において、自然パラメータ系では T_i = ∂_i log det g が恒等的に成立するため dT = 0 が普遍的に成立する（Worked Example 2, Appendix B）。この主張は Lethe 側の再現スクリプト `categorical_simplex_dT_check.py` により n=2,3,4 の複数 sample 点で max|dT| = 0.0 (閾値 10⁻¹², det g と ∏p_i の一致誤差 ≤ 10⁻¹⁷) として数値的に確認された。したがって指数型分布族の範囲では、Chebyshev ねじれ項は常に不活性であり、力の生成は方向的不整合 dΦ∧T ≠ 0 に完全に帰着する。** 非指数型分布族では dT ≠ 0 が**実際に立ち上がる**: Student t $(\nu, \gamma)$ 族について `non_exponential_dT_counterexample.py` は $|dT| = 3.27 \times 10^{-1}$ 〜 $9.41 \times 10^{-1}$, $|dT|/|T| = 4.56 \times 10^{-2}$ 〜 $2.91 \times 10^{-1}$ を与えた (Appendix F)。ゆえに $dT = 0$ は指数型 class に特有の構造であり、非指数型では Chebyshev ねじれ項 $\Phi \cdot dT$ が方向的不整合項 $d\Phi \wedge T$ と並んで力の生成に寄与する。

**注: Cauchy 位置スケール族の鈍さ.** 初期候補として検討した Cauchy 位置スケール族は数値 pilot で $T \approx 0$, $dT \approx 0$ を返し、非指数型でありながら Chebyshev ねじれの反例としては鈍い。これは「非指数型なら常に $dT \neq 0$」という一般命題が成立**しない**ことを示唆し、どの class で $dT = 0$ が保たれどの class で破れるかの**体系的分類**が残された問題として提起される (**Open Problem OP-I-7**)。

### 5.5 選択的忘却定理への接続

方向性定理は直接的な実験的対応物を持つ:

**定義 (局所忘却曲率密度).** 統計多様体上の点 $\theta$ における局所忘却曲率密度を次で定義する:

$$\kappa(\theta) := \|d(\Phi T)\|_g^2 = \frac{4}{\alpha^2}\|F\|_g^2$$

これは定理 5.1 のゼロ曲率条件 $F_{ij} = 0 \iff d(\Phi T) = 0$ の直接的定量化である (Appendix D line 1412-1441 のノルム分離と完全整合)。指数型分布族では $dT = 0$ が成立するので (§5.4, Appendix B)、$\kappa$ は

$$\kappa(\theta) = \|d\Phi\|_g^2 \|T\|_g^2 - \langle d\Phi, T\rangle_g^2$$

に分解される — 「忘却勾配の大きさ」「Chebyshev 感受性」「両者の斜交性」の積構造。

**定義 (理論的不均一性).** 観測設計 $\nu$ (turn, trajectory unit, layer, coordinate 等) に対し、理論的不均一性を次で定義する:

$$\Xi_{\text{theory}}(\nu) := \operatorname{Var}_{u \sim \nu}[\kappa(\theta_u)]$$

これは line 266 の「分散」直観を保存しつつ、定理 5.1 を定量化する量である。

**経験的 proxy の位置づけ.** §5.5 以降で用いる $\Xi_{\text{Var}}, \Xi_{\text{Gini}}, \Xi_{\text{CV}}, \Xi_{\text{impl}}$ は、それぞれ異なる観測モデルの下で $\Xi_{\text{theory}}(\nu)$ の **estimator** または **lower-bound estimator** として再定義される (詳細な定式化と仮定 H1-H1''' は Appendix E に与える)。raw observable と estimator の区別を明示することで、「proxy が悪かった」による退避を構造的に不可能にする。

**予測 (選択的忘却定理, estimator 版).** 忘却を操作しない系では $d\Phi \approx 0$ であるから $\kappa \approx 0$、したがって $\Xi_{\text{obs}} \approx 0$ が予測される。逆に、task-relevant structure と整列した忘却により $\kappa$ の heterogeneity が増大する系では、sufficiently large $n$ と仮定 H1-H1''' の成立のもとで

$$\operatorname{Corr}(\Xi_{\text{obs}}, P) > 0$$

が期待される。これは数学的な曲率 $F_{ij}$（統計多様体上で測定）と経験的な観測量（ニューラルネットワーク上で測定）を、定量的誤差評価付きで接続する。

**統制実験結果 (N=416).** 忘却パターンを直接操作した統制実験で本予測を検証した。

*実験設計.* 各タスクは長コンテキスト LLM 対話セッション (20-50 ターン) であり、ターン列の中に 3-8 個の事実 (名前・数値・関係) が埋め込まれる。忘却はコンテキスト圧縮として操作化される: 各ポリシーは保持するターンの選択規則を定義し、圧縮率 r ∈ {0.3, 0.4, 0.5, 0.6, 0.7} でターン列を縮約する。タスク持続性 P は、圧縮後のコンテキストを与えた LLM が埋め込まれた事実を正確に回答できた割合（事実保持率）として測定される。不均一性 Ξ_Var は、保持されたターン群と棄却されたターン群の間の情報量分散として計算される。

6種の忘却ポリシーは以下の通り: (i) *selective*: 事実含有ターンを優先保持、(ii) *importance*: 各ターンの情報量（トークン長 × 名詞密度）で重み付けしてサンプリング、(iii) *uniform*: 一様ランダムにターンを選択、(iv) *recency*: 最新 M ターンのみ保持、(v) *random_selective*: ランダムな偏りでターンを選択（方向性なき不均一性の対照条件）、(vi) *none*: 全ターン保持（コントロール）。

合成タスク (20-50 ターン, 3-8 事実埋め込み) に対し、6種の忘却ポリシー × 5種の圧縮率 × 16シード = 416 条件を実行:

| 忘却ポリシー | $\Xi_\text{Var}$ | P (事実保持率) | N |
|:------------|:---:|:---:|:---:|
| selective (事実含有 turn 優先保持) | 0.209 | 0.712 | 80 |
| importance (情報量で重み付け) | 0.221 | 0.648 | 80 |
| uniform (一様確率) | 0.225 | 0.527 | 80 |
| recency (最新 M turn のみ) | 0.230 | 0.492 | 80 |
| random_selective (ランダム偏り) | 0.214 | 0.342 | 80 |
| none (全保持, コントロール) | 0.000 | 1.000 | 16 |

**中心的発見: 定理 5.1 対偶の実証.**
`selective` と `random_selective` はともに高い不均一性を持つが P は正反対に分かれた:

| ポリシー | $\Xi_\text{Var}$ | P | 忘却の方向性 |
|:--------|:-----:|:-:|:------------|
| selective | 0.209 | 0.712 | 整列 ($d\Phi \wedge T \neq 0$、順方向) |
| random_selective | 0.214 | 0.342 | 非整列 ($d\Phi \wedge T \neq 0$、逆方向) |

これは方向性定理の**対偶**の直接的な経験的対応物である:
不均一性の大きさが同程度でも、Chebyshev 構造との整列の有無が力の**符号**を決定する。
selective vs uniform: Cohen's $d = 0.76$, $p = 3.8 \times 10^{-6}$（大効果量）
random_selective の反転: $r(\Xi_\text{Gini}, P) = -0.43$, $p < 10^{-19}$

$$\frac{\partial P}{\partial \text{Var}(\Xi)} > 0 \iff \text{Align}(\Xi, \mathcal{T}) > 0$$

方向なき不均一性は力を生むが、その力はタスク持続を**害する**。

**補足統計（全ポリシー混合）.**
$r(\Xi_\text{Var}, P) = +0.28$, $p = 1.3 \times 10^{-8}$（Pearson）— 整列・非整列を混ぜた場合の希釈された相関値。
$\rho(\Xi_\text{Var}, P) = +0.31$, $p = 2.6 \times 10^{-10}$（Spearman）

**境界条件の成功確認: 全保持エージェント (SWE-bench, N=500).** 忘却を操作*しない*条件の対照実験として、nebius/SWE-agent-trajectories (N=500) の全保持エージェントから Ξ-P 相関を測定した。結果は $r(\Xi_\text{CV}, P) = -0.007$, $p = 0.87$ であった (詳細は Appendix D)。これは方向性定理の**境界条件の成功確認**である: 忘却を操作しない ($d\Phi \approx 0$) 系では $\kappa(\theta) = \|d(\Phi T)\|_g^2 \approx 0$ が成立し、したがって $\Xi_{\text{theory}}(\nu) \approx 0$、結果として $\operatorname{Corr}(\Xi_{\text{CV}}, P) \approx 0$ が**予測**される。

**定量的検証.** Fisher の $z$ 変換による有限標本誤差範囲を計算すると、$z_{\text{obs}} = \operatorname{arctanh}(-0.007) = -0.0070$, $\operatorname{SE}_z = 1/\sqrt{500-3} = 0.0449$。したがって $\pm 2\sigma$ 帯域は $r \in [-0.0964, +0.0825]$ となる。観測値 $-0.007$ はこの予測帯域の**ほぼ中心**にあり、ゼロから実質的に区別できない。これは「理論が予測した通りの帯域内に観測値が落ちた」ことを意味し、N=500 という大規模サンプルで予測が確認されたと読むべきである。従来の読み方 (「相関が見つからなかった」) ではなく、**ゼロ近傍に落ちるはずの相関が理論通りゼロ近傍に落ちた**という読み方が正しい。

唯一有意な変数はターン数 ($r = -0.116$, $p = 0.009$) であり、全保持設定ではコンテキスト長が性能を支配することを示す。これは方向性定理の射程外 (忘却操作による力の生成を扱う定理は、忘却を操作しない系の性能変動を説明しない)。

**体系内検証 (N=229 CCL式).** 認知ハイパーバイザー体系 HGK [6] は6つの修飾座標 (Value, Function, Precision, Scale, Valence, Temporality) に張られた36の認知動詞を持つ。各動詞は座標空間の部分集合を暗黙的にカバーし、CCL 式（ワークフロー定義）が使用する動詞の合併集合が**カバーされない座標**を決定する。座標 $k$ の忘却率 $p_k$ を全 CCL 式における座標 $k$ の欠落頻度として定義し、raw observable として $\Xi_\text{impl} := \text{Gini}(p_1,...,p_6)$ を座標的不均一性スコアとする。

36動詞の暗黙座標は X-series 結合張力 $w_{ij} \geq 0.4$ から演繹的に決定される（v5.1 マッピング）。均等使用の仮定の下で:

$$\Xi_{\text{coord,theory}} = \text{Gini}(\bar{p}_1,...,\bar{p}_6) = 0.271$$

229 個の実際の CCL 式（ワークフロー定義およびマクロ）を測定した結果:

| 座標 | $p_k$ (忘却率) | 解釈 |
|:---|:---:|:---|
| Precision | 0.629 | 最も想起される — FEP の precision weighting と整合 |
| Value | 0.673 | |
| Function | 0.690 | |
| Valence | 0.716 | |
| Scale | 0.777 | |
| Temporality | 0.790 | 最も忘却される — 時間的推論の構造的欠落 |

$$\Xi_\text{impl} = \text{Gini}(p_k) = 0.045 \ll \Xi_{\text{coord,theory}} = 0.271$$

**乖離の原因.** 理論的 Gini は36動詞の均等使用を仮定するが、実際の CCL 式は Telos 族 (/noe, /bou) と Methodos 族 (/ske, /sag) に集中し、Diástasis 族 (/lys, /ops) と Chronos 族 (/hyp, /prm) の出現頻度が低い。これは体系の**学習曲線効果**を反映する: 使用者が全動詞を習得する過程で、慣れた動詞への偏りが $\Xi_\text{impl}$ を平坦化させる。

**共起構造.** 座標 Scale と Temporality の同時欠落確率 M(Sc, Te) = 0.755 は、独立仮定の期待値 $p_\text{Sc} \cdot p_\text{Te} = 0.614$ を大幅に上回る。X-series における Sc×Te 辺の結合張力 $w = 0.35$ が低いことと整合する: 結合の弱い座標対ほど同時に忘却される傾向がある。

**Schur-Horn との関係.** $\Xi_\text{impl}$ は §5.6 の diagonal observable に対応し、$\Xi_\text{impl} \leq \Xi_{\text{spec,proj}}$ が成り立つ。ここで $\Xi_{\text{coord,theory}}$ は36動詞を均等使用したときの coordinate-level benchmark であり、実測値との差は使用頻度の偏りと basis alignment の破れの双方を含む。

### 5.6 Schur-Horn 橋渡し: 理論と観測の不等式

忘却のスペクトラム Ξ = spec(G)（G は忘却関手のグラム行列）は忘却関手の内部構造であり、外部から直接観測できない。外部で観測可能なのは、忘却関手を通過した後の特徴量ベクトルの共分散行列 Σ = Cov(Ux) の**対角成分**である。

**定理 5.6.1 (Schur-Horn 橋渡し).** 忘却関手 U: ℝ^D → ℝ^d の線形近似の下で:

$$\Xi_\text{impl} \equiv \text{Gini}(\text{diag}(\Sigma)) \leq \text{Gini}(\text{spec}(\Sigma)) = \Xi_{\text{spec,proj}}$$

等号条件: 特徴次元が Σ の固有方向と一致するとき（PCA 後）。

**等号条件の意味.** 観測 basis が Σ の principal directions と一致するとき、対角成分から計算した Gini は単なる影ではなく projected spectral quantity $\Xi_{\text{spec,proj}}$ そのものを回収する。この regime では Appendix E.3 の $\widehat{\Xi}_{\text{Gini}}$ と Appendix E.5 の $\widehat{\Xi}_{\text{impl}}$ は、選んだ観測設計 $\nu$ に対する full target の estimator と読める。逆に basis alignment が破れると、対角観測は固有方向の混合で平坦化され、recover されるのは lower-bound quantity のみである。したがって、実験設計者は「見ている座標が principal directions に整列しているか」を確認することで、full estimator と lower-bound estimator の境界を判定しなければならない。

**証明 (5段階).** (i) U を接空間での線形写像で近似する。(ii) Σ = UΣ_xU† と書く。(iii) Schur-Horn の定理により diag(A) ≺ spec(A)（弱優越）[Marshall & Olkin, 2011]。(iv) Gini 係数は Schur-凸関数である（Lorenz 優越の下で単調; cf. Marshall & Olkin, §3.A, Prop. A.1）ため Gini(diag) ≤ Gini(spec)。(v) 等方ソース Σ_x = σ²I の下では spec(Σ) ∝ spec(G)。ここで G := U†U は忘却関手のグラム行列であり、その固有値が忘却のスペクトラム Ξ = spec(G) を定義する。したがって $\text{Gini}(\text{spec}(\Sigma))$ を projected spectral quantity $\Xi_{\text{spec,proj}}$ とおけば主張が従う。□

**帰結.** $\Xi_\text{impl}$ は常に $\Xi_{\text{spec,proj}}$ を過小評価する。測定の不均一性は projected spectral quantity の下界であり、$\Xi_{\text{coord,theory}}$ はそのさらに外側に置かれた equal-usage benchmark である。影はつねに実物より平板に見える。§5.5 の統制実験で観測された r = +0.28 は、理論が予測する相関の lower-bound 側に位置する。

### 5.7 Embedding 空間における蒸留の方向選択性 (E9/E10)

§5.5 は忘却ポリシーの操作による間接的検証であった。ここでは index_op ⊣ Search 随伴 (Hyphē [7]) における G∘F の蒸留操作が embedding 空間で方向選択的であることを、直接測定・方向別 ablation・座標相関の 3段階で示す。

**実験設計.** 13セッション (871 steps, 768次元 Gemini embedding) に対し 4つの手法を適用した:

- **M1 (Boundary Drift)**: 各チャンク境界 step の embedding からチャンク centroid への移動ベクトル $\Delta e_i = \bar{e}_\text{chunk} - e_i$ を計算 (n=96)。centroid は G∘F の出力そのものであり、$\Delta e_i$ は G∘F が境界 step の情報を**実際に移動させた方向**。
- **M2 (Within-Chunk Residual)**: 各 step の embedding からチャンク centroid を引いた残差 $r_i = e_i - \bar{e}_\text{chunk}$ を計算 (n=867)。G∘F が「同一チャンク」と判定した後に残る変動 = G∘F が**保存しなかった情報**。
- **M3 (Fisher Ratio)**: Between-chunk scatter $S_b$ と within-chunk scatter $S_w$ の比を PCA 方向ごとに計算。τ ∈ {0.70, 0.75, 0.80} の 3条件。
- **M4 (Direction-Level Ablation, E10)**: M1 PCA の上位 k 方向を embedding から射影除去 ($P_\text{out} = I - \sum_{i=1}^{k} v_i v_i^T$) し、同一 chunker を再実行。k ∈ {1,2,3,5,10}, τ ∈ {0.70,0.75,0.80}。対照: 同数のランダム PCA 方向を除去 (10回平均)。

**CKA route の位置づけ (partial closure).** CKA-based Fisher ratio は、$\Phi$ 側を $\Phi_{\text{CKA}}(l) := 1-\operatorname{CKA}(h_l,h_0)$ で観測し、$\operatorname{image}(G)$ への射影で $d\Phi$ の主成分を抽出する限りで、projected quantity $\Xi_{\text{spec,proj}}$ の lower-bound estimator を partial に与える。しかし現行設計には $\widehat T_l$ の独立 estimator がなく、$\Pi_{\operatorname{image}(G)} d\Phi_l \wedge \widehat T_l$ を layerwise に再構成できない。したがって CKA route は $d(\Phi T)$ 全体の full estimator ではなく、$\Phi$ 側の projected component のみを捉える partial closure に留まる。この未閉鎖性を Open Problem OP-I-5 として記録する。

**解釈.**

1. **方向性定理の embedding 空間における partial analogue**: G∘F の蒸留は等方的ではない (結果 1)。その方向は G∘F の結晶化に因果的に必要である (結果 2, direction-level ablation)。G∘F の存在自体が方向選択性を生む (結果 3, G∘F ON/OFF)。ただし現在 recover できるのは $d\Phi$ の projected component までであり、$d(\Phi T)$ 全体の estimator は OP-I-5 に残る。
2. **§5.5 との相補**: §5.5 は忘却の**符号** (方向なき不均一性 vs 整列した不均一性) を測定。本節は忘却の**方向** (embedding 空間の少数主成分) を測定し、その方向の**因果的必要性** (ablation) を検証。両者は方向性定理の異なる側面を照らす。
3. **ker(G) = 座標外ノイズ**: 蒸留と無関係な方向はどの座標 proxy とも相関しない (結果 4)。image(G) 方向を除去すると意味的構造が消失する (結果 2)。G∘F は座標的構造を保存し、座標に乗らない表層的変動を捨てる — これは忘却の「商空間を開く」操作 (Paper II) と一致する。

**Cross-τ PCA 検証.** E10 の PCA 方向 (boundary drift) が τ=0.70 の特定 chunking に overfitting している可能性を排除するため、τ=0.70/0.75/0.80 の独立な chunking 結果から PCA を別々に計算し、方向間のコサイン類似度を測定した。PC-1/PC-2 方向は τ 間で |cos| = 0.66-0.97 と高度に整列し、特に τ=0.75 ↔ τ=0.80 では |cos| > 0.94。image(G) 方向は τ 非依存の embedding 空間の構造的性質であり、特定の chunking 結果への circular fitting ではない。

**既知の制限.**
(i) |r| = 0.2-0.4 は弱-中程度。proxy の質 (Scale=文書内位置/chunk_size, Temporality=セッション順序, Valence=キーワードヒューリスティック) に律速される。Scale proxy の一つ (chunk_size) は τ と循環的に依存する。
(ii) 13 sessions (768d, 48-261 chunks) での単一 embedding モデル (gemini-embedding-001) の結果。独立データセット・モデルでの再現が望ましい。
(iii) Function, Value, Precision の 3座標は proxy 不在。6座標中 3座標が未検証。

### 5.8 Coherence Invariance Theorem (E11)

§5.5-§5.7 は忘却の**方向**を検証した。本節で扱う coherence は $\Xi$ の estimator ではなく、G∘F 不動点の post-distillation diagnostic である。したがって以下の主張は「曲率 heterogeneity を推定する」ことではなく、「蒸留後にどの quality が保存されるか」を述べる。

**定理 5.8.1 (Coherence Invariance).** index_op ⊣ Search 随伴 (Hyphē) における蒸留操作 G∘F の不動点 Fix(G∘F; τ) の平均 coherence は、閾値 τ に対して近似的に不変である:

$$\bar{C}(\text{Fix}(G \circ F; \tau)) \approx \text{const} \quad \forall \tau \in [0.50, 0.95]$$

**実験的証拠 (E11).** 13 sessions × 46 τ 点 (τ = 0.50 ~ 0.95, Δ = 0.01) の連続掃引:

```
τ = 0.50: N = 1.0 chunks,  C̄ = 0.8067
τ = 0.75: N = 11.3 chunks, C̄ = 0.8145
τ = 0.95: N = 33.1 chunks, C̄ = 0.8074
```

chunk 数 (秩序パラメータ) が 1 → 33 と**33倍**変動する間、coherence は 0.806-0.818 (±0.7%) の狭帯域に収まる。全 598 データ点 (13 sessions × 46 τ) で成立。

**先行検証との接続.** §3.7 の 104 実験 (13 sessions × 4τ × G∘F ON/OFF) では G∘F ON 条件の coherence range = 0.008 であった。E11 はこの結果を 4τ → 46τ に拡張し、Coherence Invariance が τ の離散サンプルではなく連続全域で成立することを確認した。

**解釈.**

1. **G∘F の不動点構造**: G∘F は embedding 空間の特定方向 (§5.7, image(G)) に沿って蒸留を行う。証明が示すように、τ は G∘F の**初期条件**であり、experimental knob / instrument variable であって、不動点 quality の estimator ではない。不動点の coherence $c^*$ は embedding 分布の固有量 $[C_\text{global}, \bar{s}_\text{adj}]$ 内の引力点で決まり、coherence は G∘F が到達した fixed point を読む量的 invariant / post-distillation diagnostic である。
2. **方向性定理との関係**: 定理 5.1 は $F_{ij} \neq 0 \Leftrightarrow d(\Phi T) \neq 0$ (方向が力を決める) と述べる。Coherence Invariance はこの原理の帰結として理解できる: G∘F の蒸留は**方向を選択する** (§5.7) が、蒸留後の within-chunk quality は方向選択の結果として embedding 分布から一意に決まり、τ の選択に依存しない。
3. **α-τ 対応との接続**: §6.7.1 (E11) で τ 掃引が Paper III の α = 0 臨界点に対応する相転移を示すことを報告した。Coherence Invariance は**相転移の非自明側 (τ > τ_c) 全域で成立する**。自明側 (τ ≤ τ_c, N=1) では $c^* = C_\text{global}$ に退化するが (証明 注(ii))、これは定理と矛盾しない。

**証明 (非自明レジーム τ > τ_c).**

embedding 列 $\{e_i\}_{i=1}^N$ (N steps, d 次元) が与えられたとき、隣接類似度 $s_i = \cos(e_i, e_{i+1})$ とチャンク分割 $\mathcal{P} = \{C_1, \dots, C_K\}$ を考える。各チャンクの coherence は $\bar{C}_k = \frac{2}{n_k(n_k-1)} \sum_{i<j \in C_k} \cos(e_i, e_j)$、全体の平均 coherence は $\bar{C}(\mathcal{P}) = \frac{1}{K} \sum_k \bar{C}_k$ である。

G∘F は以下の 2 操作の合成:
- **F (merge)**: $n_k < n_\min$ のチャンクを隣接チャンクに統合 → チャンク数 K を減らし、平均 $n_k$ を増やす
- **G (split)**: 内部に $s_i < \tau$ なるペアを持つチャンクを分割 → チャンク数 K を増やし、平均 $n_k$ を減らす

**Step 1: Coherence の上下界.**

(a) *上界*: 任意の分割に対し $\bar{C}(\mathcal{P}) \leq \bar{s}_\text{adj}$ (隣接類似度の平均)。等号は全チャンクが 2 steps のとき。$\bar{s}_\text{adj}$ は embedding 列の固有量であり τ に依存しない。

(b) *下界*: 全体を 1 チャンクにした自明分割で $\bar{C}(\mathcal{P}_\text{trivial}) = C_\text{global}$ (全ペア平均類似度)。$C_\text{global}$ も embedding 列の固有量であり τ に依存しない。

(c) よって $C_\text{global} \leq \bar{C}(\text{Fix}(G \circ F; \tau)) \leq \bar{s}_\text{adj}$。本データでは $C_\text{global} \approx 0.807$, $\bar{s}_\text{adj} \approx 0.85$ であり、両界とも τ に依存しない。

**Step 2: G∘F の不動点の coherence は帯域 $[C_\text{global}, \bar{s}_\text{adj}]$ 内の引力点に収束する.**

G∘F を coherence 空間上の写像とみなす。Iteration n での平均 coherence を $c_n = \bar{C}(\mathcal{P}_n)$ とする。

- G が split を実行すると、分割されたチャンクの coherence は元のチャンクより高い (低 coherence のサブチャンクが分離されるため)。よって $c_{n+1}^{(G)} \geq c_n$。
- F が merge を実行すると、統合されたチャンクの coherence は構成チャンクの重み付き平均に近い。チャンクサイズが増えるため、coherence は $C_\text{global}$ 方向に引き戻される。

不動点条件 $G \circ F(\mathcal{P}^*) = \mathcal{P}^*$ は、split による上昇圧と merge による降下圧が均衡する状態であり、$c^* = \bar{C}(\mathcal{P}^*)$ は $[C_\text{global}, \bar{s}_\text{adj}]$ の内部に一意に定まる。

**Step 3: τ 非依存性.**

τ は **初期分割** $\mathcal{P}_0$ (境界を $s_i < \tau$ に配置) を制御するが、G∘F の反復は $\mathcal{P}_0$ から出発して不動点 $\mathcal{P}^*$ に収束する。不動点の coherence $c^*$ は Step 1-2 より $[C_\text{global}, \bar{s}_\text{adj}]$ の引力点であり、この帯域は embedding 列 $\{e_i\}$ のみに依存し τ に依存しない。τ は experimental knob として引力盆地内の初期点を選ぶに過ぎず、coherence 自体を推定する量ではない。

よって $\bar{C}(\text{Fix}(G \circ F; \tau)) \approx c^*$ は τ に対して近似的に不変である。$\square$

**注.** (i) 近似的不変 ($\approx$) は、有限回の G∘F 反復 (max 10) で厳密な不動点に到達しない場合の残差による。(ii) 自明レジーム (τ ≤ τ_c) では G∘F の初期分割が1チャンクとなり、G の split 条件を満たさないため $c^* = C_\text{global}$ に退化する — これは定理の自明なケースであり、証拠からは除外すべきだが主張とは矛盾しない。(iii) $c^*$ が帯域内で一意の引力点であることの厳密証明は、G∘F の Banach 不動点定理適用条件 (Lipschitz 定数 < 1) の検証を要する。実験的にはすべてのケースで 2 回以内に収束 (100% convergence rate) しており、収縮性は経験的に確認されている。

**既知の制限.** (i) 13 sessions, 単一 embedding モデル (gemini-embedding-001)。(ii) coherence の定義は pairwise cosine similarity であり、他の coherence 定義 (kNN, mutual information) での普遍性は未検証。(iii) ±0.7% は厳密なゼロではなく近似的不変。有限サイズ効果 vs 真の不変量の区別が必要。

### 5.9 忘却の二重性: 広義忘却と狭義忘却

§5.1-§5.8 の実験的・理論的知見を統合する。

#### 5.9.1 等方的忘却と方向的淘汰

定理 5.1 は「忘却の*方向*が力を決める」と主張し、§5.5-§5.7 はこれを実験的に確認した。しかし E12 実験 (30 sessions, 6053 steps, 248 chunks) は一見矛盾する知見を示す: 忘却関手 $G$ の核 $\ker(G)$ は embedding 空間上で*等方的*であり (participation ratio = 106.4, dim = 768)、いかなる方向にも集中しない。一方 $\text{image}(G)$ は 6 方向に強く集中する (Fisher ratio > 2×中央値)。

この非対称性は次のように統合される: **忘却自体は等方的に作用する。方向が生じるのは、忘却が作用する圏の構造が非対称だからである。**

ニュートン力学との対応がこの構造を定量化する:

| 力学 | 記号 | 忘却論 | Paper I |
|:---|:---|:---|:---|
| 重力加速度 (定数) | $g$ | 広義忘却 (全要素に MECE) | $\alpha$ (忘却強度) |
| 質量 (対象の性質) | $m$ | 忘却感受性 (射の密度の逆数) | $\|T\|_g$ (Chebyshev ノルム) |
| 力 (方向を持つ結果) | $F = mg$ | 狭義忘却 (方向的淘汰) | $F_{ij} = \frac{\alpha}{2}[d(\Phi T)]_{ij}$ |

**ノルムレベルの分離.** この対応は比喩ではない (導出は Appendix D)。指数型分布族上では $dT = 0$ であるから $F_{ij} = (\alpha/2)(d\Phi \wedge T)_{ij}$。Cauchy-Binet 恒等式を2-形式のノルムに適用すると:

$$\|F\| = \frac{\alpha}{2} \|d\Phi\| \cdot \|T\|_g \cdot |\sin\theta|$$

ここで $\theta$ は 1-形式 $d\Phi$ と $T$ のなす角である。$g_{\text{eff}} := (\alpha/2)\|d\Phi\||\sin\theta|$ とおけば:

$$\boxed{\|F\| = g_{\text{eff}} \cdot \|T\|_g}$$

ノルムレベルで $F = mg$ が正確に成立する。$g_{\text{eff}}$ は $\alpha$ (忘却強度) と $\|d\Phi\||\sin\theta|$ (忘却場の方向的不均一性) の積であり、$\|T\|_g$ (Chebyshev ノルム = 多様体の幾何学的感受性) が質量 $m$ に対応する。$|\sin\theta| = 0$ ⟺ $d\Phi \wedge T = 0$ ⟺ $F = 0$ は定理 5.1 の力の消滅条件に他ならない。

$g$ は物体を区別しない。落下の速さが異なるのは空気抵抗 — すなわち物体の側の性質 — による。同様に、広義忘却 $g_{\text{eff}}$ は情報を区別しない。「何が忘れられるか」は情報の側の性質 — 圏における射の密度、すなわち Chebyshev ノルム $\|T\|_g$ — が決める。

#### 5.9.2 忘却耐性と射の密度

**定義 5.9.1 (忘却耐性).** 圏 $\mathcal{C}$ の対象 $X$ の忘却耐性を $R(X) := |\text{Hom}(X, -)|$ (X から出る射の数) と定義する。

**公理 5.9.2 (忘却の単調性).** 忘却関手 $G: \mathcal{C} \to \mathcal{C}/{\sim}$ は忘却耐性に対して単調である: $R(X) \geq R(Y)$ かつ $X \in \ker(G)$ ならば $Y \in \ker(G)$。対偶として $Y \notin \ker(G)$ かつ $R(X) \geq R(Y)$ ならば $X \notin \ker(G)$ — 射が密な構造が忘却されるなら、射が疎な構造も忘却される。

**正当化.** この公理は「忘却とは何か」の定義的内容を述べている: 射が密な構造は射が疎な構造より忘却されにくい。これは「よく使うものは忘れにくい」という認知科学の基本的知見と同型であり、忘却関手を忘却関手たらしめる条件である。公理としての独立性はほぼ持たず、忘却関手の定義に近い — しかし定理 5.9.3 の証明に必要な命題を明示的に隔離することに意味がある。定理の前提が何であるかが明確になるからである。

**定量的構造 (予想).** 公理 5.9.2 は忘却関手の定性的性質を述べる。その定量的精密化として、閾値 $R_{\text{crit}}(\alpha)$ が存在し:

- $R(X) > R_{\text{crit}}(\alpha) \implies X \in \text{image}(G)$ (保存)
- $R(X) < R_{\text{crit}}(\alpha) \implies X \in \ker(G)$ (忘却)

が成立すると予想する。$R_{\text{crit}}$ は $\alpha$ の単調増加関数: 忘却が強いほど、生存に必要な射の密度が高くなる。E12 はこの閾値構造を支持する (下記)。

**R(X) の embedding 空間における実現.** Fisher ratio $\text{FR}(v) := \sigma^2_{\text{between}} / \sigma^2_{\text{within}}$ は $R(X)$ の proxy ではなく、embedding 圏 $\mathcal{C}_E$ における $R$ の**実現** (realization) である。embedding 圏を次のように構成する:

| 圏 $\mathcal{C}$ | embedding 圏 $\mathcal{C}_E$ | 実現 |
|:---|:---|:---|
| 対象 $X$ | embedding 空間の方向 $v$ | 情報の単位 |
| 射 $X \to Y$ | 方向 $v$ がチャンク $Y$ で統計的に有意 | 接続 = 文脈内の有意な出現 |
| $R(X) = |\text{Hom}(X, -)|$ | $\text{FR}(v) \propto$ 方向 $v$ が有意なチャンク数 | 射の数 = 有意出現チャンク数 |

$\mathcal{C}_E$ において Fisher ratio は $R$ の単調な関数である: $\text{FR}(v)$ が高い方向は多くのチャンクに射を持ち、$\text{FR}(v)$ が低い方向は少数のチャンクにしか射を持たない。$\mathcal{C}_E$ の構成は「統計的に有意」の閾値選択に依存する — この恣意性は $R$ の**順序**には影響しない (閾値を変えても FR の大小関係は保存される) が、$R$ の**値**には影響する。公理 5.9.2 は順序のみを要求するため、この恣意性は定理 5.9.3 の適用を妨げない。E12 の結果 — image(G) = 高 FR 方向 6 本、ker(G) = 低 FR 方向 — は、$\mathcal{C}_E$ における公理 5.9.2 の直接的実証である。

**実験的裏付け (E12).** E12 (30 sessions, 6053 steps, 248 chunks) で image(G) 方向は 6 本 (Fisher > 2×中央値)、ker(G) 方向は 0 本 (Fisher < 0.5×中央値)。Fisher ratio = 2×中央値が閾値 $R_{\text{crit}}$ の empirical estimate を与え、この閾値を超える 6 方向のみが $\text{image}(G)$ に属する。

#### 5.9.3 忘却と抽象化の同一性

**定理 5.9.3 (忘却-抽象化定理).** 忘却関手 $G$ が公理 5.9.2 (単調性) を満たすとき、圏 $\mathcal{C}$ の普遍的構造は $\ker(G)$ に属することができない。

**証明.** 普遍的構造 $U$ は定義により圏 $\mathcal{C}$ の全ての対象への射を持つ: $R(U) = \sup_{X \in \mathcal{C}} R(X)$。任意の $X \in \mathcal{C}$ に対して $R(U) \geq R(X)$。公理 5.9.2 の対偶より、$U \in \ker(G)$ ならば $R(U) \leq R(X)$ なる全ての $X$ も $\ker(G)$ に属するが、$R(U)$ は最大であるから $\ker(G) = \mathcal{C}$ — すなわち $G$ は全てを忘れる自明な関手。$G$ が非自明であれば $U \notin \ker(G)$。□

$$\alpha \uparrow \implies R_{\text{crit}} \uparrow \implies \dim(\text{image}(G)) \downarrow \implies \text{image}(G) \text{ はより抽象的な構造のみで構成}$$

**忘却の強化は抽象化を必然的に生産する。** この主張は2層の構造を持つ:

1. **定性的層 (定理 5.9.3).** 非自明な忘却関手の下で、普遍的構造は必ず生存する — 忘却がどれほど強くても、最も接続された構造は最後まで残る。これは証明済みであり、忘却の単調性公理の直接的帰結である。

2. **定量的層.** $\alpha$ の増大に伴い $R_{\text{crit}}(\alpha)$ が単調に上昇し、$\dim(\text{image}(G))$ が単調に減少する — すなわち忘却が強まるほど「生存者」が少なくなり、残る構造がより抽象的になる。含意チェーン $\alpha \uparrow \implies \dim(\text{image}(G)) \downarrow$ は定理 5.9.3 単独からは従わず、閾値の単調性を必要とする。

**実験的裏付け (E15+E16).** 定量的層を直接検証するため、固定 PCA 基底上で $\tau$ を 0.50→0.95 に掃引し、正規化 Fisher ratio ($\text{FR}_{\text{norm}}(i, \tau) := \text{FR}(i, \tau) / \overline{\text{FR}}(\tau)$、粒度 confound を除去) による方向追跡を実施した (30 sessions, 200 PCA 方向, 46 τ 点)。

| 検証項目 | 結果 | 解釈 |
|:---|:---|:---|
| **単調性** | $\dim(\text{image}(G))$ の非増加率 86.7% (閾値 2.0) | $\tau \uparrow \implies \dim(\text{image}(G)) \downarrow$ が概ね成立 |
| **脱落順序** | $\rho(\text{dropout}_\tau, \text{FR}_{\text{base}}) = 0.73$ ($p = 0.011$) | $\text{FR}$ が低い方向から先に脱落 — 公理 5.9.2 の直接的実証 |
| **Last Survivor** | $\tau = 0.90$ の生存者 9 方向中 top-3 が baseline $\text{FR}$ 上位 5 に含まれる | 定理 5.9.3 の実験的対応: 普遍的構造が最後まで生存 |
| **順序安定性** | $\text{FR}$ 順位の $\tau$ 間 Spearman $\bar{\rho} = 0.90$, 91.3% の $\tau$ 点で $\rho > 0.7$ | embedding 圏 $\mathcal{C}_E$ での公理 5.9.2 (順序保存) が成立 |
| **閾値ロバスト性** | 閾値 1.5-3.0 の全てで $\rho < -0.84$, 単調率 $\geq 80\%$ | 特定閾値に依存しない構造的パターン |

特に脱落順序 ($\rho = 0.73$) は、公理 5.9.2 の embedding 圏における直接的な実証として重要である: baseline で Fisher ratio が高い方向 (= $R(X)$ が大きい = 射が密な構造) ほど、忘却の強化に対して遅く脱落する。

定性的層と定量的層の間に位置する核心的洞察は次の通り: 定理 5.9.3 は「最後の生存者は普遍的構造である」と保証する。E15 は「忘却が強まるほど生存者が減り、脱落は射の密度の逆順で進む」ことを実験的に確認する。両者を合わせると、**忘却は構造を抽象化の方向に選択する — 忘却が強いほど、抽象的な構造だけが残る場 (image(G)) が先鋭化する**。

これは選択と忘却の同一性を意味する: 「何かを選ぶ」ことは「他の全てを捨てる」ことであり、「何かを忘れる」ことは「残ったものを選んだ」ことである。忘却関手 $G$ は選択関手に他ならない。$\alpha$ は選択圧の強度であり、結果の方向は選択対象の射の密度 $R(X)$ のみが決定する。

#### 5.9.4 認知科学への含意

上記の構造は認知系に直接的な予測を与える:

1. **ワーキングメモリ容量と抽象的思考の反相関.** $\alpha$ が大きい系 (ワーキングメモリが小さい) では $R_{\text{crit}}$ が高く、具体的情報が急速に忘却される。残るのは射が密な抽象的構造のみ。注意欠如・多動性 (AuDHD) 特性を持つ個体が具体の保持に困難を持ちながら抽象的パターン認識に秀でるのは、この構造の帰結として予測される。

2. **エピソード記憶と抽象化の逆相関.** $\alpha \approx 0$ (忘却が弱い) の系では $R_{\text{crit}} \approx 0$ であり、$\text{image}(G) \approx \mathcal{C}$ — 具体が全て保存される。抽象化の*圧力*がゼロ。チンパンジーの瞬間的エピソード記憶の優位性と抽象的思考の欠如は、本理論からの対偶的予測と整合する。

3. **LLM の extended thinking と構造的性能.** 推論中の情報選択を強化する機構 (extended thinking) は $\alpha$ の動的増大に対応する。コーディング性能の向上 (構造的操作 = 射が密な型・パターン・制約の操作) は定理 5.9.3 の直接的帰結。

**反証条件.** 上記予測の独立検証には $\alpha$ の外的測定が必要である。ワーキングメモリ容量 (WM span) のタスクスコアを $\alpha$ の proxy として採用できる: $\alpha_{\text{proxy}} \propto 1/\text{WM span}$。予測 1 は「WM span が低い個体ほど抽象的パターンマッチングタスクの成績が高い」という反証可能な形をとる。WM span が低いにもかかわらず抽象課題の成績も低い個体が有意に観測されれば、本理論の予測は棄却される。

#### 5.9.5 忘却の自己参照構造: テンソルからスカラーへ

§5.9.1 の $\|F\| = g_{\text{eff}} \cdot \|T\|_g$ はテンソル方程式 $F_{ij} = (\alpha/2)(d\Phi \wedge T)_{ij}$ のノルム射影である。この「テンソル → スカラー」の射影は、ニュートン力学と一般相対論の関係に正確に対応する:

| 理論 | テンソル方程式 | スカラー射影 |
|:---|:---|:---|
| 重力 | $R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R = \frac{8\pi G}{c^4}T_{\mu\nu}$ | $F = mg$ (弱場極限) |
| 忘却 | $F_{ij} = \frac{\alpha}{2}(d\Phi \wedge T)_{ij}$ | $\|F\| = g_{\text{eff}} \cdot \|T\|_g$ (ノルム射影) |

ここで注目すべきは、ノルム射影 $\|\cdot\|: \Omega^2(M) \to C^\infty(M)$ 自体が**方向情報を忘却する関手**であるという事実である。2-形式 $F_{ij}$ は忘却の方向と大きさを保持するが、スカラー $\|F\|$ は大きさのみを保持し方向を捨てる。すなわち:

$$F_{ij} \xrightarrow{\text{忘却}} \|F\| = g_{\text{eff}} \cdot \|T\|_g$$

**F=mg は忘却理論の自己適用である** — 忘却曲率テンソルに忘却関手 (ノルム射影) を適用した結果がニュートン的スカラー方程式を生む。理論が自分自身を記述する。

**ニュートン近似の限界.** $\|F\| = g_{\text{eff}} \cdot \|T\|_g$ は $g_{\text{eff}}$ が $\|T\|_g$ に依存しないことを仮定する — $g$ が $m$ に依存しないニュートン近似。E12 の participation ratio = 106.4 (完全等方なら 768) は、$\ker(G)$ が厳密には等方的でないことを示唆し、一般相対性理論における質量-時空の双方向的結合に対応する第一次補正項の存在を示唆する。

忘却の場 $\Phi$ が作用する対象の圏 $\mathcal{C}$ の構造が $\Phi$ 自体に影響を与える — すなわち「忘却の対象が忘却の場を変形する」 — 場合、$g_{\text{eff}}$ は $\|T\|_g$ の汎関数 $g_{\text{eff}} = g_{\text{eff}}[\|T\|_g]$ となる。この双方向的結合の定量化は §6 の α-動力学の射程に属し、今後の課題とする。

## §6. α-動力学: 精度場と予測

### 6.1 動機

標準的な情報幾何では α ∈ ℝ は α-接続族から1つの接続を選択する固定パラメータである。しかし、多様体全体にわたる観測が単一の接続を使うべき先験的理由はない。パラメータ空間の異なる領域は異なる観測レジームに対応しうる — 顕微鏡と望遠鏡は異なる α を使う。

α を M 上の動力学場 α(θ) に昇格させることを提案する。

### 6.2 拡張された作用と場の方程式

$$S[\Phi, \alpha] = \int_M \left( \frac{1}{4} F_{ij}[\Phi,\alpha] F^{ij}[\Phi,\alpha] + \frac{\kappa}{2} g^{ij} \partial_i\alpha \, \partial_j\alpha + \frac{\lambda}{2} \Phi^2 \right) \sqrt{g} \, d^n\theta$$

Φ と α の結合は曲率を通じて入る:

$$A_i = \partial_i\Phi + \frac{\alpha(\theta)}{2} \Phi \, T_i$$

ℋ² 上では:

$$F_{12} = \frac{1}{2}\left[(\alpha \, \partial_\mu\Phi + \Phi \, \partial_\mu\alpha)T_2\right] = \frac{3}{\sigma}(\alpha \, \partial_\mu\Phi + \Phi \, \partial_\mu\alpha)$$

変分により連立 PDE が得られる:

- δS/δΦ = 0: 忘却場方程式（α(θ) と結合）
- δS/δα = 0: 精度場方程式（Φ(θ) と結合）

### 6.3 予測 1: α-遷移層力

**本論文の中心的予測。** ガウス族上（T₁ = 0, T₂ = 6/σ）で:

$$\boxed{F_{12} = \frac{3}{\sigma}(\alpha \, \partial_\mu\Phi + \Phi \, \partial_\mu\alpha)}$$

第2項 **Φ ∂_μα** は α 一定の理論には類似物がない:

> **忘却場が均一 (∂Φ = 0) であっても、観測フレームワーク自体の空間変動 (∂α ≠ 0) が曲率を生成する。**

**具体例.** m-接続 (α → -1) から e-接続 (α → +1) への遷移 α(μ) = tanh(μ/μ₀) を考える:

$$F_{12}\big|_{\partial\Phi=0} = \frac{3\Phi}{\mu_0 \sigma \cosh^2(\mu/\mu_0)}$$

μ ≈ 0（遷移層）に局在し、幅は ∼ μ₀。

![Fig.3: α-dynamics 遷移層でのΦ∂_μα項の局在化](./experiments/fig3_alpha_dynamics.png)

**物理的解釈:** 観測フレームワークが変わるところ — ある見方から別の見方へ移行するところ — で力が生成される。これはゲージアーティファクトではなく、α を場に昇格させた幾何学的帰結である。

**なぜ驚くべきか:** ゲージ理論では力は*冗長性*（ゲージ自由度）を表す接続の曲率から生じる。ここでは力は*観測の仕方の変化*から生じ、これは冗長性ではなく物理量である。α-遷移層力はゲージ理論の枠組みからは決して導出できない予測である。α-パラメータの類似物がゲージ理論には存在しないからだ。

**観測可能な領域の予測.** Appendix C.5 のスケーリング則 $|\lambda_c^\text{eff}| \propto n$ により、α-遷移層力が観測可能な領域は有効自由度 $n_\text{eff}$ が小さい統計多様体に限定される: 低ランクの注意ヘッド、小次元の潜在空間、文脈圧縮後の縮約空間がその主要な候補である。大語彙モデル ($n \sim 50{,}000$) では $|\lambda_c^\text{eff}| \approx 7{,}300$ となり自発的忘却相は実質不可能だが、有効ランクの低いサブシステムでは α-遷移層力が支配的になりうる (§C.5 参照)。

### 6.4 予測 2: 自己無撞着な遷移幅

精度場方程式 δS/δα = 0 において、tanh プロファイル α(θ) = tanh((θ - θ_c)/θ₀) は次のとき自己無撞着に満たされる:

$$\kappa = \frac{9\Phi(\theta_c)}{2\theta_0^2}$$

**導出の概要.** ガウス族 ℋ² 上で拡張作用 S[Φ,α] の α に関する Euler-Lagrange 方程式は:

$$-\kappa \, \nabla^2 \alpha + \frac{\partial}{\partial\alpha}\left(\frac{1}{4}F_{ij}F^{ij}\right) = 0$$

F_{12} = (3/σ)(α∂_μΦ + Φ∂_μα) のとき、第二項は Φ²T²α を含む項を生む。Φ が θ_c 近傍で緩やかに変化すると仮定し、tanh'' = -(2/θ₀²)sech²(·)tanh(·) を代入すると、sech² の係数比較から κ = 9Φ(θ_c)/(2θ₀²) が得られる。

**解釈:** 遷移幅 θ₀ は自由パラメータではない — 局所的な忘却強度と精度場の硬さの比で決定される。精度が硬いほど（κ が大きいほど）→ 遷移が狭くなる → 観測レジーム間の境界がより鋭くなる。

ガウス族上の数値検証でこの関係式は相対誤差 1.4 × 10⁻¹⁰ で確認済み。[SOURCE: oblivion_field_gaussian.py]

**α-動力学フレームワークの間接検証.** 自己無撞着条件（予測 P5）の高精度検証は、§6.1 の α-動力学フレームワーク全体の整合性を間接的に確認する。P5 は $\Phi$ と $\alpha$ の結合構造から導出される — P3（α-遷移層力、§6.3）も同じ結合から従う。したがって P5 の数値的確立は、P3 が依拠するフレームワークの信頼性の間接的支持である。

### 6.5 FEP の埋め込みと精度場対応

**主張の水準.** 本節の主張は「FEP と構造的に同型」という対称的な対応ではなく、より強い非対称的関係である: **FEP は本理論の $\alpha > 0$ 部分構造として埋め込まれる**（§6.5.1 で詳述）。$\alpha < 0$ 側（m-接続）には FEP の対応物が存在せず、本理論は FEP より広い構造を持つ。さらに、§6.6 の命題 6.6.1 により $F_{ij}$ は $\alpha$ に対して**正確に線形**であり、以下の対応表は曲率レベルでは近似ではなく構造的に正確である。

> **主張水準ラベル (批判反証レジストリ §1b.1 準拠).** 本節の FEP 対応は**構成的命題**（命題水準）である: CPS 内での構成は提示されているが、FEP 側の構成要素との等価性は外部査読を経ていない。ただし命題 6.6.1 (曲率の α-線形性) は**厳密対応**（定理水準）として証明済みである。

自由エネルギー原理 (FEP) では、ベイズ的エージェントは精度パラメータ π を最適化する:

$$\dot{\mu} = -\partial F/\partial\mu, \quad \dot{\pi} = -\partial F/\partial\pi$$

統計多様体上で α は類似の役割を果たす:

| FEP | 情報幾何 |
|:---|:---|
| 精度 π | α-パラメータ |
| 高精度 = 活用（データを信じる） | α → +1 = e-接続（データに忠実） |
| 低精度 = 探索（事前分布を信じる） | α → -1 = m-接続（事前分布に忠実） |
| π 最適化 = ∂F/∂π = 0 | α 最適化 = ∂S/∂α = 0 |

**用語の明確化.** 以下で「線形化」は2つの異なる意味で現れる。混同を避けるため明示する: (i) **動力学の線形化** — 固定点 (α₀, π₀) まわりで E-L 方程式を δα について展開する標準的な操作。これは近似である。(ii) **曲率の α-線形性** — 忘却曲率 F_{ij} が α に対して正確に線形であるという数学的事実（命題 6.6.1）。これは近似ではない。表の対応は (i) の線形化レベルだが、曲率に関する対応は (ii) により正確である。

**固定点まわりの動力学的同型.** 固定点 (α₀, π₀) まわりで:

- 忘却: κσ²∂²(δα) + (9σ²μ²/2)δα ≈ 0
- FEP: κ̃ d²(δπ)/dt² + γ(δπ) ≈ 0

いずれも調和振動子: 線形化レベルで同一の数学構造。

![Fig.4: Euler-Lagrange 精度場方程式の振動解（理論値波長との一致）](./experiments/fig4_euler_lagrange.png)

**誠実な限界:**

| 側面 | 本理論 | FEP |
|:---|:---|:---|
| 定義域 | α ∈ ℝ | π > 0 |
| 空間/時間 | α(θ) は*空間*場 | π(t) は*時間*発展 |
| 同型の水準 | 線形化レベルの構造的類似 **[構成的命題]** | 圏的同値ではまだない |

#### 6.5.1 限界の克服と射程

**定義域の不一致 (α ∈ ℝ vs π > 0).** α > 0 の部分領域に制限すれば、FEP 対応は定義域を含めた構造的類似となる。α < 0（m-接続側）の領域は FEP に対応物を持たない。これは限界ではなく、忘却場理論が FEP より広い構造を持つことの証拠である: FEP は忘却場理論における α > 0 セクターの特殊化として埋め込まれる。

**空間 vs 時間の不一致.** α(θ) は統計多様体上の空間場、π(t) は時間発展である。統一するには拡張多様体 (M × ℝ, g̃) 上で α(θ, t) を定義し、エージェントの位置 θ_agent(t) における値 π(t) = α(θ_agent(t), t) として FEP の π を回復する構成が必要となる。この時間拡張は本論文の範囲外であり、Paper II §4 で扱う。Paper II 命題 3.3.1 は α(θ) と CPS の離散的非対称性パラメータ Δd の層化構造 (Δd = 底空間、α(θ) = ファイバー) を示す。

**線形化の正確さ.** §6.6 で F_{ij} が α に対して正確に線形であることを証明する。曲率レベルでの FEP 対応は近似ではなく、構造的に正確である。

### 6.6 曲率の α-線形性: 構造的証明

§6.5 の対応表は固定点まわりの動力学的線形化（上記 (i)）に基づく。しかし、忘却曲率 $F_{ij}$ の $\alpha$ に対する依存性は近似ではなく正確に線形である（上記 (ii)）。この事実を証明する。

**命題 6.6.1 (F の α-線形性).** 忘却曲率 F_{ij} は α に対して正確に線形である。すなわち、任意の忘却場 Φ(θ) と α-プロファイル α(θ) に対して:

$$F_{ij}(\lambda\alpha) = \lambda F_{ij}(\alpha), \quad \forall \lambda \in \mathbb{R}$$

**証明.** 忘却接続を A_i = ∂_i Φ + (α/2)ΦT_i と書く。A_i は α の1次式である。忘却曲率 F_{ij} = ∂_i A_j - ∂_j A_i は A_i の微分の差であるから、α に対する線形性を継承する。□

**系.** §6.5 の FEP 対応において、忘却曲率に関する全ての対応は、近似ではなく正確な構造的対応である。§6.5 で「線形化レベル」と述べたのは、α の力学（E-L 方程式 δS/δα = 0）が非線形であることのみを指す。曲率そのものは α に対して正確に線形である。

**非線形性の所在.** 非線形性は以下にのみ現れる:
- α の E-L 方程式: δS/δα の Φ²T²結合項に Φ² が含まれる
- Φ の E-L 方程式: F_{ij}F^{ij} ∝ α² が入る
- α の自己一貫的決定は非線形だが、「与えられた α(θ) に対する F_{12}」は正確に線形

**数値的確認.** ガウス族 (ℋ²) 上で tanh プロファイル α(θ) = α₀ + (Δα/2)tanh((θ-θ₀)/ε) に対し、F_{12} の計算を ε/σ₀ ∈ [0.05, 4.0] の範囲で実行した。全てのケースで F_{12}(α) の α-線形性が機械精度で保持されることを確認した。

### 6.7 応用: α-接続近傍の最適化 (α-SAM)

命題 6.6.1 の α-線形性と §6.1 の α-フィールドは、深層学習における最適化手法への直接的な応用を可能にする。本節では、Foret et al. [7] の Sharpness-Aware Minimization (SAM) と Kim et al. [8] の Fisher SAM を、α-接続族の枠組みで一般化する手法 **α-SAM** を提案する。

**背景.** SAM は損失関数の近傍における最悪ケース損失を最小化する:

$$\min_\theta \max_{\|\epsilon\| \leq \rho} L(\theta + \epsilon)$$

ここで摂動 ε の近傍はユークリッド計量で定義される。Fisher SAM [8] はこれを Fisher 情報行列 $g^{(0)}(\theta)$ の近傍に拡張し、パラメータ空間の情報幾何学的構造を考慮する:

$$\max_{\epsilon^T g^{(0)}(\theta) \epsilon \leq \rho} L(\theta + \epsilon)$$

しかし、Fisher 計量 $g^{(0)}$ は Amari の α-接続族において α = 0 (Levi-Civita 接続) に対応する特殊ケースに過ぎない。

**定義 6.7.1 (α-接続近傍).** 統計多様体 $\mathcal{M}$ 上のパラメータ $\theta$ の α-接続近傍を以下で定義する:

$$B_\rho(\theta) = \{\theta + \epsilon : \epsilon^T g^{(0)}(\theta) \epsilon \leq \rho\}$$

近傍の計量は Fisher 計量 $g^{(0)}$ に固定する。α-パラメータは α-接続 $\nabla^{(\alpha)}$ の指数写像 $\text{Exp}^{(\alpha)}_\theta$ を通じて**内部更新方向**に入り、計量そのものを動かさない。α-接続族 $\nabla^{(\alpha)}$ は Amari の枠組みに従う: α = 0 は Fisher 計量の Levi-Civita 接続、α = 1 は e-平坦接続 (指数族の自然パラメータ座標で Christoffel 記号が消える)、α = -1 は m-平坦接続 (混合パラメータ座標で平坦)。

**計量一意性の根拠.** Ay, Jost, Lê, Schwachhöfer (2017) の定理 2.10(2) は、十分統計で不変な weakly continuous quadratic form が Fisher quadratic form の定数倍に限ることを示す (Čencov 1982 の強化版)。これは統計多様体上に α ∈ ℝ でパラメータ化された**計量族** $g^{(\alpha)}$ を独立に構成することが不変性原理のもとで不可能であることを意味する。したがって本稿は $g^{(0)}$ を唯一の計量として採用し、α は接続側 (更新方向の選択) に限定する。これは Ay et al. の一意性定理と整合的な**唯一の defensible な定式化**である。

**α-SAM の定式.** α-接続近傍を用いた最適化問題を定義する:

$$\min_\theta \max_{\epsilon \in B_\rho(\theta)} L\bigl(\text{Exp}^{(\alpha)}_\theta(\epsilon)\bigr)$$

ここで近傍 $B_\rho$ は Fisher 計量 $g^{(0)}$ で測定され、α は指数写像 $\text{Exp}^{(\alpha)}_\theta$ の選択 (更新方向) を通じて入る。標準 SAM は $g^{(0)} = I$ かつ $\text{Exp}^{(\alpha)} = $ 恒等的平行移動に対応し、Fisher SAM は $\text{Exp}^{(0)}$ (Levi-Civita 測地線) に対応する。

ここで α を**層ごとのフィールド** $\alpha(l)$ として設定する。これは §6.3 の α-フィールド:

$$\alpha(l) = \frac{1}{2}\left(1 + \tanh\left(s(l - l_c)\right)\right)$$

の直接的操作化であり、α(l) は学習可能なパラメータとして更新される。

**命題 6.7.1 (α パラメータ特殊化による Fisher SAM の回収).** α = 0 かつ α を全層で一定とすると、α-SAM の指数写像 $\text{Exp}^{(0)}$ は Fisher 計量の Levi-Civita 測地線に還元され、α-SAM は Fisher SAM [8] に一致する。さらに $g^{(0)} = I$（単位行列）とすると、標準 SAM [7] を回収する。

$$\text{SAM} \subset_\alpha \text{Fisher SAM} \subset_\alpha \alpha\text{-SAM}$$

ここで $\subset_\alpha$ は**α パラメータ特殊化による包含**を意味する: Fisher SAM は α-SAM の α=0 断面であり、標準 SAM はさらに $g^{(0)} = I$ の特殊ケースである。これは set-theoretic inclusion ではなく、**更新方向の族のパラメータ化による従属関係**である。

**命題 6.7.2 (表現制御定理).** α-SAM の内部最大化が生成する更新 $\epsilon^* = \arg\max_{\epsilon \in B_\rho} L(\text{Exp}^{(\alpha)}_\theta(\epsilon))$ は、α に対して微分すると:

$$\frac{\partial \epsilon^*}{\partial \alpha}\bigg|_{\alpha=0} \propto (\nabla^{(\alpha)} - \nabla^{(0)})\epsilon^* \quad(\text{α-接続差の方向})$$

すなわち α の変化は更新方向を $\nabla^{(\alpha)}$ 測地線に沿ってシフトさせる。命題 6.6.1 の α-線形性と組み合わせると、α の変化は忘却曲率 $F_{ij} = (\alpha/2) d(\Phi T)_{ij}$ を線形に変化させ、これが representation 空間に $d(\Phi T)$ 方向の構造的シフトを誘導する。

**方向性定理との関係.** 方向性定理（定理 5.1）は、忘却の方向（α が決定する接続の選択）が力を決めることを述べる。α-SAM はこの原理の実装であり、**最適化における摂動の方向を α-接続を通じて制御する**。representation 制御は定理 5.1 と命題 6.7.2 から理論的に保証される ( $\alpha \neq 0$ のもと)。

**実験的検証の 2 段階構造 (予測).**

- **Primary**: α を動かすと representation が $d(\Phi T)$ 方向に構造的にシフトするか (CKA / 表現類似度の直接測定)。この一次予測は命題 6.7.2 の直接的帰結であり、統計多様体の曲率理論の**実用的検証**となる
- **Secondary**: primary が verify された場合、その系として汎化性能が改善するか (CIFAR/ImageNet 実験)。最適 α(l) プロファイルは §6.3 の理論的予測 (tanh 型遷移) と整合すると期待される

Secondary の性能主張は primary の成否に**条件付き**である: representation 制御が理論予測どおり起きていることが前提となる。

**注意.** α-SAM は現時点では理論的提案であり、指数写像 $\text{Exp}^{(\alpha)}$ の効率的計算と、CIFAR-10/ImageNet 規模での primary/secondary 両面の実験的検証が不可欠である。特に、α-接続係数 $\Gamma_{ij,k}^{(\alpha)}$ の計算コストが実用性の鍵となる。

### 6.7.1 α-τ 対応の実験的定量化 (E11)

§6.5 の FEP 対応表は理論的対応であった。ここでは Hyphē [7] の結晶化閾値 τ と α-パラメータの定量的対応を、τ の連続掃引 (0.50 ~ 0.95, 46 点, 13 sessions) で実験的に検証する。

**理論的予測.** Paper III [9] は α > 0 でのみ copy/del (Markov 圏公理) が well-defined であり、α = 0 が臨界点であることを証明した。Hyphē では ρ_MB > τ でのみ結晶化 (安定チャンク形成) が起きる。したがって、τ の掃引において結晶化の秩序パラメータ N(τ) (チャンク数) に相転移が観測され、その臨界点 τ_c が α = 0 に対応すると予測される。

**結果 1: 相転移曲線.** N(τ) は3レジームのシグモイド型相転移を示す:

| レジーム | τ 範囲 | N (mean) | 対応する α |
|:--|:--|:--|:--|
| 未分化 (trivial) | ≤ 0.62 | 1.0 | α < 0 (anti-copy) |
| 相転移 (critical) | 0.63-0.75 | 1.1 → 11.3 | α ≈ 0 |
| 構造化 (structured) | > 0.75 | 11.3 → 33 | α > 0 (copy well-defined) |

臨界閾値 τ_c = 0.75 (max |dN/dτ| = 223 chunks/unit-τ)。

**結果 2: Coherence Invariance の再発見.** τ を 0.50 から 0.95 まで変化させても、mean coherence は 0.806-0.818 の狭い帯域 (±0.7%) に収まる。秩序パラメータ (チャンク数) が 1 → 33 と 33倍に変動する間、coherence は**不変**に保たれる。これは §3.7 の 104 実験で発見した Coherence Invariance $\bar{C}(\text{Fix}(G \circ F; \tau)) \approx \text{const}$ の τ 全域版であり、G∘F が任意の τ で coherence を保存しながら結晶の粒度のみを調整することを示す。

**解釈: α-τ 対応表.**

| Paper III | Hyphē (E11) | 物理的意味 |
|:--|:--|:--|
| α < 0: anti-copy 領域 | τ < 0.63: N=1 (全体が1塊) | 構造的分解が不可能。情報は「全か無」 |
| α = 0: 臨界点 (copy 退化) | τ ≈ 0.68-0.75: 急速な構造化 | 結晶核の形成。Copy/del が可能になる閾値 |
| α > 0: Markov 圏公理成立 | τ > 0.75: 安定チャンク群 | 独立操作可能な知識単位。Index_op ⊣ Search が機能 |

**§6.5 との接続.** §6.5 の FEP 対応表では α → +1 = exploitation (データに忠実)、α → -1 = exploration (事前分布に忠実) とした。E11 はこの対応に実験的証拠を与える: 高 τ (= 高 α) では fine-grained chunks が形成され、各 chunk は局所的なデータ構造に忠実な exploitation モード。低 τ (= 低 α) では全体が1つの chunk に融合し、局所構造を無視した exploration モード。

**Coherence Invariance の α-動力学的意味.** Coherence が τ に対して不変であることは、§6.3 の α-遷移層力の帰結として解釈可能である。α(τ) が遷移層を横切る際、遷移層力 $F_{12} \propto \Phi \, \partial_\tau \alpha$ が結晶の粒度変化を駆動するが、同時に Coherence を保存する拘束条件として機能する。これは §6.4 の自己無撞着条件の類似物であり、遷移幅が κ と Φ の比で決定されるのと同様に、Coherence 不変性は G∘F の自己無撞着条件から導かれる可能性がある。この仮説は今後の理論的検証を要する。

**既知の制限.** (i) 13 sessions, 単一 embedding モデルでの結果。(ii) τ_c の特定は dN/dτ の最大値点であり、有限サイズ効果の影響を受ける。(iii) α_eff の正確な定量的マッピング (α_eff(τ) の関数形) は未確立。

### 6.8 応用: 忘却場勾配の直接最適化 (Oblivion-Aware SAM)

§6.7 の α-SAM は近傍の形状（計量）を制御する。本節では、§3.4 の作用汎関数から自然に出現する別の最適化手法——忘却場 Φ の勾配を直接正則化する **Oblivion-Aware SAM**——を提案する。

**動機.** 作用汎関数 S[Φ] の場の方程式 δS/δΦ = 0 は、忘却場が自発的に F_{ij} ≠ 0（力あり）の配位を選ぶ条件を与える。この場の方程式を深層学習の目的関数に翻訳すると、以下の正則化付き損失を得る:

$$\min_\theta \Big[ L(\theta) + \lambda \|\nabla \Phi(\theta)\|_{g^{(0)}}^2 \Big]$$

ここで $\|\nabla \Phi\|_{g^{(0)}}^2 = g^{(0)ij} (\partial_i \Phi)(\partial_j \Phi)$ は Fisher 計量 ($g^{(0)}$, §6.7.1 の計量一意性に従う) による忘却場の勾配ノルムであり、λ は §3.4 の質量項と同一の物理的起源を持つ。共変微分 $\nabla$ は α-接続 $\nabla^{(\alpha)}$ を用いても同じ値 $\partial_i \Phi$ を与える (スカラー場の共変微分)。

**方向性定理との関係.** 定理 5.1 により、F_{ij} ≠ 0 ⟺ dΦ ∧ T ≠ 0（指数族の場合）。忘却場の勾配 ∇Φ が Chebyshev 形式 T に整列 (∇Φ ∝ T) するとき力はゼロになるが、直交するとき最大の力を生む。λ < 0（自発的忘却レジーム）のもとで $\|\nabla\Phi\|^2$ を最小化する方向へ θ を駆動することは、忘却場の勾配を T に対して最大の不整合を持つ方向——すなわち最大の力を生む方向——に揃えることに相当する。

**ガウス族上の具体的計算.** §4 の異方的忘却場 Φ_B(μ,σ) = -log σ + (σ² + μ²)/2 - 1/2 に対し:

$$\|\nabla \Phi_B\|_{g^{(0)}}^2 = \sigma^2 \mu^2 + \frac{(\sigma^2 - 1)^2}{2}$$

μ = 0（等方的忘却、F₁₂ = 0）のとき勾配ノルムは σ 寄与のみを持ち、μ ≠ 0（異方的忘却、F₁₂ = 3αμ/σ ≠ 0）のとき μ² 項が出現する。Oblivion-Aware SAM は後者の配位（力を生む忘却パターン）を選好する。

**命題 6.8.1 (α-Oblivion SAM の α パラメータ特殊化による包含).** α-SAM（§6.7）と Oblivion-Aware SAM を合成した完全形態:

$$\min_\theta \max_{\epsilon \in B_\rho(\theta)} \Big[ L\bigl(\text{Exp}^{(\alpha)}_\theta(\epsilon)\bigr) + \lambda \|\nabla \Phi(\theta)\|_{g^{(0)}}^2 \Big]$$

において、パラメータの特殊化により以下の従属関係を持つ (§6.7 の $\subset_\alpha$ と同様):

$$\text{SAM} \subset_\alpha \text{Fisher SAM} \subset_\alpha \alpha\text{-SAM} \subset_\alpha \alpha\text{-Oblivion SAM}$$
$$\text{SGD} \subset_\alpha \text{Oblivion-Aware SAM} \subset_\alpha \alpha\text{-Oblivion SAM}$$

が厳密な包含関係として成立する。特に (α=0, λ=0, ρ>0) で Fisher SAM、(α学習, λ=0, ρ>0) で α-SAM、(α=0, λ<0, ρ=0) で Oblivion-Aware SAM をそれぞれ回収する。

**注意.** Oblivion-Aware SAM の実装には、忘却場 Φ(l) の操作的定義——例えば CKA に基づく $\Phi(l) = 1 - \text{CKA}(h_l, h_0)$——と、λ < 0 レジームでの学習安定性の検証が不可欠である。λ の最適スケーリングは |λ| ∼ ρ · ‖T‖²_g と予想される（SAM の平坦性要求と忘却の方向性要求のバランス）。CKA ベースの操作的定義と理論的 KL 定義の関係は §6.8.1 で厳密に解析する。

#### 6.8.1 CKA ↔ KL 橋渡し: 操作的忘却場の理論的保証

本節では、操作的忘却場 $\Phi_\text{CKA}(l) = 1 - \text{CKA}(h_l, h_0)$ が §3-5 の理論的忘却場 $\Phi_\text{KL}(\theta)$ をどの程度忠実に近似するかを解析する。主結果は、方向性定理（定理 5.1）の判定——$d\Phi \wedge T \neq 0$ か否か——が CKA ベースの忘却場でも有効であることの理論的保証である。

**定理 6.8.1 (CKA-KL 分離).** 層 $l$ の表現がガウス $h_l \sim \mathcal{N}(\mu_l, \Sigma_l)$ に従い、$\Sigma_l$ と $\Sigma_0$ が同時対角化可能（固有値比 $r_i = \lambda_i^{(l)}/\lambda_i^{(0)}$）とする。このとき:

(i) CKA は共分散の**形状**（固有値比の非一様性）のみを測定し、一様スケール変化（$r_i \equiv c$）に対して恒等的に CKA = 1 となる。

(ii) KL はスケール+形状の両方を測定する: $D_\text{KL} = \frac{1}{2}\sum_i [r_i - 1 - \ln r_i]$。

(iii) 忘却場は二成分に直交分解される: $\Phi_\text{KL} = \Phi_\text{shape} + \Phi_\text{scale}$。CKA は $\Phi_\text{shape}$ に対応する。

**命題 6.8.2 (方向保存条件).** $|\partial_l \Phi_\text{scale}| \ll |\partial_l \Phi_\text{shape}|$ のとき、$\text{sign}(\partial_l \Phi_\text{CKA}) = \text{sign}(\partial_l \Phi_\text{KL})$。すなわち方向性定理の判定は CKA ベースでも有効:

$$d\Phi_\text{CKA} \wedge T \neq 0 \iff d\Phi_\text{KL} \wedge T \neq 0$$

BatchNorm 使用時、一様スケール成分 $\bar\delta = d^{-1}\sum_i \delta_i \approx 0$ となるため、この条件は**環境的に強制**される。

**命題 6.8.3 (比例定理).** $r_i \in [1-\epsilon, 1+\epsilon]$（隣接層間の変化が小さい）のとき、$\Phi_\text{CKA}$ と $\Phi_\text{KL}$ の間に次元依存の比例関係が成立する:

$$\Phi_\text{CKA} = \frac{C_\star}{d} \cdot \Phi_\text{KL} + O(\epsilon^3)$$

ここで $d$ は表現次元、$C_\star$ は次元に依存しない普遍定数である。解析的 CKA（$\|\Sigma_0 \Sigma_l\|_F^2 / (\|\Sigma_0^2\|_F \cdot \|\Sigma_l^2\|_F)$）の2次展開と $\Phi_\text{KL} \approx \frac{1}{2}\sum_i \delta_i^2$ の比較により、$C_\star$ は $\|\Sigma_0^2\|_F^2$ の正規化項から生じる。数値検証（§6.8.1 注2）により $C_\star \approx 6.3$（変動係数 1.3%、$d \in [10, 500]$ で安定）が確認された。

Oblivion-Aware SAM の目的関数において CKA を用いる場合、$C_\star$ を $\lambda$ のスケーリングに吸収する:

$$\min_\theta \Big[ L(\theta) + \frac{\lambda d}{C_\star} \|\nabla \Phi_\text{CKA}(\theta)\|_{g^{(0)}}^2 \Big]$$

により、CKA ベースの最適化と KL ベースの理論が定量的に整合する。$C_\star$ は次元 $d$ に依らず安定であるため、実装上は単一のハイパーパラメータ $\tilde\lambda = \lambda d / C_\star$ として扱える。

**仮定の除去.** 定理 6.8.1 は技術的にガウス仮定と同時対角化可能性を用いたが、いずれも結果の有効性に不要である:

(a) **ガウス仮定の除去 (情報幾何学的ピタゴラス定理, Amari 1985).** ガウス族は指数型分布族であるから、任意の分布 $p_l$ の m-射影 $p_l^* = \mathcal{N}(\mu_l, \Sigma_l)$ に対して $D_\text{KL}(p_l \| p_0) = J(p_l) + D_\text{KL}(p_l^* \| p_0)$ が**厳密に**成立する。ここで $J(p_l) = D_\text{KL}(p_l \| p_l^*)$ は negentropy（非ガウス性の尺度）。忘却場は三成分に分解される:

$$\Phi_\text{KL} = \underbrace{J(p_l)}_{\text{非ガウス性}} + \underbrace{\Phi_\text{shape}}_{\text{CKA が測定}} + \underbrace{\Phi_\text{scale}}_{\text{CKA が見逃す}}$$

条件 D ($|\partial_l J| \ll |\partial_l \Phi_\text{shape}|$) のもとで方向保存が成立する。この条件は BatchNorm（信号対雑音比の安定化）、高次元 $d \gg 1$（大数の法則により $|\partial_l J| \sim O(1/\sqrt{d})$）、残差接続（$\partial_l J$ の抑制）の三つの機構で環境的に保証される。

(b) **同時対角化の除去 (正準相関分解).** CKA は交差共分散 $\Sigma_{l0} = \text{Cov}(h_l, h_0)$ の Frobenius ノルムで計算され、基底に依存しない。一般ケースでは固有方向の回転が $\Phi_\text{rotation}$ として出現するが、この成分は CKA と KL の**共通成分**であるため、方向保存は同時対角化可能な場合**以上に強化**される。

(c) **非線形カーネル CKA (RKHS リフト原理).** 任意のカーネル $k$ に対して $\text{CKA}_k(H_l, H_0) = \text{CKA}_\text{lin}(\varphi(H_l), \varphi(H_0))$ であり、§2-10 の理論は RKHS 上で再利用可能。特に arc-cosine カーネル $K_1$ [Cho & Saul 2009] は ReLU に正確に対応し、整流効果をカーネルに吸収する代替アプローチを提供する。実装上は線形 CKA + ピタゴラス分解が推奨される。

**条件体系.** 方向保存 ($d\Phi_\text{CKA} \wedge T \neq 0 \iff d\Phi_\text{KL} \wedge T \neq 0$) を保証する条件:

| 条件 | 内容 | 保証する機構 |
|:---|:---|:---|
| A (形状優位) | 表現変化が主に形状変化 | 深層ネットワーク構造 |
| B (スケール変化小) | $\|\partial_l\Phi_\text{scale}\| \ll \|\partial_l\Phi_\text{shape}\|$ | BatchNorm |
| C (局所近似) | $\epsilon \ll 1$ | 残差接続 |
| D (negentropy 安定) | $\|\partial_l J\| \ll \|\partial_l\Phi_\text{shape}\|$ | BN + 高次元 + 残差 |

条件 B, C, D は深層ネットワークの標準構成により環境的に強制される（詳細は calculations/cka_kl_bridge.md §8-11 を参照）。

**注1 (ReLU による CKA の系統的シフト).** 非線形活性化（ReLU）は CKA 値に系統的シフトを導入する: 数値検証により $\text{CKA}_\text{ReLU} / \text{CKA}_\text{Gauss} \approx 0.74$（$\sigma = 0.0015$）が観測された。絶対値のシフトは大きいが、標準偏差が極めて小さい（変動係数 0.2%）ことから、シフトは**一定の倍率**であり、方向保存（sign の一致）には影響しない。Oblivion-Aware SAM は $\Phi_\text{CKA}$ の**勾配の方向**を使用するため、系統的定数倍は $\tilde\lambda$ に吸収される。代替的に、(c) の arc-cosine カーネル CKA を用いればシフトはカーネルに吸収される。

**注2 (数値検証).** 本節の理論的保証は experiments/cka_kl_bridge_numerical.py による6項目の数値検証で確認された:

| 検証項目 | 結果 | 状態 |
|:---|:---|:---|
| P1: スケール不変性 | CKA(cI, I) = 1.0（差 < 10⁻¹⁶） | 理論通り |
| P2: 方向保存率 | BN あり 89.4%, BN なし 62.6%（差 +26.7%） | BN が方向保存を強化 |
| P3: 誤差上界 | d 増大で |err|/ε³ 比率が低下 | 高次元改善を確認 |
| P4: ピタゴラス（ReLU） | CKA(ReLU)/CKA(Gauss) = 0.74, σ = 0.0015 | 安定なシフト（注1） |
| P5: 回転不変性 | CKA, KL ともに完全不変（$\Sigma_0 = I$ 下） | 理論通り |
| P6: 次元補正 $C_\star$ | $d \cdot \Phi_\text{CKA}/\Phi_\text{KL} \approx 6.3$（CV = 1.3%） | 次元非依存を確認 |

#### 6.8.2 実験的検証: CKA ベース Oblivion-Aware SAM (E12)

§6.8 の理論的枠組みと §6.8.1 の CKA-KL 橋渡しを、ResNet-18/CIFAR-10 上の4条件比較実験で検証する。全条件で CKA ベースの操作的忘却場 $\Phi(l) = 1 - \text{CKA}(h_l, h_0)$ を用い、正則化項 $R = \lambda_\text{sign} \cdot \lambda_\text{base} \cdot \sum_l (\nabla_l \Phi)^2$ を損失に加算した。

**実験条件.** ResNet-18（conv1=3×3, maxpool 削除）、CIFAR-10、200 epochs、cosine annealing、5 seeds（42-46）。

| 条件 | SAM (ρ) | λ_base | λ_sign | 目的 |
|:---|:---|:---|:---|:---|
| SGD | — | 0 | 0 | ベースライン |
| SAM | 0.05 | 0 | 0 | Foret et al. 2021 |
| OA-SAM | 0.05 | 0.01 | −1 | 忘却促進（§6.8 の λ < 0 レジーム） |
| 反転制御 | 0.05 | 0.01 | +1 | 忘却抑制（対照実験） |

**結果 1: 精度.** SAM 系3条件はいずれも SGD に対して有意に優れる（p < 0.001）。OA-SAM と SAM の間に有意差はない（p = 0.067）。

| 条件 | Test Accuracy (ep200) | 90% 到達 epoch |
|:---|:---|:---|
| SGD | 95.48 ± 0.09 | 103 ± 8 |
| SAM | 96.05 ± 0.10 | 66 ± 5 |
| OA-SAM (λ<0) | 95.91 ± 0.09 | 68 ± 9 |
| 反転制御 (λ>0) | 95.91 ± 0.07 | 70 ± 7 |

忘却場の正則化（λ ≠ 0）は精度に対して中立である。これは命題 6.8.1 の包含関係と整合する: OA-SAM は SAM を含むより広い最適化空間で探索するが、最終精度は SAM と統計的に区別できない。

**結果 2: SAM は忘却パターンを変えない.** 4層の CKA プロファイル（20 epoch 毎に計測）は、SGD と SAM で全層にわたり有意差がない。

| Layer | SGD (CKA, ep200) | SAM (CKA, ep200) | 差 | p 値 |
|:---|:---|:---|:---|:---|
| L1 | 0.424 ± 0.012 | 0.426 ± 0.026 | +0.002 | 0.913 |
| L2 | 0.443 ± 0.021 | 0.428 ± 0.011 | −0.016 | 0.228 |
| L3 | 0.338 ± 0.016 | 0.319 ± 0.017 | −0.019 | 0.143 |
| L4 | 0.221 ± 0.005 | 0.229 ± 0.006 | +0.008 | 0.052 |

SAM の sharpness-aware perturbation は loss landscape の曲率を制御して汎化を改善するが、層ごとの忘却パターン（CKA プロファイル）には作用しない。SAM と忘却場は直交する2つの経路——SAM は precision の経路、Φ は representation の経路——であり、§6.7-6.8 の理論的区別に実験的根拠を与える。

**結果 3: λ の符号が忘却を直接制御する.** OA-SAM（λ < 0）と反転制御（λ > 0）は、SGD/SAM とは質的に異なる CKA プロファイルを示す。効果は Layer 1（入力に最も近い層）に集中する:

| 条件 | L1 CKA (ep200) | 解釈 |
|:---|:---|:---|
| SGD / SAM | 0.42 | 中程度の忘却 |
| OA-SAM (λ<0) | **0.051 ± 0.021** | 初期表現の 95% を書き換え |
| 反転制御 (λ>0) | **0.774 ± 0.012** | 初期表現の 77% を保存 |

L4 は全条件で CKA ≈ 0.22-0.25 と安定であり、正則化の効果は浅い層に集中する。これは $\Phi(l) = 1 - \text{CKA}(h_l, h_0)$ の L1 における感度が最大であること（入力 $h_0$ との直接比較）から説明される。

**結果 4: 忘却の縮退 (Degeneracy).** OA-SAM（λ < 0）の 10 seeds を個別に分析すると、L2/L3 の「高分散」はランダムなばらつきではなく、**2 つの離散的な attractor への二分岐** であることが判明した:

| 経路 | N | Seeds | L2 CKA (ep200) | L3 CKA (ep200) | Test Acc |
|:---|:---|:---|:---|:---|:---|
| 経路 A（L2 保存） | 7 | 42,44,45,46,47,48,51 | 0.926 ± 0.005 | 0.040 ± 0.004 | 95.99 ± 0.10 |
| 経路 B（L3 保存） | 3 | 43,49,50 | 0.052 ± 0.005 | 0.939 ± 0.003 | 95.91 ± 0.10 |

分岐比率は **7:3** (N=10)。各経路の内部分散は極めて小さく（CKA の標準偏差 ≤ 0.005）、中間状態は存在しない。2 つの attractor は明確に離散的である。Path A の basin of attraction が Path B より広い（約 70:30）。

両経路の最終精度に有意差はない（Welch's t-test: p = 0.372）。L1（全 seed で忘却、CKA→0.05）から伝播する忘却の波は、必ずどこか 1 層で堰き止められる。堰の位置（L2 vs L3）は初期条件（seed）で決まるが、ネットワークは「L2 保存 + L3 忘却」と「L2 忘却 + L3 保存」のどちらでも等しく機能する。これは忘却パターンに**縮退**——同一の精度を達成する内部構造が複数存在すること——があることを示す。

この縮退は §3.4 の作用汎関数 S[Φ] が複数の極小点を持つことの直接的証拠と解釈できる。basin の非対称性（70:30）は、S[Φ] の 2 つの極小が同じ深さだが basin の体積が異なることを示唆する。反転制御（λ > 0）ではこの分岐が発生しない（全 seed で低分散）ことから、λ の符号が縮退の有無も決定する。

**結果の要約.**

| 記号 | 主張 | 検証状態 |
|:---|:---|:---|
| E12a | SAM は忘却パターンを変えない（CKA: SGD ≈ SAM） | ✓ N=5, 全層 p > 0.05 |
| E12b | λ の符号が L1 の忘却/保存を直接制御する | ✓ CKA 0.05 vs 0.77 |
| E12c | 忘却場の正則化は精度に対して中立 | ✓ OA-SAM vs SAM: p = 0.067 |
| E12d | 忘却パターンに縮退がある（L2/L3 二分岐） | ✓ N=10, 7:3 分岐, p = 0.372 |

**既知の制限.** (i) ResNet-18/CIFAR-10 の単一設定。より大規模なモデル/データセットへの一般化は未検証。(ii) CKA プロファイルの計測にテストバッチ1つ（128 samples）を使用。バッチ間のばらつきは未検証。(iii) 分岐比率 7:3 の信頼区間は N=10 では広い（二項分布 95% CI: 0.35-0.93）。N=30+ で精度向上が望ましい。

### 6.9 予測 4: 曲率選択則

方向性定理は曲率の異なる多様体上で質的に異なる振る舞いを予測する:

| 多様体 | ガウス曲率 | Chebyshev 形式 | 予測 |
|:---|:---|:---|:---|
| ガウス族 (ℋ²) | K = -1/2 | T = (0, 6/σ) | Θ = 0 は**安定** — 自発的忘却は起きない |
| カテゴリカルシンプレックス (Δⁿ) | K = +1/4 | T_i = 1-(n+1)p_i | 正曲率は Θ = 0 の**不安定化傾向**を示す |
| 指数型分布族（一般） | K が変化 | dT = 0 (普遍) | K の符号が安定性の傾向を決定 |

全指数型分布族において dT = 0 が成立するため（§5.4 注）、力の生成は第1メカニズム（方向的不整合 dΦ ∧ T ≠ 0）に完全に帰着する。カテゴリカルシンプレックス上での具体的結果は Appendix B で詳述する。

**注意.** Θ = 0 の安定性に関する厳密な主張には以下が必要であり、現時点では定性的議論に留まる:

1. **作用汎関数 S[Φ] の λ 閾値**: 質量項 λ が曲率結合項 α²|T|²_g/4 を上回る場合、正曲率でも Θ = 0 は安定化されうる。λ の物理的起源は §3.4 で導出した VFE の Accuracy-Complexity バランスであり、臨界条件 λ_c = 0 は N·ḡ_eff = d_eff/L² に対応する（Appendix C 参照）
2. **Chebyshev 力との結合**: 球面上の自由 scalar field は安定である。不安定性の真のドライバーは曲率そのものではなく **T_i と Φ の結合項** Φ T_i ∂^i Φ である
3. **スカラー場の伝搬**: Φ の摂動伝搬は Laplace-Beltrami 固有値空間で特徴づけられ、安定性には Ricci 曲率（sectional curvature ではなく）が関与する可能性がある
4. **n 次元スケーリング**: Appendix C.5 の Liouville 変換解析により、有効臨界閾値は |λ_c^eff| ∝ n（simplex 次元に線形）であり、大語彙空間では自発的忘却に極めて大きな VFE 曲率が必要である

定性的には、正曲率多様体上では Chebyshev 力 T_i が摂動を集中させる傾向があり、負曲率では発散させる傾向がある。この直感は正しいが、Appendix C の変分的安定性解析により、安定性の一次的決定因子は sign(λ) であり K ではないことが示された。

**物理的意味（弱い形式）:** 正曲率を持つモデル空間は Θ = 0 の不安定化を*促進する傾向がある*。§3.4 の VFE 解析により、λ < 0 はデータ密度 N が十分に高い場合（Bayesian Occam's Razor の閾値超過）に成立し、このとき正曲率が幾何学的脆弱性（ker(T) 近傍でのカイネティック項の消失）を提供する。

**数値的確認（Appendix C.4）.** Δ² 上の ODE シミュレーションにより、Φ = 0 からの pitchfork 分岐を有効臨界値 λ_c^eff ≈ -0.301 で観測した。理論的臨界値 λ_c = 0（C.2、ker(T) の一点での局所条件）との差は、空間的に拡がった摂動が拡散に打ち勝つために必要な追加の駆動力を反映している: λ_c^eff = -α²⟨|T|²_g⟩_eff/4。不安定な摂動モードは ker(T)（均等分布）に局在化し、C.3 の幾何学的脆弱性予測を直接的に支持する。

## §7. ゲージ理論との関係

本理論はゲージ理論**ではなく**、そのような主張も行わない。ゲージ対称性の不在は*欠点*ではなく*特徴*である。

| 性質 | Yang-Mills | 忘却場 |
|:---|:---|:---|
| 動力学変数 | 接続 A_i（独立） | スカラー Φ（A_i を決定） |
| ゲージ対称性 | A → A + dχ | なし — A_i は完全に決まる |
| 分類 | ゲージ理論 | 非最小結合スカラー場 |
| 曲率の意味 | 平行移動の失敗 | **忘却の方向性** |
| α-動力学 | 類似物なし | α(θ) を精度場として |

独自の貢献:

1. **方向性定理** — 任意の統計多様体上で適用可能な、忘却からの力の判定基準
2. **α-遷移層力** — ゲージ理論に類似物のない予測
3. **FEP 対応** — 情報幾何を認知的精度に接続

## §8. 認識論的射程 (非防衛的宣言)

> **⚠️ 本節の位置づけ.** 以下の段落は**本稿で防衛されない**。忘却論シリーズ (Paper I-XII) が目指す長射程の地平を宣言としてのみ置く。形式的な正当化は各項に記した後続稿および Open Problem に委ねる。批評・反論は歓迎するが、本稿はこれらの主張を防衛しない — これらは方向性定理の**先にある景色**であって、方向性定理そのものの成立には依存しない。

**前層テーゼと自己適用**: 各物理理論を前層 $\tilde{\text{Th}}(-, \Omega)$ の切断として捉えれば、米田の補題により完全な構造 $\Omega$ は切断族全体によってのみ決定される。忘却場理論はその 1 つの切断であり、ゲージ理論は別の切断である。この認識論的主張はそれ自身にも適用され、方向性定理は正確な意味で自己参照的である: 観測の**方向**が大きさよりも重要だと述べる定理を定理自身に適用すれば、定理をどの角度から見るかがそこに何を見出すかを決める。

**EFE 統一問題との構造的対応** **[構造的類似]**: Champion et al. [7] が提示した期待自由エネルギー (EFE) の 4 定式化の統一ジレンマ — Setting 1 の制約 $C_o \in \text{Image}(A)$ — は、方向性定理の力の生成条件 $d\Phi \wedge T \neq 0$ と構造的に対応する。Paper II [10] は離散 POMDP を連続統計多様体に持ち上げ、$\Phi > 0 \Leftrightarrow $ Markov blanket の存在として再定式化した (命題 3.7.4b-f)。ジレンマは情報幾何学的に解消される見込みだが、本稿の範囲を超える。

**Rate-Distortion × Levins GRP**: 抽象度 $\alpha \in [0,1]$ に対し、説明力 $E(\alpha)$ と特定的予測力 $P_s(\alpha)$ は $E \times P_s \approx$ const の反比例制約を受ける (高い場所に立つほど遠くまで見渡せるが足元の石ころは見えなくなる)。これを Rate-Distortion 理論で形式化すると、$R(\alpha) = \dim(\text{image}(G))$, $D(\alpha) = |F_{ij}|$ の対応が立ち、パラダイム内は $R(D)$ 曲線固定、パラダイム交替 (Paper V) は曲線シフトに対応する。Levins の Generality-Realism-Precision 同時最大化不可能性は前層テーゼの帰結として回収される。$R(D) \leftrightarrow F_{ij}$ の関手的証明は **Open Problem OP-I-1** として記録する。

## §9.5. 圏論のゼロ曲率極限 — 忘却場理論からの展望 (非防衛的宣言)

> **⚠️ 本節の位置づけ.** 以下は**本稿で防衛されない**。方向性定理の先にある景色を Conjecture として置く。形式的証明は **Open Problem OP-I-2, OP-I-3** として記録し、部分的先行実装 (HGK 弱 2-圏, [0,1]-豊穣圏, Paper VIII §6.7-6.8 の Grothendieck 位相 $J_\alpha$) にポインタを置く。批評は歓迎するが、本稿はこの節を防衛しない。

**圏論 = $\Phi=0$ 極限という仮説** **[構造的類似]**. 方向性定理 (定理 5.1) より、$\Phi = 0$ なら忘却曲率は消える。さらに $G$ は恒等へ退化するから、$\ker(G)=\{0\}$、したがって合成ドリフト $\delta = G(f \circ g) - G(f) \circ G(g)$ も消える。ここまでは既存枠内で押せる。だが最後の $\delta = 0 \Rightarrow \text{Hom}_\Phi \in \{0,1\}$ は自動ではない。**⚠️ 精度注記 (2026-04-14):** 一対象圏 $\text{Hom}(*,*) = [0,1]$、合成 $= \min$、$G = \text{id}$ は OP-I-3 と $\delta=0$ を共に満たしつつ $\text{Hom}(*,*) = 1/2$ を許す (Codex GPT-5.4 形式化検証)。したがって Hom の離散化には追加公理 **ZeroForgetCollapse** — 「$\Phi=0 \Rightarrow [0,1]$-豊穣化が $\{0,1\}$-豊穣化に崩壊する」— が要る。これは FEP 的直観の形式化として自然だが、OP-I-3 からの導出ではない。アナロジー: ユークリッド幾何がリーマン幾何の曲率ゼロ特殊ケースであるように、**圏論は忘却場理論の $\Phi = 0$ 特殊ケースである**。ただし現在この比喩が保証するのは、Boolean 回復を含む全鎖ではなく、$\Phi=0 \Rightarrow \ker(G)=\{0\} \Rightarrow \delta=0$ の一方向鎖までである。逆転して $\Phi$ による圏論の拡張 (忘却豊穣圏 $\mathcal{C}_\Phi$, 対象 = 統計多様体上の分布, $\text{Hom}_\Phi(A,B) \in [0,1]$) はなお定義可能だが、well-definedness は未検証 (OP-I-3)。

> **注意 ($\Phi = 0$ と $F = 0$ の区別).** $\Phi = 0$ (忘却なし) と $F = 0$ (力なし) は同値ではない。§4 の二分法が示す通り、均一忘却 ($\Phi = \text{const} \neq 0$, $d\Phi = 0$) は $d\Phi \wedge T = 0$ を満たすため $F_{ij} = 0$ だが、$\ker(G) \neq \{0\}$ である。上記の予想で $\Phi \to 0$ を取るのは（$F \to 0$ ではなく）、$\Phi = 0$ のみが $\ker(G) = \{0\}$ を直接保証するからである。$\Phi = 0$ → $G$ は恒等に退化 → 忠実 → $\delta = 0$ という一方向含意鎖が OP-I-2 の本来の構造であり、$F = 0$ からの導出は不成立。なお、$\delta = 0$ から先の $\text{Hom}_\Phi \in \{0,1\}$ へのステップは OP-I-3 単独では導けず、**ZeroForgetCollapse** 公理の追加が必要であることが Codex GPT-5.4 (2026-04-14) の形式化検証で確認された (反例: min合成の一対象 $[0,1]$-豊穣圏; 詳細は automath bridge `dictionary.md` §2.B)。OP-I-2 の解決には ZeroForgetCollapse の正当化も含まれる。離散的検証: The Omega Project (automath) の carry defect 定理 (`restrict_stableAdd_carry_defect`, Lean 4 証明済み [2]) は、有限体上の離散版としてこの一方向含意鎖を機械検証で支持する。

> **OP-I-2 の open は三分割される.** 第一に**公理の穴**: `ZeroForgetCollapse` の正当化。第二に**極限の穴**: Conjecture 9.5.2 は $X_\infty(\lambda)$ 自体の連続変形では閉じず、固定された ambient profinite 空間 $\{0,1\}^{\mathbb{N}}$ 上の弱*連続測度族 $\mu_\lambda$ へ再定式化が必要である。第三に**関手の穴**: 離散化関手 $D$ は現時点で $\mathbf{Man}_{\mathrm{No11}}$ に限って厳密であり、$\mathbf{Man}$ 全体への拡張には `Discretizable` / `DescendsToCube` の data を要する。実装戦略としては $\Delta^n$ 直行より逆極限経由の Strategy B が主経路であり、Appendix B の幾何を $(\Delta^1)^n$ に組み替える Strategy A' は代替経路として残る。

**圏論が表現できない因果の矢 — E12 による実証**. 圏論の関手 $G$ は $\ker$ と $\text{image}$ を定義するが、**$\ker$ の内部構造が $\text{image}$ の形を決める**という因果関係を記述する道具を持たない。E12 実験 (30 sessions, 6053 steps) はこの因果を直接観測した: $\ker(G)$ は 768 次元で等方的 ($\text{PR} = 106.4$, 座標相関 $r < 0.08$) であるのに対し、$\text{image}(G)$ は 6 方向に集中 (Temporality $r=-0.26$, Valence $r=0.22$, Scale $r=0.27$, effective rank $33/768 = 4.3\%$)。等方的忘却から低ランク保存が**必然化**される — 忘却場の曲率 $F_{ij}$ がこの因果を幾何学的に記述する。

**RAG ノイズと等方性の有益性 (OP-I-4) [構造的類似].** $F = \nabla\Phi$ の技術的インスタンスとして RAG 検索ノイズの効果が記述できる可能性がある: 無関係文書の混入が性能を**向上させる**現象 (Cuconasu et al., arXiv 2401.14887, 2024) は、過確信状態 ($|F_{ij}|$ が高い) の正則化として解釈される。チャンクサイズ最適化は $\nabla\Phi$ の極小点に対応し、ノイズ種別 (矛盾 vs 無関係 vs ランダム) の方向性比較により直接検証可能。[確信度 70%: 定性的推論による; 定量的導出は §6 動力学方程式と過確信度の操作的定義を要する]

## §10. 結論

本稿は、統計多様体上における忘却——確率分布のパラメータを粗視化する操作——から力が創発される機構を定式化し、以下の4つの主要結果を確立した。

**第一に、方向性定理 (定理 5.1)** は、忘却曲率 $F_{ij} \neq 0$ の必要十分条件が忘却場の勾配と Chebyshev 1-形式の方向的不整合にあることを示した。力の源泉は忘却の大きさではなく方向にある。この結果はガウス族上の解析計算 (§4) で導出され、その算術的健全性は独立な有限差分計算との一致 (相対誤差 $1.4 \times 10^{-10}$, §4.5) で確認された。定理そのものの証明は §5.2 に与えられる。

**第二に、α-遷移層力 (§6.3)** として、Amari の α-パラメータを空間場 α(θ) に昇格させることで、統計的フレームワーク間の遷移領域に力が局在する予測を導出した。この力は Yang-Mills 型のゲージ冗長性とは本質的に異なり（§7）、観測フレームワーク自体に伴う幾何学的効果である。

**第三に、FEP との構造的対応 (§6.5) [構成的命題]** において、忘却曲率が α に対して正確に線形であること（命題 6.6.1 **[厳密対応]**）を証明し、精度場方程式が自由エネルギー原理 [2,3] の精度最適化と線形化レベルで構造的に同型であることを示した。この対応は曲率レベルでは近似ではなく正確である。

**限界と今後の展望.** (i) 本理論の FEP 対応は空間場 α(θ) と時間発展 π(t) の間の同型であり、統一には拡張多様体 M × ℝ 上の時間拡張が必要である（Paper II）。(ii) 選択的忘却の経験的検証（予測 P2）は N=416 の統制実験で予備的支持を得たが [5]、実データ（Transformer 層分析）による検証はまだ提案段階にある。(iii) 一般の統計多様体における dT ≠ 0 の条件が力に及ぼす影響の系統的分類は今後の課題である。(iv) α-動力学の操作化として、§6.7-6.8 で提案した α-SAM および Oblivion-Aware SAM の実験的検証が、方向性定理の実用的インパクトを示す最も直接的な経路である。$\text{SAM} \subset_\alpha \text{Fisher SAM} \subset_\alpha \alpha\text{-SAM} \subset_\alpha \alpha\text{-Oblivion SAM}$ の α パラメータ特殊化による従属関係は理論的に確立された (§6.7-6.8, Ay et al. 2017 Thm 2.10(2) に従う Fisher 計量一意性のもと)。実験的検証は**2 段階構造**で進める: **(Primary)** α の変化が representation 空間を $d(\Phi T)$ 方向に構造的にシフトさせるかを CKA 等で直接測定する (命題 6.7.2 の直接検証); **(Secondary)** primary が verify された場合の系として、最適 α(l) プロファイル (tanh 型遷移)、最適 λ スケーリング、および汎化性能改善を CIFAR/ImageNet 規模で検証する。Secondary の性能主張は primary の成否に条件付きである。

**第四に、圏論のゼロ曲率極限 (§9.5) [構造的類似]** として、忘却場 Φ → 0 の極限で圏論の公理 (合成の正確性、射の保存、離散的 Hom) が回収されることを示した。ユークリッド幾何がリーマン幾何の曲率ゼロの特殊ケースであるように、圏論は忘却場理論の Φ = 0 の特殊ケースとして位置づけられる。ker(G) の内部構造が image(G) の方向を決定するという因果関係は、圏論の枠内では表現不能であり、忘却場の曲率 F_{ij} によって初めて幾何学的に記述される。

力は忘却の方向的不均一性から生まれる。この原理は、物理学における力と認知科学における推論を、情報幾何学の統一的枠組みで橋渡しし、圏論を含む数学的基盤そのものを忘却場の視点から再構成する道を開く。

## Appendix B. Worked Example 2: カテゴリカルシンプレックス Δⁿ

### B.1 設定

n+1 カテゴリのカテゴリカル分布 p = (p₁,...,p_{n+1}), Σp_i = 1 のパラメータ空間 Δⁿ を自然パラメータ θ_i = log(p_i/p_{n+1}) (i = 1,...,n) で座標化する。対数正規化定数は ψ(θ) = log(1 + Σexp(θ_i))。

### B.2 Fisher 計量と Amari-Chentsov テンソル

$$g_{ij} = \frac{\partial^2\psi}{\partial\theta_i\partial\theta_j} = p_i(\delta_{ij} - p_j)$$

$$C_{ijk} = \frac{\partial g_{jk}}{\partial\theta_i}$$

ここで ∂p_m/∂θ_i = p_m(δ_{mi} - p_i)。逆計量は g^{jk} = (1/p_j)δ_{jk} + 1/p_{n+1}。

変換 x_i = 2√p_i により Δⁿ は半径 R=2 の球面 S^n の正象限と等長になり、ガウス曲率は:

$$K = \frac{1}{R^2} = \frac{1}{4} > 0$$

### B.3 Chebyshev 1-形式

指数型分布族の自然パラメータ系では C_{ijk} = ∂_ig_{jk} が恒等的に成立するため:

$$T_i = g^{jk}C_{ijk} = \partial_i\log\det g$$

det g = p₁p₂···p_{n+1} より:

$$\boxed{T_i = 1 - (n+1)p_i}$$

**ker(T) の幾何学的意味.** T_i = 0 ⟺ p_i = 1/(n+1) (均等分布)。最大エントロピー点において Chebyshev 力が消失する。

### B.4 dT = 0 の検証

T_i = ∂_i log det g は完全微分であるから:

$$(dT)_{ij} = \partial_i T_j - \partial_j T_i = \partial_i\partial_j\log\det g - \partial_j\partial_i\log\det g = 0$$

直接計算でも確認: ∂T_i/∂θ_j = -(n+1)p_i(δ_{ij}-p_j) は (i,j) 対称であり、(dT)_{ij} = ∂_iT_j - ∂_jT_i = 0。

**数値確認.** Lethe 側の再現スクリプト `categorical_simplex_dT_check.py` は n=2, 3, 4 の複数 sample 点 (均等点, 偏り点, 符号交替点) すべてで max|dT| = 0.0 (閾値 10⁻¹²) を与えた。副次確認として det g = ∏p_i の一致誤差は 10⁻¹⁷ 以下であり、T = ∂ log det g の前提式そのものが数値的に噛み合っている。

**帰結.** 方向性定理 (§5.1) により、F_{ij} = (α/2)[(∂_iΦ)T_j - (∂_jΦ)T_i]。力の生成は第1メカニズム (dΦ∧T ≠ 0) のみに依存し、Chebyshev ねじれ項は常に不活性である。この結果はガウス族 (§4) と構造的に同一であり、指数型分布族の普遍的性質を反映している。

### B.5 ガウス族との対比

| 量 | ガウス族 (ℋ², §4) | カテゴリカル (Δⁿ) |
|:---|:---|:---|
| 多様体 | ポアンカレ上半平面 | S^n_+ (半径2の正象限球面) |
| ガウス曲率 K | -1/2 | +1/4 |
| T_i | (0, 6/σ) | 1-(n+1)p_i |
| ker(T) | ∂/∂μ 方向 | 均等分布 p_i = 1/(n+1) |
| dT | = 0 | = 0 |
| F_{ij} 構造 | ∝ (∂_μΦ)(6/σ) | ∝ (∂_iΦ)T_j - (∂_jΦ)T_i |

**共通点**: 両方とも dT=0 であり、力は方向的不整合のみから発生。

**相違点**: K の符号が反転。§3.4 の VFE 解析により、この符号反転は Φ=0 近傍の変分的安定性に幾何学的脆弱性 (ker(T) 近傍でのカイネティック項の消失) を通じて影響する (Appendix C)。

## Appendix C. Φ=0 の変分的安定性

### C.1 Hessian の構造

作用 S[Φ] = ∫_M [(1/4)F_{ij}F^{ij} + (λ/2)Φ²]√g d^nθ に対し、Φ=0 まわりの二次変分を計算する。

F_{ij} は Φ の1次式 (A_i = ∂_iΦ + (α/2)ΦT_i を通じて) であるから、F_{ij}F^{ij} は Φ の2次式。従って S[Φ] は Φ=0 の周りで:

$$S[\delta\Phi] = \int_M \left[\frac{\alpha^2}{4}|T|^2_g(\delta\Phi)^2 + \text{(勾配項)} + \frac{\lambda}{2}(\delta\Phi)^2\right]\sqrt{g}\,d^n\theta + O((\delta\Phi)^3)$$

### C.2 安定性条件

Hessian の固有値は摂動の空間モードに依存する。均質摂動 δΦ = const に対して:

$$\sigma_0 = \lambda + \frac{\alpha^2}{4}\langle|T|^2_g\rangle$$

ここで ⟨·⟩ は M 上の平均。α²|T|²/4 ≥ 0 であるから:

- **λ > 0**: σ₀ > 0 が保証され、Φ=0 は安定
- **λ = 0**: σ₀ = α²⟨|T|²⟩/4 ≥ 0 であり、ker(T) 上でのみ σ₀ = 0 (限界安定)
- **λ < 0**: |λ| > α²⟨|T|²⟩/4 のとき σ₀ < 0 となり、Φ=0 は不安定

**臨界条件**: λ_c = 0。安定性の一次的決定因子は sign(λ) であり K ではない。

### C.3 幾何学的脆弱性

正曲率 K > 0 は ker(T) の近傍 (均等分布近傍) で α²|T|²_g → 0 となるため、カイネティック項による安定化が消失する。この「幾何学的脆弱性」は:

1. λ < 0 のとき、ker(T) 近傍から不安定性が成長する開始点を提供する
2. 負曲率 K < 0 では ker(T) が存在しない (T は任意の方向で非零) ため、カイネティック項が常に正の安定化寄与を持つ

§3.4 の VFE 解析により、λ < 0 はデータ密度 N が臨界値 N_c = d_eff/(L²·ḡ_eff) を超えるときに成立する。

### C.4 数値検証: Δ² 上の相転移

C.2 の安定性条件を Δ² (2-simplex) 上の ODE シミュレーションで数値的に検証する。

**方程式.** 自然パラメータ θ₁ ∈ [-4, 4] 上で:

$$\frac{\partial\Phi}{\partial t} = \alpha^2\nabla^2\Phi - \lambda\Phi - \kappa\Phi^3 + \frac{\alpha^2}{4}|T|^2_g\,\Phi$$

ここで |T|²_g = g^{ij}T_iT_j, T_i = 1-3p_i, κ = 1 (自己相互作用), α = 1。

**結果.** λ を [-2, +2] で走査し、定常状態の max|Φ|∞ を測定:

| λ 領域 | 振る舞い | 理論 (C.2) との整合 |
|:-------|:---------|:-------------------|
| λ > 0 | Φ → 0 (指数減衰) | ✓ 完全一致 |
| -0.3 < λ < 0 | Φ → 0 (|T|² 安定化) | ✓ C.3 の予測通り |
| λ ≈ -0.301 | 臨界転移 (pitchfork 分岐) | C.2 を精密化 |
| λ < -0.3 | Φ ≠ 0 (自発的対称性の破れ) | ✓ 定性一致 |

**有効臨界値 λ_c^eff.** C.2 の「臨界条件 λ_c = 0」は ker(T) の一点 (均等分布 p_i = 1/3) での局所安定性を述べている。空間的に拡がった摂動は拡散により |T|² > 0 の安定領域にも侵入するため、不安定化には拡散に打ち勝つだけの駆動力が必要である:

$$\lambda_c^{\text{eff}} = -\frac{\alpha^2}{4}\langle|T|^2_g\rangle_{\text{eff}}$$

ここで ⟨·⟩_eff は摂動モードの空間構造で重み付けした有効平均。Δ² 上では λ_c^eff ≈ -0.301 (数値). これは λ_c = 0 と矛盾しない — 均一摂動 (C.2) は δΦ = const を仮定しており、実際の摂動は ker(T) に局在化する (下記参照)。

**幾何学的脆弱性の確認.** 数値的に観測された Φ(θ) のプロファイルは ker(T) (θ₁ = 0) に明瞭に局在化しており、C.3 の「ker(T) 近傍から不安定性が成長する」という予測を直接的に支持する。

### C.5 n 次元スケーリング則: Liouville 変換

安定性閾値 λ_c^eff の n 依存性を解析的に導出する。C.4 の Schrödinger 型固有値問題:

$$-D\Delta_g\phi + V(\theta)\phi = E\phi, \qquad V(\theta) = \frac{\alpha^2}{4}|T|^2_g$$

において、変数係数 $g^{11}(\theta)$ を Liouville 変換 (測地座標変換) で除去する。

**測地座標 s.** $ds = \sqrt{g_{11}(\theta)}\,d\theta$ と定義すると、$\Delta_g\phi = d^2\phi/ds^2$ となり:

$$-D\frac{d^2\phi}{ds^2} + V(s)\,\phi = E\,\phi \qquad \text{(定数係数 Schrödinger 方程式)}$$

**命題 C.5.1 (測地長の普遍性).** 全ての $n$ に対して $L(\Delta^n) = \pi$。

*証明.* $dp_1/d\theta = p_1(1-p_1) = g_{11}$ より、$L = \int_{-\infty}^{\infty}\sqrt{g_{11}}\,d\theta = \int_0^1 dp/\sqrt{p(1-p)} = \pi$. □

**命題 C.5.2 (ポテンシャル曲率の n² スケーリング).** $V''(s_0) \to \alpha^2 n^2/2$ as $n \to \infty$.

*証明.* θ座標のポテンシャル曲率は $V''_\theta \propto n$ であり、座標変換因子 $(d\theta/ds)^2 = 1/g_{11}^* \approx n$ が追加の $n$ 因子を寄与する。□

**系 (線形スケーリング則).** 測地座標の調和振動数 $\Omega^2 = V''(s_0) \propto n^2$ に対して:

$$E_0 = \frac{1}{2}\sqrt{D \cdot \Omega^2} \propto n \qquad \Longrightarrow \qquad |\lambda_c^{\text{eff}}(n)| \propto n$$

数値的確認 ($D=0.05$, $\alpha=1$):

| n | E₀ (eigh) | 測地調和 | 比率 |
|:--|:----------|:---------|:-----|
| 2 | 0.396 | 0.158 | 2.50 |
| 10 | 1.383 | 0.791 | 1.75 |
| 50 | 6.347 | 3.953 | 1.61 |
| 100 | 12.553 | 7.906 | 1.59 |

べき乗フィット: $E_0 = 0.146 \times n^{0.967 \pm 0.012}$。非調和補正因子 ($\kappa \approx 1.59$) は $n \to \infty$ で定数に収束し、スケーリング指数に影響しない。**注:** Paper II §5.4 では測地座標 (Liouville 変換) 離散化によるより高精度な検証を行い、$\kappa_\infty \approx 1.46$, 指数 $0.993 \pm 0.003$ を得ている。θ座標法との差 (約 8%) は座標選択による系統誤差に起因する。

**物理的意味.** n 個のカテゴリの各自由度が $\sim 0.146$ ずつ安定化に寄与する。語彙 $V = 50{,}000$ のモデルでは $|\lambda_c^{\text{eff}}| \approx 7{,}300$ であり、自発的忘却は実質不可能。Paper II で証明の一般化と他の統計多様体への拡張を行う。

## Appendix D. 忘却曲率のノルム分離 (Cauchy-Binet)

§5.9.1 で用いた $\|F\| = g_{\text{eff}} \cdot \|T\|_g$ の導出を、一般の統計多様体と指数型分布族の両ケースで与える。

### D.1 一般ケース

**補題 D.1.1 (忘却曲率のノルム分離).** $(M, g, C)$ を Chebyshev 1-形式 $T_i = g^{jk}C_{ijk}$ を持つ $n$ 次元統計多様体、$\Phi$ を $M$ 上の滑らかな忘却場、$\alpha \in \mathbb{R} \setminus \{0\}$ とする。忘却曲率 $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$ のノルムは:

$$\|F\|_g = \frac{|\alpha|}{2} \|d(\Phi T)\|_{\wedge^2}$$

ここで $\|d(\Phi T)\|_{\wedge^2}^2 = \sum_{i < j} [d(\Phi T)]_{ij}^2 \cdot g^{ii}g^{jj}$ (正規直交基底)。

**証明.** $F_{ij} = (\alpha/2)[d(\Phi T)]_{ij}$ から $\|F\|_g^2 = (\alpha/2)^2 \|d(\Phi T)\|_{\wedge^2}^2$。$\alpha \neq 0$ で平方根をとると結論を得る。□

Leibniz 規則 $d(\Phi T) = d\Phi \wedge T + \Phi \cdot dT$ により:

$$\|F\|_g = \frac{|\alpha|}{2} \|d\Phi \wedge T + \Phi \cdot dT\|_{\wedge^2}$$

非指数型では $dT \neq 0$ の寄与があり、$\|T\|_g$ の乗法的分離は成立しない。

### D.2 指数型分布族ケース

**補題 D.2.1 (F=mg のノルム的成立).** $(M, g, C)$ が指数型分布族の統計多様体であり、自然パラメータ座標で $dT = 0$ が成立するとき:

$$\|F\|_g = \frac{|\alpha|}{2} \|d\Phi\|_g \cdot \|T\|_g \cdot |\sin\theta_{d\Phi, T}|$$

ここで $\theta_{d\Phi, T}$ は 1-形式 $d\Phi$ と $T$ の Fisher 計量に関する角度: $\cos\theta_{d\Phi, T} := \langle d\Phi, T \rangle_g / (\|d\Phi\|_g \|T\|_g)$。

$g_{\text{eff}} := (|\alpha|/2)\|d\Phi\|_g |\sin\theta_{d\Phi, T}|$ とおけば:

$$\boxed{\|F\|_g = g_{\text{eff}} \cdot \|T\|_g}$$

**証明.** $dT = 0$ のとき $d(\Phi T) = d\Phi \wedge T$。2-形式 $\omega = u \wedge v$ ($u, v$ は 1-形式) のノルムに Cauchy-Binet 恒等式を適用する:

$$\|u \wedge v\|_{\wedge^2}^2 = \|u\|_g^2 \|v\|_g^2 - \langle u, v \rangle_g^2 = \|u\|_g^2 \|v\|_g^2 (1 - \cos^2\theta_{u,v}) = \|u\|_g^2 \|v\|_g^2 \sin^2\theta_{u,v}$$

$u = d\Phi$, $v = T$ として:

$$\|d\Phi \wedge T\|_{\wedge^2} = \|d\Phi\|_g \cdot \|T\|_g \cdot |\sin\theta_{d\Phi, T}|$$

補題 D.1.1 と合わせて結論を得る。□

### D.3 定理 5.1 との接続

補題 D.2.1 は定理 5.1 の定量的な補完である:

| 定理 5.1 (定性的) | 補題 D.2.1 (定量的) |
|:---|:---|
| $F = 0 \iff d\Phi \wedge T = 0$ | $\|F\| = 0 \iff |\sin\theta_{d\Phi,T}| = 0 \iff d\Phi \parallel T$ |
| $F \neq 0 \iff d\Phi \wedge T \neq 0$ | $\|F\| = g_{\text{eff}} \cdot \|T\|_g$, 最大値は $d\Phi \perp T$ で達成 |

力の消滅条件 $|\sin\theta| = 0$ は、忘却の勾配 $d\Phi$ と Chebyshev 形式 $T$ が整列する ($d\Phi \parallel T$) ことと同値であり、これは系 5.1.1 の $d\Phi \wedge T = 0$ に他ならない。力が最大となるのは $d\Phi \perp T$ のとき — 忘却の方向が多様体の幾何学的構造に対して最も「斜交」する場合である。

### D.4 自己参照的構造に関する注記

ノルム射影 $\|\cdot\|: \Omega^2(M) \to C^\infty(M)$ は、2-形式から方向情報を捨てスカラーを返す写像である。忘却曲率 $F_{ij}$ は「忘却がどの方向にどれだけの力を生むか」の完全な情報を保持するが、$\|F\|_g$ は「力の大きさ」のみを保持し方向を忘れる。

$$F_{ij} \xrightarrow{\|\cdot\| \text{ (方向の忘却)}} \|F\|_g = g_{\text{eff}} \cdot \|T\|_g$$

この射影自体が忘却関手の適用であることは、$F = mg$ が忘却理論の**自己適用** (self-application) であることを意味する: 忘却の理論に忘却を適用した結果がニュートン的スカラー方程式を生む。Einstein 方程式 $G_{\mu\nu} = 8\pi G T_{\mu\nu}$ とニュートン方程式 $F = mg$ の関係が「弱場極限」(物理的近似) であるのに対し、$F_{ij}$ と $\|F\|_g = g_{\text{eff}} \cdot \|T\|_g$ の関係は「ノルム射影」(代数的忘却) であり、近似を含まない正確な写像である。

## Appendix E. Estimator 化の詳細 — 仮定 H1-H1''' と命題

§5.5 の $\Xi_{\text{Var}}, \Xi_{\text{Gini}}, \Xi_{\text{CV}}, \Xi_{\text{impl}}$ は、raw observable をそのまま理論量と同一視するための記号ではない。本 Appendix では、それぞれがどの観測モデルの下で $\Xi_{\text{theory}}(\nu)$ の estimator もしくは lower-bound estimator と読めるかを明示する。

### E.1 共通設定

局所忘却曲率密度は

$$
\kappa(\theta) := \|d(\Phi T)\|_g^2 = \frac{4}{\alpha^2}\|F\|_g^2
$$

であり、観測設計 $\nu$ に対する理論的 target は

$$
\Xi_{\text{theory}}(\nu) := \operatorname{Var}_{u \sim \nu}[\kappa(\theta_u)]
$$

である。ここで observation unit $u$ は実験ごとに異なり、turn, trajectory, layer, coordinate などをとる。したがって「同じ $\Xi$ を測っている」とは、同じ latent law を見ているという意味ではなく、同じ curvature density $\kappa$ を**異なる observation design で集約した量**を見ているという意味である。

本 Appendix では、観測される raw quantity を $Z_u$、その背後の latent curvature density を $\kappa_u := \kappa(\theta_u)$ と書く。raw observable から latent target へ戻すには、(i) 観測モデル、(ii) noise baseline、(iii) basis alignment の 3 点が必要である。特に basis alignment とは、観測に用いる coordinate system が共分散行列 $\Sigma$ の principal directions と一致している条件であり、これが破れると recover されるのは full target ではなく lower-bound quantity になる (§5.6)。

### E.2 $\widehat{\Xi}_{\text{Var}}$ — N=416 統制実験

§5.5 の N=416 統制実験では observation unit は turn $u=t$ であり、turn ごとに局所パッチ $\theta_t$ を考える。

**仮定 H1.**
1. 各 turn $t$ には curvature density $\kappa_t := \kappa(\theta_t)$ が対応する。
2. turn-level score $Y_t$ は線形観測モデル

$$
Y_t = \beta_0 + \beta_1 \kappa_t + \varepsilon_t,
\qquad
\mathbb{E}[\varepsilon_t \mid \kappa_t] = 0,
\qquad
\operatorname{Var}(\varepsilon_t)=\sigma_\varepsilon^2 < \infty
$$

を満たす。
3. 比較する policy 間で compression budget は固定され、$\beta_1$ は共通である。
4. $(\kappa_t,\varepsilon_t)$ は i.i.d. にサンプルされ、四次モーメントが有限である。

このとき de-noised variance

$$
\widehat{\Xi}_{\text{Var}}
:=
\frac{1}{\beta_1^2}
\left[
\frac{1}{n-1}\sum_{t=1}^n (Y_t-\bar Y)^2 - \widehat{\sigma_\varepsilon^2}
\right]
$$

を $\Xi_{\text{theory}}(\nu_{\text{turn}})$ の estimator と定義できる。

**命題 V.** H1 のもとで $\widehat{\Xi}_{\text{Var}}$ は unbiased であり、$n \to \infty$ で consistent である。plug-in な noise 推定 $\widehat{\sigma_\varepsilon^2}$ を用いる場合も bias は $O(n^{-1})$、variance は $O(n^{-1})$ である。

*証明スケッチ.* 線形観測モデルより

$$
\operatorname{Var}(Y_t) = \beta_1^2 \operatorname{Var}(\kappa_t) + \sigma_\varepsilon^2
= \beta_1^2 \Xi_{\text{theory}}(\nu_{\text{turn}}) + \sigma_\varepsilon^2.
$$

したがって sample variance から noise baseline を差し引き、$\beta_1^2$ で割れば target に戻る。sample variance の標準漸近論により bias/variance はともに $O(n^{-1})$ であり、consistency が従う。□

### E.3 $\widehat{\Xi}_{\text{Gini}}$ — SWE-bench

SWE-bench 系では observation unit を trajectory-level positive score $W_u$ とし、multiplicative observation model を採る。

**仮定 H1'.**
1. 観測 unit $u$ の正値スコア $W_u$ は

$$
\log W_u = \mu + \beta \kappa_u + \varepsilon_u,
\qquad
\varepsilon_u \sim \mathcal{N}(0,\sigma_\varepsilon^2)
$$

に従う。
2. 観測 basis は $\Sigma$ の principal directions と一致する。あるいは PCA whitening 後に Gini を計算する。
3. $\kappa_u$ の分散が有限で、$n \to \infty$ の極限を考える。

このとき $W_u$ は lognormal family に入り、

$$
\tau_{\log}^2 := \beta^2 \Xi_{\text{theory}}(\nu_{\text{traj}}) + \sigma_\varepsilon^2
$$

とおけば population Gini は

$$
G_\infty = 2\Phi_{\mathcal N}\!\left(\frac{\tau_{\log}}{\sqrt{2}}\right)-1
$$

となる。したがって calibrated estimator を

$$
\widehat{\Xi}_{\text{Gini}}
:=
\frac{1}{\beta^2}
\left(
2\left[\Phi_{\mathcal N}^{-1}\!\left(\frac{\widehat G_n+1}{2}\right)\right]^2
- \widehat{\sigma_\varepsilon^2}
\right)
$$

と定義できる。ここで $\Phi_{\mathcal N}$ は標準正規分布の累積分布関数である。

**命題 G.** H1' のもとで $\widehat{\Xi}_{\text{Gini}}$ は asymptotically unbiased かつ consistent であり、bias は $O(n^{-1})$、variance は $O(n^{-1})$ である。

*証明スケッチ.* sample Gini $\widehat G_n$ は U-statistic 型であり、$G_\infty$ の estimator として bias/variance ともに $O(n^{-1})$ を持つ。逆写像 $G \mapsto 2[\Phi_{\mathcal N}^{-1}((G+1)/2)]^2$ は $(0,1)$ 上で滑らかなので、delta method により $\widehat{\Xi}_{\text{Gini}}$ も同じ次数で収束する。□

**但し書き.** §5.6 の Schur-Horn 条件を満たさないとき、この estimator が回収するのは full target ではなく projected spectral quantity $\Xi_{\text{spec,proj}}$ の lower bound である。すなわち $\widehat{\Xi}_{\text{Gini}}$ は basis alignment が成立する regime でのみ full estimator となる。

### E.4 $\widehat{\Xi}_{\text{CV}}$ — 全保持対照

全保持対照では H1' と同じ lognormal family を使いつつ、CV を zero-curvature limit の diagnostic から calibrated estimator に引き戻す。

**仮定 H1''.**
1. $W_u$ は H1' と同じ lognormal observation model に従う。
2. mean budget $m := \mathbb{E}[W_u]$ は policy 間で固定される。
3. noise baseline $\sigma_\varepsilon^2$ は full-retention control から独立に推定可能である。

lognormal family では

$$
\operatorname{CV}^2 = e^{\tau_{\log}^2} - 1,
\qquad
\tau_{\log}^2 = \beta^2 \Xi_{\text{theory}}(\nu_{\text{traj}}) + \sigma_\varepsilon^2.
$$

よって calibrated estimator は

$$
\widehat{\Xi}_{\text{CV}}
:=
\frac{1}{\beta^2}
\left(
\log(1+\widehat{\operatorname{CV}}^2) - \widehat{\sigma_\varepsilon^2}
\right).
$$

**命題 C.** H1'' のもとで $\widehat{\Xi}_{\text{CV}}$ は asymptotically unbiased かつ consistent である。特に no-forgetting control では $\kappa_u \approx 0$ が全 unit で成立するので $\Xi_{\text{theory}}(\nu_{\text{traj}}) \approx 0$、したがって $\widehat{\Xi}_{\text{CV}} \approx 0$ が予測される。

*証明スケッチ.* $\widehat{\operatorname{CV}}^2$ は plug-in estimator なので有限標本 bias を持つが、CV の漸近展開と delta method により bias/variance は $O(n^{-1})$ に抑えられる。zero-curvature limit では $\tau_{\log}^2 \to \sigma_\varepsilon^2$ であり、noise baseline を引けば target はゼロへ収束する。□

### E.5 $\widehat{\Xi}_{\text{impl}}$ — N=229 CCL 式

CCL 式の測定では observation unit は expression $j=1,\dots,N$、観測座標は $k \in \{1,\dots,6\}$ である。

**仮定 H1'''.**
1. omission indicator $M_{jk} \in \{0,1\}$ は

$$
M_{jk} \sim \operatorname{Bernoulli}(p_k),
\qquad
\operatorname{logit}(p_k) = a + b\kappa_k
$$

に従う。ここで $\kappa_k := \mathbb{E}[\kappa_u \mid \text{coordinate}=k]$ は coordinate-level curvature energy である。
2. 6 座標 basis が principal directions と一致する場合に限り equality が成立し、一致しない場合は §5.6 の lower-bound regime に落ちる。
3. small-spread regime で $p_0 := (1/6)\sum_k p_k$ の近傍に線形化できる。

各座標の omission rate estimator は

$$
\widehat p_k = \frac{1}{N}\sum_{j=1}^N M_{jk}
$$

であり、small-spread regime では

$$
p_k
=
p_0 + b\,p_0(1-p_0)(\kappa_k-\bar\kappa) + O((\kappa_k-\bar\kappa)^2)
$$

となる。したがって Gini は first order で $\sqrt{\operatorname{Var}_k(\kappa_k)}$ に比例するので、

$$
\widehat{\Xi}_{\text{impl}}
:=
c_{\text{impl}}(p_0)^{-2}\,
\operatorname{Gini}(\widehat p_1,\dots,\widehat p_6)^2
$$

を coordinate-level target

$$
\Xi_{\text{theory}}(\nu_{\text{coord}})
:=
\operatorname{Var}_{k=1,\dots,6}[\kappa_k]
$$

の estimator と定義できる。

本文 §5.5 で報告した

$$
\Xi_{\text{coord,theory}} := \operatorname{Gini}(\bar p_1,\dots,\bar p_6)
$$

は、36 動詞を均等使用したときの coordinate-level benchmark を raw Gini scale で書いたものであり、上の calibrated target とは representation scale が異なる。両者を区別することで、raw observable・calibrated estimator・benchmark の三層が分離される。

**命題 I.** H1''' のもとで $\widehat{\Xi}_{\text{impl}}$ は coordinate-level heterogeneity $\Xi_{\text{theory}}(\nu_{\text{coord}})$ の consistent estimator である。basis alignment が壊れている場合は lower-bound estimator に降格する。

*証明スケッチ.* $\widehat p_k$ は unbiased で $\operatorname{Var}(\widehat p_k)=p_k(1-p_k)/N = O(N^{-1})$。Gini は滑らかな対称関数なので multivariate delta method により $\widehat{\Xi}_{\text{impl}}$ の bias/variance は $O(N^{-1})$。basis alignment が成り立つときは coordinate basis が principal directions を表すため full target を recover し、そうでなければ §5.6 の Schur-Horn 不等式により lower bound のみが見える。□

### E.6 推定量間の関係

4 つの estimator はすべて同じ latent curvature density $\kappa$ から出発するが、observation unit と観測モデルが異なるため、target は observation design $\nu$ を通して現れる。

| estimator | observation unit $u$ | raw observable | latent target | full / lower-bound の境界 |
|:---|:---|:---|:---|:---|
| $\widehat{\Xi}_{\text{Var}}$ | turn | de-noised sample variance | $\Xi_{\text{theory}}(\nu_{\text{turn}})$ | noise baseline が既知なら full |
| $\widehat{\Xi}_{\text{Gini}}$ | trajectory | sample Gini of positive score | $\Xi_{\text{theory}}(\nu_{\text{traj}})$ | basis alignment があれば full, なければ lower bound |
| $\widehat{\Xi}_{\text{CV}}$ | trajectory | calibrated CV | $\Xi_{\text{theory}}(\nu_{\text{traj}})$ | noise baseline が既知なら full |
| $\widehat{\Xi}_{\text{impl}}$ | coordinate | Gini of omission rates | $\Xi_{\text{theory}}(\nu_{\text{coord}})$ | basis alignment があれば full, なければ lower bound |

したがって「どの $\Xi$ が正しいか」という問いよりも、「どの observation design の下で何を推定しているか」を明示する方が重要である。本文で導入した $\Xi_{\text{coord,theory}}$ と $\Xi_{\text{spec,proj}}$ は、この設計依存性を可視化するための補助ラベルである。前者は equal-usage benchmark、後者は spectral projection であり、いずれも fundamental quantity $\Xi_{\text{theory}}(\nu)$ そのものではなく、その可観測化の仕方を示している。

## Appendix F. 非指数型反例 — Student t $(\nu, \gamma)$ 族

§5.4 は指数型分布族において $dT = 0$ が普遍的に成立することを主張するが、この主張の**境界**を示すには非指数型の具体例が必要である。本 Appendix は Student t 位置スケール族を用いてこの境界を数値的に実証する。

### F.1 設定

位置を 0 に固定した Student t 分布:

$$p(x; \nu, \gamma) \propto \gamma^{-1}\left(1 + \frac{x^2}{\nu \gamma^2}\right)^{-(\nu+1)/2}$$

を考える。パラメータは自由度 $\nu > 0$ とスケール $\gamma > 0$ の 2 次元。この族は指数型分布族ではないため、Appendix B で用いた簡約 $T_i = \partial_i \log \det g$ は**適用できない** — 一般定義から直接構成する必要がある。

### F.2 計算方法

score ベクトル $s = (\partial_\nu \log p, \partial_\gamma \log p)$ を閉形式で書き、Fisher 計量と Amari-Chentsov 3-tensor を数値積分で直接構成する:

$$g_{ab} := \mathbb{E}[s_a s_b], \qquad C_{abc} := \mathbb{E}[s_a s_b s_c]$$

そこから Chebyshev 1-形式 $T_a = g^{bc} C_{abc}$ を構成し、外微分 $dT = \partial_\nu T_\gamma - \partial_\gamma T_\nu$ を中心差分で評価する。実装は `non_exponential_dT_counterexample.py` (scipy.integrate.quad による積分)。

### F.3 結果

| $(\nu, \gamma)$ | $T = (T_\nu, T_\gamma)$ | $dT$ | $|dT|/|T|$ |
|:---|:---|---:|---:|
| $(3.0, 1.0)$ | $(-0.9639, +3.0902)$ | $9.41 \times 10^{-1}$ | $2.91 \times 10^{-1}$ |
| $(5.0, 2.0)$ | $(-0.5190, +2.3303)$ | $3.27 \times 10^{-1}$ | $1.37 \times 10^{-1}$ |
| $(10.0, 0.5)$ | $(-0.1925, +13.8788)$ | $6.33 \times 10^{-1}$ | $4.56 \times 10^{-2}$ |

全サンプル点で $|dT| \gg 10^{-3}$ かつ $|dT|/|T| \gg 10^{-3}$ を満たし、数値誤差帯を大きく上回る。

### F.4 含意

Student t $(\nu, \gamma)$ 族は、$dT = 0$ が**指数型 class に特有**の構造であり非指数型では一般に破れることの具体例である。Chebyshev ねじれ項 $\Phi \cdot dT$ は非指数型では方向的不整合項 $d\Phi \wedge T$ と並んで力の生成に寄与する (§5.4)。

### F.5 Cauchy 位置スケール族の鈍さ

初期候補であった Cauchy 位置スケール族 $p(x; x_0, \gamma) \propto (1 + (x-x_0)^2/\gamma^2)^{-1}$ は、数値 pilot で $T \approx 0$ および $dT \approx 0$ を返した。Cauchy は Student t の $\nu = 1$ 極限であるが、位置・スケール 2 パラメータ化の対称性により Chebyshev 感受性がほぼ消えるためと推測される。これは「非指数型なら常に $dT \neq 0$」という直感的一般命題が**成立しないこと**を示唆し、$dT = 0$ の保持 / 破れの**体系的分類**が残された問題として提起される (**Open Problem OP-I-7**)。
