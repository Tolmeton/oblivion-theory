## 編集規律

1. **編集はすべて `論文I_力としての忘却_publication_draft.md` 上で行う**。`../論文I_力としての忘却_草稿.md` は read-only ソースとして維持
2. **meta は本 workspace 同梱版を更新** (Kalon 判定履歴・±3σ ゲート・Gauntlet ログ・§M6 充填)
3. **大規模変更ごとに git commit** で復元面を残す
4. **Yugaku 5 規律の遵守**: notebook-sourcing / kalon-check / provocation-gauntlet / sigma-heuristic / style-discipline
5. **HGK 内部用語 (Handoff/CCL/Daimonion/F⊣G/Kalon/Gauntlet 等) を本文に書かない** — meta のみで使用可
6. **査読版完成時に `_publication_v0.1.md` 等として版を切る**
