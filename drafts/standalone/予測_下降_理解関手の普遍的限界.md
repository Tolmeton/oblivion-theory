## §1 結論先行 (7 段開口部 — Understatement + Axiom-First 合成)

### §1.1 課題 (Understatement)

物理学者は加速器の検出器読み出しから粒子の存在を推論する。直接観測しているのは画素強度の時系列であり、粒子そのものは観測していない。Bogen と Woodward (1988) はこの段階構造を **data → phenomena → theory** として整理し、phenomena は data を介して間接的に検出されるが、ほとんどの場合は直接観測可能ではないと述べた [SOURCE: Bogen-Woodward 1988 §I, transitive 強候補, G-ζ 独立検証義務]。

> "Phenomena are detected through the use of data, but in most cases are not observable in any interesting sense of that term." (Bogen-Woodward 1988)

本稿はこの段階構造を **関手的下降** として読む。すなわち、

```
理論層  ─(下降)→  真理₁ (= phenomena)  ─(下降)→  真理₀ (= data)
```

の各矢は **下降関手** である。理論層から phenomena への矢は、理論が記述する現象の選び取り (具体化) であり、phenomena から data への矢は、現象が観測可能な記録に痕跡として残ること (測定) である。

提示するのはこの読み方であって、新しい現象ではない。だが、この読み方を採った瞬間、いくつかの認識論的限界定理が不可避に従う。

### §1.2 公理 (Axiom-First) — F⊣G

本稿は次の随伴対を **構成的定義** として置く:

- **L** (左随伴, 発散側) = 理解の生成。対象 $c$ に対し、$c$ を別のもっと簡単な対象 $R(L(c))$ に対応づける関手
- **R** (右随伴, 収束側) = 理解の評価。その対応が両方向に矛盾なく成立するかを検査する関手
- **η_unit: Id ⟹ R∘L** (随伴単位) = $c$ が $R(L(c))$ に「内側に持たれている」度合い

**理解の操作的定義**: 「$c$ を理解している」⟺ η_unit(c) が同型 (i.e., $R(L(c)) \cong c$)。

[構成的命題, 推定 70%]

aletheia §1 L99-L107 の **U⊣N と本稿 L⊣R は同型対応** (L ↔ U 左随伴 / R ↔ N 右随伴)。aletheia の VFE 減少定理 $F[N(q_{\text{poor}})] \leq F[q_{\text{poor}}]$ は本稿 η_unit の収束性に対応する [SOURCE: aletheia.md §1 L99-L107]。

存在条件は Mac Lane CWM §V.6 GAFT (Solution Set Condition) または §V.8 SAFT (cogenerator) を満たす圏で well-defined [SOURCE: Mac Lane CWM, Buzzard 2012 triangulation, transitive 中候補, G-ζ]。これらの条件を満たさない $L_i$ は射程外と宣言する (§M5.1 Round 1 r1 対応)。

### §1.3 結論 1 (C1) — 理解 = L⊣R 内在化

> 科学における「理解」とは、随伴対 L⊣R の内在化として定義される **関手的操作** である。
>
> [構成的命題, 推定 70%]

| 領域 | $L$ (発散) | $R$ (収束) |

## §2 随伴構造導入 (F の最初の展開)

§1.2 で公理として置いた随伴対 L⊣R を、Bogen-Woodward 三層と aletheia U⊣N に橋渡しし、視覚の逆問題のメタファーで読者の直観を起動する。本節の役割は構造の証明ではなく、構造が **どこに現れるか** を 3 つの異なる場所で並列的に示すことである。

### §2.1 Bogen-Woodward 三層の関手的読み替え

§1.1 で導入した三層

```
理論層  ─(下降)→  真理₁ (= phenomena)  ─(下降)→  真理₀ (= data)
```

を関手的に書き直す。下降関手を 2 段に分け、

- $L_{\text{th→ph}}$: 理論から phenomena を取り出す関手 (具体化)
- $L_{\text{ph→da}}$: phenomena から data を取り出す関手 (測定)

と置く。両者の合成 $L_{\text{th→da}} = L_{\text{ph→da}} \circ L_{\text{th→ph}}$ が、Mangalam 予測至上主義 (§7.2) や benchmark culture が暗黙裏に仮定する「理論 ↔ データ」の単一線形対応である。Bogen-Woodward が示すのは、この合成が **2 つの関手の合成として分解されるべき** であって、単一線形対応として読むべきではないということだ:

> "Data, which play the role of evidence for the existence of phenomena, for the most part can be straightforwardly observed. However, data typically cannot be predicted or systematically explained by theory. By contrast, well-developed scientific theories do predict and explain facts about phenomena. Phenomena are detected through the use of data, but in most cases are not observable in any interesting sense of that term." (Bogen-Woodward 1988 §I)

[SOURCE: bogen.txt L101-130 verbatim, transitive 強候補, G-ζ 独立検証義務]

例として彼らが挙げるのは weak neutral currents, proton decay, chunking and recency effects である (L122-130) — いずれも data から間接的に検出されるが直接 observable ではない。

各下降関手には対応する右随伴が存在する:

- $R_{\text{ph→th}}$: phenomena から理論を構成する (帰納的構成)
- $R_{\text{da→ph}}$: data から phenomena を抽出する (測定理論)

各層で $L_i \dashv R_i$ が独立に成立する。これが本稿の **多層随伴構造** である。

### §2.2 aletheia U⊣N との同型対応

aletheia.md §1 (L99-L107) は、視覚-推論二相における随伴定理 U0' を次のように定式化している:

> 「U⊣N: U が左随伴 (忘却関手) / N が右随伴 (回復関手) / VFE 減少定理 $F[N(q_{\text{poor}})] \leq F[q_{\text{poor}}]$ を満たす」

[SOURCE: aletheia.md §1 L99-L107]

本稿の L⊣R はこれと同型対応する:

| aletheia | 本稿 | 役割 |
|:---|:---|:---|
| $U$ (左随伴, 忘却) | $L$ (左随伴, 発散) | 単純化への射 |
| $N$ (右随伴, 回復) | $R$ (右随伴, 収束) | 整合性検査 |
| $\eta: \text{Id} \Rightarrow N \circ U$ | $\eta_{\text{unit}}: \text{Id} \Rightarrow R \circ L$ | 内在化度合い |
| $F[N(q_{\text{poor}})] \leq F[q_{\text{poor}}]$ | $\eta_{\text{unit}}$ の収束性 | VFE 減少 |

「忘却」と「発散」の対応に注意が必要である。aletheia の $U$ は精度を落とす方向の関手 (例えば α-忘却で予測を粗くする) であり、本稿の $L$ は対象を別のもっと簡単な対象に対応づける関手である。両者は **情報を捨てる方向** で同型である。

aletheia §2.1 (L126-141) は $U$ の 8 段パターン生成テーブルを与え、§2.3 (L154-195) は Bohr 太陽系モデルの worked example を提示している。後者は §3.2 (ゲージ理論) と §3.3 (統計力学) でも参照する。

### §2.3 GAFT / SAFT による L⊣R 存在条件

L⊣R の well-defined 性は無条件には成立しない。Mac Lane CWM §V.6 GAFT (General Adjoint Functor Theorem, p.117) は次の条件で存在を保証する:

> $A$ が small-complete かつ small hom-sets を持つとき、関手 $G: A \to X$ が左随伴を持つ ⟺ $G$ が連続 (small limit 保存) かつ Solution Set Condition (SSC) を満たす。SSC: $\forall x \in X$ に対し small set $I$ と $\{a_i \in A\}_{i \in I}$ および $\{f_i: x \to G(a_i)\}$ が存在し、任意の arrow $x \to G(a)$ が $h = G(t) \circ f_i$ ($t: a_i \to a$) の合成として書ける。

[SOURCE: Mac Lane CWM §V.6 p.117, Buzzard 2012 lecture notes p.2 Theorem 1.1 verbatim 引用, transitive 中候補, G-ζ 独立検証義務]

§V.8 SAFT (Special Adjoint Functor Theorem, p.125-126) はより強い前提を要求する:

> $C$ が small-complete, locally small, well-powered, small cogenerating set を持ち、$D$ が locally small なら、極限保存関手 $G: C \to D$ は左随伴を持つ。

[SOURCE: Mac Lane CWM §V.8 p.125, Buzzard 2012 L185-194 + nLab 公式, transitive 中候補, G-ζ]

本稿は **GAFT (SSC) を主条件**、SAFT (cogenerator) を補助条件として置く。GAFT を満たさない $L_i$ は射程外と宣言する (§M5.1 Round 1 r1 対応)。これは「全関手で L⊣R が成立する」という overclaim を予防する措置である (Z-03 圏論 overclaim 反論への防御)。

### §2.4 視覚の逆問題メタファー

読者の直観を起動するため、視覚の逆問題で随伴構造を例示する。

網膜に届く光強度のパターンは data (真理₀) である。だが網膜パターンから「何を見ているか」を直接読むことはできない。我々は無意識のうちに 2 段階の関手を経て概念に到達している:

1. $L_{\text{retina→percept}}$: 網膜パターン → 知覚オブジェクト (例: 「猫がいる」)
2. $L_{\text{percept→concept}}$: 知覚オブジェクト → 概念ラベル (例: 「Schrödinger の猫」)

それぞれに右随伴が存在する:

1. $R_{\text{percept→retina}}$: 概念から予測される網膜パターン (image prediction; 内部生成モデル)
2. $R_{\text{concept→percept}}$: 概念から予測される知覚パターン

理解とは、各層で $\eta_{\text{unit}}: \text{Id} \Rightarrow R \circ L$ が **構造** に対しては iso に近づくが、**値** (例えば瞬間的な網膜画素強度) に対しては iso にならない、という構造的不等式の中に住んでいることである。

これが §1.4 C2 の構造保存定理の直観的な絵である。網膜画素強度の完全予測 (η_unit = iso for all values) は不可能であり、また不要である。我々が予測しているのは **構造としての世界** であって、画素値ではない。

ここで重要なのは、この絵が脳科学・認知科学の経験的事実というより、**Bogen-Woodward 段階構造の必然的帰結** だということである。data (網膜) から phenomena (知覚) を経て theory (概念) に至る経路は、関手として読めば 2 段の合成であり、単一線形対応ではない。

## §7 corollary (3 誤配位独立展開)

### §7.1 Popper 反証可能性

(骨格) ポパーは「真理₁ (phenomena) で反証可能性を要求」する。だが真理₀ (data) との下降関係を見ない。反証可能性は phenomena 層で測ろうとするが、phenomena は data からしか検出できないという Bogen-Woodward の不可避な間接性を無視する。

→ **C4 の特殊例**: 真理₁ への下降関手の痕跡 (= 予測₁) を真理₁ 自体と取り違える。

[SOURCE: Popper 1934/1959 + 反証可能性は死んだ.md §2-§3, fep_epistemic_status §ポパー適用不能 v2.7.0]

### §7.2 Mangalam 予測至上主義

(骨格) 「予測精度こそ良い理論の指標」とする立場 (benchmark culture)。予測₁ (data 痕跡) を真理₀ の指標と読み、phenomena 層の存在を見落とす。

→ **C4 同型**: 関手の方向を逆転している (data から theory への遡上を、theory の data 上での適合度として誤測定)。

[SOURCE: 反証可能性.md §9 + Mangalam 予測至上主義の批判文献]

### §7.3 超ひも landscape

(骨格) 超ひも理論の $10^{500}$ 真空が「予測を消失させる」批判 (Hossenfelder 2018)。理論層の冗長性で予測₁ が一意に定まらないことを「悪い理論」と判定するが、これは真理₀/真理₁ の下降を **線形視** している。

→ **C4 異型**: 理論層から data 層への単一線形写像を期待する誤配位。

[SOURCE: Hossenfelder 2018 + 反証可能性.md §8 §8.6, transitive 中候補]

## §8 結語 — Predictions Descend 単一図式

3 誤配位は Bogen-Woodward 三層を共通構造 anchor として **単一図式の派生形** として閉じる:

```
       理論層 (laws / models)
        │
        ▼ (下降関手 1: 具体化)
  真理₁ = phenomena
        │
        ▼ (下降関手 2: 測定)
  真理₀ = data
   ↑    ↑    ↑
   │    │    └─ Mangalam 予測至上主義: 予測₁ (data 痕跡) を真理₀ の指標と誤読
   │    └─── Popper 反証可能性: 真理₁ で反証可能性要求、真理₀ 下降を見ない
   └────── landscape: 理論層冗長で予測消失を「悪い」と判定 (下降を線形視)
```

3 誤配位は「下降関手 (predictions descend) の方向逆転誤読」の単一形態の派生形である。
