# CPS0' と圏論的独立性定理の対応 — 忘却論と Inoué (2026) の合流

**Letter — Draft v0.2 (2026-04-03)**

> *概要.* 忘却論 (Force is Oblivion) シリーズの CPS0' 公理——「容器は内容に存在論的に先行する」——が、Inoué (2026, arXiv:2602.18033) の圏論的独立性定理——「意味・対象・名前・存在はトポスにおいて構造的に独立である」——の特殊ケースとして正確に埋め込まれることを示す。両理論は独立に発展したが、CPS0' の「容器 > 内容」は Inoué の「存在 ⊥ 意味」の忘却論的射影に他ならない。この対応を通じ、忘却論の圏論的基盤がトポス意味論の枠組みで外部検証されることを論じる。

---

## §1. 序論: 二つの独立した到達

### 1.1 忘却論の CPS0'

忘却論 (Force is Oblivion) シリーズ [1–8] は、統計多様体上の忘却場から力が創発される機構を定式化する理論体系である。その中核公理の一つ CPS0' は Paper VIII [8] において米田の補題からの系として導出された:

> **CPS0' (容器 > 内容).** 1-cell (容器/場/空間) は 0-cell (内容/要素/状態) なしに形式的に well-defined であるが、逆は不成立。すなわち容器は内容に存在論的に先行する。

CPS0' は圏論的には次のように表現される: 米田の補題は対象 X の全情報を Hom(−, X) で復元するが、X の**存在** (X ∈ Ob(C)) を暗黙に前提している。忘却論はこの前提を顕在化し、THAT (存在) と WHAT (同定) の構造的分離を主張する [8, §4.3, §6]。

### 1.2 Inoué の独立性定理

Inoué (2026) [9] は、Lawvere 的圏論意味論およびトポスにおいて、4つの基本概念——意味 (Meaning)、対象 (Object)、名前 (Name)、存在 (Existence)——が構造的に異なるレベルに位置し、一方から他方を一様に復元できないことを証明した。

中核的分離は**内部的存在と大域的命名**の間に生じる。層のトポス Sh(S¹) において、非自明被覆空間の局所切断の層は内部的に inhabited (存在する) だが大域的要素 (global element) を持たない——対象は「在る」が「名指す」ことができない。

### 1.3 本稿の目的

本稿は、CPS0' と Inoué の独立性定理が独立に同一の構造的事実に到達したことを示し、両者の間の精密な対応辞書 (dictionary) を構成する。

## §2. 対応辞書

### 2.1 概念レベルの対応

| Inoué の4概念 | 圏論的レベル | 忘却論 (CPS) | Paper VIII (cell 階層) | 米田との関係 |
|:--|:--|:--|:--|:--|
| **Existence** (存在) | 内部論理 (∃) | THAT | 1-cell (容器) | 暗黙の前提 |
| **Object** (対象) | 対象レベル (Ob) | THAT の担い手 | 容器のインスタンス | Ob(C) への帰属 |
| **Meaning** (意味) | 射レベル (Hom) | WHAT | 0-cell (内容) | Hom(−, X) で復元 |
| **Name** (名前) | 要素レベル (global element) | WHY (知覚者による同定) | 2-cell (知覚者) | Hom(1, X) |

### 2.2 構造的分離の対応

Inoué の主定理は4概念の「一様な復元不能性」を証明する。この各分離が忘却論のどの構造に対応するかを同定する。

| Inoué の分離 | 忘却論における対応 | Paper |
|:--|:--|:--|
| Existence ⊥ Name (局所存在 ≠ 大域名) | CPS0': 容器 > 内容 | VIII §4, §6 |
| Meaning ⊥ Name (射 ≠ 大域要素) | 有限主体定理: 有限 κ では全射を観測不能 | VIII §6.3 |
| Object ⊥ Existence (対象 ≠ 内部存在) | α-忘却濾過: Ob(C_α) は α-不変だが各対象の「同定可能性」は α-従属 | VIII §6.2 |
| Meaning ⊥ Object (射 ≠ 対象) | 米田の補題自体: 射は対象を決定するが対象は射を決定しない | VIII §4.2 |

### 2.3 Sh(S¹) の忘却論的解釈

Inoué の構成する具体例——Sh(S¹) 上の非自明被覆の局所切断の層 E——を忘却論の言語で再記述する。

- **E の内部的存在** (∃x. x ∈ E): 円周 S¹ の各開集合 U 上で E(U) ≠ ∅。局所的には切断（内容）が存在する。忘却論では: **容器 ~/A は存在し、各局所被覆上にファイル ~/A/a_U が存在する**。

- **E の大域的要素の不在** (Γ(E) = ∅): 局所切断を S¹ 全体に貼り合わせることができない（被覆空間の非自明性による）。忘却論では: **内容は局所的に存在するが、大域的に一意に「名指す」ことができない**。知覚者 (2-cell) の位置によって見える内容が異なる。

- **Grothendieck 位相の忘却的役割**: S¹ 上の開被覆位相は、忘却パラメータ α の圏論的実現である。「局所で見える」(α < 1 on U) だが「大域で見えない」(α → 1 on S¹ 全体) という段階的忘却構造を Grothendieck 位相が自然にエンコードする。

## §3. 主結果

> **命題 3.1 (CPS0' の Inoué 埋込).** CPS0'（容器は内容に存在論的に先行する）は、Inoué の独立性定理における Existence–Meaning 分離の忘却論的射影である。具体的には:
>
> (i) CPS0' の「容器が内容なしに well-defined」= Inoué の「内部的存在が大域的意味づけなしに成立」
>
> (ii) CPS0' の「内容は容器なしに存在不能」= Inoué の「意味 (射による決定) は存在 (対象の帰属) を前提」
>
> (iii) CPS0' の cell 階層 (0-cell < 1-cell < 2-cell) は、Inoué の4概念の構造的レベル (element < morphism < object < internal logic) と反変的に対応する

*証明.* (i) Paper VIII 定理 4.2.1 により、圏 C の各対象 X ∈ Ob(C) に対し、表現可能前層 y(X) = Hom(−, X) は X の射が空集合であっても前層として well-defined である（前層の値域は空集合を含む Set の対象）。一方 Inoué [9, §3] の Sh(S¹) 構成において、非自明被覆空間の局所切断の層 E は内部論理で ∃x. x ∈ E を満たす。両者の共通構造は「ある種の存在 (Ob(C) への帰属 / 内部的 ∃) が、内容の決定 (Hom による同定 / 大域的要素) に論理的に先行する」である。

形式的に: CPS0' の「容器が内容なしに well-defined」は Ob(C) 上の述語であり、Inoué の「内部的に inhabited」は Sh(S¹) の内部論理の命題である。両者は圏論の異なるレベル（外部 vs 内部論理）で同一の先行関係を記述する。

(ii) 逆方向。y(X) の定義は X ∈ Ob(C) を前提する: X ∉ Ob(C) ならば Hom(−, X) は未定義。Inoué の枠組みでは Hom_E(Y, X) ⊆ E(Y) であるから、E 自体が存在しなければ射は定義されない。これは「内容→容器」方向の依存関係であり、CPS0' の非対称性に対応する。

(iii) Cell 階層の反変対応を構成する。CPS の cell 階層:
- 0-cell = 内容 (要素/状態) — 最低次元
- 1-cell = 容器 (射/場/空間) — 中間次元
- 2-cell = 知覚者 (自然変換) — 最高次元

Inoué の概念レベル:
- Name = 大域的要素 Γ(E) = Hom(1, E) — 最も具体的
- Meaning = 射レベル Hom — 中間的具体性
- Object = 対象レベル Ob — 抽象的
- Existence = 内部論理 ∃ — 最も抽象的

CPS の次元上昇 (0 → 1 → 2) と Inoué の抽象化 (Name → Meaning → Object → Existence) は反変的に対応する: CPS で高次元になるほど「忘却に強い」= Inoué で抽象度が上がるほど「一様復元から遠い」。

具体的に: CPS の 1-cell (容器) = Inoué の Existence (内部的存在) はともに忘却/抽象化に対して不変な最高層であり、CPS の 0-cell (内容) = Inoué の Meaning (射による決定) はともに忘却/抽象化に従属する。□

### 3.2 Sh(S¹) における CPS0' の具体的計算

Inoué [9] の中心的構成を忘却論の言語で再計算し、CPS0' の実現を明示する。

**設定.** S¹ を円周とし、p: ℝ → S¹ を被覆写像 p(t) = e^{2πit} とする。ℝ の S¹ 上の局所切断の層を E ∈ Sh(S¹) とする:

$$E(U) = \{s: U \to \mathbb{R} \mid p \circ s = \mathrm{id}_U\} \quad (U \subseteq S^1 \text{ 開集合})$$

**内部的存在.** 任意の開集合 U ⊊ S¹ (真の開部分集合) に対し、E(U) ≠ ∅ — 被覆の局所切断は常に存在する。例えば U = S¹ \ {pt} ≅ (0,1) に対し s: U → ℝ, s(e^{2πit}) = t は E(U) の要素。Sh(S¹) の内部論理で ∃x. x ∈ E が成立。

**大域的不在.** E(S¹) = Γ(E) = Hom(1, E) = ∅ — 大域的切断は存在しない。連続写像 s: S¹ → ℝ で p ∘ s = id_{S¹} を満たすものは、S¹ がコンパクトで ℝ が S¹ に同相でないこと（基本群: π₁(S¹) = ℤ ≠ 0 = π₁(ℝ)）から存在しない。

**CPS0' の実現.** E は CPS0' の圏論的実現である:
- **容器** (1-cell): E ∈ Ob(Sh(S¹)) は well-defined — E は層として存在する ✓
- **内容** (0-cell): Γ(E) = ∅ — 大域的な「中身」は存在しない ✓
- **知覚者** (2-cell): 局所切断 s ∈ E(U) は「開集合 U に位置する知覚者が見る内容」に対応。U の選択（知覚者の視点）によって見える内容が異なる ✓

**α-忘却濾過的解釈.** Grothendieck 位相 J_{S¹} の被覆篩が忘却パラメータの離散化を与える:
- U = S¹ (全体): α = 0 に対応。Γ(E) = ∅ であるから、「α = 0 でも大域的同定が不可能」= 非自明な忘却構造
- U ⊊ S¹ (局所): α > 0 に対応。E(U) ≠ ∅ であるが、U → S¹ への拡張で内容が失われる

これは Paper VIII 定理 6.2.3 (ii) の「部分的同定」の具体例である: 局所的には同定可能 (E(U) ≠ ∅) だが大域的には同定不可能 (Γ(E) = ∅)。Grothendieck 位相のコサイクル条件の非自明性が、α-忘却濾過の単調性公理 (F4) に対応する。

> **系 3.2 (独立な再発見).** CPS0' (2026年3月, Paper I–VIII) と Inoué の独立性定理 (2026年2月, arXiv:2602.18033) は、互いの存在を知らずに同一の構造的事実に到達した。CPS0' は忘却場理論と認知科学の実験から帰納的に抽出され、Inoué の定理は Lawvere 的意味論とトポスの内部論理から演繹的に導出された。帰納と演繹の合流は、この構造が理論の選択に依存しない数学的事実であることを強く示唆する。

## §4. 含意と今後の課題

### 4.1 忘却論への含意

1. **外部検証**: CPS0' は忘却論の内部公理にとどまらず、独立な圏論的結果として外部検証された。理論の信頼性が上がる。

2. **トポス意味論への接続**: Inoué の結果は Lawvere 的圏論意味論の枠組みにある。CPS0' をこの枠組みに埋め込むことで、忘却論がトポス理論の道具立てを利用できるようになる。特に:
   - Grothendieck 位相 ↔ α-忘却濾過の接続 (Paper VIII OP-VIII-1) に Inoué の構成が具体例を提供する
   - HoTT の (-1)-トランケーション ↔ U_cnt の対応 (Paper VIII OP-VIII-5) が、Inoué の内部存在の扱いを経由して精密化できる可能性

3. **4概念辞書の応用**: Inoué の Existence/Object/Meaning/Name の4分類は、忘却論の THAT/WHAT/WHY 分類をより精緻にする。特に Object と Existence の分離（Inoué では異なるレベル）は、CPS0' の「容器」概念の内部構造を明らかにする。

## §5. 結論

CPS0' と Inoué の独立性定理は、「存在は意味に先行する」という同一の数学的事実の、異なる言語による表現である。忘却論は認知科学・物理学の実験から帰納的に、Inoué はトポス理論の内部論理から演繹的にこの事実に到達した。両理論の合流は CPS0' の圏論的正当性を大きく強化する。

今後は、この対応を足がかりとして、忘却論のα-忘却濾過 (Paper VIII 定義 6.2.1) と Grothendieck 位相の正準的接続 (OP-VIII-1) の解決に取り組むことが最優先課題となる。
