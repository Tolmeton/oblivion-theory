# T9 自己診断的反論マッピング

## 概要

本文書は、査読者から想定される異議に対する構造化された応答資源である。ただし、現在の主目的は「反論の言い返し」ではない。論文内の負結果・限界・潜在的循環性を、T9 枠組み（$U_i \dashv N_i$ 随伴）の下で、どこまで言えるか、次に何を検証するか、何が出たら主張を下げるかまで管理する **typed recovery ledger** として扱う。

T9 基準（§3.6, 式 T9）は次を要請する。

$$\forall i: \rho_i := N_i \circ U_i(x) - x \geq 0$$

ある枠組みが T9 健全であるとは、各構造層について、忘却の検出可能性と回復操作の両方が与えられていることを意味する。本台帳では、それを次の 5 面に分解する。

| 面 | 役割 |
|:---|:---|
| Claim Cap | 現在の SOURCE から言える上限。ここを超えたら過剰主張。 |
| Next-N | 次に実行すべき回復操作。研究プログラムとしての次単位。 |
| Observable | Next-N が観測すべき信号。 |
| Falsifier | これが出たら当該主張を維持できない、という明示的な失敗条件。 |
| Downgrade Rule | 失敗時にどの射程へ下げるか。撤回ではなく、主張の型を下げる規則。 |

## 反論テンプレート

### R1: 「この枠組みは反証不可能だ」

T9 枠組み（§3.6）は Popper の反証可能性を包含するが、そこに留まらない。反証可能性は単一の $U_{\mathrm{causal}} \dashv N_{\mathrm{causal}}$ 対に対応するにすぎない。T9 が要求するのは、すべての構造層について検出可能性と回復可能性の双方を与えることである（式 T9）。

本台帳は、各 $U_i$ に Claim Cap、Next-N、Falsifier、Downgrade Rule を付けている。したがって、反証不能なのではなく、失敗時にどの主張へ下げるかまで事前に指定している。反証可能性だけを要求し、診断と回復の側を見ない態度は $U_{\mathrm{adjoint}}$ を犯す。

### R2: 「負結果が主張を弱めている」

T9 の下で負結果は、理論を削る失敗ではなく、忘却パターンを特定する診断である。たとえば §5.6 の遅延予測の失敗は $U_{\mathrm{static}}$ を露出させ、より情報量の高い dynamic-range prediction へ更新された。§7.3 の CoT 劣化は $U_{\mathrm{dilution}}$ を露出させ、構造的拡張と表層的拡張の区別を鋭くしている。

ただし、負結果をすべて勝利に読み替えてよいわけではない。各 ledger 行には Falsifier と Downgrade Rule がある。失敗が出た場合は、主張を「因果的支持」から「相関的 regularity」へ、または「理論的回復」から「工程上の緩和策」へ下げる。

### R3: 「プロービングは相関しか示しておらず、因果を示していない」

その通りである。この点は本文で明示的に切り出している（§7.1, L1077）。現段階の Claim Cap は「構造信号と振る舞いが共変している」までであり、因果寄与までは先取りしていない。

T9 上の処理は、このギャップを $U_{\mathrm{causal}}$ として名指しし、MCP ablation、反事実的文脈操作、合成のみを変数とする統制訓練という Next-N を与えることにある。これらで振る舞いが動かなければ、主張は相関的 regularity へ下げる。

### R4: 「HGK 測定は循環的だ」

この残部も本文で隠していない（§7.9, L1600）。本論文はこれを $U_{\mathrm{HGK}}$、すなわち設計者循環として切り出している。現在の Claim Cap は、HGK 測定が内部整合と部分的な外部予測を持つ、までである。外部再現済みとは言わない。

Next-N は外部再現、独立予測、HGK 構成要素のアブレーションである。外部 system で再現しない、または HGK 部品を抜いても予測力が落ちないなら、主張は「HGK 内部診断器」へ下げる。

### R5: 「$\varphi$ 分解は層独立性を前提している」

この点も本文で切り出し済みである（§7.7.4, L1373）。P1 テストでは $r = 0.28$（$p = 0.27$）であり、観測域では共分散が見えないことを示している。ただし、これは機械論的独立の証明ではない。

したがって ledger 上の Claim Cap は「観測域での低共分散」までである。Next-N は、$\varphi_0$ を変えつつ合成を一定に保つ因果介入である。$\varphi_0$ 操作で $\varphi_1$ が一貫して大きく動くなら、層独立の主張は撤回し、観測域での低共分散へ下げる。
