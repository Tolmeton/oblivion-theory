## §0 問いの再設定 — Benacerraf の不満から始める

Paul Benacerraf は 1965 年、一つの不満を言語化した。自然数の集合論的構成は複数ある。von Neumann の構成では $0 = \emptyset$、$1 = \{\emptyset\}$、$2 = \{\emptyset, \{\emptyset\}\}$ と進み、Zermelo の構成では $0 = \emptyset$、$1 = \{\emptyset\}$、$2 = \{\{\emptyset\}\}$ と進む。どちらも Peano 公理を満たす。では「自然数 2 とは何か」という問いに対して、どちらの集合が本物の 2 なのか。Benacerraf の答えは明快だった — この問いは**偽問題である**。

この不満は今も未解決のまま数学基礎論の中に残っている。集合論的な解答は「どちらでもよい、Peano 公理を満たせば」という形式的な回避に終わる。圏論的な解答は「NNO (natural number object) として普遍的に定義する」という方向へ進んだ (Lawvere, 1964)。

本章は第三の方向を提案する。忘却論の観点から見ると、「0 が何であるか」という問いに複数の答えが存在することは、**驚くべき事実ではなく、予測される帰結**である。

忘却論の基本構造が示すのは、任意の情報は複数の担体に同時に存在しうるということである。glyph「0」が複数の型に分離されるのは欠陥ではなく、型の生態系の正常な状態だ。Benacerraf の問いへの忘却論的応答は「どれが本物の 0 か」ではなく「どの担体に乗った 0 の話をしているのか」である。

これを正確にするために、本章は二つの層を展開する。

## §1 辞書層 — glyph「0」の型分離 (C-S1-上)

### 1.1 型分離の原則

数学において「0」という glyph は複数の異なる役割を担う。これらを一つの概念として潰し込むことは、型システムの破壊である。

忘却論定数辞書 (v0.1 §2) は、以下の担体 (carrier) を前提する:

| 記号 | 役割 |
|:--|:--|
| $A_\text{abs}$ | 絶対量の担体。加法 chart |
| $\Delta A$ | 変化の担体 |
| $\Theta$ | 角度の担体。位相の additive chart |

この上で、glyph「0」は**少なくとも三つの typed token** に分離される:

| typed token | 本籍 | 正準生成 | 役割 |
|:--|:--|:--|:--|
| $0_{abs}$ | $A_\text{abs}$ | primitive | 加法単位元。絶対量ゼロ |
| $0_\Delta$ | $\Delta A$ | $d(\text{const}) = 0_\Delta$ | 変化ゼロ |
| $0_\Theta$ | $\Theta$ | primitive | 回転ゼロ |

[SOURCE: `calculations/構成_忘却論定数辞書_v0.1.md` §3.1]

この三分離は恣意的ではない。それぞれ**異なる射**によって到達され、**異なる法則**に参加する。$0_{abs}$ は加法 $a + 0_{abs} = a$ を満たす。$0_\Delta$ は $d(\text{const}) = 0_\Delta$ により生まれ、定数関数の微分という操作的意味を持つ。$0_\Theta$ は角度 chart における回転なし、すなわち $\text{cis}(0_\Theta) = 1_\text{phase}$ を意味する。

### 1.2 禁則 — 何をしてはいけないか

型分離の裏返しとして、以下の潰し込みを禁則とする:

- $0_{abs} = 0_\Delta$ と書かない (加法単位元と変化ゼロは型が異なる)
- $0_\Delta = 0_\Theta$ と書かない (変化ゼロと回転ゼロは担体が異なる)
- $e^{i\pi}$ を最初から $A_\text{abs}$ の式とみなさない (複数の担体をまたぐ複合表現)

[SOURCE: `calculations/構成_忘却論定数辞書_v0.1.md` §5.1, §5.5]

### 1.3 Benacerraf への応答の第一層

Benacerraf が問うたのは「自然数 2 とは何か」だった。忘却論的な読み直しをすれば、「0 とは何か」という問いも同型の問いになる。

辞書層の答えは: **「0」は担体選択によって異なる型に着地する**。担体選択が $A_\text{abs}$ を選べば $0_{abs}$ が現れ、$\Delta A$ を選べば $0_\Delta$ が現れ、$\Theta$ を選べば $0_\Theta$ が現れる。「どれが本物の 0 か」は問いの立て方が間違っている — 担体を固定してから始めて初めて「その担体での 0」が意味を持つ。

(注: たたき台 `四則演算は忘却の選択である_たたき台.md` L16 ではこの担体選択を表す便宜的記号として $U_{or}$ が導入されているが、本稿では定数辞書 v0.1 に未登録の非正本記号として扱い、中立語「担体選択」を用いる。$U_{or}$ の正本化は別稿課題。)

これは Benacerraf の問いに形式的に答えている: 「自然数が何であるか」ではなく「どの圏の射が自然数の役割を担っているか」が問いの正しい形である。

## §3 二層整合 — 「0 が何か」は偽問題 (C-S1 総括)

辞書層 (§1) と CCM 層 (§2) の関係は、**射影の手前と後**という構造として読める。

- **手前**: 辞書層が型分離を課す。$0_{abs}$, $0_\Delta$, $0_\Theta$ はそれぞれ異なる担体に属し、潰し込みは禁則。
- **後**: CCM 層が単一機械性を主張する。5 面は CCM1-4 という一つの機械の異なる投影として現れる。

矛盾しない。手前で分けて、後で一つの源泉に遡る — これは地図の縮尺が変わっても同じ地形を描いているようなものである。

**Benacerraf 応答の第一層の総括**: 「0 とは何か」は偽問題である。正確には — 担体を指定せずに問うている。担体を指定すれば型は一意に決まる。担体を超えた問いに答えようとすれば、CCM1-4 という機械を提示し「あなたが問うていた 0 は、この機械のどの面から見た投影か」を聞き返すことが忘却論の答えである。

この答えは Benacerraf の「どちらの集合論的構成も恣意的」という不満に対して、**恣意性は担体選択の問題であり、担体を超えた単一機械が存在する** [仮説 70%] という形で正面から応じる。ただし「単一機械が存在する」の側は C-S1-下として本章では sketch レベルに留まる (§2.3 の既存同一視 (a)≡(e) が最強の先行証拠)。完全な応答の完成には 5 面 ≅ CCM1-4 の射影等式の形式化が必要で、これは第 0 章以降の課題として §7 に残す。

## §4 自然数の α-濾過読解 (C-S2a)

### 4.1 α-忘却濾過の離散骨格

忘却論の核定義 (Paper VIII §6.2.1) によれば、α-忘却濾過は連続パラメータ $\alpha \in [0, 1]$ を持つが、その離散的骨格は n-cell tower として構造化されている。

普遍フィルトレーション (Paper VII 由来) は:

$$U_\text{arrow}(1) \leq U_\text{compose}(1.5) \leq U_\text{id}(2) \leq U_\text{inv}(3) \leq U_\text{sym}(4) \leq \cdots \leq U_\text{self}(\omega)$$

[SOURCE: `drafts/series/論文VIII_存在は忘却に先行する_草稿.md` 系 6.5.3]

接続写像 $\alpha(n) = n/\omega$ がこの離散骨格と連続 $\alpha$ を橋渡しする。

### 4.2 cell 次元と「自然数」の違い

**重要な厳密化**: 上の骨格で $n$ が走る値は $\{1, 1.5, 2, 3, 4, \infty - 1, \infty, \omega\}$ であり、**自然数のみではない**。$n = 1.5$ は矢印圏と合成圏の中間的構造を、$n = \omega$ は全 n-cell の極限を表す。

したがって「自然数 = α-濾過段数」という読解は厳密には「**cell 次元 = α-濾過段数**」と言うべきである。自然数はこの cell 次元の**整数部分**に対応する — より正確には、整数 cell 次元を持つ段が Peano 的な succession によって連鎖する。

### 4.3 読解の意義

この再整理によって何が得られるか。

自然数を「ある離散的なものを数える方法」として抽象的に定義するとき、忘却論は一つの具体的候補を提示する: **情報を保存するために必要な cell の次元数が、alpha-filtration の段数として自然数を生成する**。

これは自然数の「存在理由」についての一つの解釈である。自然数は神秘的な抽象的対象ではなく、圏の cell tower を indexing するために出てくる。

## §5 Peano Succession との同型性 (C-S2b)

### 5.1 同型の候補

Peano の succession $s: n \mapsto n + 1$ は、cell 次元の 1 段刻みと対応する:

$$s: n \mapsto n + 1 \quad \longleftrightarrow \quad \alpha(n+1) - \alpha(n) = \frac{1}{\omega}$$

[仮説 75%: 以下の同型命題は忘却論体系内で明示されていない。新規接続の明示化]

**命題 C-S2b** (暫定): F5 制約を無視した骨格上で、$(\mathbb{N}, 0, +1)$ と $(\mathbb{N}_\text{cell}, 1, \text{step})$ は canonical に同型である。ここで $\mathbb{N}_\text{cell} = \{1, 2, 3, 4, \ldots\} \subset$ cell 次元 (整数部分) であり、$\text{step}$ は接続写像 $\alpha(n) \mapsto \alpha(n+1)$ を cell 側に引き戻した後者関数である (半整数 $1.5$ や超限 $\omega$ は本同型の範囲外)。

**同型射の定義**:

$$h: \mathbb{N} \to \mathbb{N}_\text{cell}, \quad h(0) = 1, \quad h(n+1) = \text{step}(h(n))$$

$h$ は以下の可換図式を満たす:

$$
\begin{array}{ccc}
\mathbb{N} & \xrightarrow{\;+1\;} & \mathbb{N} \\
\scriptsize h \downarrow & & \downarrow \scriptsize h \\
\mathbb{N}_\text{cell} & \xrightarrow{\;\text{step}\;} & \mathbb{N}_\text{cell}
\end{array}
$$

すなわち $h \circ (+1) = \text{step} \circ h$。$h$ は全単射であり、生成元 ($0 \mapsto 1$) と後者関数 ($+1 \leftrightarrow \text{step}$) の対で**モノイド同型**として機能する。

**同型の意味論的核**: Peano の $+1$ と cell tower の $\text{step}$ はどちらも「定数ステップ幅の離散的前進」を表す射であり、$h$ によって canonical に対応する。同型は構造レベルであって glyph レベルではない — Peano が $0$ を始点とするのに対し cell tower は $1$-cell (矢印段) を始点とするため、$h(0) = 1$ という **基点オフセット** が常に伴う。この基点オフセットは「自然数 $0$ と cell tower の $1$-cell は情報源的に同等」という意味で非自明であり、§1 の $0_{abs}$ と区別される (cell tower に「$0$-cell」は本稿の範囲では存在しない)。

[この同型射 $h$ は忘却論体系内で明示的には構成されていない。本命題の寄与は canonical 同型の存在主張 + $h$ の明示的構成である。厳密な証明義務は別稿 C-S2b' に委譲する。]

### 5.2 排他制約 F5 との関係 — 直交軸の発見

cell tower には排他制約 F5 (no-consecutive-1s 制約) が課されている [SOURCE: `drafts/series/論文XIV_曲率は忘却の繰り上がりである_草稿.md` §7.3 Route D]。これは Fibonacci 漸化式を導く:

$$|A(n)| = |A(n-1)| + |A(n-2)|$$

注意すべきは **F5 が作用する軸** である。論文 XIV §7.3 Route D L367 では:

> 「$m$ 個の格子点を持つ 1 次元離散格子で、この隣接排他を課すと、許される配置数はちょうど $F_{m+2}$ になる。」

すなわち F5 は cell 次元 $n$ の succession (整数前進) ではなく、**各 cell 段階 $n$ の公理集合 $A(n)$ の濃度 $|A(n)|$ の成長率** に作用する。Fibonacci 漸化式 $|A(n)| = |A(n-1)| + |A(n-2)|$ は **公理数の漸近成長率** を支配し、cell 次元 $n$ の +1 という前進そのものは F5 と独立である。

#### F5 と本主張 C-S2b は直交する

整数 cell 次元 $\mathbb{N}_\text{cell} = \{1, 2, 3, 4, \ldots\}$ の succession (1 段刻み $n \mapsto n+1$) は、F5 制約と **直交する別軸** である:

| 軸 | 構造 | F5 の作用 |
|:---|:---|:---|
| cell 次元 $n$ | 整数前進 (Peano succession と canonical 同型) | **作用しない** |
| 公理数 $|A(n)|$ | Fibonacci 漸近成長率 → 黄金比 φ | **作用する** |

両者は同一の cell tower の上で並存する独立構造であり、本主張 C-S2b の Peano canonical 同型 (cell 次元 軸) は F5 制約 (公理数 軸) と矛盾しない。

#### 公理数 軸の Fibonacci 成長は本シリーズの射程外

公理数 $|A(n)|$ の Fibonacci 漸近成長 → 黄金比 φ という構造は、本シリーズ (数の基礎) では扱わず、論文 XIV または スピンオフ `number_theory_oblivion/` (Abel-Ruffini と φ の文脈) に委譲する。本章の C-S2b は cell 次元 軸の Peano canonical 同型に集中する。

これにより、忘却論が標準的な自然数論を **2 つの独立な軸で組織する** 構造が見える: cell 次元 軸 (Peano arithmetic と整合) と公理数 軸 (Fibonacci → φ 成長)。両者の独立性自体が、忘却論が応用理論ではなく基礎理論候補である一つの根拠となる。

## §6 接続命題 — Lawvere NNO (C-S3 sketch)

### 6.1 Lawvere NNO の構造

Lawvere (1964) の自然数対象 (NNO) は、圏 $\mathbf{C}$ における初期対象として定義される:

- 始点 $0: 1 \to N$
- 後者関数 $s: N \to N$
- 普遍性: 任意の $(A, a: 1 \to A, f: A \to A)$ に対して唯一の $h: N \to A$ が存在し $h \circ 0 = a$、$h \circ s = f \circ h$

この普遍性は、自然数の「再帰的定義」に対する圏論的な表現である。
