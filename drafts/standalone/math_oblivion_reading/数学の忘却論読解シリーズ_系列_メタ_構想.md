### C-S2a: 自然数の $\alpha$-濾過段数読解 (既存設定の明示化)

v0.2 (2026-04-21) で NLM Query 2 の結果を受けて分解した。この面は既存体系の再整理に近い。

**主張**: **cell 次元 $n$** は $\alpha$-忘却濾過の離散段数として読める。接続写像 $\alpha(n) = n/\omega$ が **Paper VIII 系 6.5.3** で明示されており、普遍フィルトレーション $U_{\text{arrow}}(1) \leq U_{\text{compose}}(1.5) \leq \dots \leq U_{\text{self}}(\omega)$ (Paper VII 由来) が α-忘却濾過の「離散的骨格」として位置づけられている。

**DISCOVERY 2 による厳密化**: Layer 2 Read (Paper VIII 系 6.5.3) により、$n \in \{1, 1.5, 2, 3, 4, \infty-1, \infty, \omega\}$ — すなわち **cell 次元は自然数のみではなく、半整数 (1.5) や超限値 ($\omega$) を含む**。「自然数 = α-濾過段数」という読解は正確ではなく、「cell 次元 = α-濾過段数」と厳密化が必要。

**新規性**: 低い。既存設定の再整理。**±1.5〜2σ 級**。

**意義**: 本シリーズの基盤として必要だが、単独では σ を稼がない。C-S2b と C-S1/C-S3 と統合して初めて射程が立つ。

**章**: 第 0 章「0 と自然数」§(前置節として)
