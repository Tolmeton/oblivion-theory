# C4 整合担体 (`coherence carrier`)

**作成日**: 2026-04-23  
**役割**: C4 の selector / lift / weak-package / counterexample / guard-probe surface を一つの中間対象へ束ね、何が theorem 化を止めているのかを「補題不足」ではなく「器の未固定」として切り出す。  
**位置づけ**: backstage 正本。本文への侵入前に、命名と境界だけを固定する。

---

## 0. 一文核

**整合担体**とは、

> **局所 forgetting law が global law へ貼り合わさるか、残差 class を残すかを問うための、support-local な最小の器**

である。

`貼合側 / 残差側` はこの器の**読み**であり、`A'' / B''` はこの器の**guard 付き両極実現**である。

---

## 1. なぜ新しい名前が必要か

いま C4 の calculations 面には次の層がある。

- selector surface
- lift surface
- weak-package surface
- counterexample surface
- guard-probe surface

だが、これらはまだ「何を一つの対象として theorem 化したいのか」を共有名で持っていない。

その結果、阻害要因が

- 補題不足
- 翻訳不足
- 本文語不足

のいずれなのかが曖昧に見える。

実際には、止まっているのは

> **局所 law / defect support / guarded re-lift or witness を一つの object として束ねる面**

である。

この欠けていた object を、ここで **整合担体** と呼ぶ。

---

## 2. 最小定義

対象 `x` に対し、整合担体 `K_Θ(x)` は次の 4 つからなる。

```text
K_Θ(x) = ( S_x, Σ_x, Θ_x, Γ_x )
```

### 2.1 `S_x` — defect support

- `ω_Θ` が非自明になりうる support
- あるいは、その候補 support

ここで大事なのは、圏全体を持ち歩かないことである。  
C4 が見ているのは global world 全体ではなく、**比較面が立ち、defect が意味を持つ support** だけである。

### 2.2 `Σ_x` — generating comparison faces

- `S_x` 上で comparison を生成する非退化 2-simplex の族
- `B_1^{ver}` や defect support を支える face 生成系

Face Lemma が与えるのは comparison surface の最小成立条件であり、整合担体はその成立した面を「どこで問うか」の器として保持する。

### 2.3 `Θ_x` — local forgetting law

- `Σ_x` の 1-simplex 上で書ける局所 forgetting law
- 各 face 上で 1-cocycle 的に読まれる law

ここでの `Θ_x` は、まだ global law ではない。  
整合担体は、**局所 law を global law 候補へ運ぶ前段**に立つ。

### 2.4 `Γ_x` — guarded realization

- trivial side なら support-local re-lift
- nontrivial side なら support-faithful witness

を与える guarded datum。

`Γ_x` は decode や recovery を与えるものではない。  
その仕事は、

> **この局所 law が貼り合わさる側に寄るのか、残差側に寄るのかを、support 上で正当に判定できるだけの視界を与える**

ことにある。

---

## 3. 両極実現

整合担体は単独では neutral である。  
それがどちらの side に読むべきかは、`Γ_x` の型で分かれる。

### 3.1 貼合型整合担体

`Γ_x` が support-local re-lift 側にあるとき、整合担体は **貼合型** に分極する。

現時点の guarded name は `A''` であり、最低限の読みは次である。

- support-saturated
- triple-overlap-consistent
- gauge-stable

これは

> **局所 law を support 上で re-lift し、comparison face の重なりを一周しても drift が閉じる**

という型を表す。

### 3.2 残差型整合担体

`Γ_x` が support-faithful witness 側にあるとき、整合担体は **残差型** に分極する。

現時点の guarded name は `B''` であり、最低限の読みは次である。

- coboundary-conservative
- boundary-closed
- refinement-stable

これは

> **向き付き defect が、support 上で faithful に見えており、gauge の取り直しや refinement をしても対称 coboundary へ潰れない**

という型を表す。

---

## 4. 境界

整合担体が**閉じるもの**:

1. `A'' / B''` を同じ object の両極として読める  
2. FTC / QM を「別々の例」ではなく「同じ器の異なる分極」として置ける  
3. `貼合側 / 残差側` を front-stage の読者語として残しつつ、backstage で guard を保持できる  
4. calculations 面の 5 層を「何を theorem 化したいのか」の一点へ収束できる

整合担体が**まだ閉じないもの**:

1. 任意の CPS 圏で canonical `ω_Θ` が一般に立つこと  
2. `A'' / B''` が必要十分条件であること  
3. `H^2_{\Theta}=0` または `[\omega_{\Theta}] \neq 0` の一般 theorem  
4. recoverability の decoder 条件  
5. C2b との UCT bridge

したがって整合担体は、

> **theorem そのものではなく、theorem が依拠すべき最小の carrier**

である。

---

## 5. 既存語との関係

### 5.1 `localizable` との関係

`localizable` は、

> 複数の face の交差によって、欠損位置が候補集合へ縮約される

という概念である。

整合担体はそれより一段上で、

> **その候補集合の上に、局所 law と guarded realization を載せて class 可否を問える状態**

を指す。

したがって:

- `localizable` = 位置候補の縮約
- `coherence carrier` = class 判定の器

であり、両者は同一ではない。

### 5.2 `recoverable` との関係

整合担体は recoverability より手前で止まる。

recoverability にはさらに、

- face の重なりの十分性
- n-cell tower の縦の可定義性
- 右随伴 `N` の decoder 条件

が必要である。

したがって:

- `coherence carrier` は class 可否の前段
- `recoverable` は再構成の後段

である。

### 5.3 `貼合側 / 残差側` との関係

`貼合側 / 残差側` は読者向け front-stage pair である。  
これは整合担体の**名前**ではなく、整合担体に乗った局所 law の**読み**である。

---

## 6. 今後の使い方

いま安全にできることは次の 3 つだけである。

1. meta で blocker を「中間型未分類」から「整合担体未固定」へ言い換える  
2. calculations / infra で `A'' / B''` を整合担体の両極実現として束ねる  
3. 本文には、必要なら「貼合側 / 残差側を支える backstage carrier がある」とだけ補注する

逆に、まだやってはいけないことは次である。

1. 整合担体を本文 theorem として宣言する  
2. `A'' / B''` を十分条件として昇格する  
3. 整合担体から recoverability を直結する  
4. FTC / QM の 2 例だけで carrier の一般存在を主張する

---

## 7. 推奨一文

meta や handoff で短く言うなら、次の文が最も誤読が少ない。

> C4 の missing intermediate object は、局所 forgetting law を support-local な比較面と guarded re-lift / witness に束ねる **整合担体 (coherence carrier)** である。`A'' / B''` はその両極実現であり、`貼合側 / 残差側` はその front-stage の読みである。
