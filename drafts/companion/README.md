# Oblivion Companion Drafts

This directory collects companion draft series for Oblivion Theory.

Companion drafts are not part of the numbered main series `0-XV`, but they are also not isolated standalone essays. Each series develops a connected branch that returns to the main Oblivion Theory papers through definitions, proof obligations, examples, or conceptual bridges.

## Public Scope

This public edition includes reader-facing draft texts and claim maps. Project-local `*.meta.md` ledgers are intentionally omitted from this repository because the public edition guard rejects internal metadata files.

## Reading Notes

These files are concept drafts. They may contain ambitious conjectural links, partial proof sketches, and unresolved obligations. Treat claim maps as navigation aids, not as settled proofs.

The numbered paper series in `papers/series/` remains the primary reading path. Companion drafts should be read as adjacent branches that clarify, pressure-test, or extend that main path.
