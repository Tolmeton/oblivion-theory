## §0 概要

結論を先に書く。

この外部論文は、忘却論にとって **根拠柱** ではない。  
むしろ **敵対的比較対象であり、同時に外部収束例でもある**。

忘却論の側から見たとき、本論文の literal な「三次元時間 ontology」はそのままでは受け入れられない。主な理由は四つある。

1. 時間をまず「座標」として立てており、忘却論の「時間の矢」概念と型がずれる
2. 時間構造と粒子・質量・空間の関係について、容器/内容の型付けが十分に明示されていない
3. 構造そのものの運動と、因果的・信号的伝搬を分離していない
4. `c` や signature を「導出した」と言うための派生規則が、忘却論が自分自身に課している基準ほどには露出していない

他方で、回収できる芯も明確にある。

1. 前幾何への志向
2. 物理定数を原始的に置かず派生値へ落とそうとする志向
3. 空間や質量を、より深い構造の二次的発現として読み替えようとする直感

したがって本 memo の基本判定は次の一文に尽きる。

> **三次元時間という名前は棄却する。だが、その背後で見ようとしている構造の一部は、忘却論では「多成分 α 空間」「容器先行の前幾何」「派生値としての定数」という上位語彙で包摂できる。**

## §2 忘却論からの反駁

ここでは「忘却論の方が偉いから却下する」のではなく、**忘却論が自分自身に課している基準で照合したとき、外部論文のどこが未閉鎖か** を示す。

### R1. 時間はまず座標ではなく「矢」である

[SOURCE] /home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/series/論文IX_エントロピーは忘却である_草稿.md:137 は「時間の矢 = 忘却の矢」と置く。  
[SOURCE] 同論文は、忘却の進行 `α ↑` とエントロピー増大の方向を結び、時間をまず **単調方向** として定式化している。

[推論] この基準から見ると、外部論文の「時間が三次元ある」は、時間の最初の定義を **方向** ではなく **座標軸** に置いている。  
[推論] したがって忘却論からの第一反駁は、「三本の時間軸」は時間の高階化ではなく、**別種の自由度を時間という名で呼んでいる可能性** をまず疑うべき、である。

> **反駁 R1**: 時間を増やす前に、何をもって時間と呼ぶのかを定義せよ。忘却論では、時間はまず向きであって、棒の本数ではない。

### R2. 容器/内容の型付けが未証明なら、「派生」は成立しない

[SOURCE] /home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/series/論文VIII_存在は忘却に先行する_草稿.md:113 は、時空 `M` が質量・エネルギーなしに存在しうることを「容器先行」の実例として扱う。  
[SOURCE] 同論文 :135 は、CPS ではまず容器が THAT として立ち、内容が WHAT としてそれを満たすと明示する。  
[SOURCE] 同論文 :293-320 は、容器 = 1-cell、内容 = 0-cell という型付けを GR にも通している。

[推論] この基準から見ると、外部論文が「三つの時間方向から粒子世代や質量差が出る」と言うためには、まず

1. 三次元時間構造が本当に **容器** なのか
2. 粒子世代・質量が本当に **内容** なのか
3. その両者の間にどの forgetful span / projection / reconstruction が立つのか

を示さなければならない。

[推論] これを飛ばしたまま「派生」と言うなら、それは生成ではなく **名称変更** に落ちる。

> **反駁 R2**: 容器/内容の型付けがない派生論は、発生論ではなく言い換えに退化する。

### R3. Level 0 / Level 1 の未分離なら、構造速度と因果速度を混同している

[SOURCE] /home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/series/論文XII_速度は忘却である_草稿.md:55-82 は、Level 0 = 構造そのもの、Level 1 = 構造の不均一 = 値 = 力、と分ける。  
[SOURCE] 同論文 :65-69 は、光速制限 `c` は Level 1 の法則であり、Level 0 の幾何学的再配置を直接拘束しないと書く。  
[SOURCE] 同論文 :191-201 は、`Δ` を Level 0、`C` を Level 1 と再配置し、`χ = v_null / v_carrier` を「異なる Level に属する速度の比」と読む。

[推論] この基準から見ると、外部論文が時間構造から空間や物理量を出すときに、もし

- 構造そのものの幾何
- 因果的伝搬
- エネルギーや信号の運搬

を同一レイヤで扱っているなら、そこで混ざっているのは「時間」ではなく **型の異なる速度** である。

> **反駁 R3**: 基底構造の再配置と、因果信号の伝搬は同じ速度ではない。ここを分けない理論は、深いようでいて速度の型を混同している。

### R4. `c` や signature を「導出した」と言うには、派生規則の明示が必要である

[SOURCE] /home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/series/論文XII_速度は忘却である_草稿.md:84-105 は、`c` を物理定数ではなく派生値へ落とそうとする。  
[SOURCE] ただし XII 自身も :111-117 で反証条件を露出し、:1000-1008 で `c = f(∇²Φ)` を **野望** として開示している。  
[SOURCE] /home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/series/論文XIII_時空は忘却である_草稿.md:285-299 も `(3+1)` signature の発生を [予想] として扱い、まだ skeleton であることを隠していない。

[推論] ここから外部論文への反駁は、かなり単純で強い。

> **反駁 R4**: 忘却論は自分自身の前幾何的野望にさえ「予想」「野望」「未閉鎖」という札を貼っている。だから外部論文の `c` や family や signature の「導出」も、派生規則を露出しない限り、同じ厳しさで未閉鎖と判定される。

### 反駁のまとめ

| ID | 忘却論の基準 | 外部論文に対する判定 |
|:---|:---|:---|
| R1 | 時間はまず矢である | 3D time は型の取り違えの疑い |
| R2 | 容器/内容の型付けが要る | 派生の前提面が未露出 |
| R3 | Level 0 / 1 分離が要る | 構造速度と因果速度の混同疑い |
| R4 | 派生規則の露出が要る | 導出宣言は現状では過剰 |
