## v3-§1  最小形式モデル

三角形 Δ は次の1つのデータとして固定する：

```
Δ = (A, B, C;  f: A→B,  g: B→C,  h: A→C;  α: g∘f ⇒ h)
```

- `A, B, C`: 0-cell (頂点)
- `f, g, h`: 1-cell (向き付き辺)
- `α`: 2-cell (面。塗り塗りされた三角形内部)

付加構造：

- 長さ: `ℓ: Hom → [0,∞]`   (metric enrichment)
- 角: `θ: boundary turns → S¹`   (phase / turning enrichment)
- 向き付き境界: `∂Δ = f · g · h⁻¹` (向きを揃えた 1-cycle)
- 非退化条件: α が可逆 2-cell ではないこと (厳密三角形ならば α は degenerate でない)

重要な設計判断: **角と長さは別々の豊穣化である**。
- 長さは `[0,∞]` 値の metric enrichment (Lawvere 1973 型)
- 角は `S¹` 値の phase enrichment
- 両者を同じ構造で扱ってはならない (v2 棄却台帳で既に明示した原則の正式化)

## v3-§2  実在化ラダー (L0–L4)

## v3-§3  関手は何に対応するのか — よくある誤解の矯正

v2 では `𝓕_△: Triangle Geometry ⤳ Metric-Enriched 2-Category` と書いたが、

### 辺と辺のつながりは関手ではない

- 辺 A→B と辺 B→C が頂点 B でつながる → **合成 g∘f** (1-cell レベル)
- 合成 g∘f と直接射 h の間に張られる面 → **2-cell α** (2-cell レベル)

**関手はここではまだ登場していない**。関手は別の階層の写像。

### 関手は何か

関手 `𝓕` は、三角形全体のパターン `(A,B,C; f,g,h; α)` を、
別の圏 `𝒟` の中へ **構造を保ったまま送る写像**：

```
𝓕: Δ-shape category  →  𝒟

    頂点 A,B,C  ↦  𝒟 の対象 𝓕(A), 𝓕(B), 𝓕(C)
    辺 f,g,h    ↦  𝒟 の射   𝓕(f), 𝓕(g), 𝓕(h)
    面 α         ↦  𝒟 の 2-cell 𝓕(α)
    合成と恒等を保存
```

直感的には「抽象的な三角形の設計図を、ある数学世界の中で実現する写像」。

- 幾何の世界へ送れば「実際の平面三角形」

**L0→L4 ラダーは、同じ Δ-shape から出発する関手の終域を変えていく操作**として再解釈できる。

## v3-§6  sin/cos/tan の型分離

v2 の拡張節では sin/cos/tan を同格に並べたが、型が混ざっている。v3 で分離する。

### sin, cos = 回転表現の**行列要素**

```
ρ: S¹ → GL₂(ℝ),  ρ(θ) = | cosθ  −sinθ |
                        | sinθ   cosθ |
```

cos, sin は **行列 ρ(θ) の成分**。
つまり sin, cos は「単位円の座標」というより、**回転表現 ρ の matrix entry** として読むのが 1 段階抽象的で正確。

### tan = **射影座標**

tan は cosθ ≠ 0 の領域で定義される。
sin/cos を `ℝ² ∖ {y-axis}` に制限したときの **射影化 `(x,y) ↦ y/x`** の像。
つまり tan は主役ではなく、sin/cos を projectivize した**派生量**。

### π, e = 対象ではなく**構造定数**

- π = ℝ を S¹ へ閉じる周期化商の定数 (θ ~ θ+2π を閉じるパラメータ)
- e = 加法モノイド (ℝ,+) を乗法モノイド (ℝ₊,×) へ送る指数写像の規格化定数

**この二つは cell (点・辺・面) のような対象ではない**。
functor の定義を成立させる **規格化パラメータ** として読む方が型が整う。

### オイラーの等式 e^(iπ)+1=0 の位置づけ

これは「特別な一点の美」ではなく、
```
e^(iθ₁) · e^(iθ₂) = e^(i(θ₁+θ₂))
```
という **回転合成の一般法則** の、θ=π での半回転特殊化。
つまり：

- オイラー公式 `e^(iθ) = cosθ + i sinθ` = 回転の一般法則
- オイラーの等式 `e^(iπ)+1=0` = その半回転特殊化

階層構造にすると、`(e, π, i)` は「美しい数字の集合」ではなく「**回転合成 functor の一つの機械**」になる。

## v3-§7  φ, ψ の Q-matrix 固有値化

v2 で φ, ψ を「再帰 endomorphism のスペクトル」と書いたが、行列表示で具体化する。

Fibonacci 再帰 `F_{n+1} = F_n + F_{n-1}` の **Q-matrix 表示**:

```
Q = | 1  1 |,     Qⁿ = | F_{n+1}  F_n     |
    | 1  0 |           | F_n      F_{n-1} |
```

この行列 Q の特性方程式 `λ² = λ + 1` の2つの根が `φ, ψ`:

- `φ = (1+√5)/2`   (スペクトル半径、支配的成長モード)
- `ψ = (1−√5)/2`   (|ψ|<1、従属的減衰モード)

読み替え:

- **φ は再帰的生成の支配固有値** (n→∞ で Fibonacci 数列の成長率を支配)
- **ψ は従属固有値** (補正項として指数的に減衰する寄与)

したがって黄金比 `φ` は「美しい数」ではなく、**再帰的生成 endomorphism を対角化したときの主固有値**。

## v3.1-§3  Fibonacci の inflation rule 補強

v3-§7 では Q-matrix の固有値として φ, ψ を位置づけた。v3.1 では symbolic dynamics 側からの補強を加える。

### inflation rule としての Fibonacci

Fibonacci 数列は、文字列置換系として以下の **inflation rule** で生成される：

```
S ↦ L
L ↦ LS
```

初期文字列 `L` から反復すると：
```
Step 0: L
Step 1: LS
Step 2: LSL
Step 3: LSLLS
Step 4: LSLLSLSL
Step 5: LSLLSLSLLSLLS
```

各ステップの文字列長は `1, 2, 3, 5, 8, 13, ...` = Fibonacci 数列。

```
| #L_{n+1} |   | 1  1 | | #L_n |
|          | = |      | |      |
| #S_{n+1} |   | 1  0 | | #S_n |
```

これは v3-§7 の Q 行列そのもの。つまり **Q-matrix は inflation rule の線形代数的影**。

# 拡張 v3.2：walking triangle 圏による関手 domain の標準化 (2026-04-17 追記)

v3-§3 では関手 `𝓕` の domain を「Δ-shape category」と呼んでいたが、これは独自命名で検索性が低い。
v3.2 では圏論標準用語 **walking triangle (歩く三角形)** を導入し、関手を universal property の形で書き直す。内容は変更せず、**命名と位置付けの標準化** のみ。

## v3.2-§1  walking X の一般概念

圏論における walking X とは、「構造 X を **普遍的に** 実現する最小の圏」である。Street, Leinster, nLab 等で標準的に使われる術語。

| 構造 X | walking X | 対象数 | 生成データ |
|:---|:---|:---|:---|
| arrow | walking arrow `[1]` | 2 | `f: 0→1` |
| isomorphism | walking iso | 2 | `f, f⁻¹` + `f⁻¹∘f=id, f∘f⁻¹=id` |
| 2-cell | walking 2-cell | 2 | 平行射 `f,g: 0⇉1` + `α: f⇒g` |
| composable pair | walking composition | 3 | `f: 0→1, g: 1→2, g∘f: 0→2` |
| **triangle** | **walking triangle Δ²** | **3** | **`f: A→B, g: B→C, h: A→C` + `σ: g∘f ⇒ h`** |

walking triangle `Δ²` は 2-simplex の 2-圏化で、**3 対象 `A,B,C` + 3 生成射 `f,g,h` + 1 非自明 2-cell `σ`** を持つ最小の 2-圏。

## v3.2-§2  関手 domain の昇格

v3-§3 の関手記述は次のように書き直せる：

```
𝓕: Δ²(walking triangle)  →  𝒟

    Δ² の 3 対象 A,B,C  ↦  𝒟 の対象 𝓕(A), 𝓕(B), 𝓕(C)
    Δ² の 3 射 f,g,h    ↦  𝒟 の射 𝓕(f), 𝓕(g), 𝓕(h)
    Δ² の 2-cell σ      ↦  𝒟 の 2-cell 𝓕(σ): 𝓕(g)∘𝓕(f) ⇒ 𝓕(h)
    合成と恒等を保存
```

**universal property**: 任意の 2-圏 `𝒟` の中に存在する「三角形パターン」 (3 対象 + 3 射 + 1 2-cell) は、**`Δ²` からの 2-関手と一対一に対応する**。これが「walking X」の意味するところ。
