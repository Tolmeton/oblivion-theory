## 0. 一文核

速度は忘却境界の一次運動であり、加速度はその境界運動の二階曲率である。力は、その二階曲率が対象の幾何学的感受性を通じて Level 1 側に物理量化されたものである。

ここで「曲率」は、まだ Riemann 曲率テンソルに固定しない。本線は、境界が時空内に掃く worldtube の曲がりである。level-set front の法線速度の二階変化は、この worldtube 曲率を座標や追跡規則に降ろしたときの表示として扱う。したがって本ノートでは「加速度 = 力」とは言わず、「力 = 加速度的曲率に対象感受性を掛けたもの」と読む。

## 2. 採る前提 / 採らない前提

### 採る前提

- 微分は、全体の位置や積分定数を忘れ、局所的な不均一性を抽出する操作である。
- Paper XII の速度は、物質粒子そのものの速度ではなく、`Δ` または `C` の level-set front の法線速度である。
- Paper I の力は、忘却場 `Φ` と対象側の Chebyshev 形式 `T` の非整列から生じる二階的な不均一性である。
- `F = mg` 型の関係では、`m` は対象側の感受性、`g` は忘却場側の有効勾配として分離できる。

### 採らない前提

- 「加速度には必ず絶対上限がある」とは置かない。
- 暗点や欠如境界の超光速運動を、エネルギー・信号・因果の超光速伝達とは読まない。
- 「加速度 = 力」とは置かない。加速度は運動学側、力は対象感受性を含む相互作用側である。

## 3. 三階層

| 階 | 忘却論での対象 | 物理的な読み |
|:---|:---|:---|
| 0階 | 境界 `B_t`, `Σ_t` | どこが見えなくなるか、どこに担体があるか |
| 1階 | `v_null`, `v_carrier` | 境界がどの向き・どの速さで動くか |
| 2階 | `II_null`, `II_carrier` | 境界が時空内に掃く worldtube の法線が、接方向へどれだけ変化するか |

Paper XII はすでに 1階の定義を持つ:

```math
v_{\text{null}}^\varepsilon
:=
-\frac{\partial_t \Delta_t}{\|\nabla_x \Delta_t\|}
\Bigg|_{B_t^\varepsilon},
\qquad
v_{\text{carrier}}^{\kappa_c}
:=
-\frac{\partial_t C_t}{\|\nabla_x C_t\|}
\Bigg|_{\Sigma_t^{\kappa_c}}.
```

次に必要なのは、各時刻の境界を「動いている点の集合」として追跡することではなく、それらが時空内に作る面そのものを見ることである。

```math
\mathcal{W}_{\text{null}}^\varepsilon
:= \{(t,x) \mid \Delta_t(x)=\varepsilon\},
\qquad
\mathcal{W}_{\text{carrier}}^{\kappa_c}
:= \{(t,x) \mid C_t(x)=\kappa_c\}.
```

この `W_null`, `W_carrier` が正則な hypersurface として立つとき、その第二基本形式を加速度の本体候補にする。第二基本形式は、面の法線が面に沿って動くとき、どれだけ向きを変えるかを測る量である。つまり、点の追跡ではなく、境界運動そのものの曲がりを測る。

```math
\mathrm{II}_{\text{null}}^\varepsilon
:= \mathrm{II}(\mathcal{W}_{\text{null}}^\varepsilon),
\qquad
\mathrm{II}_{\text{carrier}}^{\kappa_c}
:= \mathrm{II}(\mathcal{W}_{\text{carrier}}^{\kappa_c}).
```

加速度の「向き込みの構造」を残すときは `II` を保持する。通常の物理量として大きさだけを取り出すときは、そのノルム射影を使う。

```math
a_{\text{null}}^\varepsilon
:= \|\mathrm{II}_{\text{null}}^\varepsilon\|,
\qquad
a_{\text{carrier}}^{\kappa_c}
:= \|\mathrm{II}_{\text{carrier}}^{\kappa_c}\|.
```

この採用により、平均曲率は primitive ではなく後続の trace 射影になる。`D_t v` も primitive ではなく、座標と追跡規則を選んだ後に出てくる表示である。

```text
front 速度の時間微分を primitive にしない。
第二基本形式 II(W) を primitive にし、必要ならノルム射影で |a|、座標表示で D_t v を取り出す。
```

この順序にすると、level-set front で「同じ点を追跡する」規則を先に選ばずに済む。加速度は点の持ち物ではなく、境界運動全体の曲がりとして定義される。

## 3.5 定義パッチ: `II(W)` と `|a|`

Paper XII に移植する場合、この小節を `§3.x 加速度としての worldtube curvature` の核にする。

### 3.5.1 入力

時空化された probe 空間を

```math
\mathcal{E} := I_t \times X
```

と置く。ここで `I_t` は観測時刻の区間、`X` は Paper XII の probe space である。`\mathcal{E}` 上には、時間方向と probe space 側の距離を比較するための ambient metric `\mathcal{G}_E` を入れる。

この `\mathcal{G}_E` は本ノートでは未固定である。物理時空として読むなら Lorentz 型、抽象 probe 幾何として読むなら task-conditioned Fisher 型または Euclidean 型が候補になる。どれを採るかで `II(W)` の値は変わるため、`\mathcal{G}_E` は隠れた前提としてではなく、定義の入力として明示する。

### 3.5.2 worldtube

忘却境界と担体境界の worldtube を次で定義する。

```math
f_{\text{null}}^\varepsilon(t,x) := \Delta_t(x)-\varepsilon,
\qquad
f_{\text{carrier}}^{\kappa_c}(t,x) := C_t(x)-\kappa_c.
```

```math
\mathcal{W}_{\text{null}}^\varepsilon
:= (f_{\text{null}}^\varepsilon)^{-1}(0),
\qquad
\mathcal{W}_{\text{carrier}}^{\kappa_c}
:= (f_{\text{carrier}}^{\kappa_c})^{-1}(0).
```

正則性条件は次である。

```math
df_{\text{null}}^\varepsilon \neq 0
\quad \text{on } \mathcal{W}_{\text{null}}^\varepsilon,
\qquad
df_{\text{carrier}}^{\kappa_c} \neq 0
\quad \text{on } \mathcal{W}_{\text{carrier}}^{\kappa_c}.
```

この条件が破れる点では、境界は fold / cusp / bifurcation を持つ。そこでは通常の加速度ではなく、境界生成・消滅の特異点として扱う。

### 3.5.3 第二基本形式

`W` を `\mathcal{E}` 内の正則 hypersurface とし、`n` を `\mathcal{G}_E` に関する単位法線とする。`u, v` を `W` の接ベクトルとすると、第二基本形式は

```math
\mathrm{II}_W(u,v)
:=
\mathcal{G}_E(\nabla_u n, v)
```

である。直観的には、境界面に沿って少し動いたとき、境界の法線がどれだけ向きを変えるかを測る。

忘却境界と担体境界では、

```math
\mathrm{II}_{\text{null}}^\varepsilon
:= \mathrm{II}_{\mathcal{W}_{\text{null}}^\varepsilon},
\qquad
\mathrm{II}_{\text{carrier}}^{\kappa_c}
:= \mathrm{II}_{\mathcal{W}_{\text{carrier}}^{\kappa_c}}.
```

これを加速度の primitive とする。

### 3.5.4 加速度の大きさ

向きや断面ごとの差を保持するときは `II` をそのまま使う。通常の「加速度の大きさ」に落とすときは、metric から誘導されるノルムで射影する。

```math
|a_{\text{null}}^\varepsilon|
:= \|\mathrm{II}_{\text{null}}^\varepsilon\|_{\mathcal{G}_E},
\qquad
|a_{\text{carrier}}^{\kappa_c}|
:= \|\mathrm{II}_{\text{carrier}}^{\kappa_c}\|_{\mathcal{G}_E}.
```

したがって、忘却論での加速度は二層に分かれる。

| 層 | 記号 | 役割 |
|:---|:---|:---|
| 構造加速度 | `II(W)` | 方向込みの二階幾何。どの方向に、どれだけ曲がるかを保持する。 |
| スカラー加速度 | `||II(W)||` | 大きさだけを取り出した表示。古典的な `|a|` に対応しやすい。 |

### 3.5.5 平均曲率と `D_t v` の位置

平均曲率 `H = tr(II)` は、`II` の trace 射影である。これは有用だが、primitive ではない。最初から `H` を本体にすると、方向ごとの差を失い、Paper I の方向性定理との接続が弱くなる。

同様に、`D_t v` は `II(W)` を特定の座標、時間分解、front 追跡規則に降ろした表示である。これは計算には便利だが、定義本体ではない。

```text
II(W)       = 定義本体
||II(W)||   = 加速度の大きさ
tr(II)      = 平均曲率射影
D_t v       = 座標表示
```

## 4. Paper I との接続

Paper I のノルム射影は、

```math
\|F\| = g_{\text{eff}} \cdot \|T\|_g
```

と読める。ここで `||T||_g` は対象側の幾何学的感受性であり、質量に対応する。したがって `g_eff` は、対象感受性をまだ掛けていない「加速度的なもの」として読める。

この読みでは、対応は次になる。

| 面 | 忘却論での読み | 古典力学での影 |
|:---|:---|:---|
| `g_eff` | 忘却場の方向的不均一性が作る有効曲率 | 加速度 `a` |
| `||T||_g` | その曲率を受ける対象側の感受性 | 質量 `m` |
| `||F||` | 曲率と感受性が結合した結果 | 力 `F` |

したがって、構造的には「力と加速度は同一」ではなく、次の射影関係にある。

```text
忘却境界の二階曲率
  -> 対象感受性を忘れた表示 = 加速度
  -> 対象感受性を再挿入した表示 = 力
```

## 5. 最大加速度問題への翻訳

相対論の速度上限 `c` は、Level 1 の担体・信号・因果伝播に対する上限である。Paper XII の整理では、`v_carrier` はこの制約を受けるが、`v_null` は欠如境界の運動であり、同じ意味では `c` に縛られない。

加速度の上限を忘却論で問うなら、問いはこう変換される。

```text
忘却境界の worldtube は、どれだけ急に曲がれるか？
```

または、

```text
二階の局所的忘却、すなわち境界 worldtube の曲率には、構造的な上限があるか？
```

現段階で言えるのは次である。

| claim | 判定 |
|:---|:---|
| 速度上限 `c` は Level 1 担体の制約である | SOURCE 支持あり |
| `v_null` は Level 0 境界運動であり、`c` と同型の制約を受けない | SOURCE 支持あり |
| 加速度は忘却境界 worldtube の第二基本形式、またはそのノルム射影として定義できそうである | 作業仮説 |
| 最大加速度が存在するなら、それは速度上限ではなく、許容される二階曲率の上限である | 作業命題 |
| Planck scale 等から最大加速度が必然的に出る | 未採用。外部物理との追加 Sourcing が必要 |

## 6. 作業命題

**A1.** 速度は、忘却境界 `B_t` または担体境界 `Σ_t` の一次運動である。

**A2.** 加速度は、忘却境界運動の二階変化、つまり境界 worldtube の第二基本形式である。大きさだけを問うときは `||II||` を使う。`D_t v` はその座標表示候補であり、定義本体ではない。

**A3.** 力は、A2 の曲率が対象側の感受性 `T` を通じて Level 1 側に現れた量である。

**A4.** 最大加速度問題は、「速度の上限」ではなく「二階忘却曲率の上限」の問題として再定式化される。

## 8. 棄却された読み

| 棄却案 | 理由 |
|:---|:---|
| 加速度は力そのものである | 運動学量と相互作用量を混同する。`F = ma` では `m` の側に対象感受性が残る。 |
| 暗点の超光速運動は相対論破れである | Paper XII の区別では、`v_null` は Level 0 境界運動であり、エネルギー・信号の伝搬ではない。 |
| Planck scale があるなら最大加速度は必然である | これは外部理論の追加仮定を要する。忘却論内部からはまだ必然化されていない。 |
| 平均曲率を加速度の本体にする | trace によって方向ごとの差を先に捨てるため、Paper I の方向性定理との接続が弱くなる。平均曲率は後続の射影として扱う。 |
