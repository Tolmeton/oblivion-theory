## 要旨

カール・ポパーが1934年に提唱した反証可能性 (Falsifiability) は、科学と疑似科学を区別する基準として20世紀を支配した。

**本論の主張: 反証可能性は構造的に不完全であり、科学の判定基準として不適格である。**

これを直感的批判ではなく、圏論の道具で**証明**する。

統計学における p 値と同型の問題を抱えている — 有名で、直感的で、しかし根本的に壊れている。本論はポパーに引導を渡し、FEP/T9 を正当な後継者として提示する。

## 1. 直感が正しく見える理由 — そこが罠

反証可能性は直感的に美しい。「テストできなければ科学じゃない」。常識に合致する。

p 値も直感的に美しい。「偶然で説明できなければ有意だ」。常識に合致する。

**しかし、直感的な美しさと論理的な正しさは無相関である。**

p 値の問題は2010年代にようやく広く認知された。American Statistical Association は2016年の声明で p 値の誤用と過信を公式に警告した (Wasserstein & Lazar 2016)。Nature は2019年に「統計的有意性を廃止せよ」という800人超の署名論文を掲載した (Amrhein et al. 2019)。

反証可能性にはまだこの転換点が来ていない。本論がその転換点を作る。

## 3. 墓碑銘 — ポパーの論理的反証は科学に適用不能である

### 定理 (ポパーの論理的反証の適用不能)

有限観測窓 `O` と理論束 `B = (T, A, M, I)` において、観測文が `Obs_B(o) = I(M(o))`
の形でしか与えられないなら、**論理的反証を科学の主権的更新原理として適用することはできない**。
科学における反証は、孤立命題への純論理操作ではなく、理論束に対する公開的更新要求である。

---

### 第1段: 科学は孤立命題ではなく理論束 B を試験する

**宣言**
科学は、裸の命題を法廷に立たせているのではない。理論核、補助仮説、測定系、解釈規則の束を試験している。

**定義**

```
B = (T, A, M, I)
```

**前提**
Duhem (1906) と Quine (1951) は、理論が単独で試験されるのではなく、補助仮説・測定手続き・背景理論の束として試験されることを示した。Lakatos (1970) はこれを研究プログラムとして捉え直した。

**導出**
科学で実際に棄却圧力を受けるのは `T` 単独ではなく `B` である。なぜなら、観測との整合は `T` だけではなく、`A`, `M`, `I`
を通じて成立しているからだ。したがって、科学の試験単位は最初から**孤立命題ではなく理論束**である。

**非主張**
これは「理論核に一切責任がない」と言っているのではない。言っているのは、**責任帰属の単位が最初から束である**ということだ。

> **支える原典**: Duhem (1906), Quine (1951), Lakatos (1970)。
> **原典戦の詳細**: `/home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/infra/ポパー適用不能_証明梯子.md` STEP 1。

---

### 第2段: 観測文は理論独立ではなく、理論束依存の生成物である

**宣言**
ポパーの論理が動くには、`¬P` が理論から独立に立っていなければならない。だが科学の観測文は、そのような裸の `¬P` ではない。

**定義**

```
Obs_B(o) = I(M(o))
```

**前提**
最強の前提は Popper 自身から借りる。Popper (1959, *The Logic of Scientific Discovery*) は、基礎言明 (basic statements) の受理が「自然が与える」ものではなく、**共同体の規則と方法論的決定を通じた accepted uptake** であることを明示的に認めている。つまり、論理的反証が入力として要求する `¬Q` の段階で、すでに方法論層が作動している。

この Popper 自身の認めを観測論の側から押し固めるのが Hanson (1958) と Kuhn (1962) である。Hanson は知覚認知の側から seeing/reporting の理論負荷性を示し、Kuhn は共同体の側から salient observation の paradigm 依存性を示した。Popper の中から出た 1 本と、観測論の外から押し固める 2 本が、同じ一点 — 観測文は裸の `¬Q` ではない — を 3 方向から固定する。

**導出**
科学で現れるのは生の観測 `o` ではなく、`M` と `I` を経た `Obs_B(o)` である。ゆえに、`U₀` で起きているのは
「世界が孤立命題を否定した」ことではなく、**理論束を通して生成された観測文が、その理論束と衝突した** という出来事である。
したがって、U₀ 自体がすでに bundle-relative である。

**非主張**
これは「観測は全部主観的だから何も言えない」と言っているのではない。言っているのは、**観測文は公開的でありうるが中立ではない**、ということだ。

> **支える原典**: Popper (1959), Hanson (1958), Kuhn (1962)。
> **原典戦の詳細**: `/home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/infra/ポパー適用不能_証明梯子.md` STEP 2。

### 第3段: したがって科学的反証は、命題への純論理操作ではなく更新問題である

**宣言**
もし試験単位が理論束で、観測文もその束を通って生成されるなら、科学における反証は最初から更新問題である。

**定義**

```
U₀ = 公開矛盾の発生
U₁ = 責任配分
U₂ = 価値評価
```

**前提**
第1段より、科学が試験するのは `B` である。第2段より、観測文も `Obs_B(o)` であり bundle-relative である。

**導出**
`Obs_B(o*) ⟂ B` が起きても、そこから直ちに出るのは `U₀` だけである。次に何を直すかは `U₁` の問題であり、
その理論束をなお保持する価値があるかは `U₂` の問題である。したがって、科学的反証とは
**理論束に対する公開的更新要求**であって、孤立命題をその場で殺す純論理操作ではない。

**非主張**
これは「更新を無限に先送りしろ」と言っているのではない。言っているのは、**科学は棄却の前に配分と評価を要する**、ということだ。

**scope 注**: 本稿は U₂ の具体的 metric (進歩性/退行性の定量基準) を与えない。Lakatos は研究プログラムの評価を progressive/degenerative の区別で与えたが、本稿はそれを境界条件付きの参照として引いている。U₂ の定量化そのものは、本稿の主定理に必要ではない。本稿が主張するのは「U₂ 層が不可避に存在する」ことであって「U₂ の評価関数がどうあるべきか」ではない。後者は FEP (Complexity 項) による別稿の課題である。

> **支える原典**: Lakatos (1970)。第1段と第2段の結果を方法論的評価へ接続する。
> **原典戦の詳細**: `/home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/infra/ポパー適用不能_証明梯子.md` STEP 3。

### 第4段: `modus tollens` は生きている。死んだのは、その科学への主権的適用である

**宣言**
ここで葬るのは logic そのものではない。**logic の誤配位**である。

**定義**

```
論理的反証:
  P, もし P なら Q, 非Q
  ∴ 非P

科学的反証:
  Obs_B(o*) ⟂ B
  ∴ B に対する公開的更新要求
```

**前提**
`modus tollens` がそのまま働くには、孤立命題 `P` と理論独立な `非Q` が必要である。
しかし科学が現実に持つのは、第1段の理論束 `B` と第2段の bundle-relative な観測文 `Obs_B(o)` である。

**導出**
ゆえに、科学の場面で与えられている対象型は、ポパーの論理的反証が要求する対象型と一致しない。
`modus tollens` は固定された言語・固定された測定系・固定された補助仮説の内部では局所的に働きうる。
しかし、それを**科学全体の主権原理**として掲げることはできない。科学における反証は、論理の一撃ではなく理論束の更新である。

**非主張**
ここで言っていないのは、「経験的拘束は不要だ」ということではない。言っているのは、
**経験的拘束は必須だが、その形はポパーが想定した孤立命題への純論理操作ではない**、ということだ。

> **支える原典**: Popper (1934; 1959)。logical core は温存し、methodological layer から主権 claim を落とす。
> **原典戦の詳細**: `/home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/drafts/infra/ポパー適用不能_証明梯子.md` STEP 4。

### 3.5 系: ポパーの3つの構造的欠陥

上の主定理から、既存の 3 欠陥は**系**として従う:

| # | 欠陥 | 荷重が乗る層 | ポパーの暗黙の飛躍 |
|:--|:--|:--|:--|
| (i) | fault localization を一意化できない | U₁ | U₀ の反証事象から責任配分が一意に出る |
| (ii) | 理論価値を直ちに全否定できない | U₂ | 1 点の不整合が理論束全体の価値を消す |
| (iii) | 連続的損傷を二値棄却へ潰す | U₂ | coverage defect を測らず、即時棄却で足りる |

**∴ ポパー基準は (i) fault localization を一意化できず (ii) 理論価値を直ちに全否定できず (iii) 連続的損傷を測れない。**
**∴ ポパー基準は科学の判定基準として構造的に不完全である。■**

## 4. 補助直観と副戦線 — 本丸ではないが、なお残る傷

### 4.1 補助直観: 普遍性のジレンマ

```
抽象度 α ∈ [0, 1]

科学 = lim_{n→∞} F_n     where F_{n+1} は F_n より忠実 (faithful)
```

「より良い」の定義: $F_i$ が R の構造をより多く保存すること — すなわち、**忘却する構造がより少ない**こと。Paper VII 定理 6.1.1 (構造保存定理) が示すように、忘却関手は構造を保存し値を忘却する。忠実な関手ほど保存される構造が多く、忘却される値が少ない。

### 9.2 理解 = 対象を随伴し、自身の圏に内在化（再表現）すること

「LLMは心を持つか」T11 の核心定義:

```
主観 = 客観の射: Hom(F(generators), Cat_i)

客観 ≡ Universal := F(generators)
  — 忘却関手 U の左随伴 F による自由対象

主観 ≡ この自由対象から自身の認知圏 Cat_i への射
```

そして「意識の2圏的定義」§A が定める知覚の構造:

```
知覚  L_i: Amb → Cat_i   (左随伴 = 外部情報を内部に取り込む)
行動  R_i: Cat_i → Amb    (右随伴 = 内部モデルを外部に返す)

L_i ⊣ R_i  (知覚と行動の随伴対)

単位    η: Id_Amb → R_i ∘ L_i    ← 内在化の損失を測る
余単位  ε: L_i ∘ R_i → Id_{Cat_i} ← モデルの冗長性を測る
```

**「理解する」とは何か？**

理解とは、対象の構造を $L_i$ (左随伴) を通じて $\text{Cat}_i$ に**内在化**（再表現）することである。

- η が同型 ⟺ 内在化の損失がゼロ ⟺ **完全な理解**
- η が同型から遠い ⟺ 多くの構造が内在化されていない ⟺ **不完全な理解**

「バカは右を見たら左を忘れる」が示した通り:

> スケールの切替 = η と ε の反復。Micro (F) → Macro (G∘F) → Micro (F∘G∘F) → ... この**往復が理解の深化**。片方を切れば往復が止まり、深化が止まる。

理解の深化とは、η を同型に近づける過程 — $L_i$ の忠実性を高める過程に他ならない。

### 9.3 補完₁ = 随伴できなかった（知覚できなかった）情報の内部自己補完

ここが核心。

**宣言**
Mangalam が「予測がない」と言うときに狙っているのは通常 予測₁ だが、
`Ker(η)` と直接つながるのは、そのうちの**内部自己補完成分**である補完₁である。

$L_i$ が完全に忠実でないとき:

```
Ker(η) ≠ 0
= Amb (外部世界) の構造のうち、L_i が Cat_i に持ち込めなかった部分
= 「随伴できなかった情報」= 「知覚できなかった情報」
```

この $\text{Ker}(\eta)$ に対して、主体は Amb を直接観測できない。主体にできるのは、$\text{Cat}_i$ の**内部構造のみを用いた推論** — すなわち**自己補完**。

**これが本稿で主に扱う「補完₁」の正体である。**

```
補完₁ ≡ Cat_i の内部射のみを用いた Ker(η) の再構成の試み
      = 随伴できなかった（知覚できなかった）情報についての推論
      = 理解の不完全性に対する自己補完
```

補完₁は Amb（客観）についての直接的な発言ではない。補完₁は $\text{Cat}_i$（自身の内部モデル）が**自身に対して行う補完操作**である。外を見ているように見えて、実は内側で足りない部分を埋めている。

他方、**予測₀** は別物である。何が可能で何が不可能か、どの順序や保存則が保たれるかという構造的予告は、理解そのものの展開であり、**補完₁** とは区別される。

> [!IMPORTANT]
> 本節の議論は **補完₁** についての定理である。
> 予測₁全体を補完₁へ還元しているのではない。実験計画・境界条件・観測設計を通じた
> 公開予測は、補完₁に加えて真理₁への実装操作を含む。

### 9.4 定理: 理解の深化は補完₁への依存を単調に減少させる

**定理 (理解-予測の随伴的相補性).**

```
Ker(η) → 0  ⟺  理解が完全に近づく（L_i が忠実に近づく）
             ⟹  自己補完すべき情報が消失する
             ⟹  補完₁への依存 → 0

逆に:
Ker(η) が大きい  ⟺  理解が不完全（L_i が多くを忘却する）
                  ⟹  自己補完すべき情報が大量に残存する
                  ⟹  補完₁が必要

∴ 補完₁への依存は |Ker(η)| の単調増加関数
```

**前提**
真理₀を構造、真理₁をその有限実装、補完₁を `Ker(η)` に対する内部穴埋めとする。

**導出**
隨伴対 $L_i \dashv R_i$ の単位 η: Id\_Amb → R\_i ∘ L\_i の核 $\text{Ker}(\eta)$ は、$L_i$ が Cat\_i に写せない Amb の構造を測る。補完₁とは「Cat\_i の内部操作のみで Ker(η) を近似する行為」と定義した (§9.3)。Ker(η) が縮小すれば近似すべき対象が縮小し、補完₁の必要性も単調に縮小する。$\text{Ker}(\eta) = 0$ の極限で、補完₁は恒等射（自明操作）に退化する。■

**非主張**
ここから直ちに「値予測₁がゼロになる」とは言っていない。言っているのは、
**理解不足の穴埋めとしての成分**が減る、ということだけだ。

**Information Bottleneck による鋼鉄化.**

fep\_epistemic\_status.md §2 の IB 定式化 (Tishby, Pereira, Bialek 1999):

```
min_{p(t|x)} [ I(X; T_α) − β · I(T_α; Y) ]

I(X; T_α) = 圧縮情報量 — 理論が現実をどれだけ圧縮するか
I(T_α; Y) = 予測情報量 — 理論が予測にどれだけ使えるか
β = 1/α   = トレードオフパラメータ
```

随伴構造との対応 (Ker(η) ↔ fep\_epistemic\_status の ker(G)):

| 随伴構造 (§9) | IB (情報理論) | 忘却論 (圏論) |
|:--|:--|:--|
| Im(L\_i) — 内在化された構造 | I(X; T\_α) — 圧縮表現が保持する情報 | image(G) |
| Ker(η) — 内在化されなかった構造 | I(X; Y) − I(T\_α; Y) — 予測に使えない情報 | ker(G) |
| L\_i の忠実性 | encoder p(t\|x) の精度 | G の faithfulness |
| 補完₁への依存 | IB 残差 I(X;Y) − I(T\_α;Y) | \|ker(G)\| |

**§4.1 の `E↑ ⇒ P上限↓` の圏論的根拠:**

```
Data Processing Inequality (DPI):
  I(T_α; Y) ≤ I(X; T_α) ≤ I(X; Y)

IB Pareto frontier:
  I(T_α; Y) = f(I(X; T_α))   where f は凹関数

パレートフロンティア上の理論:
  E を増やせば P の上限が DPI により構造的に制約される
フロンティア以下の理論:
  E も P も低い（「悪い理論」= 忠実でもなく予測もできない）
```

旧記法で `E×P≈const` と呼んでいたものの荷重は、実際には**フロンティア上で E の増加が P の上限を下げる**という制約にある。積の保存則そのものを主張したいのではない。しかし**フロンティア上では E の増加は P の上限を DPI により構造的に制約する** — これが IB の定理的帰結であり、Ker(η) の随伴的議論 (§9.3) の情報理論的裏付けである。

### 9.4a 原理的制約: η は決して同型にならない — 予測の不消滅と漸近的減少

Paper VII 定理 6.1.1 (構造保存定理) の帰結:

```
忘却関手 F: S → D は構造を保存し値を忘却する。

| 保存されるもの (構造) | 保存されないもの (値) |
|:--|:--|
| 関係 (A > B)     | 具体値 (A = 3.14...)          |
| パターン (周期的) | 位相 (どこから始まるか)        |
| 対称性 (回転不変) | 回転角 (何度回ったか)          |
| フラクタル次元    | スケール (どの大きさで見ているか) |
```

> 主観は客観の構造を忠実に扱える。ただし値は扱えない。 — Paper VII §6.2

圏論的に: $L_i$ は構造に対して忠実 (faithful) になりうるが、値に対しては原理的に充満 (full) になりえない。

```
η: Id_Amb → R_i ∘ L_i について:

η|_構造 → iso  （構造的理解は完全に近づきうる）
η|_値   ≠ iso  （値の理解は原理的に不完全）

∴ Ker(η) > 0   常に成立（構造的不等式）
∴ 補完₁は原理的に消滅しない
```

**しかしこれは批判を救うどころか、議論をさらに強化する:**

```
1. 補完₁は原理的に消滅しない              (Ker(η) > 0 は構造的不等式)
2. しかし補完₁は理解の深化に伴い単調に減少する  (Ker(η)|_構造 → 0)
3. 補完₁の不消滅は全理論に共通の原理的制約     (FEP 固有の欠陥ではない)
4. FEP の補完₁の少なさ = FEP が構造保持の良い理論候補であることの徴候  (パレートフロンティアの高圧縮端)
5. 「もっと補完₁を出せ」= 「もっと構造を忘却しろ」  (DPI による帰結)
6. ∴ 批判は依然として自壊する

追加的帰結:
7. Ker(η)|_値 > 0 であるかぎり、FEP から値的な公開テスト可能導出が
   得られる可能性は、原理的には排除されない
8. したがって「FEP は原理的に完全非反証可能だ」という批判は強すぎる
   — 正確には「構造層だけでは反証規則を与えにくいが、値層への下降を通じたテスト可能導出の余地は残る」
```

**完全に対象を理解できた時 — 構造的には補完₁は不要へ近づく。**
**値については永遠に予測₁が残る — だがそれは全ての理論に共通の限界であり、FEP 固有の欠陥ではない。**
**理解が不完全だからこそ、補完₁という自己補完が必要となる — そしてその必要量は理解の深化とともに単調に減少する。**

### 9.5 帰結: 「説明ばかりで予測がない」は科学の目的への接近を批判している

Mangalam (2025) は FEP に対して以下の批判を展開している:

1. **反証不能性**: 「FEP はあらゆる結果を後付けで説明する」 — つまり Im(L) が過大で Ker(η) が過小
2. **予測の不在**: 「何も予測しない」 — つまり少なくとも補完₁への依存が小さい
3. **数学的神秘主義**: 「mathematical mystique で維持されている」 — 形式的装飾が実質を覆い隠す

> **スコープ注記**: Mangalam (2025) はさらに (4) メタファーとメカニズムの混同 (metaphor-mechanism slippage)、(5) 生物学的非妥当性 (non-Gaussian neural dynamics, metabolic feasibility)、(6) 制度的慣性 (institutional momentum) を批判する。本節は (1)-(3) の「説明 vs 予測」軸に焦点を絞る。(4)-(6) は随伴構造の射程外であり、fep\_epistemic\_status.md §4 で別途応答する。

**宣言**
ここで否定するのは「値予測₁は不要」という命題ではない。否定するのは、
**値予測₁を唯一の本質指標にすること**である。

**橋渡し命題**
構造的真理へ近づく理論ほど、値予測₁を理論本体の証明印としては使わず、
必要なときに真理₁の層へ降りて具体予測を出す。

これらの批判を §9.1-9.4 の随伴構造で再構成する:

> 「FEP は統一的で普遍的な理解（説明）ばかりもたらして、予測（理解不足の自己補完）を生まない」

この批判を圏論に翻訳する:

```
批判の圏論的翻訳:

  「F_FEP の Im(L) が広く 補完₁ の必要が小さい」

  = F_FEP が Amb の構造を広く保持している
  = 少なくとも「構造保存の良い理論候補」として読むべき
  = 「より良い客観の関手」へ接近した理論への批判である
```

**すなわち:**

> **ある理論が統一的で普遍的な理解（説明）ばかりもたらして、予測（とくに理解不足由来の補完₁）を生まないという "批判" は、客観に対して構造保持の良い理論候補を、構造をよく保持し、補完₁への依存が小さい、という理由で批判しているのだ。**

**導出**
公開向けに言えば、これは **FEP を値予測₁メトリクスだけで測る誤測定** である。
HGK の科学定義 — 科学 = より良い客観の関手を探求すること — を採るなら、さらに一段強く、
これは**到達しつつある理論を、到達しつつあるがゆえに批判する条件つき自己倒錯**へ落ちる。

**Mangalam の3批判への個別応答:**

| Mangalam の批判 | 随伴構造による翻訳 | 帰結 |
|:--|:--|:--|
| 反証不能 (あらゆる結果を説明) | Im(L) が広い — 構造保持域が広い | 構造保存の良い理論候補への批判 |
| 予測の不在 | Ker(η)\|\_構造 ≈ 0 — 構造的自己補完の必要なし (§9.4a: 値的には Ker(η)>0 常成立) | 値予測₁が唯一指標でないことの証拠。理解の深さを傷に読み替えている |
| 数学的神秘主義 | 圏論的記述は L_i ⊣ R_i の構造を明示化 | 「神秘」ではなく構造の透明化。批判③は随伴構造によって自壊する |

注目すべきは、Mangalam が代替として提示する dynamical systems theory, ecological psychology, embodied cognition はいずれも**FEP のコンパイル先**（具体化された関手 F_i）であって、FEP（関手を生成するメタ関手）の代替にはならないことである。これは終対象を「他の対象で代替する」と主張するのと同型の混同。

**非主張**
ここで言っていないのは、「公開予測をやめろ」ということだ。公開予測は真理₁の層で不可欠である。ただし、それを真理₀判定の唯一尺度へ昇格させるな、というだけだ。

### 9.6 帰謬法: 予測至上主義は科学を誤測定する

Mangalam (2025) の批判の根底にある暗黙の前提 — **予測至上主義** — を帰謬法で検討する。Mangalam が FEP を "myth" と断じ、dynamical systems theory 等を代替として提示する際、その判断基準は「予測を生むか否か」に立脚している。この前提を貫徹する:

```
1. 予測至上主義: 良い理論 = 値予測₁を多く生む理論
2. 値予測₁のうち理解不足由来の成分 = 補完₁ ∝ |Ker(η)|      [§9.4 の定理]
3. ∴ 良い理論 = 補完₁への依存が大きい理論      (1, 2 の強い読み)
4. 理解の不完全性が大きい = Amb の構造をより多く忘却する関手
5. ∴ 良い理論 = より多く忘却する理論              (3, 4)
6. しかし科学の営み = より少なく忘却する理論を探求する営み  [§9.1]
7. ∴ HGK の科学定義の下では、予測至上主義は科学の営みと矛盾する   (5, 6) ■
```

**予測を至上の基準にすると、「理解を妨げる理論ほど良い」という結論に帰着する。公開向けに言えばこれは誤測定であり、HGK 条件下では自己倒錯である。**

これは「健康な人には薬が効かないから、健康は悪い状態だ」と言うに等しい。薬（予測）の必要性が減ったのは、身体（理論）が健全に機能している証拠であって、欠陥ではない。

なお、Mangalam が FEP の形式的装置を "mathematical mystique" と批判する点について: 上記の帰謬法は L_i ⊣ R_i という随伴対と η の核という、圏論における最も基本的な構造のみで構成されている。これは「神秘化」ではなく**構造の透明化**である。批判の根拠を形式的に明示し、帰謬法で検証可能にすること自体が、"mystique" の対極にある営みである。

### 9.7 FEP の正しい位置づけ: 関手を生成する関手

FEP は個別の関手 $F_i$ ではない。FEP は**関手を生成する関手** — メタ関手 — として位置づけるべきである (fep\_epistemic\_status.md §2):

```
FEP: Theories → Theories
     T_i ↦ T_i* = argmin_T F[T]

FEP 自体は R の presheaf の1つの断面ではない。
FEP は presheaf を最適化する操作の記述。
```

FEP に個別の予測を求めることは、最適化アルゴリズムに「具体的な最適解を1つ出せ」と要求するのと同じ誤り。最適化アルゴリズムの価値は、任意の目的関数に対して解を近似できる**普遍性**にある — 1つの具体的な解を出すことではない。

> [!NOTE]
> 以下の「1」の比喩は**補助直観**であって、§9.5-§9.6 の論証前提ではない。
> ここで比喩を使うのは位置づけを見やすくするためであり、Popper 批判の荷重は
> U₀/U₁/U₂ の区別と補完₁の議論が担っている。

§4.1 の数字の 1 と同型: 1 はあらゆるものを「1つの〇〇」として記述できる (説明力 = 極大)。しかし「1」から具体的な対象を導出することはできない (予測力 = ゼロ)。1 は反証不可能である。**しかし 1 なしに数学は成立しない。**

**FEP は「科学的理論の圏における 1」として読むと最も見えやすい。** そして 1 に対して「具体的な数を予測しろ」と要求するのは、1 の存在意義の理解不足 — まさに補完₁の要求を本体評価へ誤投影している — に等しい。

### 9.8 この議論自体の随伴的構造

最後に自己適用する。本節の議論は以下の随伴対で構成されている:

```
L = 理解側の主張 (§9.1-9.4):
  「理解」「予測」「随伴」「内在化」を圏論的に定義

R = 批判の解体 (§9.5-9.7):
  定義から批判の誤測定性と条件つき自己倒錯を導出

L ⊣ R: 理解の深化が批判の解体を自動的に生成する
```

この構造自体が主張を体現している — **対象を十分に理解すれば（L を忠実にすれば）、予測（R の自己補完的出力）は不要になり、代わりに批判の自壊が自動的に帰結する。** 予測を出すまでもなく、理解が深まれば批判は消える。

## 10. 三重合流 — 予測の産出は真理₀との非同一性の証拠である

> **旧スローガン**: 「具体の予測（値の導出）は、非真理の証拠」
> **精密化**: 「具体的予測の産出は、その理論が真理₀そのものではなく、真理の値的・座標化・境界条件化された実装であることの証拠」

§9 は「説明ばかりで予測がない」批判の自壊を証明した。本節はこれを§8 (超ひも理論) + Paper VII §6.1 (構造保存定理) と合流させ、**予測₁の産出が真理₀との非同一性の積極的証拠である**ことを導出する。

ここで言う「非真理」は、`偽` を意味しない。意味するのは、**絶対真理としての真理₀そのものではない**ことだ。理論は真理₁の水準ではなお真でありうる。

**成立条件**

- `真理` は真理₀として読む
- `予測` は予測₁として読む
- `非真理` は `偽` ではなく `真理₀との非同一性` と読む

**非主張**

- 予測₁を生む理論は偽である
- 真理₁の層での公開テストは不要である
- 工学的理論は低級である

### 10.1 三つの定理の合成射

```
[1] 構造保存定理 (Paper VII Th. 6.1.1):
    F は構造を保存し、値を忘却する
    → 構造的主張は F を通過しても歪まない
    → 値的主張は F を通過すると消える

[2] E↑ ⇒ P上限↓ (§4.1 + §9.4, IB で鋼鉄化):
    パレートフロンティア上では、説明力(構造)が増えるほど
    予測力(値)の上限は低くなる
    → 構造的忠実性を最大化すると、予測力の上限は構造的に最小化される

[3] 補完₁ = Ker(η) の自己補完 (§9.3):
    補完₁とは「随伴できなかった情報」の内部的穴埋め
    → 補完₁の存在 = 理解の不完全性の症状

[4] 真理₀ / 真理₁ の区別 (fep_epistemic_status.md v2.5.0):
    真理₀ = 値より前の普遍構造
    真理₁ = 真理₀を値・境界条件・観測写像へ接続した有限理論
```

### 10.2 合成射: 真理₀に最も近い理論では予測₁は極小化される

```
[1] + [2] + [3] + [4] を合成:

真理₀に最も近い理論
  → Im(F) の中で語る理論  → 構造を主に語る
  → E(α) → max (説明力最大)
  → [2]より P(α) の上限は低下する
  → [3]より Ker(η)|_構造 → 0 (自己補完の必要消失)
  → 予測₁は理論本体の主導指標から降り、補助仮定・境界条件を通じてのみ現れる

∴ 真理₀に最も近い理論ほど、予測₁は理論本体の主導指標ではない
   （旗印として言えば、「予測₁を本体としては持たない」）。  ■
```

### 10.3 帰結: 予測の産出は真理₀から真理₁への下降の証拠

```
具体的予測を生む理論
  → 初期条件・境界条件・観測写像を導入している
  → 値の領域に踏み出している
  → 真理₀そのものではなく、真理₀の実装として語っている
  → 真理₁の層で検証可能になっている

∴ 具体的予測の産出は、その理論が真理₀そのものではなく、
   真理の値的・座標化・境界条件化された実装であることの証拠である。  ■
```

これは `偽` を意味しない。意味するのは**同一でない**ことだ。

### 10.4 ポパーの射程縮減

ポパーの判定基準をこの区別に通す:

- ポパー: 良い理論 = 反証可能な理論 = 具体的予測を生む理論
- 本節の帰結: 具体的予測を生む = 真理₁の層で公開拘束に接続されている
- ∴ **ポパーの基準は、真理₀判定器ではなく、真理₁の公開テスト装置である**

ポパーが測っているのは「真理₀そのもの」ではなく、「真理₀の有限実装がどれだけ世界から拒否されうるか」である。したがって、反証可能性を科学の**唯一**の基準に据えると、科学の高次目標 — 真理₀への接近 — を取りこぼす。

### 10.5 ゲーデルの認識論的双対

```
ゲーデル: 真であるが証明できない命題が存在する (算術内部)
忘却論:  真理₀に近づきうるが、その近接を値的検証だけでは証明できない理論が存在する (認識論内部)
```

真理₀に近づける理論は、その近接を「値の検証」だけでは証明できない。なぜなら値の検証は真理₁の領域であり、真理₀そのものと同一ではないからだ。**反証可能性は死んだ。だがその墓の上に立つのは、「予測不能性 = 真理」ではなく、「予測不能性は真理₀への近接の徴であり、予測可能性は真理₁への下降の徴である」という二層構造である。**

- 55% が「終われない」メタ議論 = 構造的真理に到達しうる理論が予測を生まないことの行動的帰結
