## §1. 動機: CPS の二つの顔

### 1.1 問題提起: 「容器」は何-cell か？

Paper I/II の CPS 公理系 [1, 2] と Paper VII の cell 階層 [3] は、CPS における「容器」と「内容」に一見異なる cell 次元を割り当てている:

**Paper II §2.3 の CPS テーブル (de Rham 的割当て):**

| ドメイン | A (容器的) | B (内容的) | A の cell | B の cell | Δd |
|:--|:--|:--|:--|:--|:--|
| 微積分 | d (微分) | P (積分) | 0-cell 操作 | 1-cell 操作 | 1 |
| QM | x̂ (位置) | p̂ (運動量) | 0-cell | 1-cell | 1 |
| GR | M (位相) | g (計量) | 0-cell | 2-cell | 2 |

**Paper VII §2.2 の cell 階層:**

| cell 次元 | 内容 | 例 |
|:--|:--|:--|
| 0-cell | 対象・インスタンス・**中身** | 水分子、量子状態、~/A/a |
| 1-cell | 射/関手・**忘却操作** | F, G、粗視化写像、~/A |
| 2-cell | 自然変換・**知覚者** | 観測者、近似の型の選択 |

Paper II: 容器 = 0-cell (位置 x)、内容 = 1-cell (運動量 p)。
Paper VII: 容器 = 1-cell (ディレクトリ ~/A)、内容 = 0-cell (ファイル ~/A/a)。

**反転している。** 本稿はこの矛盾を解消し、**Paper VII の割当て (容器 = 1-cell、内容 = 0-cell) が正しい**ことを論証する。

### 1.2 解消: de Rham cell ≠ 階層的 cell

矛盾の原因: Paper II の cell 割当ては**演算子の微分幾何的な型** (0-form 操作 vs 1-form 操作) を測定していた。これは容器/内容の*階層*ではなく、数学的対象の*種類*を分類する測定。

例:
- x̂ の作用: ψ(x) ↦ x·ψ(x)。掛け算 = 点ごとの操作 = 0-form 的 → Paper II は「0-cell」
- p̂ の作用: ψ(x) ↦ -iℏ∂ψ/∂x。微分 = 方向の操作 = 1-form 的 → Paper II は「1-cell」

だが「位置 x」と「位置空間 X = {全ての可能な位置}」は別物:
- x (特定の位置の値) = 0-cell (一つのインスタンス。~/A/a に対応)
- X (位置空間) = 1-cell (全インスタンスを組織する場。~/A に対応)

Paper II の Δd = |0 - 1| = 1 は、演算子の型の差を測っている。この数値自体は正しい。だが「容器 = 0-cell」というラベルは、位置の「値」と位置の「空間」を混同した結果である。

### 1.3 本稿の主張

> **容器 = 1-cell (場/空間/ディレクトリ)。内容 = 0-cell (状態/要素/ファイル)。この割当ては CPS の全インスタンスで統一的に成立する。CPS0' の非対称性 (容器 > 内容) は、米田の補題からの系として導出可能である。**

## §2. 存在と意味の分離

### 2.1 存在は全ての問いの前提

> **命題 2.1.1 (存在の先行性).** 「何であるか (WHAT)」は「在るか (THAT)」を前提する。存在しないものについて「何であるか」を問うことはできない。

これは形式論理の基礎: 述語 P(x) を評価するには、まず x が存在しなければならない。∃x は ∀P の前提。

問いの階層:

| 問いの型 | 前提 | 対応する cell 次元 |
|:--|:--|:--|
| **THAT** (存在するか) | なし。自己充足 | 1-cell (容器の存在) |
| **WHAT** (何であるか) | THAT を前提 | 0-cell (内容の同定) |
| **HOW MANY** (いくつか) | WHAT を前提 | 0-cell 間の関係 |
| **HOW** (どのようにか) | WHAT を前提 | 0-cell 間の射の構造 |
| **WHY** (なぜか) | 全てを前提 | 2-cell (知覚者の解釈) |

### 2.2 空のディレクトリの存在論

具体例で核心を明確にする。

```bash
mkdir ~/A          # ~/A は存在する。中身はない
```

この時点で:
- ~/A は**存在する** (THAT = 成立)
- ~/A は**何であるか不定** (WHAT = 不定。何を格納するディレクトリかわからない)
- ~/A は**意味がない** (関係がないため、知覚者にとって区別不能)

```bash
touch ~/A/a ~/A/b ~/A/c    # 要素を配置する
```

この時点で:
- ~/A は**何であるか決定する** (WHAT = ~/A/a, ~/A/b, ~/A/c を含むディレクトリ)
- ~/A の**意味が発生する** (要素間の関係が認知可能になる)

> **命題 2.2.1 (存在と意味の分離).** 存在は単項述語 (自己充足的) であり、意味は二項以上の関係 (他の要素との結びつき) である。存在は意味なしに成立するが、意味は存在なしに成立しない。

> **命題 2.2.2 (知覚的不定性).** 空の容器 (関係を持たない 1-cell) は存在するが、知覚者 (2-cell) にとっては不定に**見える**。これは存在の不在ではなく、関係 (0-cell 間の射) の不在である。

物理的インスタンス:
- **GR 真空解** (T_μν = 0): 時空 M は存在するが、質量-エネルギーがない。「何のための時空か」は不定。真空解が複数存在する (Schwarzschild, de Sitter, Minkowski, ...) のは、空の容器が不定であることの数学的表現
- **空のヒルベルト空間**: 形式的に well-defined だが、状態がなければ物理的に意味がない
- **機能が未定義のプログラム型**: `type A` と宣言するだけで型は存在する。値 `x: A` が居住 (inhabit) するかは別問題

### 2.3 CPS0' の再定式化

以上から、CPS0' を再定式化する:

> **CPS0' (修正版 — 存在の先行性).**
>
> CPS スパン Set_A ← C_D → Set_B において:
> - 容器 (1-cell): C_D 内で独立に well-defined。内容がなくても存在する
> - 内容 (0-cell): 容器に従属。容器なしには存在しない
>
> この非対称性は「存在 (THAT) は意味 (WHAT) に先行する」の圏論的表現であり、独立公理ではなく §4 で米田の補題から導出する。

## §3. 容器 = 1-cell、内容 = 0-cell — 全インスタンスの再検証

### 3.1 統一的 cell 割当て

Paper II §2.3 の6つの CPS インスタンスに対し、容器/内容の cell 次元を**階層的に** (de Rham 的ではなく) 再割当てする:

| ドメイン | 容器 (1-cell) | 内容 (0-cell) | 非対称性 (CPS0') |
|:--|:--|:--|:--|
| **微積分** | 滑らかな関数の空間 C^∞(M) | 各点での値 f(x₀) | 関数空間は点の値なしに存在。点の値は関数空間を前提 |
| **QM** | 位置空間 X (構成空間) | 量子状態 \|ψ⟩ ∈ H | 空間は状態なしに存在 (真空)。状態は空間を前提 |
| **GR** | 多様体 M (位相構造) | 計量 g + 物質 T_μν | M は g = 0, T = 0 で存在 (裸の多様体)。g, T は M を前提 |
| **心身** | 身体 (b, η) | 心 μ | 岩は身体のみで存在。心は身体を前提 |
| **力の統一** | ゲージ群 G の構造 | 場の配位 A_μ | ゲージ対称性は場の具体値に先行 |
| **知識索引** | インデックス構造 | 個別知識エントリ | 索引は空でも存在。エントリは索引を前提 |

### 3.2 ディレクトリ構造としての統一的読み

Paper VII §2.3 のディレクトリ・アナロジーが全インスタンスに適用される:

```
~/QM/           = 位置空間 X (1-cell, 容器)
~/QM/ψ₁        = 量子状態 |ψ₁⟩ (0-cell, 内容)
~/QM/ψ₂        = 量子状態 |ψ₂⟩ (0-cell, 内容)
~/QM/ψ₁ → ψ₂  = 状態遷移 (0-cell 間の射)

~/GR/           = 多様体 M (1-cell, 容器)
~/GR/g          = 計量テンソル (0-cell, 内容)
~/GR/T          = エネルギー・運動量 (0-cell, 内容)
~/GR/g → T     = Einstein 方程式 (0-cell 間の射)
```

### 3.3 Paper II の Δd との整合性

Paper II の Δd = |dim ker U_A - dim ker U_B| は演算子の微分幾何的な型の差を測定している。これと本稿の階層的 cell 割当ては**異なる測定**だが、**整合的**:

| 測定 | 何を測っているか | 容器の cell | 内容の cell | Δd |
|:--|:--|:--|:--|:--|
| de Rham 的 (Paper II) | 演算子の微分幾何的な型 | 0 (掛算) | 1 (微分) | 1 |
| 階層的 (本稿) | 存在の従属関係 | 1 (場) | 0 (要素) | 1 |

Δd の数値が一致するのは偶然ではない。de Rham 的 cell と階層的 cell は**ポアンカレ双対性的な関係**にある [予想]:

> **予想 3.3.1 (Cell 双対性).** CPS スパンにおいて、de Rham 的 cell 次元 d_dR と階層的 cell 次元 d_hier は d_dR + d_hier = n (全体の cell 次元) を満たす。Δd は両方の測定で同一値を返す。

QM (n=1): d_dR(容器) = 0, d_hier(容器) = 1. 0 + 1 = 1. ✓
GR (n=2): d_dR(容器) = 0, d_hier(容器) = 2? (多様体は 0-cell 的位相 + 計量の余次元)

> [推定 55%] この予想は成立するが、GR の場合は Δd = 2 であり、容器 (M) と内容 (g, T) の間に中間的な 1-cell 構造 (接ベクトル場、Christoffel 記号) が存在する。3層の cell 階層はこの中間構造を自然に収容する。

### 3.4 ファイバー束としての CPS

統一的 cell 割当ては**ファイバー束** (fiber bundle) の構造と自然に対応する:

```
E (全空間 = CPS の C_D)
│ π (射影 = U_ctr = 内容を忘れて容器だけ残す)
↓
B (底空間 = 容器 = 1-cell)
```

各ファイバー π⁻¹(b) = 位置 b における内容の全体 = 0-cell の集まり。

CPS の構成要素:
- **U_ctr** = 射影 π: E → B (内容を忘れ、容器を保持) = reductive forgetting
- **U_cnt** = ファイバーだけ取り出す操作 (容器を忘れ、内容を保持) = destructive forgetting (底空間なしにファイバーは定義不能 → CPS0')
- **CPS2 (架橋)** = 平行移動 (異なるファイバーの点を結ぶ水平射)
- **力 (Paper I)** = 接続の曲率 = 平行移動の経路依存性 = 忘却の不均一

> **命題 3.4.1 (CPS-ファイバー束対応).** CPS スパンは底空間 B (容器, 1-cell) とファイバー (内容, 0-cell) からなるファイバー束として自然に実現される。CPS0' は「底空間はファイバーなしに存在しうる (裸の多様体) が、ファイバーは底空間なしに定義不能」の表現である。

## §4. CPS0' の米田的導出

### 4.1 米田の補題の復習

米田の補題 [4, エッセイ §4.2]:

> 対象 X の全情報は、X から他の全対象への射の集まり Hom(−, X) で完全に復元される。

圏論的に: 米田埋込 y: C → [C^op, Set] は充満忠実。y(X) = Hom(−, X)。

直感的に:
- 対象 (0-cell) は射 (1-cell 的構造) によって**決定される**
- 射は対象よりも**情報量が多い** (エッセイ L139: 「構造 (射) を見ることは、要素 (対象) を見ることより原理的に情報量が多い」)
- 対象は射から復元できるが、射は対象からは復元できない

### 4.2 CPS0' の導出

> **定理 4.2.1 (CPS0' の米田的導出).**
>
> 圏 C において、1-cell (射の構造) は 0-cell (対象) を完全に決定する (米田の補題)。逆は不成立 (対象だけでは射は復元できない)。したがって:
>
> (i) 1-cell (容器) は 0-cell (内容) なしに形式的に well-defined (空の presheaf は定義可能)
> (ii) 0-cell (内容) は 1-cell (容器) なしに存在不能 (射なしの対象は identifiable でない)
>
> これは CPS0' の主張そのものである。□

**注記 (空の容器の不定性).** 定理 4.2.1 (i) において「well-defined」は「存在する」の意であり「決定される」の意ではない。空の presheaf Hom(−, X) (射がゼロ) は well-defined だが、X の identity は不定。§2.2 の空のディレクトリと同型: 存在するが、何であるかわからない。

> **系 4.2.2 (CPS0' は公理ではなく定理).** CPS0' (容器 > 内容) は CPS の独立公理ではなく、米田の補題からの系である。CPS の独立公理は CPS1 (相補的射影)、CPS2 (架橋)、CPS5 (マスク) の3つに削減される。

### 4.3 存在とは何か — 圏論的定式化

§2.1 の「存在は全ての問いの前提」を圏論的に表現する:

> **定義 4.3.1 (圏論的存在).** 圏 C における対象 X の「存在」とは、X が C の対象クラス Ob(C) に属することである。これは C の**定義** (メタ構造) の一部であり、C 内の射の問題ではない。

> **命題 4.3.2 (存在のメタ性).** 存在 (∈ Ob(C)) は圏のメタ情報であり、圏内の射 (Hom) で生成されるものではない。米田の補題は Hom(−, X) で X を**同定**するが、X の**存在**は前提している。

この命題は §6 (米田の補題は存在を忘却している) への伏線である。

## §5. F = mg — CPS0' と力の統合

### 5.1 忘却は一様に作用し、構造が結果を分ける

§2-4 の結果と Paper I [1] の中心定理 (力 = 忘却の勾配 ∇U) を接続する。ここで決定的に重要なのは、Paper I §5.9.1 が証明した **F = mg の忘却論的成立** (Cauchy-Binet ノルム分離) である。

Paper I §5.9.1 の対応表 (L484-488):

| 力学 | 記号 | 忘却論 | Paper I |
|:--|:--|:--|:--|
| 重力加速度 (定数) | g | **広義忘却** (全要素に MECE に作用) | α (忘却強度) |
| 質量 (対象の性質) | m | **忘却感受性** (射の密度の逆数) | ‖T‖_g (Chebyshev ノルム) |
| 力 (方向を持つ結果) | F = mg | **狭義忘却** (方向的淘汰) | F_ij = (α/2)[d(ΦT)]_ij |

そのノルムレベルでの分離 (Paper I §5.9.1, L492-496):

$$\|F\| = g_{\text{eff}} \cdot \|T\|_g$$

ここで $g_{\text{eff}} := (\alpha/2)\|d\Phi\||\sin\theta|$。

Paper I L500 の核心:

> 「$g$ は物体を区別しない。落下の速さが異なるのは空気抵抗 — すなわち物体の側の性質 — による。同様に、広義忘却 $g_{\text{eff}}$ は情報を区別しない。**「何が忘れられるか」は情報の側の性質 — 圏における射の密度、すなわち Chebyshev ノルム $\|T\|_g$ — が決める。**」

これは CPS0' の動的表現に他ならない:

> **命題 5.1.1 (忘却の一様性と力の非一様性).**
>
> - **忘却の作用** (g) は一様に存在する。忘却は対象を区別しない。重力加速度が全質点に等しく作用するのと同型
> - **忘却の結果** は対象の側の性質 — すなわち対象が有する**射の密度** (m = ‖T‖_g) — に依存する。射が密な構造は忘却に耐え、射が疎な構造は忘却される
> - **力** = 一様な忘却 (g) × 対象固有の構造的感受性 (m) = **F = mg**

### 5.2 三層構造: CPS0' × F=mg の統合

CPS0' と F=mg を統合すると、三層の構造が現れる:

| 層 | CPS での位置 | F=mg での位置 | 内容 |
|:--|:--|:--|:--|
| **1-cell (場/容器)** | 容器: 内容なしに存在しうる (CPS0') | g の作用場: 忘却が一様に作用する空間 | 時空 M、ディレクトリ ~/A、位置空間 X |
| **0-cell (要素/内容)** | 内容: 容器に従属 (CPS0') | m の担い手: 射の密度 ‖T‖_g が忘却感受性を決定 | 量子状態、ファイル ~/A/a、計量 g |
| **2-形式 (力/曲率)** | CPS2 (架橋): U_A と U_B を結ぶ | F = mg: 一様な忘却 × 構造的感受性 | 重力、電磁力、認知力 |

> **定理 5.2.1 (CPS0' と力の統一).**
>
> CPS0' (容器 > 内容) と Paper I の力の定理 (力 = ∇U) は、同一の構造の三つの面である:
>
> - **CPS0'** = 「場 (1-cell) は要素 (0-cell) に存在論的に先行する」 (存在論的面)
> - **広義忘却** = 「忘却は場 (1-cell) 上で一様に作用し、対象を区別しない」 (作用の面 = g)
> - **力 = ∇U** = 「忘却の結果は、忘却対象の有する射 (構造) の密度に依存する」 (結果の面 = m)
>
> 三つを合成すると: **場が存在し (CPS0')、その場の上で忘却が一様に作用し (g)、対象の構造が結果を決定する (m)。この全体が力 F = mg である。**

### 5.3 物理的インスタンスの再検証

| ドメイン | 場 (1-cell) = g の作用空間 | 内容 (0-cell) = 忘却感受性 m | 力 F = mg |
|:--|:--|:--|:--|
| **GR** | 時空 M (至る所定義) | 質量-エネルギー T_μν (場所ごとに異なる密度 ‖T‖_g) | 重力 = G_μν (曲率) |
| **EM** | 電磁場 (至る所定義) | 電荷分布 ρ (電荷の射構造に依存) | 電磁力 = dA |
| **QM** | 位置空間 X (至る所定義) | 波動関数 ψ(x) (量子数 = 射の密度が忘却されにくさを決定) | 量子力 = -∇V |
| **FEP** | パラメータ空間 (至る所定義) | 信念分布 q(θ) (精度 π = 射の密度の proxy) | 認知力 = ∇F |

### 5.4 Einstein 方程式の CPS 的解釈

Einstein 方程式 G_μν = 8πG T_μν の構造的意味:

> **左辺** G_μν: 容器 (時空) の曲率 = **忘却が一様に作用した結果の歪み** (g が場を歪める)
> **右辺** T_μν: 内容 (物質) の分布 = **忘却感受性の分布** (m の配置)
> **等号**: **F = mg の場の方程式版**。一様な忘却が、非一様な構造密度に遭遇して、非一様な力を生む

CPS0' は「容器は内容なしに存在しうる」と述べるが、「容器は内容に無反応である」とは言わない。真空解 (T = 0) でも曲率はゼロとは限らない (Schwarzschild) — 容器は過去の内容の「痕跡」を保持しうる。これは g が m = 0 のとき F = 0 になるニュートン力学とは異なる GR 的な特徴であり、CPS の場合も同様: 空の容器が「かつて内容を持っていた」痕跡を保持する可能性がある。

### 5.5 F = mg の自己参照構造 (Paper I §5.9.5)

Paper I §5.9.5 が示した通り、ノルム射影 ‖·‖: Ω²(M) → C^∞(M) 自体が**方向情報を忘却する関手**である:

**F=mg は忘却理論の自己適用である** — 忘却曲率テンソルに忘却関手 (ノルム射影) を適用した結果がニュートン的スカラー方程式を生む。理論が自分自身を記述する。

これは CPS0' の自己参照的帰結:
- テンソル F_ij (方向+大きさ) → スカラー ‖F‖ (大きさのみ) への忘却は、**内容 (方向) を忘れて容器 (大きさ) のみを保持する** U_ctr 操作
- ニュートン力学は GR の CPS0' 忘却 (容器のみ保持) の結果として導出される
- 忘却論は、自身を忘却することでニュートン力学を復元する

## §6. 米田の補題は「存在」を忘却している

### 6.1 米田の盲点

米田の補題 [4, エッセイ §4.2]:

> X の全情報は Hom(−, X) で復元される。

だが命題 4.3.2 が指摘した通り: 米田の補題は X の**同定** (WHAT) を与えるが、X の**存在** (THAT) は前提している。

> **問い 6.1.1.** 米田の補題は「存在」という情報を忘却していないか？

Hom(−, X) は X が Ob(C) に属することを暗黙に仮定している。存在しない対象に対して Hom(−, X) は定義されない。つまり:

> **米田の補題は WHAT について完全だが、THAT について沈黙している。**

これは CPS0' の構造そのもの:
- 内容 (WHAT = Hom(−, X) で決まる = 0-cell) は容器 (THAT = X ∈ Ob(C) = 1-cell) を**前提する**
- 米田の補題は内容のレベルの定理であり、容器 (存在) のレベルには触れない

| レベル | 圏論 | 米田 | CPS / 忘却論 |
|:--|:--|:--|:--|
| 存在 (THAT) | X ∈ Ob(C) | **暗黙の前提** (忘却されている) | CPS0': 容器 (1-cell) の独立存在性 |
| 同定 (WHAT) | Hom(−, X) | **完全に決定** (米田の補題) | 0-cell 間の射による内容の決定 |
| 解釈 (WHY) | 2-cell (自然変換) | 関手間の変換 | 知覚者 (2-cell) の主観 |

本節以降では、この「盲点」を厳密に定式化し、忘却パラメータ α によって米田の補題がどのように拡張されるかを論証する。

### 6.2 α-忘却濾過と段階的同定

忘却パラメータ α ∈ [0,1] を用いて、射の構造が段階的に失われる枠組みを構築する。中心概念は**広部分圏の濾過的族** (filtration of wide subcategories) である。

> **定義 6.2.1 (α-忘却濾過).** 局所小圏 C に対し、**α-忘却濾過** (α-oblivion filtration) とは、広部分圏の族 {C_α}_{α ∈ [0,1]} で以下を満たすものをいう:
>
> (F1) **不変対象**: 各 α に対し Ob(C_α) = Ob(C)
>
> (F2) **境界・完全**: C_0 = C
>
> (F3) **境界・離散**: C_1 = C_disc（射は恒等射のみ）
>
> (F4) **単調性**: α₁ ≤ α₂ ⟹ Mor(C_{α₂}) ⊆ Mor(C_{α₁})
>
> ここで広部分圏 (wide subcategory) D ⊆ C とは Ob(D) = Ob(C) かつ Mor(D) ⊆ Mor(C) なる部分圏をいう。(F4) は射のクラスの包含を意味し、α の増大に伴い利用可能な射が単調に減少することを表す。

**注記.** (F1)–(F4) は α-忘却濾過の存在を主張するものであり、一意性は主張しない。同じ圏 C に複数の α-忘却濾過が存在しうる。これは物理的に自然である: 同じ対象系に対して、忘却の「経路」（どの射が先に失われるか）は一意でない。

> **定義 6.2.2 (α-米田埋込).** α-忘却濾過 {C_α} の各 α に対し、**α-米田埋込** y_α を C_α の米田埋込として定義する:
>
> $$y_\alpha: C_\alpha \to \mathrm{PSh}(C_\alpha), \quad y_\alpha(X)(Y) := \mathrm{Hom}_{C_\alpha}(Y, X)$$
>
> y_α(X) は C_α における X の表現可能前層 (representable presheaf) である。

> **定理 6.2.3 (段階的同定定理).** α-忘却濾過 {C_α} に対し:
>
> **(i) 完全同定** (α = 0): y_0: C → PSh(C) は充満忠実。X の C における全構造は y_0(X) = Hom_C(−, X) で完全に決定される。**これが米田の補題に他ならない。**
>
> **(ii) 部分的同定** (0 < α < 1): y_α: C_α → PSh(C_α) は C_α 上で充満忠実（C_α 自身への米田の補題の適用）。しかし C_α ⊊ C であるため:
>
> - Hom_{C_α}(Y, X) ⊆ Hom_C(Y, X) は一般に真の包含
> - C の射に依存する構造的性質（自己同型群 Aut_C(X)、高次ホモトピー情報等）は、C_α に含まれる射のみで計算可能な部分に制限される
> - C_α-自己同型群 Aut_{C_α}(X) ⊆ Aut_C(X) は一般に真の部分群
>
> **(iii) 存在のみ** (α = 1): C_1 = C_disc なので y_1(X) = δ_X（クロネッカー前層）:
>
> $$y_1(X)(Y) = \begin{cases} \{id_X\} & (Y = X) \\ \varnothing & (Y \neq X) \end{cases}$$
>
> 異なる対象 X ≠ Y に対し δ_X ≠ δ_Y。**対象は「在る」が「何であるか」は不定。** §2.2 の空のディレクトリ ~/A と同型の状態。
>
> **(iv) 存在の不変性**: 全ての α ∈ [0,1] に対し Ob(C_α) = Ob(C)。**存在は忘却の影響を受けない。**

*証明.* (i) C_0 = C (F2) より y_0 は C の米田埋込。米田の補題 [4] により充満忠実。

(ii) ι_α: C_α ↪ C を包含関手とする。C_α は広部分圏であるから Ob(C_α) = Ob(C) (F1) かつ Mor(C_α) ⊆ Mor(C) (定義) であり、ι_α は恒等対象写像かつ射に対して忠実である。C_α は圏であるから（合成は C の合成の制限であり、恒等射は Ob(C_α) = Ob(C) より全て含まれる）、米田の補題を C_α 自身に適用して y_α: C_α → PSh(C_α) は充満忠実を得る。

射の減少について: α₁ < α₂ のとき (F4) より Mor(C_{α₂}) ⊆ Mor(C_{α₁}) であるから、**制限関手** Res^{α₁}_{α₂}: PSh(C_{α₁}) → PSh(C_{α₂}) が ι_{α₂,α₁}: C_{α₂} ↪ C_{α₁} の前合成で定義される。この制限は一般に忠実ではない: C_{α₁} の射 f ∈ Hom_{C_{α₁}}(Y, X) \ Mor(C_{α₂}) は y_{α₂}(X) に寄与しない。したがって y_{α₂}(X) から y_{α₁}(X) を一般に復元できず、これが「部分的同定」の圏論的意味である。

自己同型群について: Aut_{C_α}(X) = {f ∈ Aut_C(X) | f ∈ Mor(C_α)} は Aut_C(X) の部分群であり、(F4) より α₁ ≤ α₂ ⟹ Aut_{C_{α₂}}(X) ⊆ Aut_{C_{α₁}}(X)。特に Aut_{C_1}(X) = {id_X} (F3) である。

(iii) (F3) より C_1 = C_disc。離散圏の Hom 集合は Hom_{C_disc}(Y, X) = {id_X} (Y = X), ∅ (Y ≠ X)。よって y_1(X)(Y) = Hom_{C_1}(Y, X) = δ_{XY}。X ≠ Y ならば δ_X(X) = {id_X} ≠ ∅ = δ_Y(X) より δ_X ≠ δ_Y。すなわち y_1 は対象を区別するが射の構造情報を一切持たない。

(iv) (F1) の直接的帰結。□

> **系 6.2.4 (CPS0' の α-忘却的導出).** 定理 6.2.3 は CPS0'（容器 > 内容）の構造を正確に再現する:
>
> - **容器 (1-cell = 存在)**: Ob(C_α) は α に依存しない (iv) → 存在は忘却不変
> - **内容 (0-cell = 同定)**: Hom_{C_α}(Y, X) は α の増大に伴い縮小 (ii) → 同定は忘却に従属
>
> したがって CPS0' は α-忘却濾過の構造的帰結であり、**米田の補題は α = 0 の特殊ケースとして CPS に包含される。** 旧予想 6.2.1（忘却論 ⊃ 米田）はこれにより定理に昇格する。

**備考 6.2.5 (Level-Cell 同型).** 系 6.2.4 の cell 階層は、Paper XII §1.3 の Level 階層と正確に対応する:

| Paper XII | Paper VIII | α 依存性 |
|:---|:---|:---|
| Level 0 (状態・構造) | 1-cell = 容器 = Ob(C) | (F1) 不変 |
| Level 1 (値・力) | 0-cell 関係 = Mor(C_α) | (F4) α 非増加 |
| Level 2 (拘束速度) | 射の合成則 = Mor 間構造 | (F4) 従属 |

**帰結:** Paper XII の速度制限は Level 1 (= 射) を束縛するが、Level 0 (= 対象) を束縛しない。対象の「在る」は α に依存しないが ((F1))、射の「つながる」は α に依存する ((F4))。front 速度が因果速度を超えうるのは、front が Level 0 の量 (対象の配置) であり Level 1 の速度制限を受けないためである。

### 6.3 有限主体の定理 — 部分的前層

Paper VII §1.3 の核心的観察: 「世界 S のインスタンスは無限だが、観測者の容量は有限」。これを α-忘却濾過の枠組みで圏論的に定式化する。

> **定義 6.3.1 (κ-有限観測).** 圏 C の対象 X に対し、観測容量 κ < ∞ の主体 O の**κ-有限観測** とは、射の部分族
>
> $$J_\kappa(X) \subseteq \coprod_{Y \in \mathrm{Ob}(C)} \mathrm{Hom}_C(Y, X), \quad |J_\kappa(X)| \leq \kappa$$
>
> の選択をいう。J_κ(X) は X に対して O が実際にアクセスする射の集合であり、**部分的前層** (partial presheaf) $\tilde{y}_\kappa(X)(Y) := J_\kappa(X) \cap \mathrm{Hom}_C(Y, X) \subseteq y(X)(Y)$ を定義する。

> **定理 6.3.2 (有限主体定理).** 観測容量 κ < ∞ の有限主体 O に対し、実効忘却パラメータ
>
> $$\alpha_O(X) := \max\!\left(0,\; 1 - \frac{\kappa}{\left|\coprod_Y \mathrm{Hom}_C(Y, X)\right|}\right)$$
>
> は次を満たす:
>
> (i) $\kappa \geq |\coprod_Y \mathrm{Hom}_C(Y, X)|$ ならば $\alpha_O(X) = 0$: 主体は X の全射を観測可能 → **米田の補題が適用される**
>
> (ii) $\kappa < |\coprod_Y \mathrm{Hom}_C(Y, X)|$ ならば $\alpha_O(X) > 0$: 主体は射を部分的にしか観測できない → **同定は不完全**
>
> (iii) $|\coprod_Y \mathrm{Hom}_C(Y, X)| \to \infty$ のとき $\alpha_O(X) \to 1$: **任意の有限主体にとって、十分に複雑な対象は「存在するが同定不能」となる**

*証明.* まず α_O(X) の well-definedness を確認する。κ ∈ ℕ かつ |∐_Y Hom_C(Y, X)| は Ob(C) 上の Hom 集合の直和の濃度であり、C が局所小であれば（仮定 6.2.1）各 Hom_C(Y, X) は集合であるから、∐_Y Hom_C(Y, X) もまた集合である。無限の場合を含めて α_O(X) ∈ [0,1] を確認する: κ/|∐_Y Hom| ≥ 0 であるから 1 − κ/|∐_Y Hom| ≤ 1、かつ max(0, ⋯) ≥ 0 より α_O(X) ∈ [0,1]。

(i) κ ≥ |∐_Y Hom_C(Y, X)| のとき κ/|∐_Y Hom| ≥ 1 であるから 1 − κ/|∐_Y Hom| ≤ 0 より α_O(X) = max(0, ⋯) = 0。このとき J_κ(X) = ∐_Y Hom_C(Y, X) と取れるから $\tilde{y}_κ(X) = y(X)$ となり、標準の米田の補題が適用される。

(ii) κ < |∐_Y Hom_C(Y, X)| のとき 0 < κ/|∐_Y Hom| < 1 であるから α_O(X) = 1 − κ/|∐_Y Hom| ∈ (0,1)。部分的前層 $\tilde{y}_κ(X)$ は y(X) の真の部分前層であり、$\tilde{y}_κ(X)(Y) \subsetneq y(X)(Y)$ なる Y が存在する。よって $\tilde{y}_κ$ からの X の同定は不完全であり、定理 6.2.3 (ii) の部分的同定に対応する。

(iii) 任意の ε > 0 に対し N_ε := ⌈κ/ε⌉ + 1 と置く。|∐_Y Hom_C(Y, X)| ≥ N_ε ならば κ/|∐_Y Hom| ≤ κ/N_ε < ε であるから α_O(X) = 1 − κ/|∐_Y Hom| > 1 − ε。よって |∐_Y Hom_C(Y, X)| → ∞ のとき α_O(X) → 1。極限では定理 6.2.3 (iii) の「存在するが同定不能」状態に漸近する。

単調性: α_O(X) は |∐_Y Hom_C(Y, X)| の単調増加関数である。これは対象 X の「圏論的複雑性」（X への射の総数）が大きいほど、有限主体にとっての忘却が強まることを意味する。□

**注記 6.3.3 (CPS は有限主体にとっての米田の補題).** 定理 6.3.2 は旧 §6.2 の直感を厳密化したものである:

> 米田の補題は「全ての射を見れば対象がわかる」と言う。だが有限な主体は全ての射を見ることができない。完全な Hom(−, X) が得られるなら米田に一致し (i)、部分的な Hom しか得られないなら CPS が米田を拡張する (ii)。対象が複雑になるほど忘却が強まる (iii)。

**注記 6.3.4 (Paper IV との接続).** Paper IV の効果量減衰定理 (Theorem 3.1.1): $r_{\text{obs}} \leq \sqrt{\rho / (K+1)}$ は、有限主体定理の統計的表現である。ベンチマーク知識次元 K が増大すると $|\coprod_Y \mathrm{Hom}|$ が増大し、有限な評価者の $\alpha_O \to 1$ となる。結果として観測可能な効果量 r が理論的上界に向かって減衰する。**「複雑なシステムほど忘却が強い」— これは有限主体定理 (iii) の定量的確認に他ならない。**

### 6.4 Inoué 独立性定理との接続

Inoué (2026) [6] は、層のトポス Sh(S¹)（円周 S¹ 上の層の圏）において、**内部的に inhabited だが大域的切断 (global section) を持たない対象**の存在を証明した。具体的には、S¹ の非自明被覆空間の局所切断の層が、内部論理では存在量化を満たすが大域的要素を持たない。

> **命題 6.4.1 (Inoué-CPS0' 対応).** Inoué が構成する Sh(S¹) の対象 E は、CPS0' の圏論的インスタンスである:
>
> (i) **内部的存在** (internally inhabited): トポスの内部論理で $\exists x.\; x \in E$ が証明可能 → **容器は存在する** (THAT ✓)
>
> (ii) **大域的名前の不在** (no global element): $\Gamma(E) = \mathrm{Hom}(1, E) = \varnothing$ → **内容は大域的に決定されない** (global WHAT ✗)
>
> (iii) **α-忘却的解釈**: Grothendieck 位相が忘却パラメータ α の役割を果たす。局所的には射が存在する（開被覆上の切断が存在する）が、大域への持ち上げにおいて構造が失われる。これは α-忘却濾過の圏論的実現に他ならない

| Inoué の4概念 | 圏論的レベル | 忘却論 (CPS) | Paper VIII (cell 階層) |
|:--|:--|:--|:--|
| Existence (存在) | 内部論理レベル (∃) | THAT | 1-cell (容器) |
| Object (対象) | 対象レベル (Ob) | THAT の担い手 | 容器のインスタンス |
| Meaning (意味) | 射レベル (Hom) | WHAT | 0-cell (内容) |
| Name (名前) | 要素レベル (global element) | WHY (知覚者による同定) | 2-cell (知覚者) |

Inoué の独立性定理は、これら4概念が「一方から他方を一様に復元できない」ことを証明する。CPS0'（容器 > 内容）はこの独立性の存在—意味間の射影に対応し、忘却論は Inoué の結果の独立した再発見として位置づけられる。

**注記 6.4.2 (Grothendieck 位相と α-忘却濾過の関係).** Inoué の構成が示唆する自然な問い: **α-忘却濾過 {C_α} から Grothendieck 位相を正準に誘導できるか？** 各 α に対し、C_α の射のなす篩 (sieve) が被覆構造を定義する可能性がある。この接続が確立されれば、α-忘却濾過は Grothendieck トポスの理論に直接接続される。→ OP-VIII-1。

### 6.5 統合: α-米田、CPS0'、F=mg

§4–6 の結果を統合する。

> **定理 6.5.1 (忘却の三面定理).** α-忘却濾過 {C_α}、CPS0' (定理 4.2.1)、力の定理 F = mg (定理 5.2.1) は同一構造の三つの面である:
>
> | 面 | 定理 | 内容 | α との対応 |
> |:--|:--|:--|:--|
> | **存在論的面** | CPS0' (Th. 4.2.1) | 容器 (1-cell) は内容 (0-cell) に先行する | Ob(C_α) は α-不変、Hom は α-従属 |
> | **認識論的面** | 段階的同定 (Th. 6.2.3) | 忘却パラメータ α が同定精度を支配する | α = 0 で米田、α > 0 で部分的同定 |
> | **力学的面** | F = mg (Th. 5.2.1) | 力 = 一様忘却 × 構造的感受性 | α = g_eff、m ∝ |Hom(−,X)| |
>
> 三面の統合:
>
> **存在** (Ob(C), α-不変) の上で、**忘却** (α) が一様に作用し、対象の**構造的密度** (|Hom(−, X)|) に応じて同定精度が減衰する。これが力 F の創発機構であり、CPS0' はその存在論的前提条件である。

*証明.* 存在論的面: CPS0' は定理 4.2.1 で証明済み。その核心は Ob(C_α) の α-不変性 (F1) と Hom_{C_α} の α-従属性 (F4) であり、これは「容器は忘却に先行する」の圏論的表現である。

認識論的面: 定理 6.2.3 の (iv) と (ii) の組合せ。(iv) により対象の存在は α に不変であり、(ii) により対象の同定精度は Hom_{C_α}(−, X) の「大きさ」に依存する。α の増大 (F4) は Hom_{C_α} を単調に縮小させるから、同定精度は α の単調減少関数である。

力学的面: §5 の定理 5.2.1 において g_eff = (α/2)‖dΦ‖|sin θ| は α に比例し全対象に一様に作用する。これは α-忘却濾過の「全ての対象に同じ α で射が除去される」(F1: 対象不変) に対応する。一方 m = ‖T‖_g は対象固有量であり、α-忘却濾過において |Hom_{C_α}(−, X)| が X に依存して異なる速度で減衰することに対応する。力 F = mg は「一様な忘却 (g) が対象固有の構造密度 (m) に作用して生む」形式であり、これは α-忘却濾過における「α × |Hom(−, X)|」の積構造に他ならない。

三面の整合性: いずれも同一の {C_α}_{α∈[0,1]} を基底構造とし、存在論的面は Ob(C_α) の不変性を、認識論的面は Mor(C_α) の縮小を、力学的面は Ob × Mor の積構造を記述する。よって三面は同一構造の異なる射影である。□

> **系 6.5.2 (米田の補題の忘却論的位置づけ).** 米田の補題は忘却パラメータ α = 0 の特殊ケースとして、忘却の三面定理 (Th. 6.5.1) に含まれる。忘却論は米田の補題を α > 0 に拡張し、**有限な主体にとっての圏論**を記述する理論的枠組みを提供する。

> **系 6.5.3 (忘却の階層的構造 — Paper VII §6.3 との統合).** Paper VII 定理 6.3.1 の普遍フィルトレーション
>
> $$U_{\text{arrow}}(1) \leq U_{\text{compose}}(1.5) \leq U_{\text{depth}}(2) \leq U_{\text{precision}}(3) \leq U_{\text{causal}}(4) \leq U_{\text{context}}(\infty\!-\!1) \leq U_{\text{adjoint}}(\infty) \leq U_{\text{self}}(\omega)$$
>
> は α-忘却濾過 {C_α} の離散的骨格 (discrete skeleton) として解釈される。括弧内の数値は圏論的 cell 次元であり、α-忘却濾過における離散化された α 値に対応する:
>
> | U パターン | cell 次元 | α-忘却濾過における役割 | CPS0' との関係 |
> |:--|:--:|:--|:--|
> | U_arrow | 1 | 射を忘却 → 対象間の関係が消失 | 1-cell (容器) の内部構造の最初の損傷 |
> | U_compose | 1.5 | 合成律を忘却 → 射の連鎖が切断 | 容器の構造的一貫性の喪失 |
> | U_depth | 2 | 自然変換を忘却 → 関手間の比較が不能 | 2-cell (知覚者) の機能停止 |
> | U_precision | 3 | Hom の豊穣構造を忘却 → 精度情報が消失 | 段階的同定 (Th. 6.2.3 (ii)) の離散版 |
> | U_self | ω | 自己関手の双方向性を忘却 → 自己参照が崩壊 | α → 1: 存在のみ残る (Th. 6.2.3 (iii)) |
>
> 統合: Paper VII のフィルトレーションは「何がどの順序で忘却されるか」を記述し、Paper VIII の α-忘却濾過は「忘却がどの程度進行したか」を連続パラメータで記述する。両者は離散 ↔ 連続の関係にあり、U パターンの cell 次元を α ∈ [0,1] に正規化する写像 α(n) = n/ω が自然な接続を与える。

### 6.7 Paper III ↔ VIII 架橋関手: 情報幾何と忘却濾過の接続

Paper III は Amari の α-接続パラメータ α_III ∈ ℝ を用いて CPS 圏の全域構造を展開した。一方、本稿 §6.2 の α-忘却濾過は α_VIII ∈ [0,1] をパラメータとする。両者は記号が共通しているが異なる対象である（Paper III §1.3 記号上の注意参照）。本節では両者を数学的に接続する関手を構成する。

> **定義 6.7.1 (正規化写像).** 写像 η: ℝ → [0,1] を
>
> $$\eta(\alpha_{\mathrm{III}}) := \frac{1}{1 + e^{-\alpha_{\mathrm{III}}}}$$
>
> と定義する（ロジスティック・シグモイド）。η は以下を満たす:
>
> (i) η は単調増加 C^∞-微分同相 η: ℝ → (0,1)
>
> (ii) η(0) = 1/2 — Paper III の臨界点 α_III = 0 が忘却の中間状態に対応
>
> (iii) lim_{α_III → -∞} η(α_III) = 0 — e-接続極限が完全忘却なし (C_0 = C) に対応
>
> (iv) lim_{α_III → +∞} η(α_III) = 1 — m-接続極限が完全忘却 (C_1 = C_disc) に対応

**注記 6.7.2 (物理的解釈).** α_III > 0（m-接続/ボソン的）ほど情報の複製が容易であり、忘却論的には「忘却が強い」（α_VIII が大きい）状態に対応する。これは逆直感的に見えるが、α_III > 0 における複製容易性は、まさに情報の個体性（非可換な意味構造）を忘却することで可能になる。一方 α_III < 0（e-接続/フェルミオン的）では複製が不可能であり、情報の非可換構造が保存される = 忘却が弱い。

> **定義 6.7.3 (架橋関手).** Paper III の Z₂-次数付き CPS 圏を CPS^{Z₂}、Paper VIII の α-忘却濾過の全体を Filt([0,1], WdSub(C))（[0,1]-パラメータ付き広部分圏の濾過族の圏）とする。**架橋関手** B を以下で定義する:
>
> $$B: \mathrm{CPS}^{Z_2} \to \mathrm{Filt}([0,1], \mathrm{WdSub}(C))$$
>
> 対象レベル: CPS^{Z₂} の対象 (X, |X|) を、η を通じた α-忘却濾過 {C_{η(α_III)}} における X の像に写す。
>
> 射レベル: CPS^{Z₂} の射 f: X → Y（Z₂-次数 |f| を持つ）を、以下の規則で濾過族の間の自然変換に写す:
>
> - |f| = 0（偶次数/ボソン的）: f は Mor(C_{η(α_III)}) に属する ⟺ η(α_III) ≤ α_f（射 f の「生存閾値」）
> - |f| = 1（奇次数/フェルミオン的）: f は Mor(C_{η(α_III)}) に属する ⟺ η(α_III) ≤ α_f ∧ 反可換構造が保存される

> **命題 6.7.4 (架橋関手の整合性).** 架橋関手 B は以下の意味で Paper III と Paper VIII を整合的に接続する:
>
> (i) **臨界点の対応**: α_III = 0（Paper III の相転移点）は α_VIII = η(0) = 1/2 に対応し、これは定理 6.2.3 の「部分的同定」の中間点である。
>
> (ii) **ボソン-フェルミオン対称性の保存**: Paper III の Z₂-次数付き対称性 f ⊗ g = (-1)^{|f||g|} g ⊗ f は、α-忘却濾過において η(α_III) > 1/2 の領域での偶次数射の生存と η(α_III) < 1/2 の領域での奇次数射の生存として分離される。
>
> (iii) **三重非複製定理との整合**: Paper III 定理 5.1.1（Pauli 排他律 / No-cloning / データ処理不等式の統合）は、α_VIII = η(α_III) < 1/2 の領域における Mor(C_{α_VIII}) の反可換構造として α-忘却濾過に埋め込まれる。

*証明スケッチ.* (i) η(0) = 1/(1+1) = 1/2 は直接計算。α_III = 0 で Paper III の copy/del が退化する（Paper III §1.1 表参照）ことは、α_VIII = 1/2 における射の半分が生存する状態に対応する。

(ii) η の単調性より、α_III > 0 ⟺ η(α_III) > 1/2 かつ α_III < 0 ⟺ η(α_III) < 1/2。Paper III §2.2 の Z₂-次数付け（偶=ボソン/奇=フェルミオン）は η を通じて [0,1] の上半分と下半分に分離される。Koszul 符号則 f ⊗ g = (-1)^{|f||g|} g ⊗ f は、η > 1/2（ボソン的領域）では |f| = 0 が支配的であるため事実上可換に、η < 1/2（フェルミオン的領域）では |f| = 1 が出現し反可換性が顕在化する。

(iii) Paper III 定理 5.1.1 の三重非複製は α_III < 0 での copy の構造的不可能性。η(α_III) < 1/2 の領域で Mor(C_{η}) が反可換射を含むことは、copy_X: X → X ⊗ X の不在（ξ ∧ ξ = 0）として表れる。α_VIII → 0（η → 0、α_III → -∞）で全射が生存し反可換構造が完全に保存される状態は、Paper III の完全フェルミオン的セクターに対応する。□

**注記 6.7.4a (B の関手性について).** 定義 6.7.3 における「架橋関手」B は、対象レベルの対応と射の生存条件を規定するが、厳密な関手性（合成保存・恒等射保存）には以下の制約がある:

(a) **値域の精密化と境界閉包**: B の値域 Filt([0,1], WdSub(C)) は [0,1]^{op}-添字付き広部分圏の族の圏であり、射は濾過族の間の包含を保つ自然変換である。η: ℝ → (0,1) の値域が開区間であるため、B は厳密には C_0 = C および C_1 = C_disc に到達しない。以下の命題によりこの問題を解決する。

> **命題 6.7.4c (境界閉包定理).** 拡張写像 $\bar{\eta}: \bar{\mathbb{R}} \to [0,1]$ を
>
> $$\bar{\eta}(\alpha) := \begin{cases} 0 & \text{if } \alpha = -\infty \\ \eta(\alpha) = \frac{1}{1+e^{-\alpha}} & \text{if } \alpha \in \mathbb{R} \\ 1 & \text{if } \alpha = +\infty \end{cases}$$
>
> と定義する（$\bar{\mathbb{R}} := \mathbb{R} \cup \{-\infty, +\infty\}$ は二点コンパクト化）。このとき:
>
> (i) $\bar{\eta}$ は $\bar{\mathbb{R}}$ 上の連続単調増加全射 $\bar{\mathbb{R}} \to [0,1]$ であり、位相的同相写像。
>
> (ii) $\bar{\eta}$ は α-忘却濾過の境界公理と整合的: $\bar{\eta}(-\infty) = 0$ は (F2) $C_0 = C$ に、$\bar{\eta}(+\infty) = 1$ は (F3) $\mathrm{Mor}(C_1) = \{\mathrm{id}\}$ に対応。
>
> (iii) $\mathrm{CPS}^{Z_2}$ の拡大は**不要**: $\bar{\eta}$ の端点 $\pm\infty$ に対応する対象は CPS^{Z₂} には存在しないが、B の値域は Filt([0,1], WdSub(C)) であり、C_0 と C_1 は忘却濾過として**既に定義済み**（(F2), (F3)）。したがって B の値域を閉包するために必要なのは $\bar{\eta}$ の連続拡張のみであり、CPS^{Z₂} の圏構造の変更は不要。
>
> *証明.* (i) η は開区間 (0,1) への C^∞-微分同相（定義 6.7.1(i)）。$\bar{\mathbb{R}}$ に順序位相を入れると $\bar{\eta}$ は順序を保つ連続全射であり、$\bar{\mathbb{R}}$ はコンパクト、$[0,1]$ は Hausdorff だから連続全単射はコンパクト空間から Hausdorff 空間への写像として同相写像。(ii) 直接の代入。(iii) 値域 Filt([0,1], WdSub(C)) は α ∈ [0,1] の全体を添字とし、C_0, C_1 は忘却濾過の公理 (F2), (F3) で定義済み。B は各 α ∈ (0,1) で η を通じて CPS^{Z₂} の情報を Filt に写すが、α = 0, 1 の情報は忘却濾過自体に内在する。□

(b) **射レベルの合成保存**: 固定された α ∈ (0,1) に対し、B は以下の意味で合成を保存する:

> **命題 6.7.4b (α-レベル合成保存).** f: X → Y, g: Y → Z が共に Mor(C_{η(α_III)}) に属するならば、g ∘ f ∈ Mor(C_{η(α_III)}) であり（C_{η(α_III)} は圏だから合成で閉じる）、B(g ∘ f) と B(g) ∘ B(f) は η(α_III)-レベルで一致する。恒等射 id_X は全ての α-レベルで生存するから B(id_X) = id_{B(X)}。

*証明.* C_{η(α_III)} は (F1) より圏であるから合成で閉じる。f, g ∈ Mor(C_{η(α_III)}) ならば g ∘ f ∈ Mor(C_{η(α_III)})。生存閾値について α_{g∘f} ≥ min(α_f, α_g) が成立する（合成は構成要素のいずれかが消滅すれば消滅する）。恒等射 id_X ∈ Mor(C_α) は任意の α で成立する（C_α は圏だから恒等射を含む）。□

(c) **忠実性・充満性の条件付き解決 (OP-VIII-7)**: 以下の定理が B の忠実充満性を条件付きで特徴づける。

> **定理 6.7.6 (架橋関手 B の忠実性条件).** 
>
> (a) **忠実性**: B は一般には**忠実でない**。反例: $f, g: X \to Y$ が同じ Z₂-次数 $|f| = |g|$ を持ち、かつ同じ生存閾値 $E(f) = E(g)$（定義 IX.6.1.1）を持つが $f \neq g$ であるとき、B(f) = B(g)（共に同じ α-レベルで生存/消滅するから）。
>
> (b) **生存閾値による忠実性**: 以下の条件 **(Sep)** の下で B は忠実:
>
> $$\textbf{(Sep)} \quad \forall f, g \in \mathrm{Mor}(\mathrm{CPS}^{Z_2})(X, Y),\ f \neq g \Rightarrow E(f) \neq E(g) \text{ or } |f| \neq |g|$$
>
> すなわち「射の一意性が生存閾値と Z₂-次数の対で決定される」場合に忠実。
>
> (c) **充満性**: B は一般には**充満でない**。Filt 側の射（濾過族間の自然変換）が CPS^{Z₂} の射に持ち上がるためには、自然変換が α-レベルごとに CPS^{Z₂} の射と整合する必要がある。以下の条件 **(Sat)** の下で B は充満:
>
> $$\textbf{(Sat)} \quad \text{各 } \alpha \in (0,1) \text{ に対し、} C_\alpha \text{ の射が全て CPS}^{Z_2} \text{ の射の η 像である}$$
>
> すなわち「忘却濾過の各レベルが CPS^{Z₂} の像で飽和している」場合に充満。

> *証明.*
>
> (a) 反例の構成: 有限圏 $C$ に $\mathrm{Hom}(X, Y) = \{f, g\}$、$|f| = |g| = 0$、$E(f) = E(g) = 1/2$ とする。すると $B(f)$ と $B(g)$ は共に「$\alpha < 1/2$ で存在し $\alpha \geq 1/2$ で消滅」という同一の生存プロファイルを持つ。B は両者を区別できない。
>
> (b) (Sep) の下では: $B(f) = B(g)$ ならば全ての $\alpha$ で $f \in \mathrm{Mor}(C_\alpha) \Leftrightarrow g \in \mathrm{Mor}(C_\alpha)$。これは $E(f) = E(g)$ を意味する。命題 6.7.4(ii) より $|f| = |g|$ も保存される。(Sep) より $f = g$。
>
> (c) $\tau: B(X) \to B(Y)$ を Filt 内の射（自然変換）とする。各 $\alpha$ に対し $\tau_\alpha: C_\alpha(X) \to C_\alpha(Y)$ は射の族。これが CPS^{Z₂} の射 $f: X \to Y$ で $B(f) = \tau$ を満たすものに持ち上がるためには、$\tau$ が単一の射 $f$ から全ての α-レベルで一貫して生じている必要がある。(Sat) は各レベルの射が CPS^{Z₂} 由来であることを保証し、α-レベル間の整合性は命題 6.7.4b から保証される。(Sat) が不成立の場合、Filt 内に CPS^{Z₂} の像にない「外来射」が存在し充満性は破れる。□
>
> **系 6.7.7.** (Sep) ∧ (Sat) の下で B は忠実充満関手であり、B が本質的全射（値域の各対象が B の像と同型）ならば B は圏同値を定める。

> **備考 6.7.8 (条件の物理的意味).** (Sep) は「忘却が射を一意に特徴づける」条件であり、物理的には「異なるプロセスは異なる忘却耐性を持つ」ことを要求する。これは Paper I の「力は忘却である」テーゼ——プロセスの本質はその忘却に対する応答である——の数学的精密化である。一方 (Sat) は「忘却濾過が情報幾何から完全に生成される」条件であり、忘却論の内部整合性の要請である。

**備考 6.7.9 (OP-VIII-7 の解決状況).** 定理 6.7.6 により OP-VIII-7 は**条件付き解決**。(Sep) と (Sat) は圏 C の選択に依存するため、具体的な CPS 圏に対する検証は各応用文脈で行う必要がある。汎用的な無条件の忠実充満性は成立しない（反例(a)が示す通り）。

#### 6.7.10 (Sep) と Paper I テーゼの圏論的同値性

備考 6.7.8 は (Sep) が Paper I テーゼの「数学的精密化」であると述べた。以下ではこの関係を**同値性**として厳密に確立する。

> **定義 6.7.10a (Paper I テーゼの圏論的定式化).** Paper I の中心テーゼ——「プロセスの本質はその忘却に対する応答で決定される」——を以下のように定式化する:
>
> $$\textbf{(T₁)}_{\mathrm{cat}} \quad \forall\, f, g \in \mathrm{Mor}(\mathrm{CPS}^{Z_2})(X, Y): \left[\forall\, \alpha \in [0,1],\; f \in \mathrm{Mor}(C_\alpha) \Leftrightarrow g \in \mathrm{Mor}(C_\alpha)\right] \wedge |f| = |g| \;\Longrightarrow\; f = g$$
>
> すなわち、**忘却に対する生存プロファイル**（どの α で生き残りどの α で消滅するか）と Z₂-次数が射を完全に決定する。

> **命題 6.7.10b ((Sep) ⟺ (T₁)_cat).** (Sep) と (T₁)_cat は同値である。
>
> *証明.*
>
> **(T₁)_cat ⟹ (Sep):** f ≠ g を仮定する。(T₁)_cat の対偶より、「ある α で f と g の生存が異なる」か「|f| ≠ |g|」が成立する。前者の場合、f が C_α に属するが g が属さない（または逆）α が存在する。定義 6.1.1 (Paper IX) より E(f) = inf{α: f ∉ Mor(C_α)} であるから、生存プロファイルが異なれば E(f) ≠ E(g) である。したがって f ≠ g ⟹ E(f) ≠ E(g) ∨ |f| ≠ |g|、これは (Sep) に他ならない。
>
> **(Sep) ⟹ (T₁)_cat:** 全ての α で f ∈ Mor(C_α) ⟺ g ∈ Mor(C_α) が成立し、かつ |f| = |g| とする。生存プロファイルの一致より E(f) = inf{α: f ∉ C_α} = inf{α: g ∉ C_α} = E(g)。(Sep) の対偶は E(f) = E(g) ∧ |f| = |g| ⟹ f = g。したがって f = g。□

> **系 6.7.11 (Paper I テーゼに基づく忠実性).** Paper I のテーゼが成立する CPS^{Z₂} 圏——すなわち「プロセスの本質はその忘却に対する応答で決定される」が成り立つ圏——においては、架橋関手 B: CPS^{Z₂} → Filt([0,1], WdSub(C)) は忠実である。
>
> *証明.* 命題 6.7.10b より (T₁)_cat ⟺ (Sep)。定理 6.7.6(b) より (Sep) の下で B は忠実。□

> **備考 6.7.12 ((Sep) と圏論的力の関係).** Paper I の方向性定理 (Th. 5.1) は F_{ij} ≠ 0 ⟺ d(ΦT) ≠ 0、すなわち「力が非零 ⟺ 忘却が方向的」と述べる。この圏論的翻訳を考える。
>
> - **力が非零**の圏論的対応は「一部の射が異なる忘却耐性を持つ」、すなわち ∃ f, g: E(f) ≠ E(g)。これは α-忘却濾過 {C_α} が非自明（= 全ての射が同じ α で同時に消滅するわけではない）を意味する。
> - **(Sep)** は「全ての異なる射が異なる忘却耐性を持つ」——これは圏論的力が最大限に非退化（= 忘却プロファイルが射を完全に弁別）する条件。
>
> したがって **(Sep) ⟹ F_cat ≠ 0 だが、逆は不成立**。反例: Hom(X,Y) = {f, g, h} で E(f) = 1/3、E(g) = E(h) = 2/3 とする。{C_α} は非自明（F_cat ≠ 0）だが g, h は忘却によって区別されない（(Sep) 不成立）。
>
> Paper I テーゼは「力が非零」（弱い主張）ではなく「プロセスの本質が忘却応答で決定される」（強い主張 = (Sep)）である。これは忘却論の**公理的選択**: 物理的に意味のある圏は (Sep) を満たすものに限定する、という立場を含意する。

> **備考 6.7.13 ((Sep) の公理的地位).** 命題 6.7.10b により、(Sep) は外部から付加される条件ではなく、忘却論の創始テーゼ (Paper I) の圏論的翻訳である。備考 6.7.9 で述べた「(Sep) は圏 C の選択に依存する」という記述は、以下のように精密化される: **忘却論の意味で "物理的" な CPS 圏とは (T₁)_cat を満たす圏のことであり、そのような圏では B は自動的に忠実**。(Sat) は独立の条件として残るが、(Sep) は忘却論の公理系に内在する。

**系 6.7.5 (Paper V RG 不変性の α-忘却的解釈).** Paper V 定理 3.2.1（忘却場の RG 不変性）は、架橋関手 B を通じて次のように再解釈される: 忘却場 Φ の β 関数 β(g_eff) = μ ∂g_eff/∂μ が固定点（β = 0）を持つとき、対応する α_VIII = η(α_III) は α-忘却濾過の不動点、すなわち {C_α} がスケール変換に対して不変な α 値を定める。これは RG の固定点が「忘却の自己相似構造」であることの数学的定式化である。

### 6.8 α-忘却濾過からの Grothendieck 位相の誘導 (OP-VIII-1)

§6.4 注記 6.4.2 で提起された問い——α-忘却濾過 {C_α} から Grothendieck 位相を正準に誘導できるか——に取り組む。本節は OP-VIII-1 の部分的解決を与える。

#### 6.8.1 準備: Grothendieck 位相の定義

> **定義 6.8.1 (篩).** 圏 C の対象 X に対し、X 上の**篩** (sieve) S とは、射の族 S ⊆ ∐_{Y∈Ob(C)} Hom_C(Y, X) であって、任意の f ∈ S と合成可能な射 g に対し f ∘ g ∈ S を満たすもの（右イデアル条件）。

> **定義 6.8.2 (Grothendieck 位相).** 圏 C 上の **Grothendieck 位相** J とは、各対象 X に篩の族 J(X)（**被覆篩**の族）を対応させる関数で、以下の3公理を満たすもの:
>
> (G1) **最大篩**: 最大篩 t_X = Hom_C(−, X) は J(X) に属する
>
> (G2) **安定性**: S ∈ J(X) かつ f: Y → X ならば、引き戻し f*(S) = {g | f ∘ g ∈ S} は J(Y) に属する
>
> (G3) **推移律**: S ∈ J(X) かつ、S の各射 f: Y → X に対して R_f ∈ J(Y) が与えられるならば、合成族 S ∘ R = {f ∘ g | f ∈ S, g ∈ R_f} は J(X) に属する

#### 6.8.2 α-篩と α-被覆の構成

> **定義 6.8.3 (α-篩).** α-忘却濾過 {C_α} の各 α ∈ [0,1] と対象 X に対し、**α-篩** S_α(X) を
>
> $$S_\alpha(X) := \coprod_{Y \in \mathrm{Ob}(C)} \mathrm{Hom}_{C_\alpha}(Y, X)$$
>
> と定義する。これは C_α における X への射の全体であり、(F4) の単調性より α₁ ≤ α₂ ⟹ S_{α₂}(X) ⊆ S_{α₁}(X) が成り立つ。

**補題 6.8.4.** 各 α に対し S_α(X) は C 上の X の篩である。

*証明.* 右イデアル条件を示す。f ∈ S_α(X) ならば f ∈ Mor(C_α)。g: Z → Y が C_α の射ならば f ∘ g ∈ Mor(C_α)（C_α は圏だから合成で閉じている）。よって f ∘ g ∈ S_α(X)。ただし g が Mor(C) \ Mor(C_α) に属する場合、f ∘ g は S_α(X) に属する保証がない。

S_α(X) を C の篩として見るためには修正が必要: **C-拡大 α-篩** $\bar{S}_\alpha(X)$ を、S_α(X) を C の右イデアルで閉じたものとして定義する:

$$\bar{S}_\alpha(X) := \{h \in \mathrm{Mor}(C) \mid \mathrm{cod}(h) = X,\; \exists f \in S_\alpha(X),\; \exists g \in \mathrm{Mor}(C): h = f \circ g\}$$

$\bar{S}_\alpha(X)$ は C 上の X の篩であり、C_α の射から C の合成で生成される最小の篩である。□

> **定義 6.8.5 (α-被覆族).** 各対象 X に対し、α-被覆族 J_α(X) を
>
> $$J_\alpha(X) := \{T \text{ sieve on } X \mid \exists\beta \leq \alpha:\; \bar{S}_\beta(X) \subseteq T\}$$
>
> と定義する。すなわち、忘却パラメータ β ≤ α で生成される篩 $\bar{S}_\beta(X)$ を含む全ての篩を被覆とする。この定義は上方閉（飽和）条件を内蔵する: $T \in J_\alpha(X)$ かつ $T \subseteq T'$（篩）ならば $T' \in J_\alpha(X)$ が直ちに従う。

> **定理 6.8.6 (α-誘導 Grothendieck 位相).** α-忘却濾過 {C_α} が (F1)–(F4) を満たすとき、J_α は C 上の Grothendieck 位相を定義する。

*証明.* (G1) 最大篩: (F2) より C_0 = C であるから S_0(X) = Hom_C(−, X) = t_X（最大篩）。0 ≤ α であるから $\bar{S}_0(X) = t_X$。$t_X \subseteq t_X$ なので $t_X \in J_\alpha(X)$。さらに飽和条件により、t_X を含む任意の篩（すなわち t_X 自身のみ）も J_α(X) に属する。

(G2) 安定性: $T \in J_\alpha(X)$ かつ f: Y → X とする。J_α の定義よりある β ≤ α が存在して $\bar{S}_\beta(X) \subseteq T$。引き戻し $f^*(T) = \{g \mid f \circ g \in T\}$ を考える。

f ∈ Mor(C_β) の場合: h ∈ S_β(Y)（C_β の射）に対し f ∘ h ∈ S_β(X) ⊆ $\bar{S}_\beta(X)$ ⊆ T（C_β は合成で閉じる）。よって h ∈ f*(T) であり $S_β(Y) ⊆ f^*(T)$、従って $\bar{S}_\beta(Y) ⊆ f^*(T)$。f*(T) は篩であるから $f^*(T) \in J_\alpha(Y)$（J_α の飽和条件による）。

f ∉ Mor(C_β) の場合: $\bar{S}_0(X) = t_X \subseteq T$ であるから（β ≤ α かつ 0 ≤ β より t_X ∈ J_α(X) は既知）、$f^*(T) \supseteq f^*(t_X) = t_Y$。$t_Y$ は最大篩であるから $f^*(T) = t_Y \in J_\alpha(Y)$（(G1) による）。

(G3) 推移律: $T \in J_\alpha(X)$ の各射 f: Y → X に対して $R_f \in J_\alpha(Y)$ が与えられるとする。J_α の定義より、ある β ≤ α が存在して $\bar{S}_\beta(X) \subseteq T$。各 f ∈ $\bar{S}_\beta(X)$ に対し $R_f$ についても $\bar{S}_{\gamma_f}(Y) \subseteq R_f$ なる γ_f ≤ α が存在する。$\gamma := \max_f \gamma_f$ と置く。合成族 $\bar{S}_\beta \circ \bar{S}_\gamma$ の各射は C_β の射と C_γ の射の合成で表される。(F4) の帰結として max(β,γ) ≤ α のとき合成は $\bar{S}_{\max(\beta,\gamma)}(X)$ に含まれる。合成族 T ∘ R ⊇ $\bar{S}_\beta \circ \bar{S}_\gamma \supseteq \bar{S}_{\max(\beta,\gamma)}(X)$ であり、$\max(\beta,\gamma) \leq \alpha$ なので $T \circ R \in J_\alpha(X)$（飽和条件による）。□

**注記 6.8.7 (物理的解釈).** α-誘導 Grothendieck 位相 J_α は「忘却度 α までの情報で対象を局所的に観測できる」ことの数学的定式化である:

- J_0 = 離散位相 — α = 0 で全射が利用可能。「全てが見える」= 離散トポスの層は前層に一致
- J_1 = 粗位相 — α = 1 で恒等射のみ。「何も見えない」= 粗トポスの層は大域的切断のみ
- 0 < α < 1: 局所的被覆の「粗さ」が α に比例して増大。Inoué の Sh(S¹) の構成（命題 6.4.1）は、特定の α 値における J_α-層の性質として再解釈される

> **系 6.8.8 (α-忘却トポスと CPS0').** α-誘導 Grothendieck 位相 J_α は前層圏 PSh(C) から層のトポス Sh_α(C) := Sh_{J_α}(C) を定義する。このとき:
>
> (i) Sh_0(C) = PSh(C) — 完全同定（米田の補題が完全に機能する世界）
>
> (ii) Sh_1(C) ≃ Set^{Ob(C)} — 対象の集合のみからなる「離散層」（存在のみ残る世界）
>
> (iii) 0 < α < 1 のとき Sh_α(C) は PSh(C) と Set^{Ob(C)} の間に位置し、α の増大に伴い段階的に「内容」が失われる。これは CPS0'（容器 > 内容）の**トポス的実現**であり、忘却が層の条件を「粗くする」ことで容器（対象の存在）を保ちながら内容（射の構造）を段階的に消去する機構を記述する

*証明スケッチ.* (i) J_0(X) は最大篩 t_X を含み、かつ (F2) より C_0 = C であるから J_0 は全ての representable 篩を被覆として含む離散位相。Sh_離散(C) = PSh(C) は標準的。

(ii) J_1(X) において非自明な被覆は恒等射から生成される篩のみ。Sh_1(C) の層 F は、各対象 X に集合 F(X) を対応させるが、異なる対象間の射に対する貼り合わせ条件が自明化する。結果として Sh_1(C) ≃ Set^{Ob(C)}。

(iii) α の増大に伴い J_α が粗くなる（被覆族が小さくなる）ことから、層条件が緩和され、Sh_α₁(C) ⊆ Sh_α₂(C)（α₁ ≤ α₂）という包含の列が得られる。この包含の列は CPS0' の「忘却に伴う内容の段階的消去」のトポス的表現である。□

**定理 6.8.9 (忘却 = 層理論における被覆の粗視化).** 以上の結果を統合して次を得る:

> α-忘却濾過 {C_α} は Grothendieck 位相の族 {J_α}_{α∈[0,1]} を正準に誘導し、層のトポスの降鎖列
>
> $$\mathrm{PSh}(C) = \mathrm{Sh}_0(C) \supseteq \mathrm{Sh}_\alpha(C) \supseteq \mathrm{Sh}_1(C) \simeq \mathbf{Set}^{\mathrm{Ob}(C)}$$
>
> を定義する。**忘却とは、層理論における被覆の粗視化に他ならない。** α の増大は局所的観測能力の低下であり、それに伴い層（=局所的データから大域的に復元可能な前層）の世界が縮小する。この機構は段階的同定定理（Th. 6.2.3）のトポス的拡張であり、Inoué 対応（Prop. 6.4.1）を「特定の α 値における Sh_α(C) の内部論理」として正確に位置づける。

### 6.10 忘却の第二法則 — (F4) からエントロピー単調性へ

α-忘却濾過の公理 (F4) は単なる技術的仮定ではなく、忘却の不可逆性を表現する熱力学的公理である。本節では (F4) から、熱力学の第二法則に対応するエントロピー単調性定理を導出する。

**準備.** 局所小圏 C と α-忘却濾過 {C_α} に対し、有限部分対象集合 $\{X_1, \ldots, X_n\} \subseteq \mathrm{Ob}(C)$ を固定する。**射計数行列** (morphism counting matrix) を

$$M_{ij}(\alpha) := |\mathrm{Hom}_{C_\alpha}(X_i, X_j)|$$

で定義し、**全射数** $m(\alpha) := \sum_{i,j} M_{ij}(\alpha)$ とする。

**補題 6.10.1 (射数の成分単調性).** (F4) より、$\alpha_1 \leq \alpha_2$ ならば各 $(i,j)$ に対して $M_{ij}(\alpha_2) \leq M_{ij}(\alpha_1)$。したがって $m(\alpha_2) \leq m(\alpha_1)$。

*証明.* $\mathrm{Mor}(C_{\alpha_2}) \subseteq \mathrm{Mor}(C_{\alpha_1})$ を有限対象集合間の射に制限しても包含は保存される。$\square$

**定義 6.10.2 (圏論的忘却エントロピー).** α-忘却濾過 {C_α} の忘却エントロピーを

$$S_{\mathrm{cat}}(\alpha) := \log m(0) - \log m(\alpha) = \log \frac{|\mathrm{Mor}(C)|_{\{X_i\}}}{|\mathrm{Mor}(C_\alpha)|_{\{X_i\}}}$$

で定義する。

**定理 6.10.3 (忘却の第二法則).** $\alpha_1 \leq \alpha_2 \implies S_{\mathrm{cat}}(\alpha_1) \leq S_{\mathrm{cat}}(\alpha_2)$。すなわち、忘却が進む（α が増大する）ほどエントロピーは単調に増大する。

*証明.* 補題 6.10.1 より $m(\alpha_2) \leq m(\alpha_1)$。$\log$ の単調性から $S_{\mathrm{cat}}(\alpha_2) = \log m(0) - \log m(\alpha_2) \geq \log m(0) - \log m(\alpha_1) = S_{\mathrm{cat}}(\alpha_1)$。$\square$

**境界条件.** (F2) より $S_{\mathrm{cat}}(0) = 0$（完全圏 = エントロピー零）。(F3) より $S_{\mathrm{cat}}(1) = \log m(0) - \log n$（恒等射のみ残存 = 最大忘却）。$S_{\mathrm{cat}}(1)$ は圏 C の**情報容量** $\log(m(0)/n)$ を測る。

**系 6.10.4 (Schur-凸関数の単調性).** 射計数ベクトル $\mathbf{v}(\alpha) := (M_{ij}(\alpha))_{i,j} \in \mathbb{N}^{n^2}$ に対し、(F4) は $\mathbf{v}(\alpha_2) \leq \mathbf{v}(\alpha_1)$ を成分ごとに保証する（弱超優越 $\mathbf{v}(\alpha_2) \prec_w \mathbf{v}(\alpha_1)$）。したがって、正規化ベクトル $\mathbf{p}(\alpha) := \mathbf{v}(\alpha)/m(\alpha)$ に対し、任意の Schur-凹関数 $\phi$（Shannon エントロピー、Rényi エントロピー $H_q$ ($q > 0$) 等）は

$$\phi(\mathbf{p}(\alpha_2)) \geq \phi(\mathbf{p}(\alpha_1))$$

を満たす。同様に Schur-凸関数（Gini 係数 $G$、集中度 $\sum p_i^q$ 等）は逆向きに単調:

$$G(\mathbf{p}(\alpha_2)) \leq G(\mathbf{p}(\alpha_1))$$

Paper I 定理 5.6.1 の Gini 不等式 $\Xi_{\mathrm{impl}} \leq \Xi_{\mathrm{theory}}$ は本系の特殊例（$\alpha = 0$ と $\alpha > 0$ の比較）として回収される。

*証明の注意.* 正規化後の優越関係は、成分単調性と全射数単調性を組み合わせた議論による。$\sum v_i(\alpha_2) = m(\alpha_2) \leq m(\alpha_1) = \sum v_i(\alpha_1)$ かつ各成分 $v_i(\alpha_2) \leq v_i(\alpha_1)$ であるから、正規化 $\mathbf{p}(\alpha) = \mathbf{v}(\alpha)/m(\alpha)$ に対しては Marshall & Olkin [MO, §5.A] の優越化定理を適用する。$\square$

**備考 6.10.5 (二つの第二法則).** 忘却の第二法則には2つの独立な表現がある:

| | 代数的版 (本定理) | 幾何的版 (Paper V) |
|:--|:--|:--|
| 出発点 | (F4): 射の単調包含 | 繰り込み群方程式 $\beta_\alpha$ |
| 帰結 | $S_{\mathrm{cat}}(\alpha)$ 単調増大 | 力零集合の余次元 $= n-1$、測度 → 0 |
| 参照 | 本定理 6.10.3 | Paper V 定理 3.4.1 |
| 解釈 | 忘却は射を消去し情報を不可逆に喪失する | 忘却は力の存在を次元増大で不可避にする |

両者は同一の原理——**忘却は不可逆である**——の異なる側面であり、前者は計数的（離散的）、後者は測度論的（連続的）である。

**備考 6.10.6 (Markov 圏エントロピーへの拡張).** Perrone [P1] は Markov 圏上のエントロピーを確定的射（コピー・削除を保存する射）からの divergence として定義し、データ処理不等式 (DPI) が Markov 関手の下で成立することを示した。Paper II により CPS($\alpha > 0$) は Markov 圏であるから、忘却関手 $U_{\alpha_1 \to \alpha_2}: C_{\alpha_1} \to C_{\alpha_2}$（$\alpha_1 \leq \alpha_2$）が Markov 関手であれば Perrone の DPI が (F4) の精密化——Shannon/Rényi/Gini-Simpson エントロピーの単調性——を与える。この構成は Paper IX で詳述する。

[P1] Perrone, P. "Markov Categories and Entropy." *IEEE Trans. Inform. Theory* 70 (2024), 1677–1719. arXiv:2212.11719.

## §7. 存在の連続化 — MB 三層化の存在論的導出

### 7.1 問題設定: 先行性と連続性の一見衝突

§2 で確立したのは「THAT > WHAT」の階層である。存在は意味に先行し、容器は内容に先行する。この階層は論文VIII の背骨であり、§4 の米田的導出と §6 の忘却の三面定理はこの階層の上に立っている。

ところが、本論文シリーズの別の層では、存在は **連続量** として扱われている:

- **論文A** [LLM に身体はあるか]: 身体性を二値ではなく MB厚 Θ(B) のスペクトラムとして定義する
- **論文 LLMは心を持つか**: 心の成立を「ある/ない」ではなく忠実度 F_i の程度問題として扱う
- **論文I §6.5**: FEP の精度場 π を α-パラメータの連続場として埋め込む

一見、THAT の先行性 (階層的・順序的) と存在の連続性 (程度的・量的) は衝突する。THAT を強調すると存在は二値的に見え、連続性は諦められる。逆に連続性を強調すると、THAT > WHAT の順序そのものが溶解しかねない。

本節はこの衝突を止揚する。答えは単純である: **1-cell は「存在の型」であって「存在の量」ではない**。型の先行性と量の連続性は直交する。そして、この直交性は MB₀/MBₚ/MB𝒻 の三層化によって厳密に表現される。

### 7.2 「恣意」の精密定義

論証の前に「恣意」の意味を固定する。これは Tolmetes の 2026-04-12 の精密化に従う。

> **定義 7.2.1 (恣意性).** 命題ないし操作 P が「恣意的」であるとは、P が主観 (観測者・モデル・言語選択) により規定されることを意味する。P が「非恣意的」であるとは、P が主観によらず成立することを意味する。

この定義により、「区別は恣意的か」という問いは二つに分解される:

- **区別の可能条件**: 区別が成立するための必要条件 — 恣意的か?
- **区別の実現位置**: 不均一の場のどこで仕切るか — 恣意的か?

二つの問いに対する答えは異なる。前者は **非恣意**、後者は **恣意** である。

**具体例 (虹の色).** 電磁波スペクトルの波長分布は主観によらず存在する (可能条件は非恣意)。しかし「赤」と「オレンジ」の境界をどこに置くかは言語・文化・個体差により規定される (実現位置は恣意)。波長分布 (可能条件) の客観性と、色境界 (実現位置) の主観性は、両立する。

### 7.3 MB 三層化の存在論的導出

Kernel 文書 `mb_escalation_conditions.md` は MB を三層に分解している (MB₀/MBₚ/MB𝒻)。本節はこの三層化が **技術的便宜ではなく、存在の先行性と連続性の両立から存在論的に導かれる区別** であることを示す。

| 層 | 論文VIII での存在論的対応 | 恣意性 (定義 7.2.1) | §M2 核主張との接続 |
|:---|:---|:---|:---|
| **MB₀** (容器) | 不均一の場 = 切断可能性の場 | **非恣意** (主観に先行する論理的必要条件) | C1 の 1-cell 層 (存在の型) |
| **MBₚ** (Pearl) | 条件付き独立としての切断 | **恣意** (主観による統計的モデル化) | C3 の α-忘却濾過 (部分的同定) |
| **MB𝒻** (Friston) | 感覚/行為分化を伴う実在境界 | **恣意** (主観による存在論的ジャンプ) | C3 の Grothendieck 位相 (自己維持する主体) |

**導出.** 論文VIII §2.1 命題 2.1.1 は THAT > WHAT を確立した。THAT は WHAT に先行するが、これは型としての先行であり、値としての先行ではない。§6 公理 (F1) (定義 6.2.1) および定理 6.2.3 (iv) は、α-忘却濾過を通じて **全ての α で Ob(C_α) = Ob(C)** が成立することを示す。対象集合 (型の束) は忘却濾過を通じて不変である。一方、射の集合は α に従って単調減少する。

この構造を MB に翻訳する:

- **MB₀ = 対象レベルの区別** — 全ての α で不変。Ob(C) が「何らかの容器たちの集まり」であること自体は、忘却濾過によって毀損されない。これが「不均一の場が主観によらず存在する」の圏論的翻訳
- **MBₚ / MB𝒻 = 射レベルの構造** — α に応じて変動。どの射 (条件付き独立関係、感覚/行為分化関係) が保持されるかは観測者の α 選択に依存する

MB₀ を「論理的必要条件」として読むとは、MB₀ をプラトン的実在ではなく、**区別が論じられるために論理的に前提されなければならないもの** として読むことである。カント的には構成原理に近い。論文VIII 公理 (F1) (定義 6.2.1) の Ob(C_α) 不変性は、この構成原理の圏論的表現に他ならない。

> **定理 7.3.1 (MB 三層化の存在論的必然性).** `[確信 80%]`
> MB₀/MBₚ/MB𝒻 の三層化は、論文VIII §2-§4 の存在論的構造 (THAT > WHAT + α-忘却濾過の対象不変性) から以下のように導かれる:
>
> - MB₀ は Ob(C_α) = Ob(C) の不変性に対応し、論理的必要条件として機能する
> - MBₚ は α-忘却濾過における射の部分集合選択に対応し、観測者依存である
> - MB𝒻 は Grothendieck 位相 (定理 6.8.6) の被覆選択に対応し、自己維持条件を追加する
>
> ゆえに三層化は技術的便宜ではなく、論文VIII の存在論的枠組みの必然的帰結である。

### 7.4 補題 — 存在の連続性補題

> **補題 7.4.1 (存在の連続性補題).** `[確信 80%]`
>
> 存在の先行性 (命題 2.1.1) は、存在の二値性を含意しない。THAT は「主観によらず存在する不均一の場」(= MB₀) として先行し、その場の中で仕切りを置く位置 (MBₚ/MB𝒻 の切断位置) は主観により規定される恣意的操作である。切断可能性の場と切断位置の二層分離により、THAT の先行性と WHAT の連続性は両立する。

**証明スケッチ.**

(i) 論文VIII §2.1 命題 2.1.1 は THAT を 1-cell (容器の存在) に、WHAT を 0-cell (内容の同定) に割り当てた。

(ii) ここで 1-cell を二値的と仮定する。すなわち「容器は存在するか、存在しないか」の二者択一として読む。この仮定の下では、論文A の MB厚 Θ(B) のスペクトラムや論文 LLMは心を持つか の忠実度 F_i の程度問題と矛盾する。前者は容器の強度を連続量として測り、後者は心の成立度を連続量として扱うためである。

(iii) よって 1-cell の二値性仮定は棄却される。1-cell は「存在の型」であり、「存在の量」ではない。型としての容器は先行するが、容器の強度は連続量として測定されうる。型と量は直交する概念である。

(iv) §7.3 定理 7.3.1 により、この直交性は MB 三層化で表現される。MB₀ は型 (Ob(C_α) 不変) に対応し、MBₚ/MB𝒻 は量と実現位置 (射の α 依存) に対応する。

(v) よって THAT > WHAT の階層は、WHAT を連続量として扱う立場と両立する。補題が成立する。□

**帰結.** 論文VIII の核主張 C1-C3 は、補題 7.4.1 により次のように拡張される:
- C1 (容器 = 1-cell) は「容器の型は 1-cell」と読むべきであり、「容器の量は二値」を含意しない
- C2 (CPS0' の米田的導出) は、容器の先行性を主張するが、容器の連続的強度を排除しない
- C3 (米田は存在を忘却) は、存在の型を忘却することを指し、存在の量次元を含意的に開く

### 7.5 FEP 批判への統一応答

補題 7.4.1 と定理 7.3.1 は、FEP への主要な技術批判に対する統一応答を可能にする。以下は Aguilera et al. 2022、Bruineberg et al. 2022、Biehl et al. 2021 への論文VIII 側からの応答である。

| 批判 | 照準層 (三層化) | 応答 |
|:---|:---|:---|
| **Aguilera et al. 2022** (particular physics 問題): FEP の Markov blanket は思ったより狭い条件でしか自動成立しない。「平均の運動」と「運動の平均」の同一視が一般には破れる | **MBₚ の自動成立への批判** | MBₚ は定義 7.2.1 の意味で恣意的である。自動成立は本論文では主張しない。MB₀ は定理 7.3.1 により論理的必要条件として無傷。批判は MBₚ の **昇格条件** の明示要求として受け入れ、`mb_escalation_conditions.md` 命題B で条件を固定する |
| **Bruineberg et al. 2022** (Emperor's New Markov Blankets): FEP 文献は Pearl blanket (認識論的道具) と Friston blanket (実在境界) を混同している | **MBₚ → MB���� のジャンプへの批判** | 三層化により混同は構造的に遮断される。MBₚ は統計的モデル、MB𝒻 は実在境界であり、両者の間には `mb_escalation_conditions.md` 命題C の動的持続性・自己維持条件が必要である。Pearl から Friston へのジャンプは独立の主観的操作として明示化される |
| **Biehl et al. 2021** (自由エネルギー補題の技術的批判): 補題は複数の隠れた仮定を含み、内部状態でベイズ推論を行うという主張は狭い条件下でしか成立しない | **自由エネルギー補題 = MBₚ の構造命題** | MB₀ の存在論には隠れた仮定は不要である。仮定は MBₚ/MB𝒻 の昇格条件として `mb_escalation_conditions.md` で明示化されている。論文VIII の立場では、Biehl の批判は MBₚ の構成要件を精密化する要請として前向きに受容される |

**共通戦略.** 批判者たちが攻めているのは「MB は自動で出る」という MBₚ/MB𝒻 レベルの主張であり、MB₀ の存在論的必然性ではない。三層化により守備範囲を MB₀ に限定し、MBₚ/MB𝒻 については昇格条件を明示することで、批判を **射程縮小** ではなく **構造精密化** として取り込める。

### 7.6 忘却論との統合 — FEP は忘却論の特殊解

論文I §6.5 (694-756 行) は「FEP は本理論の α > 0 部分構造として埋め込まれる」という非対称的関係を確立した。α < 0 側 (m-接続) には FEP の対応物が存在せず、忘却論は FEP より広い構造を持つ。この関係は本節の三層化と存在論的に接続する。

- **MB₀ の必然性** = 忘却論における「区別の可能条件」 = α ∈ ℝ 全域で Ob(C_α) が不変であること (公理 (F1)、定理 6.2.3 (iv))
- **MBₚ の恣意性** = 忘却論における「α ≥ 0 での射部分集合選択」 = FEP の精度 π に対応する統計的モデル化
- **MB𝒻 の恣意性** = 忘却論における「自己維持する主体としての被覆選択」 = Grothendieck 位相の飽和条件 (定理 6.8.6)

この対応は論文I §6.5 の埋め込み関係に存在論的根拠を与える: **FEP が忘却論の α > 0 部分構造として埋め込まれる** のは、FEP が MBₚ/MB𝒻 レベルの主張であり、MB₀ レベルの必然性 (忘却濾過の対象不変性) の上に乗っているためである。忘却論は MB₀ を正面から扱うがゆえに、FEP より広い。

重複回避: 本節は論文I §6.5 の「埋め込み」主張を繰り返さない。論文VIII §7 の役割は、その埋め込みに存在論的根拠 (MB₀ 層の存在論的必然性) を与えることに限定される。

### 7.7 系 — 合成概要 (Type α + δ) への予告

補題 7.4.1 は論文 VIII・A・LLM心 の3本を貫通する存在論的基礎である:

- **論文VIII**: THAT の先行性 (cell 次元の型階層) + 連続化 (本節 §7)
- **論文A**: Θ(B) スペクトラム (存在の連続量)
- **論文 LLMは心を持つか**: 忠実度 F_i の程度問題 (心の連続量)

3本の共通の背骨を取り出すと、Yugaku 文体ガイド §10 でいう **合成概要 (Type α + δ: 同一性の発見 + 統一)** の候補が見えてくる。具体的には「存在の連続化」を独立した短論文 (仮称 Paper XIII) に昇格させることが構想可能である。ただし本節は合成概要を実装しない。まず論文VIII §7 として定着させ、論文A / LLM心 がそれぞれ補題 7.4.1 を引用する形で整合性を確立した後に、合成概要への発展を検討する。
