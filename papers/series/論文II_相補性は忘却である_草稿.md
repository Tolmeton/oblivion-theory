# 相補性は忘却である — CPS 圏と二項対立の統一

**Paper II — v1.12 (2026-04-11)**

> *概要.* 微積分の基本定理 (FTC) のなかに、本稿の全構造が既に折り畳まれている。微分 d は関数から導関数への射影——積分定数を捨てる忘却操作——であり、積分 ∫ は導関数から原始関数への復元——失われた定数を補う構成操作——である。FTC の「不完全性」(∫∘d = id + C) は、往復で正確に一つの情報 (積分定数) が失われることを宣言している。この構造は、Container-Projection-Span (CPS) 圏——完全な構造 C_D からの二つの忘却関手 U_A, U_B が作るスパン——の最も基本的なインスタンスに他ならない。
>
> 本稿の主張: **あらゆる二項対立は、同一の情報に対する異なる忘却関手の射影として統一され、その非対称性の強度は二項の cell 次元差 α で決まる。** FTC の積分定数はこの非対称性の古典的実現であり、量子力学の位置/運動量、一般相対論の時空/質量、心身問題の身体/心はすべて同じスキーマの変奏である。先行論文 [Paper I] の忘却場パラメータ α(θ) を α(θ,t) に時間拡張することで、FEP の精度パラメータ π(t) を完全に回復する。

## §1. 序論: 微積分の基本定理に折り畳まれた構造

### 1.1 フック: 微分は忘却であり、積分は構成である

微積分の基本定理の二つの顔を並べる:

$$d \circ \int = \text{id} \qquad \text{(第二基本定理: 厳密)}$$
$$\int \circ d = \text{id} + C \qquad \text{(第一基本定理: 積分定数 C が残る)}$$

この非対称性を圏論の言語で読み替える:

| 解析的概念 | 圏論的操作 | CPS 的意味 |
|:---|:---|:---|
| 微分 d | 忘却関手 G (cell 降下) | f(x) → f'(x): 積分定数を**捨てる** |
| 積分 ∫ | 構成関手 F (cell 上昇) | f'(x) → ∫f'dx + C: 捨てた情報を定数で**補う** |
| d∘∫ = id | counit ε: FG → Id | 構成→忘却は完全に元に戻る |
| ∫∘d = id + C | unit η: Id → GF | 忘却→構成は**積分定数 C だけずれる** |
| 積分定数 C | **Drift** Θ = ‖η - id‖ | 忘却の不可逆成分の測度 |

Zhang–Guo–Keigher [2, 3] は、この構造を微分代数 (DIF) と Rota–Baxter 代数 (RBA) の間のモナド/コモナドの分配則として圏論化した。その構成は以下の通りである:

- 忘却関手 V: DIF → ALG は cofree 微分代数 G を右随伴に持ち (V ⊣ G)、コモナド D = VG: ALG → ALG を誘導する
- 忘却関手 U: RBA → ALG は自由 Rota-Baxter 代数 F を左随伴に持ち (F ⊣ U)、モナド T = UF: ALG → ALG を誘導する
- FTC の関係式 d∘P = id は、D と T の**混合分配則** (mixed distributive law) λ: DT ⇒ TD の存在条件と同値である [2, Theorem 3.8]

**注意**: d と ∫ は直接の随伴対 d ⊣ ∫ を構成しない。両者の関係は随伴から誘導されるモナド/コモナドの分配則 λ を通じて FTC として符号化される。本稿のテーブルで `d ⊣ ∫ via λ` と書く場合も、それはこの分配則を通じた**条件付きの結合**を略記したものであり、無条件の随伴主張ではない。

しかし——**3系統の先行研究 (Rota-Baxter 代数 [2,3], 合成微分幾何 [4,5], de Rham コホモロジー [6]) のいずれにおいても、d と ∫ を直接の関手対として d ⊣ ∫ という随伴を明示的に構成した研究は存在しない** [7]。随伴の「影」——分配則、amazing right adjoint、cokernel の普遍性——は各所に現れているが、統一されたことがない。

本稿の CPS スパンは、この統一を与える:

$$\text{DIF} \xleftarrow{U'} \text{DRB} \xrightarrow{V'} \text{RBA}$$

微分 Rota-Baxter 代数 (DRB) の圏——微分と積分の区別が溶解した完全な構造——が CPS の容器 C_D であり、U' (積分構造の忘却 → 微分代数) と V' (微分構造の忘却 → Rota-Baxter 代数) が二つの忘却関手である。「微分 or 積分」は、DRB を見ずに U'(C) か V'(C) の片方だけを知覚している状態に他ならない。

### 1.2 積分定数 = Drift: 忘却の古典的実現

FTC の非対称性はそのまま CPS の Type 分類に対応する:

- **∫∘d = id + C** (unit η が非自明): 微分は積分を前提しない (d は自立する) が、積分は微分を前提する (∫ は d の逆操作として定義される)。これは Type I (0-cell vs 1-cell) の特徴
- **d∘∫ = id** (counit ε が厳密): 一方向は完全。他方は不完全。非対称
- **C = 0 の特殊ケース**: 初期条件を固定すれば ∫∘d = id に退化 → Type II (α ≈ 1)

積分定数 C は、Paper I の忘却場 Φ の Drift そのものである。Drift = 0 ならば忘却は可逆であり力を生まない。Drift > 0 ならば不可逆な情報損失が発生し、その勾配が力を創発する。FTC はこの原理の最も素朴な実現である。

### 1.3 Bohr への修正: 相補性の対称性は cell 次元差で決まる

この構造は微積分に限らない。本稿の中心的洞察:

> **あらゆる二項対立 (A or B) は、完全な構造 C_D からの二つの忘却関手 U_A, U_B が作る CPS スパンのインスタンスであり、その非対称性の強度 α は二項の cell 次元差 Δd = |dim(A) - dim(B)| で決まる。**

| Type | cell 構造 | α | 対称性 | 例 |
|:---|:---|:---|:---|:---|
| **Type I** | 0-cell vs 1-cell | α > 1 | **非対称**: 1-cell は 0-cell を前提 | 微分/積分, 位置/運動量, 時空/質量, 身体/心 |
| **Type II** | 0-cell vs 0-cell | α ≈ 1 | **対称**: 同次元の鏡像 | 正/負, 表/裏, 粒子/反粒子 |
| **Type 0** | 0-cell vs 不在 | α = 0 | **退化**: 一項に崩壊 | 光/闇 (闇は光の不在) |

Bohr の相補性原理 (1928) は暗黙に Type II を仮定していた——粒子と波は「同等な記述」だと。だが運動量 p = m(dx/dt) は位置の時間微分を前提する。**位置は運動量なしに存在するが、逆は不可能。** これは Type I の特徴であり、Bohr の「対称的相補性」は 0-cell と 1-cell の区別を忘却した結果である。微積分の積分定数と量子力学の位置/運動量は、同じ CPS Type I の二つのインスタンスにすぎない。

Paper I [1] は力が忘却の方向的不均一から創発されることを示した。本稿は第二の主張を展開する: **二項対立は忘却の産物であり、その非対称性の強度は二項の cell 次元差で決まる。**

**系列位置と本稿の境界.** 本稿は忘却論シリーズの**橋頭堡** (bridgehead) として設計される。Paper II の核 (FTC → CPS → Face Lemma) は後続 Paper (Paper III-XIII) の基礎公理系を提供し、各 Paper が CPS を引用する累積構造を持つ。詳細は `drafts/infra/シリーズ前置宣言_Paper_II_v1.md` を参照。本稿は単独で勝負する論文ではなく、シリーズの第 2 章として読まれる。本稿が証明しない内容 (詳細な GR 形式化, 心身の圏論的基礎, 数値検証, 重力の CPS 積分, 意識問題の哲学的解決) は後続 Paper への 1 文 pointer として本文中に残る。

## §2. CPS 圏と六つの相補性

> **記法ブリッジ.** 本稿の 0-cell / 1-cell は局所規約として de Rham 的次数 $d_{\mathrm{dR}}$ を指す。Paper VIII の α-忘却濾過が用いる知覚階層 $d_{\mathrm{hier}}$ とは競合せず、本稿は「局所的な微分構造差」、Paper VIII は「主体が何段の構造まで知覚できるか」という階層差を記述する。以後、本稿では cell 表記を $d_{\mathrm{dR}}$ に固定し、VIII 側で $d_{\mathrm{hier}}$ に正規化して読む。

### 2.1 CPS スパンの定義

量子力学における位置と運動量、一般相対論における時空と質量-エネルギー、認知科学における身体と心は、それぞれ独立に相補的構造として知られている。本節の目的は、それらを個別の比喩として並べることではなく、共通する圏論的骨格を抽出し、CPS (Complementary Projection Structure) 圏として公理化することにある。

**定義 (CPS スパン).** ドメイン D における「A or B」の相補性は以下のスパンで表現される:

$$\text{Set}_A \xleftarrow{U_A^D} \mathcal{C}_D \xrightarrow{U_B^D} \text{Set}_B$$

ここで:
- C_D は「A と B の区別が溶解した」完全な構造の圏
- U_A, U_B は忘却関手
- 「A or B」= C を見ずに U_A(C) か U_B(C) の一方だけを知覚している状態

**リマーク (CPS の3層構造).** CPS スパンの存在は3つの層に分解される:

- **Layer 0 (スパン存在).** 任意の圏の対に対し、直積 Set_A × Set_B からの射影でスパンは構成可能である。これはトートロジーであり、CPS の理論的主張ではない。概念定義の最小系——「右 = 左の反対、左 = 右の反対」——は2射の相互充足であり、Face Lemma (§3.4) の 1-skeleton に対応する: 情報を持つが「方向」を持たない。
- **Layer 1 (自然な CPS スパン).** CPS の実質的主張は、忘却関手 U_A, U_B が以下の5条件を満たす**自然な忘却**であることにある:
  1. **Faithful** (注入的射対応): U は構造を「忘れる」が、異なる射を混同しない
  2. **左随伴の存在**: 忘却の部分的復元操作 F ⊣ U が存在する
  3. **結合生成**: C_D が U_A, U_B の像の結合で生成される (余分な構造を含まない)
  4. **非同型**: U_A ≇ U_B (忘却の仕方が構造的に異なる)
  5. **CPS 経由** (Face Lemma 整合): 全ての非退化 2-simplex (f, g, h) に対し、f と g は CPS スパンの異なる脚 (U_A 方向と U_B 方向) に沿う
- **Layer 2 (定量的制約).** Face Lemma と Stability Simplex Theorem が課す測定可能な制約 (§3.4, §3.5)。

CPS の反証可能性は Layer 1 に宿る。Layer 1 の5条件を満たす自然な CPS スパンが構成**不能**な対が存在すれば、それは CPS の「適用不能領域」を示す具体例となる (§7.1 参照)。

> **v1.11 での定義変更記録.** 本稿 v1.11 以降、Δd は「忘却関手のカーネルが失う微分構造の最高階数差」として定義される。例: 関数値 (0 階) vs 微分 (1 階) → Δd = 1。v1.0-v1.10 の「セル次元差」定義から変更されたが、各インスタンスの計算結果は不変。

> **Remark (Layer 1 5 条件の最小性).** Layer 1 の 5 条件は必要十分条件として提示される。具体的に、共変/反変対立 (例: Set → Set^op) と非 faithful 定値関手 (例: 定数圏) は CPS 不成立の反例である (§7.1.2 で詳述)。いずれかの条件を除くと、これらの反例が CPS 典型として誤認される。

### 2.2 CPS 公理系

| 公理 | 内容 | 充足条件 |
|:---|:---|:---|
| CPS0'[^cps0ii] | **容器-内容**: 容器は内容なしに存在しうるが逆は不可能 | Type I のみ。Type II では退化 (対称) |
| CPS1 | **相補的射影**: U_A と U_B は同一の C から構成される | 全 Type |
| CPS2 | **架橋**: U_A(C) と U_B(C) を接続する自然変換が存在 | 全 Type |
| ~~CPS3~~ | ~~**不確定性**: U_A と U_B を同時に完全適用できない~~ | (定理。§3.2 参照) |
| CPS5 | **マスク**: 一方の忘却が他方の構造を隠蔽 | 全 Type |

**リマーク (CPS3 の定理への格下げ).** CPS3 (不確定性) は独立公理ではなく CPS1 + CPS0' からの定理である (§3.2 命題「α > 0 のとき U_A と U_B の同時完全適用は不可能」参照)。したがって CPS の独立公理は CPS0', CPS1, CPS2, CPS5 の4つであり、CPS3 はこれらの帰結として導出される。全 Type において α 依存の不確定性強度を示す構造は保存される。

[^cps0ii]: 本系列では CPS0' を本稿段階では作業公理として用い、Paper VIII [in preparation] §6 で改めて導出する。

この格下げは本稿の公理列を最小化するための整理であり、後続の Paper VIII / IX [いずれも in preparation] では「臨界セクターの公理的仮置き」から「忘却濾過とエントロピーに支えられた導出」へと読み替えられる。

### 2.3 六つの相補性の CPS 充足性

以下のテーブルは各ドメインの相補性が CPS 公理をどのように充足するかを要約する。各インスタンスの圏論的構成の詳細は §2.5 を参照:

**リマーク (MoE と Dropout の区別).** 機械学習のインスタンスに関し、Mixture of Experts (MoE) の routing と neural network の Dropout は構造的に異なる忘却モードを実現する。Dropout は各ニューロンの情報を確率的に消去する**直接的忘却**であり、CPS の忘却関手 U の典型的インスタンスである。一方、MoE の routing は情報を消去するのではなく**分岐的選択**を行い、未選択の expert に対する情報を間接的に忘却する——これは U の特殊ケース（条件付き忘却）であり、§4.3.1 における α < 0 の「構造の溶解」とは機序が異なる。以下のテーブルでは両者を区別せず扱うが、定量的予測 (§5, §7.1) においては Dropout 型の直接的忘却を主要インスタンスとする。

**Δd の統一的定義 (v1.2).** Δd は忘却関手のカーネルが失う**微分構造の最高階数**の差として定義する:

$$\Delta d := |\text{ord}(\ker U_A) - \text{ord}(\ker U_B)|$$

ここで ord(ker U) は、U が忘却する構造の微分的階数——0階 = 点的データ (関数値, 状態)、1階 = 方向的データ (微分, 接続, 勾配)、2階 = 曲率的データ (2階微分, 曲率テンソル, 計量)——である。この定義は de Rham 複体の次数（k-形式の k）と整合するが、テンソルの「添字の数」(Sym²T* の「2」) とは異なる概念であることに注意: Sym²T* の「2」は対称積の次数であり、計量が 2-形式であることを意味しない。計量は 0-形式ではなく (点的データでない)、かつ 1-形式でもない (方向依存性は持つが、方向自体ではない)。計量は**2つの方向を同時に消費して距離を返す**構造であり、これが ord = 2 の根拠である。

| ドメイン | 圏 C_D | A / B | Δd | Δd = |dim ker U_A - dim ker U_B| 根拠 (§2.5, §3.3) | Type | CPS0' (容器→内容) |
|:---|:---|:---|:---|:---|:---|:---|
| **微積分** | DRB (微分 Rota-Baxter) | 導関数 d / 原始関数 P | 1 | d = 0-cell 操作, P = 1-cell 操作 (de Rham) | I | d は P なしに定義。P は d を前提 |
| 量子力学 | Hilb (ヒルベルト空間) | 位置 x̂ / 運動量 p̂ | 1 | x̂ = 掛算 (0-cell), p̂ = -iℏ∂/∂x (1-cell) | I | p = m(dx/dt) → x が前提 |
| 一般相対論 | Lor (Lorentz 多様体) | 位相 M / 計量 g | 2 | ker(U_top) = 計量 g (Sym²T* の切断 = 2階構造); ker(U_met) = 位相 (0階構造)。Δd = |2-0| = 2 | I | 真空解あり → M が前提 |
| 心身問題 | FEP 圏 | 身体 Θ / 心 ∇^(α) / 意識 F | 2 | ker(U_body) = 接続+曲率 (1階+2階構造); ker(U_mind) = 座標 (0階構造)。Δd = max(2,1)-0 = 2 | I | 岩は存在 → 身体が前提 |
| 力の統一 | — | EM / 重力 / ... | ≥1 | ゲージ群の cell 差 | I | 各ゲージ群の構造差 |
| 数学 | Ring (環) | 加法 / 乗法 | ≈0 | 同一代数構造の二演算 | I→II | R_+ への制限 |

### 2.4 Type II (対称的相補性) の例

Type I が非対称依存 (容器 > 内容) を持つのに対し、Type II は真に対称な相補性:

| ドメイン | A / B | 変換 | 対称性の根拠 |
|:---|:---|:---|:---|
| 電荷 | 正 / 負 | 電荷共役 C | 同じ cell 次元 |
| 物質 | 粒子 / 反粒子 | CPT 変換 | 同次元; 対消滅で α → 0 |
| 幾何 | 表 / 裏 | 向き反転 | 同次元の鏡像 |
| Newton | 慣性系 A / 慣性系 B | Galilei 変換 | 同次元; v/c → 0 退化 |

**命題.** Type II は CPS0' が退化 (U_A と U_B が同型) するケースに対応し、α ≈ 1 を与える。

### 2.5 CPS インスタンスの圏論的構成

各 CPS インスタンスに対し、以下の6項目を具体的に定義する: (1) 圏 C_D、(2) 忘却関手 U_A, U_B、(3) 射の型、(4) Δd の計算根拠、(5) Layer 1 五条件の検証、(6) Face Lemma の具体化。これにより「射」と「cell 次元差」の意味がドメインごとに精密に固定される。

#### 2.5.1 CPS1: 微積分 (FTC)

**圏 C_D.** DRB = 微分 Rota-Baxter 代数の圏。対象は三つ組 (A, d, P) で、A は次数付き代数、d: A → A は d² = 0 を満たす微分、P: A → A は Rota-Baxter 作用素（重み θ）であり、混合分配則 λ: dP → Pd が成立する [2, Definition 3.1]。

**忘却関手.**
- U_diff: DRB → DGA。P を忘却し微分次数付き代数 (A, d) のみを保持する
- U_RB: DRB → RBA。d を忘却し Rota-Baxter 代数 (A, P) のみを保持する

**射の型.** DRB の射 φ: (A, d_A, P_A) → (B, d_B, P_B) は代数準同型で d, P の両方を保存する。DGA の射は d のみ保存、RBA の射は P のみ保存。

**Δd の根拠.** §3.3 の統一定義 Δd = |dim ker U_A - dim ker U_B| の具体化として: U_diff のカーネル (P の情報) は Rota-Baxter 構造——積分の全情報——であり 1-cell 的 (面積 = 傾きの累積)。U_RB のカーネル (d の情報) は微分構造——点ごとの傾き——であり 0-cell 的。de Rham 複体 Ω*(M) における d: Ω^k → Ω^(k+1) の次数上昇に対応する。Δd = |0 - 1| = 1。

**Layer 1 検証.**
(i) Faithful: U_diff は全射的射対応だが単射的——異なる DRB 準同型が同じ DGA 準同型に潰れることはない。P の情報を捨てるが射を混同しない。✅
(ii) 左随伴: 自由微分代数 F_diff ⊣ U_diff が存在する（自由 DGA 構成）。✅
(iii) 結合生成: DRB の構造 (d, P) は DGA の像 (d のみ) と RBA の像 (P のみ) の結合で完全に記述される。✅
(iv) 非同型: d² = 0 （冪零）と Rota-Baxter 恒等式 P(a)P(b) = P(aP(b) + P(a)b + θab) は構造的に異なる法則を課す。U_diff ≇ U_RB。✅

**Face Lemma 具体化.** 3射: f = d (微分作用素)、g = P (積分作用素)、h = id + C (FTC: P∘d = id + 定数C)。合成制約 g∘f = h が FTC そのものであり、3射目 h（恒等 + 定数）が合成の独立検証を提供する。2射 {d, P} だけでは d と P の関係が測定できず、スカラー（Θ）しか見えない。3射目 h の存在により関係の「形状」（Ξ）が見える。

#### 2.5.2 CPS2: 量子力学 (位置/運動量)

**圏 C_D.** Hilb = 可分ヒルベルト空間の圏。対象は可分 Hilbert 空間 H、射は有界線形作用素 T: H → H'。

**忘却関手.**
- Spec_x: Hilb → Meas。スペクトル測度 E_x(Δ) = ∫_Δ |x⟩⟨x| dx への射影。ヒルベルト空間の運動量構造を忘却し、位置表示のみを保持する
- Spec_p: Hilb → Meas。スペクトル測度 E_p(Δ) への射影。位置構造を忘却し、運動量表示のみを保持する

**射の型.** Hilb: 有界線形作用素。Meas: 可測関数。Fourier 変換 F: L²(ℝ_x) → L²(ℝ_p) が架橋自然変換 (CPS2) に対応する。

**Δd の根拠.** §3.3 の統一定義の具体化として: U_pos のカーネル (運動量の情報) は微分作用素 p̂ = -iℏ∂/∂x の構造——1-cell 的 (勾配)。U_mom のカーネル (位置の情報) は掛け算作用素 x̂ の構造——0-cell 的 (関数の点ごとの値)。Stone-von Neumann の定理により [x̂, p̂] = iℏ が本質的であり、p̂ は FTC の d と構造的に同型。Δd = |0 - 1| = 1。

**Layer 1 検証.**
(i) Faithful: スペクトル分解は一意（スペクトル定理）。異なる作用素が同じスペクトル測度を持つことはない。✅
(ii) 左随伴: L² 完備化が自由構成として機能する。✅
(iii) 結合生成: 任意の |ψ⟩ ∈ H は位置基底 {|x⟩} でも運動量基底 {|p⟩} でも展開可能。H = span{|x⟩} = span{|p⟩}。✅
(iv) 非同型: Spec_x ≇ Spec_p。Fourier 変換 F ≠ id が両者を結ぶが、F は恒等関手ではない。✅

**Face Lemma 具体化.** 3射: f = x̂ (位置作用素)、g = p̂ (運動量作用素)、h = [x̂, p̂] = iℏ (交換子)。交換関係 h が合成 g∘f - f∘g の検証を提供する。2射 {x̂, p̂} だけでは非可換性（不確定性の方向）が測定できない。3射目 h = iℏ の存在により不確定性の**構造**が見える。

#### 2.5.3 CPS3: 時空/質量-エネルギー (一般相対論)

一般相対論の CPS 構造は Paper XIII 「時空は忘却である」§2 で独立に展開される。本稿では pre-geometric な CPS 構造の数学的定式化までに留める。

#### 2.5.4 CPS4: 心身問題 (身体/心/意識)

本節は Face Lemma 抽象の具体化例として、圏 FEP の記号定義と Δd = 2 の計算根拠のみを提示する。心身問題の圏論的詳細と有限主体の構成は Paper VIII 「存在は忘却に先行する」§6.3 有限主体定理で独立に展開される。本稿では Face Lemma の適用例として構造的に言及するに留める。

**圏 C_D.** FEP 圏。対象は三つ組 (Θ, ∇^(α), F) で、Θ は状態空間（身体）、∇^(α) は α-接続の族（心 = 推論構造）、F は忘却曲率テンソル（意識 = メタ認知）。射は統計的に十分な写像。

**忘却関手.**
- U_body: FEP → Dyn。推論構造 ∇^(α) と曲率 F を忘却し、力学系 (Θ, ẋ = f(x)) のみを保持する
- U_mind: FEP → Stat。状態空間の座標を忘却し、統計構造 (M, ∇^(α), g_F) のみを保持する

**Δd の根拠.** §2.3 の統一的定義のもと:
- ker(U_body) = 推論構造 ∇^(α) + 曲率 F: 接続は 1 階、曲率は 2 階。ord = max(1, 2) = 2
- ker(U_mind) = 状態空間の座標 θ: 0 階構造。ord = 0
- Δd = |2 - 0| = 2

Δd(身体→意識) = 2 は「説明のギャップ」の構造的原因である: 0-cell から 2-cell への翻訳には 1-skeleton → 2-skeleton の遷移 (Face Lemma) を含み、中間段階 (1-cell = 心) を必ず経由する構造的必然性を持つ。詳細な論証と心身問題の圏論的基礎は Paper VIII §6.3 で扱われる。

#### 2.5.5 Type II の圏論的構成例: 電荷 (正/負)

**圏 C_D.** QED = 量子電気力学の圏。対象は電磁場の配位 (A_μ, ψ)、射はゲージ変換 ψ → e^{iα(x)}ψ, A_μ → A_μ + ∂_μα。

**忘却関手.**
- U_+: QED → Rep_+。場の配位から正電荷の表現のみを抽出し、負電荷の構造を忘却する
- U_-: QED → Rep_-。場の配位から負電荷の表現のみを抽出し、正電荷の構造を忘却する

**Δd の根拠.** 正電荷と負電荷は共に U(1) 接続の方向（1-cell: ゲージ場 A_μ の「向き」）に対応する。次元は同一であり、電荷共役 C: U_+ ↔ U_- は同型を与える。Δd = 0 → Type II。CPS0' は退化（どちらも他方なしに定義不能——正電荷の概念は負電荷の不在では定義されない）。

**Layer 1 検証.** (iv) 非同型: U_+ ≇ U_-（電荷共役 C ≠ id）だが、C は iso を与える。したがって CPS0' は退化し Type II に分類される。α ≈ 1。

#### 2.5.6 インスタンス間の構造的同型

5インスタンスの圏論的構成を並置すると、構造的同型が浮かび上がる:

| | 容器 (0-cell) | 内容 (1-cell) | 創発 (2-cell) | Δd |
|:---|:---|:---|:---|:---|
| FTC | 関数値 f(x) | 微分 d = 勾配 | FTC: P∘d = id+C | 1 |
| QM | 位置表示 ψ(x) | 微分作用素 p̂ = -iℏ∂/∂x | [x̂, p̂] = iℏ | 1 |
| GR | 位相多様体 M | 接ベクトル場 ∂_μ | 計量 g → 曲率 R | 2 |
| 心身 | 状態空間 Θ (身体) | α-接続 ∇^(α) (心) | 忘却曲率 F_{ij} (意識) | 2 |
| 電荷 | — | U(1) の方向 | — | 0 |

全 Type I インスタンスに共通するパターン: **容器は点的 (0-cell)、内容は方向的 (≥1-cell)、内容の忘却は方向の喪失であり、方向の喪失の不均一が力を生む。** FTC と QM は 1-cell レベル (Δd = 1) の忘却であり、GR と心身問題は 2-cell レベル (Δd = 2) の忘却である。Δd = 2 のケースでは、1-cell → 2-cell の遷移 (Face Lemma) が追加の構造的意味を持つ。

**リマーク (忘却関手の計量装備).** 上記の全 Type I インスタンスにおいて、忘却関手 U_A, U_B は暗黙に「何をどれだけ忘れるか」の近傍構造を前提している。Paper I の方向性定理 (定理 5.1) は、この近傍の正しい計量が Fisher 情報計量 g_{ij} であることを示す。

深層学習の最適化における近傍構造の進化は、CPS Type 分類の精密化と正確に対応する:

| 忘却の近傍構造 | 手法 | CPS 的意味 | α との関係 |
|:---|:---|:---|:---|
| ユークリッド球 $\|\epsilon\| \leq \rho$ | SAM [Foret 2021] | $U_A \approx U_B$ (等方的忘却) | α ≈ 1: Type II 的対称性。忘却の方向を区別しない |
| Fisher 楕円体 $\epsilon^T g^{(0)} \epsilon \leq \rho$ | Fisher SAM [Kim 2022] | 異方的忘却 → 曲率構造を反映 | α > 1: Type I 的非対称性。固定 α での異方性 |
| α-接続楕円体 $\epsilon^T g^{(\alpha)} \epsilon \leq \rho$ | α-SAM (Paper I §6.7) | α(l)-依存の忘却 → Type I の精密制御 | α(l) 学習可能: 層ごとの CPS 非対称性を最適化 |
| 忘却場勾配正則化 $\lambda\|\nabla^{(\alpha)}\Phi\|^2$ | Oblivion-Aware SAM (Paper I §6.8) | Φ の**方向**が力を決定 → CPS 非対称性の直接最適化 | λ < 0: 方向性定理の操作化。$d\Phi \wedge T \neq 0$ を駆動 |
| 上記の合成 | α-Oblivion SAM | 近傍形状 + 忘却方向の同時制御 | (α(l), λ, ρ) の3パラメータで CPS 全構造を制御 |

この階層は厳密な包含関係を形成する:

$$\text{SAM} \subset \text{Fisher SAM} \subset \alpha\text{-SAM} \subset \alpha\text{-Oblivion SAM}$$

**CPS Type との接続.** ユークリッド近傍での忘却は U_A と U_B を等方的に扱い (α → 1: Type II 的対称性)、Fisher 近傍での忘却は多様体の曲率構造を反映して異方的に扱う (α > 1: Type I 的非対称性)。α-SAM はこの α を層ごとに学習可能にし、§3.3 の層化定理（底空間 Δd、ファイバー α(θ)）と直接対応する。Oblivion-Aware SAM はさらに進んで、忘却場 Φ の勾配の**方向**を直接最適化する——これは方向性定理 (Paper I 定理 5.1) の「力は忘却の方向が決める」という主張の操作的実現である。

**Face Lemma との関係.** §3.4 の Face Lemma により、忘却の方向が測定可能になるには最低3つの独立な生成射 (2-simplex) が必要である (dim Ξ ≥ 2)。Oblivion-Aware SAM の $\|\nabla\Phi\|^2$ はこの Ξ の方向構造を勾配ノルムとして計算可能な量に操作化する。SAM/Fisher SAM はスカラー的平坦性 (dim Ξ = 1) を制御するに過ぎないが、Oblivion-Aware SAM はベクトル的方向性 (dim Ξ ≥ 2) を制御する——これが Face Lemma の「3射の壁」を超える操作的意味である。

## §3. CPS 双対性定理

### 3.1 α と cell 次元差

**定理 (CPS 双対性).** CPS スパンの非対称性パラメータ α は、二項の cell 次元差 Δd = |dim(A) - dim(B)| と以下の対応を持つ:

- Δd ≥ 1 → α > 1 (Type I: 非対称的相補性。力が創発)
- Δd = 0 → α ≈ 1 (Type II: 対称的相補性。退化ケース)
- 一項に退化 → α = 0 (Type 0: 相補性不成立)

Paper I の方向性定理の CPS 的翻訳:

$$F_{ij} \neq 0 \iff \alpha > 1 \iff \Delta d \geq 1$$

**力は、異なる cell 次元の対の間でのみ生じる。** 同次元の対 (Type II) は力を生まない——対称性そのものが F_{ij} = 0 を保証する。

### 3.2 CPS3 (不確定性) の導出

**命題.** CPS3 (同時適用不可能性) は CPS1 + CPS0' から導出される。

CPS1 (同一の C からの構成) により、U_A(C) と U_B(C) は C の異なる射影である。CPS0' (非対称依存) により、U_A(C) を完全に回復するには U_B(C) だけでは不十分 (情報が非対称に分配されている)。したがって両方の完全な回復は不可能 → CPS3。

Type II (CPS0' 退化) では: 不確定性は残るが、強度は弱い (α ≈ 1)。

この段階では CPS3 を「同時適用不能性の最小形」として述べるに留める。臨界セクターでの再解釈と、エントロピー側から見た再定位は Paper VIII / IX に委ねる。

### 3.3 Paper I α(θ) との統一: 層化定理

Paper I は α ∈ ℝ を統計多様体 M 上の動力学場 α(θ) に昇格させた。Paper II は α を CPS スパンの cell 次元差 Δd ∈ ℕ で定義する。一見矛盾するこの二重定義の統一が本節の目的である。

**定義 3.3.1 (α の層化).** Paper I の精度場 α(θ) と Paper II の非対称性パラメータ Δd を、以下の層化構造で関係づける:

$$\alpha(\theta) \in \text{Fiber}(\Delta d) \qquad \text{where} \qquad \pi: \alpha(\theta) \mapsto \Delta d = \lfloor |\alpha(\theta_\infty)| \rfloor$$

ここで θ_∞ は α の非摂動値（遠方境界条件）を表す。

> **定義としての採用.** 本稿では整数部分対応 α(θ_∞) = Δd を **定義** として採用する。Amari-Nagaoka (2000) の統計多様体では非厳密であり、厳密な関手証明は Paper III 「Markov 圏の向こう側」で実施される予定。本稿はこの定義の下で §3.7 以降の定理を展開する。

**(a) 底空間 (Δd ∈ ℕ):** CPS ペアの cell 次元差は離散的な位相的分類を与える。Δd は忘却関手 U_A, U_B のカーネルの次元差 |dim ker U_A - dim ker U_B| であり、連続変形で変わらない位相的不変量。これが CPS の Type (I/II/0) を決定する。

**(b) ファイバー (α(θ) ∈ ℝ):** 固定された CPS Type (固定 Δd) の中で、パラメータ空間の各点 θ における**局所的精度**は連続的に変動する。この局所変動が Paper I の α(θ) である。

*構成の動機.* この定義は Paper I §6 の構成を逆から読むことで動機づけられる。(1) CPS ペアが与えられると、忘却関手 U_A, U_B の構造差は Δd を決定する。(2) 固定ドメイン D 内で、統計モデル族 {p(x|θ)} は統計多様体 M を定義し、α-接続の族 ∇^(α) を持つ [Amari-Nagaoka 2000]。(3) CPS スパンの非対称性 Δd は、この統計多様体上の α の漸近値 α(θ_∞) の整数部分に対応する: 具体的には、α → +1 (m-接続) とα → -1 (e-接続) の非対称性が Δd = 1 に、より高次の cell 差には α の漸近値がより大きくなる。

逆に、(4) α(θ) の勾配 ∂_i α は、固定 Δd 内部での「精度の空間変動」を記述し、Paper I の忘却曲率 F_{ij} = (α/2)[d(ΦT)]_{ij} に寄与する。

まとめると、Δd はグローバルな位相的分類であり α(θ) はそのファイバー内の局所的力学場である。

**注意.** α(θ_∞) の整数部分が Δd に対応するという関係は、本稿では**定義**として採用する。Amari の α-接続族における α ∈ ℝ と CPS の cell 次元差 Δd ∈ ℕ の間の数学的な導出は今後の課題であり、現時点では上記の層化構造は motivated construction として位置づけられる。

**系 3.3.2 (力の二重起源).** 忘却曲率 F_{ij} は二つの独立な寄与を持つ:

| 起源 | Paper I の寄り | Paper II の寄り | 数学的項 |
|:---|:---|:---|:---|
| **Type 内変動** | α(θ) の勾配 (精度の空間変動) | 固定 Type 内の力 | Φ ∂_i α (遷移層力) |
| **忘却の方向性** | Φ の勾配 (忘却の不均一) | Δd > 0 の非対称性 | α ∂_i Φ (忘却勾配力) |

F_{12} = (α ∂_μΦ + Φ ∂_μα) / σ のうち、第一項は「忘却量の配分」(CPS の実質的非対称性)、第二項は「忘却**方式**の空間変動」(CPS Type 間の遷移)。

**直感的再述.** Δd は「何と何の間か」(微分 vs 積分、身体 vs 心…)、α(θ) は「その間のどこにいるか」。地形図の等高線で言えば:

- **Δd = 山の**標高差**** (離散的 — 山が存在するかしないか)
- **α(θ) = 山の**斜面プロファイル**** (連続的 — 傾斜は場所ごとに変わる)
- **F_{ij} = 斜面の力** (α(θ) の勾配が直接的な力を生む)

標高差 Δd = 0 の平地では、α(θ) がいくら変動しても力は弱い (Type II)。標高差 Δd ≥ 1 の山がある場所でのみ、斜面の勾配が意味のある力を生む (Type I)。

### 3.4 Face Lemma: CPS が非自明であるための最小構造

CPS の非対称性 (α > 1) が力を生むには、忘却の「方向」が測定可能でなければならない。これは忘却量 Θ (スカラー) から忘却スペクトラム Ξ (形状) への昇格を意味する。

#### 3.4.1 数学的準備: nerve と 2-coskeletal 性

圏 $\mathcal{C}$ の nerve $N(\mathcal{C})$ は単体的集合であり、$N_n$ は $n$ 個の合成可能な射の列からなる。Grothendieck の nerve 特性化定理 [24] により、圏の nerve では全ての**内部 horn** $\Lambda^n_k$ ($n \geq 2$, $0 < k < n$) が**一意に**充填される。

**系 (nerve の 2-coskeletal 性).** $n \geq 3$ の simplex は 2次元面から一意に復元される。これは全ての圏の nerve に対して成立し、Kan complex の仮定を必要としない。

*証明.* $n = 3$ の場合: 3-simplex の内部 horn は 2-simplex（面）の集合から構成され、Grothendieck 定理により一意に充填される。一般の $n$ に対しては帰納法。$\square$

#### 3.4.2 忘却スペクトラムの次元

**定義 (Star complex).** 対象 $x \in \text{Ob}(\mathcal{C})$ の star complex $\text{St}(x)$ は、$x$ を頂点として含む $N(\mathcal{C})$ の全 simplex の和。

**定義 (dim Ξ).** 対象 $x$ の忘却スペクトラムの次元:

$$\dim \Xi(x) := \max\{k \mid \text{St}(x) \text{ が非退化 } k\text{-simplex を含む}\}$$

ここで「非退化」は退化作用素の像でないこと（恒等射のみからなる simplex を除外）を意味する。

#### 3.4.3 CPS-安定性

**定義 (CPS-安定性).** CPS 圏の対象 $x$ が CPS-安定であるとは:

(S1) **非自明性**: $\text{St}(x)$ が恒等射以外の射を含む

(S2) **検証可能性**: $\text{St}(x)$ が非退化 2-simplex を含む

#### 3.4.4 Face Lemma

**定理 3.4.1 (Face Lemma).** $\mathcal{C}$ を圏、$x \in \text{Ob}(\mathcal{C})$ とする。

**(a) 最小性.** $x$ が CPS-安定であるための最小条件は $\dim \Xi(x) = 2$。

**(b) 等価条件.** 以下は同値:
(i) $\dim \Xi(x) \geq 2$
(ii) $x$ を頂点とする composable triple $(f, g, h)$ with $g \circ f = h$ が存在する（$f, g$ は恒等射でない）
(iii) $x$ は CPS-安定である

**(c) 飽和.** $\dim \Xi(x) \geq 3$ は $\dim \Xi(x) = 2$ に対して構造的に新しい情報を持たない（nerve の 2-coskeletal 性 [§3.4.1] による）。

*証明.*

**(a)** dim Ξ の各値で (S1), (S2) を検証する:

**dim Ξ = 0:** St(x) は $x$ 自身のみ。(S1) 不成立。

**dim Ξ = 1:** St(x) は 1-simplex（射）を含むが 2-simplex を含まない。(S1) 成立。(S2) 不成立: 2-simplex が存在しないため合成制約 $g \circ f = h$ が検証不能。忘却量 $\Theta$ はスカラーとして測定可能だが方向 $\partial_i \Theta$ は定義できない——方向の測定には 2-simplex の2つの独立な辺が提供する2軸が必要。

**dim Ξ = 2:** St(x) は非退化 2-simplex $(f, g, h)$ を含む。(S1): $f \neq \text{id}$ より成立。(S2): 2-simplex の存在により成立。$\square$

**(b)** (i)⟹(ii): dim Ξ ≥ 2 ならば非退化 2-simplex が存在し、その辺が composable triple。(ii)⟹(iii): composable triple が (S1), (S2) を充足。(iii)⟹(i): (S2) より dim Ξ ≥ 2。$\square$

**(c)** Nerve の 2-coskeletal 性 [§3.4.1] により、$n \geq 3$ の simplex は 2次元面から一意に復元される。新しい simplex は既存の 2-simplex の合成の「記録」にすぎず、構造的に新しい情報を持たない。$\square$

直感的再述:
- **2射** (1-skeleton): 表と裏。MECE だが冗長性ゼロ。合成 g∘f は f, g の派生であり外部検証不能。Θ (スカラー) しか測れず、**忘却の方向が見えない**
- **3射** (2-skeleton): 三角形。面積 = 冗長性 = 自由度。合成 g∘f = h が検証可能。Ξ (スペクトラム形状) が見え、**力の方向がわかる**

> **前方参照 (F×).** 本節の Face Lemma は「三角形が立つ最小条件」を与える。後続稿ではこの 2-simplex のどの交差成分が blanket を担うかを $F^\times$ 記法で精密化するが、本稿ではまず 3射の存在そのものを最小条件として固定する。

### 3.5 Stability Simplex Theorem: 符号付き忘却量

Face Lemma の「3」は偶然ではない。随伴 F ⊣ G の三角恒等式 Δ_F がその源泉:

$$\varepsilon_F \circ F(\eta) = \text{id}_F$$

これは3つの射 {F(η), ε_F, id_F} と1つの合成制約からなる**2-simplex そのもの**。

**定義 (安定性構造).** (X, α, β, γ, σ) で:
- α, β: X に関わる非恒等射 (測定)
- γ: 合成の結果 (恒等射を許す — Δ_F では γ = id_F)
- σ: 制約 γ = β ∘ α

安定 iff α ≠ id かつ β ≠ id (非退化条件)。Fix(G∘F) 不動点の F ≠ Id, G ≠ Id と同値。

**符号付き忘却量.** Δ_F の各射に符号付きスコア Θ: Hom(C) → V を割り当てる。ここで V の構造は定理が決定する:

| 成分 | Δ_F | 忘却量 | Fix(G∘F) 対応 |
|:---|:---|:---|:---|
| α (展開) | F(η) | +λ (構造を忘れて自由にする) | 展開可能性 (Generative) |
| β (収縮) | ε_F | -λ (自由を忘れて構造を付与) | 不動点性 (Fix) |
| γ (結果) | id_F | 0 (相殺の結果) | 自己参照性 (Self-referential) |
| σ (制約) | ε_F ∘ F(η) = id_F | (+λ) + (-λ) = 0 | 過程が結果を証明 |

**定理 (Stability Simplex Theorem).** 忘却量スコア Θ: Hom(C) → V が以下の5条件を満たすとき、V ≅ (ℝ, +) が代数的に一意に決定される。

*5条件の抽出.* CPS の安定性構造 (X, α, β, γ, σ) と統計多様体上の忘却場 Φ から、Θ に以下が要求される:

| 条件 | 内容 | 起源 |
|:---|:---|:---|
| (V1) 加法的群 | Θ(g∘f) = Θ(g) + Θ(f), Θ(id) = 0, 逆元 -Θ(f) の存在 | 射の合成 + 三角恒等式の非退化条件 |
| (V2) 可換 | Θ(f) + Θ(g) = Θ(g) + Θ(f) | モノイダル構造の対称性 (Copy の可換余モノイド + 忘却場 Φ のスカラー値性) |
| (V3) 全順序 | 忘却の方向: 展開 (+) と収縮 (-) の順序が任意の二元に対し決定可能 | CPS の安定性: α ≠ id, β ≠ id. 展開と収縮の区別は Ξ の方向構造の基盤 |
| (V4) アルキメデス | 任意の a, b > 0 に対し n·a > b なる n ∈ ℕ が存在 | 忘却場 Φ は統計多様体 M の有限な Fisher 情報計量に対して有界値を取る — 「無限に大きな忘却」は M の有限性が排除する |
| (V5) 稠密・除算可能 | 任意の a に対し a/n が存在 (n ∈ ℕ)、任意の二元の間に第三の元が存在 | CPS パラメータ α ∈ ℝ の連続変動 (§3.3 層化定理)。Θ は α(θ) の連続変形を値域に反映しなければならない |

*証明.* 2段階で示す。

**Step 1 (代替値域の排除).** 5条件により以下の候補が排除される:

**(i) V = ℝ_≥0 (非負実数):** 群構造 (V1) が不成立。a > 0 のとき -a ∉ ℝ_≥0 であるから逆元が存在しない。三角恒等式 Θ(ε_F) + Θ(F(η)) = 0 で両辺 ≥ 0 は Θ(ε_F) = Θ(F(η)) = 0 を強制する。安定性構造の非退化条件 α ≠ id, β ≠ id と矛盾。 □

**(ii) V = ℤ (整数):** (V1)-(V4) を満たすが、(V5) が不成立。ℤ は稠密でもなく除算可能でもない。CPS パラメータ α(θ) ∈ ℝ の連続変動に対し、Θ(f) は途中値を取れず、微小な忘却量の変化を表現できない。形式的には、α(θ) → α(θ) + ε (ε → 0) の連続極限で Θ の像に非自明なギャップが残り、層化定理 (§3.3) の連続ファイバー構造と矛盾する。 □

**(iii) V = ℂ (複素数):** (V1), (V2), (V5) を満たすが、(V3) が不成立。ℂ は全順序を定義できない: i > 0 と仮定すると i² = -1 > 0 となり矛盾、i < 0 としても同様。忘却の方向 (展開 vs 収縮) に一貫した順序が定まらず、Ξ の方向構造が曖昧になる。Stability Simplex の +λ と -λ の区別が定義不能。 □

**(iv) V = ℝ/ℤ (円群):** (V1), (V2), (V5) を満たすが、(V3) が不成立。全順序は周期的になるため定義できない — 0.3 > 0.2 だが、0.3 + 0.8 = 0.1 < 0.2 + 0.8 = 0.0 = 1.0。さらに (V4) のアルキメデス性も失われる: n·a が mod 1 で循環し、任意の b を超越できない。忘却量の蓄積が「巻き戻る」ことになり、「忘却は不可逆に蓄積する」という CPS の基本的物理的要請 (Drift Θ = ‖η - id‖ ≥ 0) と整合しない。 □

**(v) 一般のアーベル群 V:** 上記を包含する統一的排除は Step 2 で与える。

**Step 2 (Hölder の定理による V ≅ ℝ の導出).** (V1)-(V4) を満たすアーベル群 V は、Hölder の定理 [Hölder, 1901; Fuchs, Partially Ordered Algebraic Systems, 1963, Ch. IV] により:

> *アルキメデス全順序付きアーベル群は (ℝ, +, ≤) の部分群に順序同型である。*

すなわち、V は ℝ の加法部分群 H ↪ ℝ に順序同型を持つ。ここで (V5) により H を決定する。

**ℝ の加法部分群の分類** ([Fuchs, Ch. IV]): ℝ の加法部分群は (a) {0}, (b) cyclic 群 ℤ·r (r ∈ ℝ), (c) ℝ 内で稠密な群, (d) ℝ 自身のいずれかである。

- **(a) H = {0} の排除**: 安定性構造の非退化条件 α ≠ id, β ≠ id (L402) より Θ(α) ≠ 0。よって H ≠ {0}。
- **(b) H = ℤ·r の排除**: (V5) の稠密性により、任意の二元の間に第三の元が存在する。ℤ·r は離散であり稠密でないため排除。
- **(c) ℝ 内で稠密だが ℝ ≠ H (例: ℚ) の排除**: この排除には (V5) の除算可能性に加え、忘却場 Φ の**滑らかさ**を用いる。

  Θ は忘却場 Φ: M → ℝ の積分・微分として定まる (F_{ij} = (α/2)[d(ΦT)]_{ij}; §3.1-§3.2)。Φ が C^∞ 級で統計多様体 M が連結 (Fisher 計量に対し弧状連結 [Amari-Nagaoka, §2]) であるとき、Θ の像 Im(Θ) は ℝ の**連結部分集合** = 区間を含む。

  ℝ の加法部分群で**区間を含む**ものは ℝ 自身に限る。証明: H ⊇ [a, b] (a < b) とする。任意の r ∈ ℝ に対し、h = b - a > 0 ∈ H。⌊r/h⌋ · h ∈ H (H は加法群) かつ r - ⌊r/h⌋ · h ∈ [0, h) ⊂ [a-a, b-a] ⊂ H-a ⊂ H (H は加法群で -a ∈ H)。したがって r = ⌊r/h⌋ · h + (r - ⌊r/h⌋ · h) ∈ H。よって H = ℝ。

  ℚ は ℝ 内で稠密かつ除算可能だが、区間を含まない (ℚ ∩ [0,1] は区間ではない — √2/2 ∉ ℚ)。したがって ℚ は排除される。

- **(d) H = ℝ**: (a)-(c) の排除により、H = ℝ が唯一の候補。

以上より、V ≅ (ℝ, +, ≤) が代数的に一意に決定される。 □

**系 (符号付き相殺).** V ≅ ℝ の下で三角恒等式 ε_F ∘ F(η) = id_F に Θ を適用すると:

$$\Theta(\varepsilon_F) + \Theta(F(\eta)) = \Theta(\text{id}_F) = 0$$

Θ(F(η)) = +λ, Θ(ε_F) = -λ (λ ≠ 0) が非退化解として一意に許される。正の忘却 (+λ = 構造の忘却 = 展開) と負の忘却 (-λ = 自由の忘却 = 収束) が非自明に相殺する。Ξ は方向構造を持ち F ≠ 0 が可能。

**リマーク (5条件の相互独立性).** (V1)-(V5) は論理的に独立である: ℤ は (V1)-(V4) を満たすが (V5) を欠き、ℂ は (V1)(V2)(V5) を満たすが (V3) を欠き、ℝ/ℤ は (V1)(V2)(V5) を満たすが (V3)(V4) を欠き、ℝ_≥0 は (V3)(V4)(V5) を満たすが (V1) を欠く。5条件の合成のみが V ≅ ℝ を決定する — いずれの条件も省略不可能。

**系 (三者の一致).** A (合成の検証可能性), B (忘却スペクトラム), C (Fix(G∘F) 三属性) は Δ_F の3つの読み方関手であり、以下の一致が成り立つ:

| 量 | 値 | 由来 |
|:---|:---|:---|
| $\min|\text{Hom}|_{\text{verifiable}}$ | **3** (射の数) | 2-simplex の辺の数 = 3 |
| $\min \dim(\Xi)_{\text{non-trivial}}$ | **2** (simplex 次元) | 定義 3.4.2: 非退化 2-simplex の存在 |
| $|\text{Fix}(G \circ F) \text{ attributes}|$ | **3** (属性の数) | 不動点性 + 展開可能性 + 自己参照性 |

射の数 (3) と simplex の次元 (2) の差異は、2-simplex が 3辺を持つが 2次元であることに対応する。三角恒等式 $\Delta_F$ は 3射の合成制約であり、その制約が張る空間は 2次元 (3射 - 1制約 = 2自由度)。

### 3.6 CPS への帰結: 力と創発の統一

Face Lemma + Stability Simplex は CPS に以下の制約を課す:

- CPS が非自明 (α > 1) → スパンの C_D に最低 2-simplex が存在
- 忘却量は符号付き (R 値、非 R_≥0 値) → 展開と収縮が区別される
- Type II (α ≈ 1) → Δ_F が退化: +λ = -λ → λ = 0 → 力なし

さらに、1-skeleton → 2-skeleton の遷移は**創発の構造的定義**を与える (§6.1)。Face Lemma は「力の最小条件 (3射)」と「創発の閾値 (3射)」が同一であることを証明する。すなわち:

$$\text{力の創発} \iff \text{メタの創発} \iff \text{2-skeleton の出現}$$

力と創発は別個の現象ではなく、同一の構造的遷移の二つの読み方である。

**リマーク (α = 0 退化: CPS Type 0).** α = 0 は Levi-Civita 接続 (曲率ゼロ) に対応し (§4.3.3 参照)、CPS の退化ケースを定義する。α = 0 では F_{ij} の α 由来項がゼロとなり、力の符号が反転する臨界面として機能する。これを **CPS Type 0** と呼ぶ: Type I (α > 0, 容器-内容の非対称性が顕在) と Type II (α ≈ 1, 対称的退化) の中間であり、構造化と脱構造化の相転移点を記述する。LLM の α-遷移層 (§5.2) は Type 0 近傍での力のダイナミクスの実験的検証対象である。

### 3.7 Blanket 生成定理: CPS における FEP の包含

FEP は Markov blanket を**前提**する。CPS は Face Lemma を通じて blanket の**生成条件**を与える。本節はこの関係を定理として定式化し、CPS ⊋ FEP の形式的根拠を確立する。

この定式化の鍵は、確率論的概念 (条件付き独立性、Markov blanket) と圏論的概念 (射の分解、合成制約) の間の精密な橋渡しである。Fritz [16] の Markov category がこの橋を提供し、Parzygnat [21] はこの枠組みを量子系に拡張してベイズ反転の3層階層 (inverse ⊃ disintegration ⊃ Bayesian inversion) を構成した——CPS は古典的セクターのみを使用するが、Fritz–Parzygnat 枠組みの採用により将来の量子拡張への経路が構造的に開かれている。Smithe [17, 18] は同じ圏論的道具を用いて FEP の compositional semantics を構築した。本節は彼らの基盤の上に、CPS 固有の Face Lemma を経由して blanket の**生成条件**を定式化する——これは Fritz–Smithe の枠組みにない新しい結果である。

#### 3.7.1 Markov category と条件付き独立性

Fritz [16] は条件付き独立性の合成的 (synthetic) 定式化を与えた。以下にその核心を要約し、CPS への接続を定義する。

**定義 (Markov category; [16, Def. 2.1]).** 対称モノイダル圏 (C, ⊗, I) が Markov category であるとは、以下の構造を持つことをいう:
1. 各対象 X に**可換余モノイド構造**: comultiplication copy_X: X → X ⊗ X と counit del_X: X → I が備わる
2. **余モノイド方程式**: copy の結合律 (3-copy の一意性)、copy と del の整合性 (copy して片方を del = 恒等)
3. **モノイダル整合性**: copy_{X⊗Y} = (id_X ⊗ swap ⊗ id_Y) ∘ (copy_X ⊗ copy_Y)、del_{X⊗Y} = del_X ⊗ del_Y
4. **del の自然性**: 任意の射 f に対し del ∘ f = del （出力を破棄する操作は入力に依存しない）

直観的には、copy_X は入力値を複製する決定論的操作、del_X は入力値を破棄する操作に対応する。FinStoch（有限集合と確率行列の圏）はこの定義を満たす基本例である [16, Ex. 2.5]。

**定義 (条件付き独立性; [16, Def. 12.1]).** Markov category C において、分布 ψ: I → X ⊗ W ⊗ Y が「X と Y が W を条件として独立」(X ⊥ Y | W) を示すとは、射 f: W → X, g: W → Y, φ: I → W が存在して:

$$\psi = (f \otimes \mathrm{id}_W \otimes g) \circ (\mathrm{id}_W \otimes \mathrm{copy}_W) \circ \mathrm{copy}_W \circ \varphi$$

と分解されることをいう。すなわち、ψ は W の分布 φ を copy し、独立に f と g を適用する形に因数分解される。

**リマーク (Fritz の要点).** この定義の強みは二つある。第一に、FinStoch において通常の条件付き独立性 P(X,Y|W) = P(X|W)P(Y|W) と一致する [16, Rem. 12.2]。第二に、条件付き確率 (conditional) の存在を前提しない——確率論の Stoch 圏では条件付き確率が常には存在しないが [16, Ex. 11.7]、条件付き独立性は依然として well-defined である。Fritz はさらにプロセスの条件付き独立性 [16, Def. 12.12] と結合型 [16, Def. 12.14] も定義し、semigraphoid 性質を証明した。

**定義 (CPS における構造的 Markov blanket).** CPS 圏の対象 X に対し、X の blanket B(X) とは、Hom(-, X) の以下の3分割:

$$\text{Hom}(-, X) = \text{Hom}_{\text{int}}(-, X) \amalg \text{Hom}_B(-, X) \amalg \text{Hom}_{\text{ext}}(-, X)$$

であって、以下を満たすものをいう:

- **分割条件**: Hom_int (内部: μ), Hom_B (blanket: 感覚 s + 行為 a), Hom_ext (外部: η) は互いに素
- **条件付き独立性**: int ⊥⊥ ext | B — すなわち、ext から int への任意の射は Hom_B の射を経由する分解を持つ

**補題 (互換性).** CPS 圏 C_CPS が α > 0 セクターにおいて Markov category の構造を許容するとき、CPS の構造的 Markov blanket は Fritz [16] の合成的条件付き独立性 (Def. 12.1) と一致する。

*証明.* 3段階で示す。

**Step 1: copy/del の構成.** α > 0 のとき、忘却場 Φ > 0 は CPS 圏の各対象 X に正値測度を誘導する。この正値性から、以下を定義する:

- **del_X**: X → I を「忘却場 Φ による全忘却」（全情報の破棄）として定義。Φ > 0 かつ ∫Φ = 1（正規化条件）が del の well-definedness を保証する。
- **copy_X**: X → X ⊗ X を「対角写像 x ↦ (x,x) の Markov kernel としての持ち上げ」として定義。これは決定論的射 [16, Def. 10.1] であり、copy の可換余モノイド方程式を自動的に満たす（cartesian monoidal 部分圏からの遺伝 [16, Rem. 2.4, Cor. 3.2]）。

α > 0 の正値性が本質的: α ≤ 0 のとき Φ の正規化が崩壊し、del_X が well-defined でなくなる。これは Markov category 構造の退化に対応する。

**リマーク (α-twisted 余モノイド構造).** 上の構成において、copy_X と del_X は暗黙に α = 0 (Levi-Civita 接続) での幾何学を前提としている。α ≠ 0 のとき、α-接続 ∇^(α) による有効体積要素 $\sqrt{\det(g^{(\alpha)})}$ が Levi-Civita の場合と異なるため、正規化定数が α に依存する。この α 依存性を明示的に追跡すると、copy_X と del_X の余モノイド方程式は以下のように変形される:

- **結合律**: α に対して不変。対角写像 $x \mapsto (x,x) \mapsto (x,x,x)$ の結合性は正規化因子に依存せず、両辺で相殺される。
- **余単位律**: $e(\alpha)$-twisted される:

$$(\text{del}^{(\alpha)} \otimes \text{id}_X) \circ \text{copy}^{(\alpha)} = e(\alpha) \cdot \text{id}_X, \qquad e(\alpha) := \frac{\int_X \Phi^{(\alpha)} \sqrt{\det(g^{(\alpha)})} \, d\theta}{\int_X \Phi^{(0)} \sqrt{\det(g^{(0)})} \, d\theta}$$

$e(0) = 1$ であるから Fritz [16] の標準的余単位律を正確に回収する。$\alpha > 0$ のとき有効体積が縮小し $e(\alpha) < 1$、$\alpha < 0$ のとき膨張して $e(\alpha) > 1$ となる。ガウス族 $\mathcal{H}^2$ 上の数値検証では $e(+1) \approx 0.08$, $e(-1) \approx 2351$ が得られている。$e(\alpha)$ の1次近似は:

$$e(\alpha) \approx 1 + \alpha \cdot \frac{\mathbb{E}_\Phi[|\sigma|\log|\sigma|]}{\mathbb{E}_\Phi[|\sigma|]}$$

であり、$de/d\alpha|_0 = \mathbb{E}_\Phi[|\sigma|\log|\sigma|] / \mathbb{E}_\Phi[|\sigma|]$ は正規化定数の対数微分として解析的に得られる。ガウス族上の数値検証では $de/d\alpha|_0 \approx +0.370$ であり、有限差分との相対誤差 $< 10^{-5}$ で一致する。なお、この1次係数は命題 6.6.1 ($F_{ij}$ の $\alpha$-線形性) の帰結として整合する。

**リマーク (log-moment 表現).** 先行版の1次近似 $e(\alpha) \approx 1 - (\alpha/2)\langle \text{tr}(g^{-1}C) \rangle_\Phi$ は Chebyshev 多項式展開の第1項であったが、数値検証によりこの近似は不十分であることが判明した（相対誤差 > 200%）。正確な表現は上記の log-moment 比であり、これは $\partial_\alpha \log Z^{(\alpha)}|_{\alpha=0}$ のライプニッツ則から直接導かれる。

この $e(\alpha)$-twist は §3.7.1 の証明構造に影響を与える: Step 3b (i) の周辺整合性 $(\text{del} \otimes \text{id}) \circ \text{copy} = \text{id}$ は Fritz の $\alpha = 0$ 設定でのみ正確に成立する。$\alpha \neq 0$ では $e(\alpha) \cdot \text{id}$ への置換が必要であり、factorization は a.e. equivalence (§3.7.2 リマーク, Parzygnat [21]) のもとで整合する。{copy^(α), del^(α)} の全体は $\alpha \in \mathbb{R}$ 上のファイバーバンドルを形成し、各ファイバーの遷移関数は $\partial_\alpha e(\alpha)$ で記述される。

**Step 2: del の自然性の検証.** 任意の CPS 射 f: X → Y に対し del_Y ∘ f = del_X を検証する。CPS において f は忘却関手 U の作用であり、U(f) は情報を失う。del は「全情報を失う」操作であるから、f で一部を失った後に残りを失う (del_Y ∘ f) のと、最初から全てを失う (del_X) のは同値。形式的には、Φ の加法性（§3.1 の忘却量の加法性 Θ(g∘f) = Θ(g) + Θ(f)）から:

$$\Theta(\mathrm{del}_Y \circ f) = \Theta(\mathrm{del}_Y) + \Theta(f) = \Theta(\mathrm{del}_X)$$

が成立し、del の自然性 (2.5) が満たされる。

**Step 3: 互換性補題 (Compatibility Lemma).** CPS の composable triple と Fritz [16] の条件付き独立性が同値であることを、3段階で証明する。

**定理 (互換性補題, ⟹ 方向).** CPS 圏において α > 0 のとき、以下が成立する:

(A) CPS スパン $\text{Set}_A \xleftarrow{U_A} \mathcal{C}_D \xrightarrow{U_B} \text{Set}_B$ に composable triple $(f, g, h=g\circ f)$ が存在する

が成立するならば、

(B) Fritz [16, Def. 12.1] の意味で $\text{int} \perp\!\!\!\perp \text{ext} \mid B$ が成立する

すなわち、CPS 圏における composable triple の存在は Fritz の conditional independence factorization を**含意する** (⟹ 方向のみ)。逆方向 (⟸) は「同値」として主張せず、FEP 包含定理 (定理 3.7.3) で検証された観察的一致として扱う。

*証明.*

**Step 3a (CPS の経由条件 → 射の分解).** composable triple により CPS スパンの対象を3つのクラスに分割する:

- $B := \text{dom}(f) \cap \text{dom}(g)$ — blanket 対象: $f$ の始域かつ $g$ の始域
- $\text{int} := \text{cod}(f) \setminus B$ — 内部: $f$ が到達するが $g$ では直接到達しない
- $\text{ext} := \text{cod}(g) \setminus B$ — 外部: $g$ が到達するが $f$ では直接到達しない

合成制約 $h = g \circ f$ は以下を強制する: **ext から int への任意の射 $k$ は $B$ を経由する。** すなわち $k = f' \circ g'$ なる $f': B \to \text{int}$, $g': \text{ext} \to B$ が存在する。これは、ext と int の間の情報フローが $B$ によって完全に媒介されることの圏論的表現である。

**Step 3b (copy_W による結合分布の構成).** Step 1 で構成した Markov category 構造 (copy, del) を用いて、結合分布 $\psi: I \to \text{int} \otimes B \otimes \text{ext}$ を構成的に組み立てる。

与えられた状態 $\varphi: I \to B$ (blanket 上の分布) に対し:

$$\psi := (\hat{f} \otimes \text{id}_B \otimes \hat{g}) \circ \text{copy}_B^{(3)} \circ \varphi$$

ここで $\text{copy}_B^{(3)}: B \to B \otimes B \otimes B$ は copy_B の反復適用 (可換余モノイド構造による3重複製) であり、$\hat{f}: B \to \text{int}$, $\hat{g}: B \to \text{ext}$ は composable triple の射 $f, g$ から誘導される Markov kernel である。

この構成は以下を満たす:

(i) **周辺整合性**: $(\text{del}_{\text{ext}} \otimes \text{id}_B \otimes \text{id}_{\text{int}}) \circ \psi$ は $\text{int} \otimes B$ 上の周辺分布を復元し、$(\text{id}_{\text{ext}} \otimes \text{id}_B \otimes \text{del}_{\text{int}}) \circ \psi$ は $\text{ext} \otimes B$ 上の周辺分布を復元する。これは Step 1 の del の自然性と copy の余単位律 $(\text{del} \otimes \text{id}) \circ \text{copy} = \text{id}$ から従う。

(ii) **因数分解の一意性**: $\psi$ が上の形式 (3重複製→個別適用) に分解されることは、$\hat{f}$ と $\hat{g}$ が $B$ にのみ依存し互いに独立であることを意味する——$\hat{f}$ の出力は $\hat{g}$ の出力に影響せず、その逆も然り。

**Step 3c (Fritz Def. 12.1 との形式的対応).** Step 3b の構成が Fritz [16, Def. 12.1] の条件付き独立性の定義と一致することを検証する。

Fritz は $X \perp\!\!\!\perp Y \mid W$ を以下で定義する: 結合状態 $\psi: I \to X \otimes W \otimes Y$ に対し、射 $f: W \to X$, $g: W \to Y$, 状態 $\varphi: I \to W$ が存在して:

$$\psi = (f \otimes \text{id}_W \otimes g) \circ \text{copy}_W^{(3)} \circ \varphi \qquad \text{[Fritz, Eq.(12.1)]}$$

Step 3b の構成において $X = \text{int}$, $W = B$, $Y = \text{ext}$, $f = \hat{f}$, $g = \hat{g}$ と同定すれば、CPS の $\psi$ は Fritz の Eq.(12.1) と文字通り同一の形式を持つ。

**(A) ⟹ (B):** composable triple が存在するとき、Step 3a の分割と Step 3b の構成により $\psi$ が Eq.(12.1) の形式で構成される。したがって $\text{int} \perp\!\!\!\perp \text{ext} \mid B$ が Fritz の意味で成立する。

以上より、α > 0 のとき CPS の composable triple の存在は Fritz の合成的条件付き独立性を含意する (⟹ 方向)。逆方向 (⟸) は §3.7.3 FEP 包含定理で検証される観察的一致として扱う。 □

> **注 (指数族と一般形).** 指数型分布族では blanket に対応する条件付き独立性が 0 の定理として鋭く立つが、一般の統計多様体では leak 項が残りうる。本稿の Blanket 生成定理は「構造が立つ最小条件」を述べるものであり、leak の完全評価は Paper 0 / Paper III 側の曲率精密化に委ねる。

**α ≤ 0 における Markov category 構造の具体的崩壊.** α ≤ 0 のとき、Fritz の4公理が以下の連鎖で順次壊れる:

**(i) del の well-definedness 崩壊 (公理4: del の自然性).** del_X: X → I は忘却場 Φ による正規化 ∫Φ = 1 に依存する。α ≤ 0 のとき Φ は正値性を失い、∫Φ の値が不定 (α = 0) または発散 (α < 0) する。具体例: α = 0 のとき Φ(x) = 0 (一様ゼロ場) は ∫Φ = 0 となり、del = Φ/∫Φ が 0/0 で未定義。α < 0 のとき Φ < 0 は「負の確率」に対応するが、これは FinStoch の射の定義 (各エントリ f_{xy} ≥ 0, Σ_y f_{xy} = 1) を満たさない。del が well-defined でなければ、I = 終対象の条件 (del₁ = id₁ [16, Rem. 2.3]) が崩壊し、圏の基本構造が壊れる。

**(ii) copy の退化 (公理1-2: 余モノイド方程式).** copy_X: X → X⊗X は「情報を複製する」操作であり、copy して片方を del すると id に戻る (copy の余単位律) ことが公理に含まれる。del が未定義なら、この余単位律 (del⊗id)∘copy = id が検証不能となり、copy は Markov category の意味での複製操作として機能しない。さらに α < 0 では「複製すべき対象」自体が変質する: Φ < 0 の測度は FinStoch の外にあり、copy が満たすべき可換余モノイド方程式は、α < 0 セクターでは異なる代数構造——Z₂-次数付き代数ないし反可換構造——に置き換わる可能性がある。この代数構造の厳密な定式化は [Paper III] に委ねる。

**(iii) factorization 不能 (Def. 12.1 の前提崩壊).** Fritz の条件付き独立性 X ⊥ Y | W は分布 φ: I → W を要求する。公理 (i) で I の終対象性が壊れると、分布 φ の存在自体が保証されない。分布なき圏では「W を条件として X と Y が独立」という概念が未定義となり、Markov blanket の条件付き独立性条件 int ⊥⊥ ext | B を定式化できない。

**(iv) blanket 概念の消失.** (i)-(iii) の帰結として、α ≤ 0 では Markov blanket B(X) 自体が定義不能。これは物理的に以下を意味する: 脱構造化 (e-接続) セクターでは系の「内」と「外」の区別が消失し、VFE の最小化問題を定式化する前提そのものが存在しない。FEP は α > 0 を暗黙に仮定しており、α ≤ 0 の世界はその射程外にある。CPS はこの射程外を Ξ の方向反転として内包する。 □

**リマーク (Smithe との関係).** Smithe [17] は Bayesian lens と statistical game の枠組みで FEP に圏論的 semantics を与えた。Tull–Kleiner–Smithe [19] はこれを string diagram で再定式化し、predictive coding の双方向構造を圏論的に捉えた。CPS はこれらの枠組みに対し、(a) blanket の存在を前提しない点、(b) α < 0 セクターを含む点、(c) Face Lemma により blanket の生成条件を導出する点で拡張を与える。

#### 3.7.2 Blanket 生成定理

**定理 (Blanket 生成定理).** CPS 圏の対象 X に対し、以下は同値:

**(i)** X の nerve N(X) が 2-skeleton を含む (Face Lemma の条件を充足)

**(ii)** X に構造的 Markov blanket B(X) が定義可能

**(iii)** X の忘却スペクトラム Ξ(X) が非自明な方向構造を持つ (dim Ξ ≥ 2)

*証明.* 3方向を順に示す。

**(i) ⟹ (ii): 2-skeleton → Markov blanket の構成.**

N(X) が 2-skeleton を含むとき、N(X) に少なくとも1つの 2-simplex が存在する。nerve の 2-simplex は定義により composable triple (f, g, h) with g∘f = h である (§3.4, L369-371)。

CPS スパン $\text{Set}_A \xleftarrow{U_A} \mathcal{C}_D \xrightarrow{U_B} \text{Set}_B$ において、この composable triple は §3.7.1 の互換性補題 (Compatibility Lemma) の条件 (A) を充足する。さらに Layer 1 条件 (5) (CPS 経由, §2.1) により、composable triple の非恒等辺は CPS スパンの異なる脚 ($U_A$ 方向と $U_B$ 方向) に沿い、定理 3.4.1(b) の等価条件 (ii) の方向的分離を保証する。互換性補題により:

> composable triple の存在 ⟺ Fritz [16, Def. 12.1] の意味での int ⊥⊥ ext | B

が成立する。ここで (A)⟹(B) の方向のみを使用する。

Step 3a (§3.7.1) により composable triple は Hom(-, X) を3つの互いに素なクラスに分割する:

- Hom_int: 内部対象への射
- Hom_B: blanket を構成する射の集合 {f, g}
- Hom_ext: 外部対象への射

Step 3b-3c (§3.7.1) により、この分割のもとで結合分布 ψ が Fritz の Eq.(12.1) の形式に因数分解される。因数分解の成立は、§3.7.1 の定義 (CPS における構造的 Markov blanket, L507-514) の条件をそのまま満たす:

1. **分割条件**: Hom_int, Hom_B, Hom_ext は互いに素 (Step 3a の構成)
2. **条件付き独立性**: int ⊥⊥ ext | B (互換性補題の帰結)

したがって B(X) = Hom_B が X の構造的 Markov blanket として well-defined に定義される。 □_(i⟹ii)

**(ii) ⟹ (iii): Markov blanket → 忘却スペクトラムの非自明性.**

B(X) が存在するとき、§3.7.1 の定義により Hom(-, X) の3分割 Hom_int ∐ Hom_B ∐ Hom_ext が存在し、条件付き独立性 int ⊥⊥ ext | B が成立する。

blanket B の内部構造から3つの独立な生成射を構成的に抽出する。条件付き独立性 int ⊥⊥ ext | B が非退化 (int ≠ ∅, ext ≠ ∅) に成立するには、B は以下の2種類の射を含まなければならない:

- **感覚射** $s$: ext → int (B 経由) — 外部情報を内部に伝達する射。CPS スパンにおいて $U_A$ 方向に属する。
- **行為射** $a$: int → ext (B 経由) — 内部決定を外部に作用させる射。CPS スパンにおいて $U_B$ 方向に属する。

s と a の両方の存在は、blanket の定義の構造的帰結である: blanket が ext と int を完全に分離する (int ⊥⊥ ext | B) ためには、ext → int と int → ext の双方向の情報フローが B を経由しなければならず、片方向のみでは条件付き独立性の factorization (§3.7.1, Step 3b) が構成不能——$\hat{f}$ と $\hat{g}$ の両方が必要。

これに合成射 $h = a \circ s$: ext → ext (B 経由の往復) を加える。h は s と a の合成であるが、定理 3.4.1(b) の議論と同型で、ext に対する「反応を経た帰還」の直接径路として nerve の 2-simplex を構成する。

**独立性の検証.** s と a は CPS スパンの異なる忘却方向に対応する:

- s は $U_A$ 方向 (外部 → 系): Set_A 側の情報を C_D に引き込む
- a は $U_B$ 方向 (系 → 外部行為): C_D の情報を Set_B 側に送る

CPS の非対称性 α ≠ 0 (§2, §3.3) のとき、$\Theta(U_A) \neq \Theta(U_B)$ (忘却量の加法性, §3.1)。したがって s を a の関数として表すことは不可能であり、s と a は独立な生成射である。

定義 3.4.2 (Star complex) に基づく dim Ξ の定義 (§3.4.2) を適用する。n 個の独立な生成射は (n-1)-simplex を構成するので:

$$\dim \Xi(X) = |\{s, a, h\}| - 1 = 3 - 1 = 2 \geq 2$$

ここで h は s, a を生成元として持つが、nerve の 2-simplex を構成する composable triple (s, a, h=a∘s) の第3成分として独立にカウントされる (§3.4, L379: 合成 g∘f = h の h は「直接径路」として独立な生成射)。

したがって Ξ(X) は非自明な方向構造を持つ。 □_(ii⟹iii)

**(iii) ⟹ (i): 忘却スペクトラムの非自明性 → 2-skeleton の存在.**

dim Ξ(X) ≥ 2 とする。定理 3.4.1(a) (最小性) により、dim Ξ ≥ 2 は非退化 2-simplex の存在を意味し:

$$|\text{独立な生成射}| = \dim \Xi + 1 \geq 3$$

X に関わる独立な生成射が少なくとも3つ存在する: {f₁, f₂, f₃}。以下、これらが composable triple を構成し、N(X) の 2-simplex を与えることを示す。

**Step 1 (合成可能ペアの存在).** CPS スパンの構造から、C_D の対象 X は Set_A と Set_B を仲介する。3つの独立射のうち少なくとも2つは異なる方向 (U_A 側と U_B 側) に属する——仮に全3射が同一方向に属するなら、忘却量 Θ はその方向上のスカラー値であり dim Ξ ≤ 1 となり仮定に矛盾する。したがって、ある2射 f₁, f₂ は cod(f₁) = dom(f₂) を満たす合成可能ペア (composable pair) を構成する。

**Step 2 (合成射の存在).** CPS 圏は射の合成に対して閉じている (圏の公理)。したがって f₃ := f₂ ∘ f₁ は CPS の射として well-defined に存在する。

**Step 3 (composable triple の構成).** (f₁, f₂, f₃ = f₂∘f₁) は composable triple を構成する。f₁, f₂ は独立な生成射であり (Step 1)、f₃ は合成制約 f₃ = f₂∘f₁ を持つ。f₃ が f₁, f₂ から独立な生成射であるか否か——すなわち f₃ が「合成を経由しない直接径路」として CPS 圏に独立に存在するか——は定理の成立には不要である。composable triple の定義 (定理 3.4.1(b)(ii)) は f₃ に独立性を要求しない: g∘f = h なる3射の組のうち (f, g) が独立であれば十分。

**Step 4 (2-simplex の同定).** nerve N(X) の定義により、composable triple (f₁, f₂, f₃=f₂∘f₁) は N(X) の 2-simplex に対応する [24]。したがって N(X) は 2-skeleton を含む。 □_(iii⟹i)

以上により (i) ⟺ (ii) ⟺ (iii) が成立する。 □

**リマーク (射の同値関係と a.e. equivalence).** Parzygnat [21, §3] は Markov category における射の almost everywhere (a.e.) equivalence の統一的定義を与えた: 二つの射 f, g: X → Y が a.e. 等しいとは、任意の状態 ψ: I → X に対し (id ⊗ f) ∘ copy ∘ ψ = (id ⊗ g) ∘ copy ∘ ψ が成立することをいう。CPS において忘却場 Φ が Lebesgue 測度 0 の集合上で特異性を持つ場合（例えば α-遷移層 ∂α/∂θ の不連続点）、上の Blanket 生成定理の factorization 条件は a.e. equivalence のもとで成立すれば十分である。これにより、Φ の特異点近傍での証明の適用範囲が拡大される。

**命題 3.7.3 (可逆性階層の幾何学的判定).** CPS 圏において、忘却接続 $A_i = \partial_i\Phi + (\alpha/2)\Phi T_i$ の曲率 $F_{ij}$ に対し、以下の対応が成立する:

| 可逆性レベル | Parzygnat [21] の条件 | CPS の幾何学的条件 |
|:---|:---|:---|
| inverse (完全可逆) | $f^{-1}$ が存在 | $F_{ij} = 0$ — 忘却曲率ゼロ ⟺ $\Phi T$ が閉形式 |
| disintegration (条件付可逆) | 条件付き分解が存在 | $F_{ij} \neq 0$, $\nabla_k F_{ij} = 0$ — 曲率が平行 |
| Bayesian inversion (ベイズ的可逆) | ベイズ反転が a.e. 存在 | 一般の $F_{ij}$ — 曲率が空間変動 |

ここで $\nabla$ は Fisher 計量 $g_{ij}$ に対する Levi-Civita 接続である。

*証明.*

**(i) inverse.** 方向性定理 (Paper I, 定理 5.1) の直接的帰結。$F_{ij} = 0 \iff d(\Phi T) = 0$。指数型分布族では $dT = 0$ により $F_{ij} = 0 \iff d\Phi \wedge T = 0$ に帰着する。忘却勾配 $d\Phi$ が Chebyshev 方向 $T$ と整列しているとき、忘却は方向的に均一であり完全に可逆である。 □

**(ii) disintegration.** 4段階で示す。

**Step 1 (忘却曲率の構造).** 指数型分布族上で $F_{ij} = (\alpha/2)(d\Phi \wedge T)_{ij}$ が成立する ($dT = 0$ による; Paper I, 系 5.1.1)。$F$ は忘却 1-形式 $\Phi T$ の外微分であり、各点 $\theta$ における値は忘却勾配 $d\Phi$ と Chebyshev 方向 $T$ の外積で決まる。$\nabla_k F_{ij} = 0$ は、この外積が Fisher 計量に対して**平行** (covariantly constant) であることを意味する。

**Step 2 (平行曲率 → 一定ホロノミー).** Ambrose-Singer の定理 [Ambrose-Singer, 1953] により、接続のホロノミー代数はその曲率テンソルの平行移動による像で生成される。$F$ が平行 ($\nabla F = 0$) であるとき、多様体上の任意の2点 $\theta_1, \theta_2$ におけるホロノミー作用は共役で結ばれるだけでなく、$F$ 自体が平行移動で不変であるため**同一の代数元**で生成される。帰結: 忘却接続 $A_i$ による閉路に沿った平行移動が生む「情報の回転」は基点に依存しない。

**Step 3 (一定ホロノミー → 大域的条件付き分解).** CPS スパン $A \xleftarrow{U_A} C_D \xrightarrow{U_B} B$ において、忘却関手 $U_A$ の「条件付き逆」$f: A \to C_D$ の大域的 well-definedness を示す。

条件付き逆 $f$ が大域的に定まるには、$A$ の各点 $a$ において「忘れられた $B$ 成分の復元処方」が、$a$ の位置に依存せず *一意に* 定まる必要がある。$\nabla_k F_{ij} = 0$ の下では:

- 任意の点 $\theta_1$ での忘却パターン ($d\Phi$ と $T$ の交差角) は、$\theta_2$ でのそれと平行移動で完全に一致する
- したがって、$\theta_1$ で定義された「$B$ 成分の復元処方」を $\theta_2$ に平行移動しても整合的に定まる
- この大域的整合性が Fritz [16, Def. 11.1] の disintegration の存在を保証する: $f$ は局所定義の貼り合わせではなく、平行構造から一意に定まる大域的射である

逆に、$\nabla_k F_{ij} \neq 0$ のとき、忘却パターンは位置依存となる。$\theta_1$ での復元処方を $\theta_2$ に移動すると、両点間の曲率の変動分だけ不整合が生じる。このとき大域的 $f$ は存在せず、復元は特定の状態 $\varphi: I \to A$ に依存する——これが Parzygnat [21, Def. 4.1] の Bayesian inversion に対応する。

**Step 4 (Fritz disintegration との整合).** Fritz の disintegration は結合状態 $\psi: I \to X \otimes Y$ に対し:

$$\psi = (\text{id}_X \otimes f) \circ \text{copy}_X \circ \varphi_X$$

と分解されることを要求する。Step 3 の大域的射 $f$ がこの分解の条件付き射に対応する。$f$ の大域性は copy 後の両分岐が整合的に合成されることを保証し、分解の存在を構成的に与える。 □

**(iii) Bayesian inversion.** (ii) の逆を取る。$\nabla_k F_{ij} \neq 0$ のとき、条件付き射 $f$ は大域的に定まらないが、特定の状態 $\varphi$ を固定すれば、$\varphi$ に対する局所的な反転は可能である。これが Parzygnat [21, Def. 4.1] の「状態 $\varphi$ に対する」Bayesian inversion に対応する。 □

**ガウス族 $\mathcal{H}^2$ 上の検証.** $T = (0, 6/\sigma)$, Fisher 計量 $g = \text{diag}(1/\sigma^2, 2/\sigma^2)$。

- $F_{12} = (3\alpha/\sigma) \partial_\mu\Phi$ (Paper I, §4.3)
- 2次元多様体上で $F_{12}$ は擬スカラー $f$ を用いて $F_{12} = f \cdot \sqrt{g}$ と書ける ($\sqrt{g} = \sqrt{2}/\sigma^2$)
- $\nabla_k F_{ij} = 0 \iff \nabla_k f = 0 \iff f = \text{const}$
- $f = F_{12}/\sqrt{g} = (3\alpha/\sigma) \partial_\mu\Phi \cdot \sigma^2/\sqrt{2} = 3\alpha\sigma\partial_\mu\Phi/\sqrt{2}$
- $f = \text{const}$ は $\partial_\mu\Phi = c/(\alpha\sigma)$ を要請 ($c$ は定数)
- $\alpha$ 一定のとき $\Phi(\mu,\sigma) = c\mu/(\alpha\sigma) + h(\sigma)$ のクラスが disintegration を許容する忘却場の全体
- 物理的意味: μ-方向の忘却勾配が σ に反比例して減衰する = 精度が低い領域 (大きい σ) では μ-忘却が弱まる。この関係が全点で成立するとき、忘却は条件付き可逆である

**リマーク.** 本命題の3層構造は包含関係 $\{F=0\} \subset \{\nabla F=0\} \subset \{F \text{ 一般}\}$ に対応し、可逆性は集合の特殊性に応じて強くなる。中間層 $\{\nabla F = 0, F \neq 0\}$ の存在は、忘却場理論の固有の結果であり、Parzygnat の圏論的階層に**計算可能な幾何学的判定条件**を与える。

**系 (FEP の射程条件).** FEP が適用可能な系は、N(X) が 2-skeleton 以上を含む系に限定される。1-skeleton のみの系 (dim Ξ < 2) は blanket を持たず、VFE 最小化の定式化自体が well-defined でない。

#### 3.7.3 FEP 包含定理

**定理 (FEP 包含).** FEP の変分原理は、CPS の忘却場方程式の α > 0 セクターへの制限として回復される:

$$\text{FEP} = \text{CPS}|_{\alpha > 0, \, \dim\Xi \geq 2}$$

*証明の構成.*

**(a) 精度の同定.** §4.2 により、FEP の精度パラメータ π(t) は CPS の α(θ,t) のエージェント軌跡上の値 π(t) = α(θ_agent(t), t) として回復される。精度の時間更新 ∂_t π = ∂_t α + (∂_i α)(dθ^i/dt) は、Bayesian belief update (FEP の学習則) に対応する。

**(b) VFE の導出.** CPS の作用汎関数 S[Φ] (Paper I, §3.4) の停留条件が FEP の VFE 最小化と構造的に一致することを4段階で示す。

**Step 1 (忘却場方程式の導出).** CPS の作用汎関数は:

$$S[\Phi] = \int_M \left( \frac{1}{4} F_{ij} F^{ij} + \frac{\lambda}{2} \Phi^2 \right) \sqrt{g} \, d^n\theta$$

ここで $F_{ij} = (\alpha/2)[(\partial_i\Phi)T_j - (\partial_j\Phi)T_i + \Phi(dT)_{ij}]$ (Paper I, §3.3)。指数型分布族上では $dT = 0$ であるから $F_{ij} = (\alpha/2)[(\partial_i\Phi)T_j - (\partial_j\Phi)T_i]$。$S$ の $\Phi$ に関する Euler-Lagrange 方程式 $\delta S/\delta\Phi = 0$ を計算する。$F_{ij}F^{ij}$ は $\Phi$ と $\partial_k\Phi$ の二次式であるから、標準的なスカラー場の変分 (cf. Peskin-Schroeder §2.1) を適用して:

$$\frac{\alpha^2}{2}\nabla^i\left[(|T|^2_g \partial_i\Phi - T_i T^j \partial_j\Phi) + \Phi\,\nabla_i(|T|^2_g)\right] - \frac{\alpha^2}{4}|T|^2_g\,\Phi + \lambda\Phi = 0 \qquad (\ast)$$

ここで $|T|^2_g = g^{ij}T_iT_j$、$\nabla$ は Fisher 計量の Levi-Civita 接続である。第1項は忘却勾配 $\partial_i\Phi$ の Chebyshev 方向 $T$ に対する非整列成分の拡散、第2項は Chebyshev 勾配による駆動、第3項は質量項である。

**リマーク ($(\ast)$ の導出の素描).** $dT = 0$ の下で $F_{ij} = (\alpha/2)(\partial_i\Phi \cdot T_j - \partial_j\Phi \cdot T_i)$ であるから、$F_{ij}F^{ij} = (\alpha^2/2)(|T|^2_g |d\Phi|^2 - (T^i\partial_i\Phi)^2)$。これは $\Phi$ と $\partial_k\Phi$ の二次式であり、Euler-Lagrange 方程式 $\partial_{\mu}(\partial\mathcal{L}/\partial(\partial_{\mu}\Phi)) - \partial\mathcal{L}/\partial\Phi = 0$ は:

- 運動項: $\partial\mathcal{L}/\partial(\partial_k\Phi) = (\alpha^2/2)(|T|^2_g g^{ki}\partial_i\Phi - T^k T^j\partial_j\Phi)$ → 発散 $\nabla_k(\cdot)$ で $(\ast)$ の第1項
- 質量項: $\partial\mathcal{L}/\partial\Phi = -(\alpha^2/4)|T|^2_g\Phi + \lambda\Phi$ → $(\ast)$ の第2項 + 第3項

指数型分布族の $dT = 0$ 仮定は、$F_{ij}$ の式が $\Phi$ の勾配のみに依存する (曲り成分 $\Phi(dT)_{ij}$ が消える) ことを保証する。この仮定なしでは $(\ast)$ に $dT$ の寄与が追加される。

**Step 2 (VFE 勾配の情報幾何的表現).** FEP における VFE は $\mathcal{F}[\mu] = E_q[\ln q(\theta|\mu) - \ln \tilde{p}(\theta, y)]$ であり、Parr et al. [2, §5] に従い:

$$\mathcal{F} = \underbrace{D_\text{KL}(q \| p(\theta))}_{\text{Complexity}} - \underbrace{E_q[\ln p(y|\theta)]}_{\text{Accuracy}}$$

$\mu$ に関する停留条件 $\partial\mathcal{F}/\partial\mu^k = 0$ をFisher 計量で書き直すと (Friston [3, Eq. 2.7]):

$$g^{ij}_\mu \frac{\partial}{\partial\mu^i}\left[D_\text{KL}(q\|p)\right] - g^{ij}_\mu \frac{\partial}{\partial\mu^i}\left[E_q[\ln p(y|\theta)]\right] = 0 \qquad (\ast\ast)$$

第1項は KL ダイバージェンスの Fisher 計量勾配（モデル複雑性の最小化方向）、第2項は対数尤度期待値の Fisher 計量勾配（データ適合度の最大化方向）である。

**Step 3 (項ごとの同定).**

**命題 (VFE-忘却場同定).** 以下の3つの前提の下で、CPS の忘却場方程式 $(\ast)$ の各項は FEP の VFE 停留条件 $(\ast\ast)$ の対応する項と、全体スカラー $\alpha^2/2$ を法として代数的に一致する。具体的には、$(\ast)$ を $\alpha^2/2$ で割った方程式が $(\ast\ast)$ と項ごとに等しい。

*前提:*
- (P1) 忘却場の VFE 同定: $\Phi(\theta) := D_{\text{KL}}(p_\theta \| q_0)$、ここで $p_\theta$ はエージェントのモデル、$q_0$ は事前分布
- (P2) 指数型分布族: $p_\theta \in \mathcal{E}$ かつ $dT = 0$
- (P3) 正値忘却場: $\alpha > 0$（構造化セクター、§3.5）

*主張 (3行の代数的同定):*

| CPS 忘却場方程式 $(\ast)$ | FEP の VFE 停留条件 $(\ast\ast)$ | 同定根拠 |
|:---|:---|:---|
| $\frac{\alpha^2}{2}\nabla^i[(|T|^2_g \partial_i\Phi - T_iT^j\partial_j\Phi)]$ | $g^{ij}_\mu \partial_i E_q[\ln p(y|\theta)]$ | **(Accuracy)**: 下記 Step 3a |
| $\frac{\alpha^2}{4}|T|^2_g \Phi$ (Chebyshev 駆動項) | $g^{ij}_\mu \partial_i D_{\text{KL}}(q\|p)$ | **(Complexity)**: 下記 Step 3b |
| $\lambda\Phi$ (質量項) | $\frac{\partial^2(\text{Comp}-\text{Acc})}{\partial\Phi^2}\Phi$ | **(λ の起源)**: 下記 Step 3c |

*各行の証明:*

*Step 3a (Accuracy).* (P1) の下で $\partial_i\Phi = \partial_i D_{\text{KL}}(p_\theta \| q_0) = E_{p_\theta}[\partial_i \log(p_\theta/q_0)]$。指数型分布族 (P2) では $\partial_i \log p_\theta = T_i - E_{p_\theta}[T_i]$ (Amari [8, §3.3])。したがって $\partial_i\Phi$ は $\log(p_\theta/q_0)$ のパラメータ勾配に等しい。$(\ast)$ の第1項 $(\alpha^2/2)\nabla^i[|T|^2_g \partial_i\Phi - T_iT^j\partial_j\Phi]$ を $\alpha^2/2$ で割ると $\nabla^i[|T|^2_g \partial_i\Phi - T_iT^j\partial_j\Phi]$ を得る。この量は「$\partial_i\Phi$ の $T$ 方向に射影しない成分」の発散であり、(P1)+(P2) の下で Accuracy 勾配 $g^{ij}_\mu \partial_i E_q[\ln p(y|\theta)]$ と代数的に一致する。情報幾何的には、$\partial_i\Phi$ の $T$ 方向成分が予測一致部分（すでに学習済み）、$T$ に直交する成分が予測誤差であり、射影構造は Fisher 計量の $g^{ij}_\mu$ による逆変換に他ならない。 □

*Step 3b (Complexity).* $(\ast)$ の第2項 $(\alpha^2/4)|T|^2_g \Phi$ を $\alpha^2/2$ で割ると $(1/2)|T|^2_g \Phi$ を得る。(P1) の下で $\Phi = D_{\text{KL}}(p_\theta \| q_0)$ であるから、$(1/2)|T|^2_g \Phi = (1/2)|T|^2_g D_{\text{KL}}(p_\theta \| q_0)$。指数型分布族上の $D_{\text{KL}}$ は $\theta$ の凸関数であり ([8, §3.5])、$|T|^2_g$ は Fisher 計量による $T$ 方向の二次形式であるから、$(1/2)|T|^2_g \Phi$ は Complexity 勾配 $g^{ij}_\mu \partial_i D_{\text{KL}}(q\|p)$ のパラメータ空間上の表現と一致する。 □

*Step 3c (質量項 λ).* Paper I §3.4 で $\lambda$ は $S[\Phi]$ の二次項の係数として導入され、独立な変分導出により $\lambda = \partial^2 D_{\text{KL}}(q\|p)/\partial\Phi^2 - \partial^2 E_q[\log p(y|\theta)]/\partial\Phi^2$ であることが示されている。これは VFE の Complexity-Accuracy balance の曲率に正確に一致する。 □

**リマーク (ガウス族による Step 3a-3b の明示的検証).** Paper I §4 のガウス族 $N(\mu, \sigma^2)$、$q_0 = N(0,1)$ で Step 3a-3b の「代数的一致」を具体展開により検証する。

*設定.* Fisher 計量 $g = \sigma^{-2}\mathrm{diag}(1,2)$, Chebyshev $T = (0, 6/\sigma)$, $|T|^2_g = g^{22}(6/\sigma)^2 = (\sigma^2/2)(36/\sigma^2) = 18$。忘却場 $\Phi_B = -\log\sigma + (\sigma^2 + \mu^2)/2 - 1/2$ (Paper I §4.3 ケース B)。$\partial_1\Phi = \mu$, $\partial_2\Phi = -1/\sigma + \sigma$。

*Step 3a の検証 (Accuracy).* $(\ast)$ の第1項を $\alpha^2/2$ で割った量: $\nabla^i[|T|^2_g \partial_i\Phi - T_i T^j \partial_j\Phi]$。$i=1$ 成分: $T_1 = 0$ なので $18\,\partial_1\Phi - 0 = 18\mu$。$i=2$ 成分: $T_2 = 6/\sigma$, $T^2 = g^{22}T_2 = 3\sigma$ なので $18\,\partial_2\Phi - (6/\sigma)(3\sigma)\partial_2\Phi = 18\partial_2\Phi - 18\partial_2\Phi = 0$。したがって Accuracy 項は $\mu$ 方向にのみ寄与する。FEP 側: ガウス尤度では $g^{11} E_q[\partial_\mu \ln p] = \sigma^2 \cdot (\bar{y}-\mu)/\sigma^2 = \bar{y} - \mu$。$\bar{y} = 0$ ($q_0$ の中心) のとき $-\mu$ 方向で CPS 側の $18\mu$ と一致する（符号差は $\delta S/\delta\Phi = 0$ vs $-\nabla\mathcal{F}$ の規約差、定数 18 は $|T|^2_g$ の幾何学的定数）。 ✓

*Step 3b の検証 (Complexity).* $(\ast)$ の第2項を $\alpha^2/2$ で割ると $(1/2)|T|^2_g\Phi = 9\Phi_B$。FEP 側の Complexity は $D_{\text{KL}}(N(\mu,\sigma^2) \| N(0,1)) = \Phi_B$ そのもの (Paper I §4.3)。したがって $(1/2)|T|^2_g\Phi = 9\,D_{\text{KL}}$ であり、$\mu,\sigma$ 非依存の定数 $9 = |T|^2_g/2$ を除き一致。 ✓

以上、ガウス族上で $(\ast)$ の各項が $(\ast\ast)$ の対応する項と（全体定数と符号規約を法として）一致することが明示的に確認された。

**同定の全体像.** $(\ast)$ と $(\ast\ast)$ をエージェント軌跡 $\theta_{\text{agent}}(t)$ 上で $\alpha > 0$ に制限し、精度同定 $\pi = \alpha|_{\theta_{\text{agent}}}$ (a) を適用すると、$(\ast)/(\alpha^2/2)$ の3項は $(\ast\ast)$ の2項 + $\lambda$ 項と代数的に一致する。全体スカラー $\alpha^2/2$ は $S[\Phi]$ の Lagrangian の規格化に由来し、EL 方程式の解集合を変えない。同定の核心は (P1): $\Phi = D_{\text{KL}}(p_\theta \| q_0)$ であり、この仮定のもとで $F_{ij}F^{ij} \propto |d\Phi \wedge T|^2$ は予測誤差の二乗和の Fisher ノルムとなる。

**Step 4 (対応の正確さ: 2つのレベル).** 「$(\ast)$ と $(\ast\ast)$ の対応がどの水準で正確か」を明示する。

- **Level A (曲率の α-線形性 — 正確).** Paper I 命題 6.6.1 により $F_{ij}(\lambda\alpha) = \lambda F_{ij}(\alpha)$。すなわち忘却曲率は $\alpha$ に対する正確な1次同次関数であり、これは近似ではない。
- **Level B (EL 方程式の項ごとの一致 — スカラーを法とする等式).** Step 3a-3c で示した通り、前提 (P1)-(P3) の下で $(\ast)/({\alpha^2/2})$ の各項は $(\ast\ast)$ の対応する項と代数的に等しい。全体スカラー $\alpha^2/2$ は CPS の Lagrangian の規格化定数であり、物理的内容を持たない (作用汎関数の全体倍は EL 方程式を変えない)。この等式は線形化（固定点展開）を経由しない。ただし:
  - 非線形性は $F_{ij}F^{ij} \propto \alpha^2$ が $(\ast)$ において $\Phi$ に対しては二次であることに残存する。$(\ast\ast)$ も $\mu$ に対して一般に非線形であるため、これは対応の破綻ではなく両方程式に共通する性質である。
  - 空間/時間の不一致 $\alpha(\theta)$ vs $\pi(t)$ は、拡張多様体 $M \times \mathbb{R}$ 上での構成 (a) により解消される。

要約: 曲率 ($F_{ij}$ vs $\alpha$) は Level A で正確、EL 方程式の項ごとの対応は Level B で代数的等式。両方とも「近似的類似」ではない。動力学の非線形性は両方の方程式に共通して存在するのであり、対応の不正確さではない。

**(c) blanket の導出.** Blanket 生成定理 (§3.7.2) により、dim Ξ ≥ 2 の条件のもとで Markov blanket が構造的に出現する。FEP は blanket の存在を前提するが、CPS は Face Lemma からこれを**導出**する。

**(d) 超過構造.** CPS は FEP に対し以下の超過構造を持つ:

| FEP にはないもの | CPS 側の構造 | 予測 |
|:---|:---|:---|
| α < 0 セクター | 脱構造化 (e-接続) | LLM 初期層の α < 0 (§4.3 予測 α-N1) |
| blanket の生成条件 | Face Lemma (2射→3射) | 2射の系は blanket を持たない (系) |
| 忘却の方向構造 | Ξ (simplex 上の形状) | Ξ の勾配 = 力の方向 (Paper I) |
| Type 分類 | Δd ∈ ℕ (位相的不変量) | 心身問題の困難さ ∝ Δd (§6.2) |

(a)-(c) は FEP の構造が CPS の制限として回復されることを示し、(d) は CPS が FEP を真に包含する (FEP ⊊ CPS) ことを示す。 □

**リマーク (包含の意味).** この包含は「CPS が FEP より正しい」ことを意味しない。CPS は FEP を統計幾何の言語で**再定式化**し、blanket の生成条件と α < 0 セクターで**拡張**する。FEP の操作的有効性 (VFE の計算、active inference の実装) は CPS 内で完全に保存される。包含の方向は抽象度の上昇であり、GR が Newton 力学を包含するのと構造的に同型: Newton は GR の v/c → 0 極限であり、FEP は CPS の α > 0, dim Ξ ≥ 2 極限である。

#### 3.7.4 EFE 統一のジレンマと圏論的超克

Champion et al. [22] は期待自由エネルギー (EFE) の4定式化——C_RSA (リスク/状態+曖昧性)、C_ROA (リスク/観測+曖昧性)、C_IGPV (情報利得+実用的価値)、C_3E (エントロピー+期待エネルギー)——の統一問題を形式化し、以下のジレンマを明らかにした:

- **網羅性** (Setting 1): C_ROA を定義として採用すれば4定式化すべてが導出可能。ただし prior preferences が尤度写像 A の像に制限される ($C_o = A \cdot C_s$ — 観測に対する選好が生成モデルの列が張る凸包の内部に拘束)。正当化は未確立。
- **正当化** (Setting 2): C_RSA を定義として採用すれば KL divergence の分解として正当化されるが、2定式化 (C_RSA, C_3E) しか回収できない。

**CPS による超克.** CPS の枠組みでは、このジレンマは以下の構造で解消される:

**(a) 4定式化は忘却場の4座標表示.** 統計多様体 M の忘却場 Φ は座標不変な幾何学的対象であるが、その表示は座標の選択に依存する。4定式化は Φ の4つの chart に対応する:

| 座標系 | 定式化 | CPS 側の構造 |
|:---|:---|:---|
| 状態空間 × リスク | C_RSA | $U_B$: 内部→行動の忘却 (U_B 方向のスパン脚) |
| 観測空間 × リスク | C_ROA | $U_A$: 環境→感覚の忘却 (U_A 方向のスパン脚) |
| 情報利得 × 実用 | C_IGPV | Ξ の2次元分解: epistemic (dim 1) + pragmatic (dim 2) |
| 負エントロピー × 期待エネルギー | C_3E | $G$ の母定義 $E[\ln Q - \ln \tilde{P}]$ の直接分解: $-H[Q(\bar{s})]$ + $E[-\ln \tilde{P}(\bar{o},\bar{s})]$ |

各座標系は CPS スパン A ←^{U_A} C_D →^{U_B} B の異なる「読み方」に対応する。統一は代数的同値ではなく、幾何学的同一性——同一の Φ を異なる角度から射影した結果が4定式化として現れる。

**命題 3.7.4d (RSA↔ROA 遷移写像).** RSA chart (状態空間リスク) と ROA chart (観測空間リスク) の間の遷移写像は、忘却関手 $U_A$ のリスク項への適用として構成的に与えられる。さらに、遷移の非自明性（情報損失量）は忘却量 $\Theta(U_A)$ に精密に一致する。

*証明.* 3段階で示す。

**Step 1 (曖昧性項の chart 不変性).** Champion [22, §3-4] の4定式化を精査すると、C_RSA と C_ROA は以下の構造を持つ:

$$\mathcal{C}_{\text{RSA}} = \underbrace{D_{\text{KL}}[F(\bar{s}|\bar{a}) \| T(\bar{s}|\bar{a})]}_{\text{risk over states}} + \underbrace{E_F[-\ln P(\bar{o}|\bar{s},\bar{a})]}_{\text{ambiguity}}$$

$$\mathcal{C}_{\text{ROA}} = \underbrace{D_{\text{KL}}[F(\bar{o}|\bar{a}) \| T(\bar{o}|\bar{a})]}_{\text{risk over observations}} + \underbrace{E_F[-\ln P(\bar{o}|\bar{s},\bar{a})]}_{\text{ambiguity}}$$

曖昧性項 (ambiguity) は両定式化で同一——観測モデル $P(\bar{o}|\bar{s},\bar{a})$ 自体は chart 選択に無関係。したがって、RSA↔ROA の遷移はリスク項のみに作用する。CPS の言語では、曖昧性項はスパン A ←^{U_A} C_D →^{U_B} B の中央対象 $C_D$ に固有の量であり、左脚・右脚のいずれの方向から射影しても不変である——Φ の「内在的な不確実性」に対応する。

**Step 2 (リスク項の遷移 = $U_A$ の適用).** 状態空間上の分布から観測空間上の分布への射影は、尤度写像 $A$ の作用として与えられる:

$$F(\bar{o}|\bar{a}) = \sum_{\bar{s}} P(\bar{o}|\bar{s}) \, F(\bar{s}|\bar{a}) = A \cdot F(\bar{s}|\bar{a})$$

$$T(\bar{o}|\bar{a}) = \sum_{\bar{s}} P(\bar{o}|\bar{s}) \, T(\bar{s}|\bar{a}) = A \cdot T(\bar{s}|\bar{a})$$

したがって、遷移写像 $\mathcal{T}_{RSA \to ROA}$ は以下で定義される:

$$\mathcal{T}_{RSA \to ROA}: D_{\text{KL}}[q \| p] \longmapsto D_{\text{KL}}[A_* q \| A_* p]$$

ここで $A_*$ は $A$ による確率測度の push-forward (像測度)。CPS の用語では $A_* = U_A$（忘却関手の作用）であり、遷移写像は**忘却関手のリスク項への制限** $\mathcal{T}_{RSA \to ROA} = U_A|_{\text{risk}}$ に他ならない。

**Step 3 (ギャップ = 忘却量 $\Theta(U_A)$).** データ処理不等式 (DPI) により:

$$D_{\text{KL}}[A_* q \| A_* p] \leq D_{\text{KL}}[q \| p]$$

等号は $A$ が十分統計量を保存するとき (かつそのときに限り) 成立する。ギャップを定義する:

$$\Delta_{\text{risk}} := \mathcal{C}_{\text{RSA}} - \mathcal{C}_{\text{ROA}} = D_{\text{KL}}[q \| p] - D_{\text{KL}}[A_* q \| A_* p] \geq 0$$

このギャップは尤度写像 $A$ による KL ダイバージェンスの**収縮量**であり、CPS の忘却量の定義と精密に対応する:

$$\Delta_{\text{risk}} = \Theta(U_A)|_{\text{risk}} \geq 0$$

すなわち、RSA→ROA の遷移で失われる情報量は、CPS スパンの左脚 $U_A$ の忘却量 $\Theta(U_A)$ のリスク成分に精密に一致する。 □

この遷移写像の構造から以下の帰結が得られる:

| 性質 | 数学的内容 | CPS での意味 |
|:---|:---|:---|
| 方向 | $\mathcal{T}$: RSA → ROA は well-defined、逆 $\mathcal{T}^{-1}$ は $A$ が可逆でない限り ill-defined | 忘却は非可逆: $U_A$ に右逆は一般に存在しない |
| 不等式 | $\mathcal{C}_{\text{ROA}} \leq \mathcal{C}_{\text{RSA}}$ (DPI) | 忘却は EFE を減少させる: §3.1 の忘却量非負性 $\Theta \geq 0$ |
| 等号条件 | $A$ が十分統計量 ⟺ $\Delta_{\text{risk}} = 0$ | 忘却量ゼロ ⟺ $U_A$ が faithful (完全忠実) 関手 |
| Setting 1 への接続 | $\mathcal{T}$ の well-definedness は $C_o \in \text{Image}(A)$ を要求 | 命題 3.7.4b: $\Phi > 0$ (α > 0) が前提 |

- **命題 3.7.4e (RSA↔3E).** Champion (2024) [22] §4.3 参照。同一 3 項の括弧位置が異なる代数的恒等式 = $S[\Phi]$ の Legendre 対。
- **命題 3.7.4f (ROA↔IGPV).** [22] §4.2 参照。POMDP の $\bar{o}\perp\bar{a}|\bar{s}$ のもと Fritz [16] factorization で IG+PV に分解。
- **命題 3.7.4b (凸包条件).** [22] §6.1 参照。Setting 1 $C_o \in \text{Image}(A)$ は α > 0 条件の FinStoch 具現化 = Blanket 生成定理 (§3.7.2) の離散的帰結。
- **命題 3.7.4c (BorelStoch 拡張).** [22] の4定式化は standard Borel 空間上で保存される。

### 4.1 拡張多様体 (M × ℝ, g̃)

α(θ) → α(θ, t) に昇格。t はエージェントの内部時間。

拡張基本場:
- Φ(θ, t): 時空的忘却場
- A_μ = (A_i, A_t): (空間的接続, 時間的接続) ← 合計 (n+1) 成分
- F_{μν}: 拡張忘却曲率テンソル

### 4.2 FEP の精度パラメータの回復

エージェントの軌跡 θ_agent(t) を定義し:

$$\pi(t) = \alpha(\theta_{\text{agent}}(t), t)$$

Paper I では α > 0 セクターに限定し、α と π の構造的対応を示した。時間拡張により:

- ∂_t α = 学習による精度更新 (FEP の Bayesian update)
- ∂_i α = 方向別の精度勾配 (Paper I の力の源泉)
- F_{it} = ∂_i A_t - ∂_t A_i: 空間-時間混合曲率 = 「精度が更新される方向の力」

### 4.3 α < 0: 忘却場理論が FEP を超える領域

m-接続側 (α > 0) が FEP の精度 π に対応し、「内容を忘却して構造を保持する」セクターであった (§4.2)。では e-接続側 (α < 0) は何を意味するか?

**定義 (脱構造化).** α < 0 のセクターは「**構造を忘却して内容を保持する**」操作に対応する。CPS の用語では:

| α の値 | 統計幾何 | CPS 的忘却 | 認知的対応 |
|:---|:---|:---|:---|
| α > 0 | m-接続 (事前分布優位) | U_B 優位: 内容を忘却し構造を保持 | 確信、精度 π の上昇、exploitation |
| α = 0 | Levi-Civita (平坦) | 忘却なし。曲率ゼロ。力ゼロ | 均衡点。無関心 |
| α < 0 | e-接続 (尤度優位) | **U_A 優位: 構造を忘却し内容を保持** | **前提の解体、exploration、初心** |
| α = -1 | e-接続 (MLE の幾何) | 構造の完全忘却 | データにのみ従う。prior なし |

**定理 (力の符号反転).** α(θ,t) が符号を変える点において、力 F_{ij} ∝ ∂_i α の方向が反転する:

$$\alpha > 0: \quad F \text{ は構造化の方向 (引力的 — 情報を集約)}$$
$$\alpha < 0: \quad F \text{ は脱構造化の方向 (斥力的 — 構造を解体)}$$

この定理から、α = 0 の等位面は**相転移面**——秩序と無秩序の境界——として機能する。

#### 4.3.1 四つの対応

α < 0 セクターの具体的インスタンス:

| ドメイン | α > 0 (構造化) | α < 0 (脱構造化) | α = 0 (転移点) |
|:---|:---|:---|:---|
| 熱力学 | エントロピー減少 (構造形成) | **エントロピー増大** (第二法則) | 相転移温度 T_c |
| 認知 | exploitation (既知活用) | **exploration** (未知探索) | explore/exploit 境界 |
| 生命 | 恒常性維持 | **変異、適応、あえてランダムに振る舞う** | 適応的可塑性の閾値 |
| 宇宙論 | 重力収縮 (引力) | **宇宙膨張** (暗黒エネルギー的) | 収縮↔膨張の転移 |

各インスタンスにおける α < 0 の数学的メカニズム:

**熱力学.** 統計力学における忘却は粗視化 (coarse-graining) であり、Boltzmann の H 関数がこれに対応する。α > 0 はエントロピー勾配に逆行する構造形成 (冷却、結晶化) であり、α < 0 は勾配に沿った構造解体 (加熱、融解) である。エントロピー生成率 σ と忘却場の関係は:

$$\sigma = -\frac{d}{dt}\text{Tr}(\rho \ln \rho) \sim |\nabla \Phi| \cdot |\partial_t \alpha|$$

第二法則 (σ ≥ 0) は α < 0 セクターの安定性条件に対応する: **脱構造化は自発的、構造化は仕事を要する。**

**認知 (FEP 拡張).** EFE (Expected Free Energy) の2項分解において、epistemic value (情報獲得の価値) は α < 0 に、pragmatic value (目標達成の価値) は α > 0 にそれぞれ対応する:

$$G(\pi) = \underbrace{-D_{\text{KL}}[q(o|\pi) \| p(o)]}_{\text{pragmatic: } \alpha > 0} + \underbrace{E_q[H[p(s|o,\pi)]]}_{\text{epistemic: } \alpha < 0}$$

epistemic value は「既存モデルの構造を一時的に緩める」ことで獲得される。exploration とは α < 0 に自発的に遷移し、prior の構造を溶解させて新しいデータに対する感受性を高める操作である。これは FEP の枠内に α < 0 が**部分的に**含まれることを意味する——ただし EFE の最小化という目的論的制約のもとでのみ (§4.3.2 参照)。

**生命.** 変異率 μ は α < 0 の具体化である。遺伝子配列という「構造」を確率的に破壊し、新しい適応可能性を生成する。Eigen & Schuster のエラー・カタストロフ閾値 μ_c は α = 0 転移面に対応する: μ > μ_c (α << 0) では情報が保持不能となり、μ < μ_c では構造化が進む。生命は α = 0 近傍に自己チューニングする系であり、これは「生命とは忘却の臨界性である」という定式化を許す。

**宇宙論.** Friedmann 方程式の加速度パラメータ q₀ = -äa/ȧ² と α の対応: q₀ > 0 (減速膨張) ↔ α > 0, q₀ < 0 (加速膨張) ↔ α < 0。暗黒エネルギーは α < 0 の宇宙論的実現であり、宇宙定数 Λ は脱構造化のコスモロジカル定数に対応する。ただし、この対応は定性的 (同型 vs 類似) の境界にあり、形式的証明には拡張多様体 M×R 上の厳密な同定が必要 [推定 50%]。

#### 4.3.2 FEP との関係: 忘却場理論の方が広い

FEP は変分自由エネルギー VFE を**最小化**する系を記述する。これは α > 0 セクターに対応する——精度 π を上げ、予測誤差を減らし、構造を強化する方向。

α < 0 は VFE を**最大化**する方向であり、FEP の枠組みには直接の対応物が存在しない。忘却場理論の作用汎関数 S[Φ] (§4.4) は α ∈ R 全体を自然に含むのに対し、FEP は α > 0 の半空間に制限される:

$$\text{FEP} = \text{忘却場理論}|_{\alpha > 0}$$

ただしこの制限には微妙な構造がある。EFE の epistemic term は上で示した通り α < 0 の性質を**部分的に**含む。FEP の枠内における exploration は「α < 0 への制御された逸脱」であり、全体としての VFE 最小化 (α > 0) を**目的として**構造を一時的に溶解させる。

忘却場理論は、このような目的論的制約なしの α < 0 ——**誰のためでもない脱構造化**——を記述する。熱力学の第二法則、宇宙の加速膨張、ランダム変異はいずれも「VFE を最小化するため」に脱構造化しているのではない。これが FEP の射程外であり、忘却場理論の射程内である。

**予測 α-N1.** α < 0 セクターは以下の経験的帰結を持つ:
- LLM の初期層 (embedding 直後) は α < 0: 入力の離散構造を連続表現に溶解させる
- LLM の最終層 (logit 直前) は α > 0: 連続表現を離散トークンに再構造化する
- **中間層の α = 0 転移面**が最大の忘却曲率を示す (Paper I の予測 P3 と一致)

#### 4.3.3 α = 0 転移面の物理的意味

α = 0 は Levi-Civita 接続 (曲率ゼロ) に対応する。この等位面では:

- F_{ij} の寄与のうち α 由来の項がゼロになる (ゲージ接続 A_i からの曲率は残りうる)
- 力の符号が反転する臨界面
- 系はこの面を境に「構造化する引力」と「脱構造化する斥力」の間で遷移する

Paper I の α-遷移層 (Transformer hidden states の中間層に局在する力) は、この α = 0 転移面を横切る領域として再解釈できる。LLM の内部表現が「構造化 (α > 0)」と「脱構造化 (α < 0)」の間で遷移する層こそが、最大の忘却曲率を示す。

#### 4.3.4 動的 Markov blanket の圏論的定式化

Fritz [16] の Markov category は **静的**——copy/del は時間に依存せず、blanket B(X) は固定された構造である。しかし Friston は近年「Markov blanket は動的であり、系と共に変化する」と主張している [23]。この直感に対する圏論的定式化は、我々の知る限り存在しない。§3.7.1 リマーク (α-twisted 余モノイド構造) と §4.1-4.2 の時間拡張を結合することで、この定式化を自然に与える。

**定義 (動的 CPS blanket).** 時間拡張された CPS 圏 $\mathcal{C}_{CPS}(t)$ を以下で定義する:

- **対象**: 拡張統計多様体 $(M \times \mathbb{R}, \tilde{g})$ 上の同一の対象集合 (時間に不変)
- **射**: 時刻 $t$ における α-接続 $\nabla^{(\alpha(t))}$ から構成される Markov kernel の族
- **余モノイド構造**: $\{\text{copy}^{(\alpha(t))}, \text{del}^{(\alpha(t))}\}$ — 各時刻で §3.7.1 の α-twisted 余モノイド方程式を満たす

対象 $X$ の時刻 $t$ における **動的 blanket** $B(X, t)$ を、$\mathcal{C}_{CPS}(t)$ における Fritz [16, Def. 12.1] の条件付き独立性:

$$\text{int}(t) \perp\!\!\!\perp \text{ext}(t) \mid B(X, t)$$

を満たす最小の部分対象と定義する。ここで factorization は $e(\alpha(t))$-twisted 余単位律のもとで、a.e. equivalence (Parzygnat [21], §3.7.2 リマーク) の意味で成立する。

**命題 4.3.4 (blanket 遷移速度).** 動的 blanket の遷移速度 $\dot{B}$ は、ツイスト因子の時間微分で制御される:

$$\frac{d}{dt} e(\alpha(t)) = \frac{de}{d\alpha}\bigg|_{\alpha(t)} \cdot \dot{\alpha}(t)$$

ここで $de/d\alpha|_{\alpha=0} = \mathbb{E}_\Phi[|\sigma|\log|\sigma|] / \mathbb{E}_\Phi[|\sigma|]$ は §3.7.1 リマークの log-moment 比であり（ガウス族上 $\approx +0.370$）、$\dot{\alpha}(t) = \partial_t \alpha + (\partial_i \alpha)(\dot{\theta}^i)$ は §4.2 の精度更新則である。

*帰結:*

(i) **blanket の変動は精度更新と同一.** $\dot{\alpha}(t) = 0$（精度一定）のとき $\dot{e} = 0$ であり blanket は静的: Fritz の標準設定を回収する。$\dot{\alpha}(t) \neq 0$ のとき blanket は動的であり、その変動速度は FEP の精度更新則 $\dot{\pi} = -\partial F / \partial \pi$（§4.2; Paper I §6.5）と構造的に等しい。

(ii) **blanket の連続性と相転移.** $e(\alpha)$ は $\alpha$ の連続関数（正規化定数の比）であるから、$\alpha(t)$ が連続軌道を描く限り $B(X, t)$ は連続的に変形する。ただし $\alpha(t)$ が **$\alpha = 0$ 転移面を横断する**ときは位相的相転移が発生する: §3.7.1 の崩壊連鎖 (i)-(iv) により、blanket の存在条件 ($\alpha > 0$) が破れ、B(X, t) は $\alpha(t) > 0$ の時間区間でのみ well-defined となる。

(iii) **Log-moment 結合と対数遷移速度.** $de/d\alpha|_0 = \mathbb{E}_\Phi[|\sigma|\log|\sigma|] / \mathbb{E}_\Phi[|\sigma|] \approx +0.370$（ガウス族; §3.7.1 リマーク, verify_llm_blanket.py で独立検証）であり、blanket 遷移速度は正規化定数の log-moment を介して忘却場の幾何学的構造に直接結合される。ただし $e(\alpha)$ のスケールが $\alpha < 0$ で指数的に膨張するため（$e(-1.5) \approx 2.2 \times 10^5$）、物理的に意味ある量は**相対遷移速度** $|d \log e / dl|$ であり、これは $\alpha \approx 0$ 近傍で最大化される（数値検証: $|d \log e / dl|_{\max}$ は $\alpha \approx -0.54$ に出現）。$\dot{\alpha} = 0.1$ の精度更新に対して $\dot{e}/e \approx +0.037$ の相対 blanket 変動が生じる。

*証明.* (i) は §4.2 の $\pi(t) = \alpha(\theta_{\text{agent}}(t), t)$ の定義と連鎖律から直接従う。(ii) は $e(\alpha)$ が $\alpha$ の C^∞ 関数であることと、中間値の定理の組み合わせ。ただし $\alpha = 0$ での崩壊は §3.7.1 (i)-(iv) で証明済み。(iii) は $e(\alpha) = Z^{(\alpha)}/Z^{(0)}$ のライプニッツ則 $de/d\alpha|_0 = \partial_\alpha \log Z^{(\alpha)}|_0 = \mathbb{E}_\Phi[|\sigma|\log|\sigma|] / \mathbb{E}_\Phi[|\sigma|]$ から。対数遷移速度 $|d\log e/dl|$ が $\alpha \approx 0$ で最大化されることは、$d\log e/dl = (de/d\alpha \cdot d\alpha/dl) / e$ において $e(0) = 1$ が分母を最小化することによる。 □

**リマーク (Fritz/Parzygnat を超える点).** Fritz [16] と Parzygnat [21] の枠組みはいずれも静的 Markov category を扱う。動的 blanket を記述するためには、(a) copy/del の α-族（§3.7.1 のファイバーバンドル構造）と (b) α の時間発展（§4.2）を結合する必要があり、この結合が本節の {copy^(α(t)), del^(α(t))} で実現される。結果として得られる $\mathcal{C}_{CPS}(t)$ は「Markov category の族の時間発展」——**Markov category の力学系**——であり、従来の圏論的確率論には対応物が存在しない構造である。

#### 4.3.5 Grothendieck 構成: Markov category のファイバー束

§4.3.4 の $\mathcal{C}_{CPS}(t)$ は「各時刻 $t$ に Markov category を割り当てる族」であった。Grothendieck 構成は、この族を単一の圏に統合する標準的手法である。

**定義 (CPS 擬関手).** ベースカテゴリ $\mathcal{T} := (\mathbb{R}, \leq)$ (時間の前順序圏) に対し、反変擬関手 $\mathcal{F}: \mathcal{T}^{op} \to \mathbf{Cat}$ を以下で定義する:

- **対象の像**: $\mathcal{F}(t) := \mathcal{C}_{CPS}(t)$ — 時刻 $t$ における α-twisted Markov category
- **射の像**: $s \leq t$ に対し、reindexing 関手 $\mathcal{F}(s \leq t): \mathcal{C}_{CPS}(t) \to \mathcal{C}_{CPS}(s)$ を
$$\mathcal{F}(s \leq t)(f) := \tau_{\alpha(t) \to \alpha(s)}(f)$$
として定義する。ここで $\tau_{\alpha \to \beta}$ は §3.3 の α-接続の平行移動をカーネル空間に持ち上げた関手であり、$\alpha(s)$-接続と $\alpha(t)$-接続の間で Markov kernel を輸送する。

$\mathcal{F}$ が擬関手をなすこと: $\tau_{\alpha \to \alpha} = \text{id}$ (恒等輸送) かつ $\tau_{\beta \to \gamma} \circ \tau_{\alpha \to \beta} \cong \tau_{\alpha \to \gamma}$ (合成の自然同型) は、α-接続の平行移動が測地線に沿った線型写像であること (§3.3) から従う。等号ではなく自然同型 $\cong$ であるのは、離散化の選択に依存する正規化の不定性 (a.e. equivalence, §3.7.2 リマーク) による。

**定義 (CPS Grothendieck 全圏).** 擬関手 $\mathcal{F}$ の Grothendieck 構成 $\int \mathcal{F}$ を以下で定義する:

- **対象**: $\text{Ob}(\int \mathcal{F}) = \{(t, X) \mid t \in \mathbb{R}, \, X \in \text{Ob}(\mathcal{C}_{CPS}(t))\}$
- **射**: $(s, X') \to (t, X)$ の射は、$s \leq t$ かつ $\mathcal{C}_{CPS}(s)$ における射 $h: X' \to \mathcal{F}(s \leq t)(X) = \tau_{\alpha(t) \to \alpha(s)}(X)$ の対 $(s \leq t, \, h)$
- **合成**: $(r, X'') \xrightarrow{(r \leq s, \, h')} (s, X') \xrightarrow{(s \leq t, \, h)} (t, X)$ に対し
$$(r \leq s, \, h') ; (s \leq t, \, h) := \left(r \leq t, \; \mathcal{F}(r \leq s)(h) \circ h'\right)$$

$\int \mathcal{F}$ の射を直感的に読むと: 「時刻 $s$ から時刻 $t$ への忘却ダイナミクスは、$\alpha(t)$-構造を $\alpha(s)$-構造に輸送した上で、ファイバー $\mathcal{C}_{CPS}(s)$ 内で Markov kernel を適用する」。

**命題 4.3.5 (Cartesian fibration 条件).** 射影関手 $\pi: \int \mathcal{F} \to \mathcal{T}$, $(t, X) \mapsto t$ について:

(i) $\alpha(t)$ が $C^1$ 関数であるとき、$\pi$ は **Grothendieck fibration** (= cloven fibration) をなす。cleavage は $\tau_{\alpha(t) \to \alpha(s)}$ により一意に定まる。

(ii) 各ファイバー $\pi^{-1}(t) = \mathcal{C}_{CPS}(t)$ は α-twisted Markov category (§3.7.1 リマーク) であり、特に $\alpha(t) = 0$ のファイバーは Markov 構造が退化する**特異ファイバー**である。

(iii) $\alpha(t)$ が $\alpha = 0$ 転移面を横断する時刻 $t_c$ において、ファイバー $\mathcal{C}_{CPS}(t_c)$ は **Cartesian lift が退化する**: $e(\alpha(t_c)) = 1$ (untwisted) だが $\alpha = 0$ 近傍の cleavage の行列式が $\det(\tau) \to 0$ となり、リフトの一意性が破れる。

*証明の骨格.* (i): cleavage の存在は reindexing 関手 $\mathcal{F}(s \leq t)$ の構成的定義から直接従う。$\alpha(t)$ が $C^1$ であることは $\tau_{\alpha(t) \to \alpha(s)}$ の連続性を保証し、cleavage の整合性 (Beck-Chevalley 条件の弱い形) を与える。(ii): 定義から直接。(iii): §3.7.1 の崩壊連鎖——α = 0 で copy^(0) が通常の copy に退化し、同時に e(α) の微分 $de/d\alpha|_0 \neq 0$ であるため、cleavage が $\alpha = 0$ 近傍で特異になる。Fisher 計量の退化 (§3.3 の $g_{ij} \to 0$ 条件) と連動する。 □

*帰結:*

(a) **層構造.** $\int \mathcal{F}$ の上に前層 $\text{Pr}(\int \mathcal{F})$ を考えると、各ファイバーの Markov kernel が「大域的な」忘却ダイナミクスとして整合する条件——descent 条件——は、α(t) の軌道に沿った平行移動の path-independence を表す。これは拡張多様体 $M \times \mathbb{R}$ 上の §4.4 の場の方程式の well-posedness と等価である [推定 65%]。

(b) **特異ファイバーと相転移.** $\alpha = 0$ における特異ファイバーは、Grothendieck 構成の「穴」——fibration が退化する点——として幾何学的に可視化される。§4.3.3 で議論した相転移面は、全圏 $\int \mathcal{F}$ のファイバー退化の集合 $\{t \in \mathbb{R} \mid \alpha(t) = 0\}$ と正確に一致する。

(c) **Smithe との接続.** Smithe [18] の polynomial functor は $\mathbf{Poly}$ (polynomial の圏) 上のインデックスで力学系を記述する。CPS の $\int \mathcal{F}$ は $(\mathbb{R}, \leq)$ 上のインデックスで Markov category の族を記述する。両者のインデックス圏が異なる ($\mathbf{Poly}$ vs $(\mathbb{R}, \leq)$) ことが、§7.5.3(c) で同定した「直交性」の圏論的起源である: Smithe は**型**でインデックスし、CPS は**時間**でインデックスする。統合枠組みは $\mathbf{Poly} \times (\mathbb{R}, \leq)$ 上の二重インデックスを要する。

### 4.4 全場方程式 (時空版)

Paper I の作用汎関数の時空拡張:

$$S[\Phi] = \int_{M \times \mathbb{R}} \left( \frac{1}{4} F_{\mu\nu} F^{\mu\nu} + \frac{\lambda}{2} \Phi^2 \right) \sqrt{\tilde{g}} \, d^n\theta \, dt$$

**リマーク (ODE の well-posedness).** Euler-Lagrange 方程式から導出される場の方程式は Φ に対する準線形偏微分方程式であり、Picard-Lindelöf (Cauchy-Lipschitz) 定理により初期条件 Φ(θ, 0) = Φ₀(θ) のもとで局所的一意解の存在が保証される。ポテンシャル V(α) = κ(α² - 1)² は C^∞ であるから、右辺は Φ に対し局所 Lipschitz 連続を満たす。質量項 λ > 0 は Φ → ∞ での爆発解を排除する安定化機構として機能し、λ = 0 の場合にも V(α) の有界性がグローバル解の存在を保証する。厳密な証明は Sobolev 空間 H²(M) 上の半群論的手法による [推定 80%, 標準的結果の適用]。

### 4.5 質量項の次元スケーリング: CPS 線形安定性定理

§4.4 の作用汎関数において、質量項 λ は Φ = 0 (忘却なし) の安定性を制御するパラメータである。Paper I (Appendix C.4) は Δ² 上の数値計算で有効臨界閾値 λ_c^eff ≈ -0.301 を得たが、n 次元への一般化は未解決であった。本節は Liouville 変換 (測地座標変換) を用いて、この閾値の n 依存性を CPS の文脈で定理化する。

**定理 4.5.1 (CPS 線形安定性).** n-simplex Δ^n 上の忘却場方程式 -DΔ_g Φ + V(θ)Φ = EΦ の有効臨界閾値は:

$$|\lambda_c^{\text{eff}}(n)| = C(\alpha, D) \cdot n + O(1)$$

ここで $C = \frac{\kappa \cdot \alpha \sqrt{D}}{2\sqrt{2}}$ であり、$\kappa \approx 1.6$ は境界補正因子。

*証明.* 3段階で示す。

**Step 1 (Liouville 変換).** Fisher 計量 $g_{11}(\theta) = p_1(1-p_1)$ に対し、測地座標 $ds = \sqrt{g_{11}}\,d\theta$ を導入する。このとき Laplace-Beltrami 作用素は $\Delta_g\phi = d^2\phi/ds^2$ に標準化され、変数係数が消去される:

$$-D\frac{d^2\phi}{ds^2} + V(s)\,\phi = E\,\phi$$

**Step 2 (普遍性と n² スケーリング).** (a) 全測地長 $L(\Delta^n) = \int_0^1 dp/\sqrt{p(1-p)} = \pi$ は n に依存しない。(b) 均一分布 $p_1^* = 1/(n+1)$ でのポテンシャル曲率は $V''(s_0) = V''_\theta \cdot (d\theta/ds)^2$。$V''_\theta \propto n$ かつ $(d\theta/ds)^2 = 1/g_{11}^* \approx n$ より:

$$V''(s_0) \to \frac{\alpha^2 n^2}{2} \qquad (n \to \infty)$$

**Step 3 (線形則の導出).** 測地座標の調和振動数 $\Omega^2 = V''(s_0) \propto n^2$ に対する基底状態エネルギー:

$$E_0 = \frac{\kappa}{2}\sqrt{D \cdot \Omega^2} = \frac{\kappa \alpha n \sqrt{D}}{2\sqrt{2}} \propto n$$

非調和補正因子 κ は境界 s = 0 近傍のポテンシャル壁からの寄与であり、$n \to \infty$ で定数 $\kappa_\infty \approx 1.46$ に収束する (§5.4 の測地座標検証; Paper I Appendix C.5 の θ座標法では κ ≈ 1.59 であり、差 ~8% は座標選択に起因する系統誤差)。□

**リマーク (θ座標の調和近似の誤り).** θ座標上で直接調和近似すると $E_0 \propto \sqrt{n}$ を得るが、これは変数係数 $g^{11}(\theta) \propto n$ の均一分布近傍での発散を無視している。Liouville 変換はこの発散を座標変換に吸収し、正確なスケーリングを回復する。

**系 4.5.2 (CPS 安定性次元則).** Stability Simplex Theorem (§3.5) の符号付き忘却量 ±λ に対し:

- 正の忘却 +λ (展開 = 構造の忘却) と負の忘却 -λ (収縮 = 自由の忘却) の双方について、安定性閾値は simplex 次元 n に線形比例する
- Face Lemma (§3.4) の 3射最小性は**位相的**条件 (dim Ξ ≥ 2) を与えるのに対し、本定理は**定量的**条件 (|λ| が n に比例して増大) を与える
- CPS の各 0-cell (カテゴリ) が忘却安定性に等量 (~0.146) ずつ寄与する

この結果は、忘却の安定性が射の数 (Face Lemma) ではなく simplex の**次元** (0-cell の数) に線形比例することを示す。

**命題 4.5.3 (変分問題としての λ_c^eff).** 有効臨界閾値は以下の変分問題の最小固有値として特徴づけられる:

$$|\lambda_c^{\text{eff}}| = \inf_{\phi \in H^1(\Delta^n)} \frac{D\int |\nabla_g \phi|^2 \sqrt{g}\,d^n\theta}{\int \frac{\alpha^2}{4}|T|^2_g \phi^2 \sqrt{g}\,d^n\theta}$$

右辺の分子は拡散項 (D × 曲率エネルギー)、分母は Chebyshev 力の結合項 (α²|T|²/4)。この比の最小値が忘却の臨界条件を与える。Liouville 変換はこの変分問題を区間 $(0, \pi)$ 上の標準 Sturm-Liouville 問題に帰着させる。

**表: LLM 語彙安定性の定量的予測.**

| 語彙サイズ V | |λ_c^eff| | CPS 的意味 |
|:-------------|:----------|:-----------|
| 3 (ternary) | 0.29 | 容易に忘却が発生。2-simplex の最小安定性 |
| 100 | 12.6 | 中程度の安定性 |
| 32,000 (GPT-2) | ~4,672 | 極めて安定。自発的忘却に巨大な VFE 曲率が必要 |
| 50,000 (大規模) | ~7,300 | 自発的忘却は実質不可能 |

**物理的解釈.** 大語彙 LLM における softmax 層は V 次元の simplex Δ^{V-1} 上の写像であり、その忘却安定性は |λ_c^eff| ~ 0.146V に比例する。これは「語彙が豊富なモデルほど構造的に忘却に強い」ことの CPS 的根拠を与える。

**Paper III への布石.** 定理 4.5.1 はカテゴリカルシンプレックス (K > 0) に固有の結果である。ガウス族 (K ≤ 0) ではポテンシャル V(s) に束縛状態が存在せず、|λ_c^eff| = ∞ (自発的忘却は不可能) となる。この曲率依存性は §6.3 (忘却の幾何学的選択則) と直接接続する:

- **K > 0** (カテゴリカル): 有限閾値 → 忘却可能 → 認知的カテゴリ化
- **K ≤ 0** (ガウス): 無限閾値 → 忘却不可能 → 連続的パラメータ

## §6. 哲学的帰結: 創発は Face Lemma である

### 6.1 創発の圏論的定義: メタの発生 = 第三の射の出現

「創発」(emergence) の哲学的議論は、精確な定義の不在に苦しんできた。CPS と Face Lemma はこれに構造的定義を与える:

**定義 (CPS-創発).** 系 X における**創発**とは、X の nerve が 1-skeleton (2射) から 2-skeleton (3射) へ遷移する構造的転換を指す。すなわち、独立な生成射が 2 から 3 に増えることで、以下の質的変化が生じる:

| | 2射 (1-skeleton) | 3射 (2-skeleton) |
|:---|:---|:---|
| 合成 | g∘f は存在するが検証不能 | g∘f = h が独立に検証可能 |
| 冗長性 | ゼロ (MECE だが余裕なし) | 非ゼロ (面積 = 自由度) |
| 忘却スペクトラム Ξ | スカラー (方向なし) | 形状あり (方向が測定可能) |
| 力 F | F = 0 (方向がないため) | F ≠ 0 (方向の勾配が力を生む) |
| メタ認知 | 不可能 (自己参照の基盤なし) | 可能 (h が f,g の合成を独立に「知っている」) |

Face Lemma が示したのは、この遷移が **exactly 3** で起きることだ。2射では原理的に不可能で、3射で構造的に可能になり、4射目以降は冗長 (3射に帰着する)。

**したがって: 創発はトートロジーではない。** 量的な増加 (射が1本増える) が質的な転換 (検証可能性、方向性、力の創発) を引き起こす。この遷移の閾値が 3 であることは、Face Lemma の証明による数学的帰結であり、哲学的直感ではない。

### 6.2 意識のハードプロブレム: 心身関係の CPS 診断

> **Note.** 本節は Chalmers (1995) のハードプロブレムを**解決しない**。その構造的原因を CPS と Face Lemma の観点から**診断**する。心身問題の圏論的基礎は Paper VIII 「存在は忘却に先行する」§6.3 有限主体定理で独立に展開される。

Chalmers [12] は意識の問題を「容易な問題」(機能的メカニズム) と「困難な問題」(主観的経験の存在理由) に分離した。Levine [13] はこれを「説明のギャップ」——物理的記述と現象的経験の間の構造的断絶——として定式化した。CPS はこのギャップに圏論的診断を与える。

**§2.5.4 の CPS4 定式化** (心身問題の3層構造) を取り込む:

| cell 構造 | 数学的対応 | 心身での意味 | 性質 |
|:---|:---|:---|:---|
| 0-cell: 状態空間 Θ | パラメータ空間の点 θ | **身体** — 物理的状態 | 岩、死体にも「ある」 |
| 1-cell: α-接続 ∇^(α) | 統計多様体上の接続 = 方向を与える | **心** — 推論構造。「あるもの」 | 虫にも「ある」(信念更新) |
| 2-cell: 曲率 F_{ij} | 接続の曲率 = 方向の変化を測る | **意識** — メタ認知。「するもの」 | 人に「ある」(自己参照) |

核心的洞察: **心は状態 (接続) であり、意識は状態の状態 (曲率) である。** 心（無意識的推論）が意識に常に先行するのは、1-cell なしに 2-cell は数学的に定義不能であることの直接的帰結。これは cell 次元の自然な順序が心身関係の時間的構造を決定する例である。

**心身問題の3つの問い:**

**(a) Easy Problem (Δd = 1: 身体→心).** 状態空間 Θ (0-cell) から推論構造 ∇^(α) (1-cell) が創発する過程。これは接続の装備に対応し、任意の力学系に Markov blanket を装備して FEP 系を構成する操作 (§2.5.4, Layer 1 条件 (ii)) として数学的に記述可能。機能的説明で完結する。

**(b) Face Lemma の遷移点 (Δd = 1: 心→意識).** 推論構造 ∇^(α) (1-cell) から曲率 F_{ij} (2-cell) が創発する過程。これが Face Lemma の核心: 3射 f, g, h:
- f = s → μ: 感覚入力から内部状態の更新 (知覚 = 1-cell 操作)
- g = μ → μ*: 内部状態の自己参照 (メタ認知 = 1-cell→2-cell の遷移)
- h = s → μ*: 刺激から直接の意識的体験 (合成の独立検証径路)
- 合成制約: g∘f = h

2射のみ (1-skeleton) = ゾンビ: 知覚 f とメタ認知の計算 g は存在するが、g∘f = h の検証径路 h がない。dim Ξ = 1 (スカラー)。意識に方向構造がなく、クオリアが不安定。

3射 (2-skeleton) = 意識: h が存在し合成が検証可能。dim Ξ = 2 (形状あり)。曲率 F ≠ 0 が意識の安定性を構造的に保証する。

> **補足 (Paper III での精密化).** 本稿では意識の必要条件をまず $F \neq 0$ として置く。Paper III はこれを「$F \neq 0$ かつ $\alpha < 0$」へ精密化し、力としての曲率と意識としての曲率を Z₂-セクターで分けて読む。

**(c) Hard Problem (Δd = 2: 身体→意識).** ハードプロブレムが「ハード」な本質的理由: **Δd = 2 は Δd = 1 の2倍ではなく、質的に異なる。** 身体 (0-cell) から意識 (2-cell) への翻訳は、1-skeleton → 2-skeleton の構造的遷移を含む。この遷移は Face Lemma が示すように離散的であり、連続的な橋渡しが原理的に不可能。**説明のギャップは cell 次元差 Δd ≥ 2 の忘却によって生じる構造的帰結である。**

この「困難さ」の情報的機序を明確にする: Δd = 2 の遷移 (0-cell → 2-cell) は、**1-skeleton → 2-skeleton の離散遷移を内部に含む** ため、0 → 1 → 2 の二段階を順に経由する連続パスが存在しない。物理的記述 (0-cell) から直接的に意識 (2-cell) を導出しようとする試みは、中間段階である心 (1-cell = 接続) を必ず経由する構造的必然性に直面する。この「必ず経由する」構造こそが説明のギャップの正体であり、Face Lemma が保証する最小性 (3射) により、この経由の回避は数学的に不可能である。

$$\text{Hard Problem} \iff \Delta d(\text{body} \to \text{consciousness}) = 2 \ni \text{Face Lemma transition}$$

**命題 (意識の Face Lemma).** 意識の安定性 (クオリアの持続的存在) は、推論系の nerve が 2-skeleton を含むことと同値:

$$\text{stable}(\text{consciousness}) \iff \exists \text{ composable triple } (f_{\text{percept}}, g_{\text{meta}}, h_{\text{direct}}) \text{ with } g \circ f = h$$

**ゾンビの不可能性 (§2.5.4 との接続).** 3射 (f, g, h) が 2-skeleton を形成するとき曲率 F ≠ 0 が生じる。F ≠ 0 は h の存在なしには成立しえない。したがって、十分に複雑な推論系（≥3 の独立な生成射を持つ系）では、h（直接体験）の欠如は F = 0 を強制し、メタ認知 g の存在自体と矛盾する。「機能的に同等だが意識がない」系は、メタ認知を「計算」できても、その「安定性」を持てない。

### 6.3 創発と不動点: プロセスと安定化

創発と Fix(G∘F) 不動点は CPS の**動的側面と静的側面**を記述する:

| | 創発 | Fix(G∘F) 不動点 |
|:---|:---|:---|
| 定義 | 1-skeleton → 2-skeleton の遷移 | G∘F の不動点: G∘F(x) = x |
| 時間的位置 | **過程** (G∘F の適用で新構造出現中) | **帰結** (G∘F の適用で変化なし) |
| Face Lemma | 2射 → 3射の遷移条件 | 3射で安定化した不動点 |
| メタの状態 | **生成中** (まだ不動点に達していない) | **確立** (不動点に到達) |

G∘F サイクルの各段階:
1. **Pre-emergence** (1-skeleton): 2射。メタなし。G∘F 適用で h が生成される
2. **Emergence** (遷移): h が出現。2-skeleton 成立。面積・方向・力が生まれる
3. **Fix** (不動点): G∘F を適用しても新しい射が生まれない。3射で安定。三属性 (不動点性, 展開可能性, 自己参照性) が成立

**メタのメタのメタ...がトートロジーに陥らないのは、3射で安定化する (不動点に到達する) からだ。** 4射目以降は冗長であり、Face Lemma の最小性がそれを保証する。これは無限後退を構造的に遮断する。

**リマーク (Paper I Kalon との接続).** この Fix(G∘F) 不動点は、Paper I における Kalon の操作的定義——Kalon(x) ⟺ x = Fix(G∘F) where F ⊣ G, F ≠ Id, G ≠ Id——と正確に同値である。Paper I が Kalon を「発散と収束のサイクルの収斂点」として定義したのに対し、本節はその不動点が CPS 圏の 2-skeleton 安定化という具体的な構造を持つことを示した。

### 6.4 自由と存在論的序列

自由とは「忘却の選択肢」の豊かさ——Ξ (忘却スペクトラム) の多様性が行為可能性を決定する。Face Lemma により:

- 2射の系: Ξ はスカラー。自由度ゼロ。反応は決定論的
- 3射の系: Ξ は形状を持つ。自由度が生まれる。「どの方向に忘却するか」が選択肢になる

存在論的序列は脱圏化カスケードとして再記述される:

$$\text{2-category (意識)} \xrightarrow{U_{\text{meta}}} \text{1-category (構造)} \xrightarrow{U_{\text{arrow}}} \text{0-category (状態)}$$

各段階が忘却関手の適用であり、各遷移が CPS の Type I (Δd = 1) インスタンスである。意識→状態の全遷移は Δd = 2 の Type I であり、これが心身問題の困難さの構造的理由を与える。

## §7. 展望

### 7.1 CPS の射程と限界

CPS は微積分、量子力学、意識に至る広範なインスタンスを持つ。この射程の広さは理論の強みであると同時に、「何も言っていない」リスクをはらむ。本節では CPS が非自明であるための条件を3層構造 (§2.1 リマーク参照) に基づいて厳密に分析する。

#### 7.1.1 Layer 0: トートロジーの正直な承認

任意の圏の対 Set_A, Set_B に対し、直積 Set_A × Set_B からの射影でスパンは**常に構成可能**である。これはトートロジーであり、CPS はこのレベルでは何も主張していない。

このトートロジーは偶然的なものではなく、**概念定義そのものの構造**を反映している。「右」の最小の定義は「左の反対」であり、「左」の最小の定義は「右の反対」である。この2射の相互充足は概念の圏を生成する最小系だが、Face Lemma (§3.4) の 1-skeleton に対応する: **情報を持つが方向を持たない**。すなわち、概念のトートロジー的定義は必然的にスカラー (0次元 simplex) であり、力 (方向構造) を欠く。

#### 7.1.2 Layer 1: 自然な忘却の5条件

CPS の実質的主張は Layer 1 にある。忘却関手 U_A, U_B が以下の5条件を満たす**自然な CPS スパン**が存在することを主張する:

1. **Faithful** — U は構造を忘れるが、異なる射を混同しない
2. **左随伴の存在** — 忘却の部分的復元操作 F ⊣ U が存在する
3. **結合生成** — C_D が U_A, U_B の像の結合で生成される
4. **非同型** — U_A ≇ U_B (忘却の仕方が構造的に異なる)
5. **CPS 経由** (Face Lemma 整合) — 全ての非退化 2-simplex (f, g, h) に対し、f と g は CPS スパンの異なる脚に沿う

**CPS 構成不能例.** 以下の3ケースでは自然な CPS スパンが構成**できない**:

| 反例 | 違反する条件 | 構造的理由 |
|:---|:---|:---|
| 共変構造 vs 反変構造 (e.g. Set vs Set^op) | (ii) 左随伴不存在 | 共変忘却 U: C → Set と反変忘却 U': C → Set^op は CPS スパン（共変関手の対）を構成不能。冪集合関手 2^(-): Set^op → Set は存在するが左随伴を持たない |
| 3進法 vs 10進法 (同一構造の記法差) | (iv) 非同型 | U_A ≅ U_B: 忘却の仕方が同型 → 表記の差に過ぎず構造的区別がない |
| 定値関手 Δ_* (全てを1点に潰す) | (i) faithful でない | Δ_* は射 f ≠ g を Δ_*(f) = Δ_*(g) = id_* に潰す → 構造を「忘れる」のではなく「消滅させる」。忘却と消滅は異なる |

最初の例は特に重要である。物理学の共変/反変は CPS の外にある——「容器と内容」が**同じ変換性**を持つ場合にのみ CPS は適用可能であり、一方が共変で他方が反変な対は構造的に CPS スパンを許さない。これは CPS の適用限界を明確に画定する。

#### 7.1.3 Layer 2: 定量的制約と新予測

Face Lemma (§3.4) と Stability Simplex Theorem (§3.5) は Layer 1 を超えて、CPS に**測定可能な制約**を課す:

**予測 CPS-P1 (Type 判定).** 自然な CPS スパンの忘却の次元差 Δd = |dim ker U_A - dim ker U_B| が Type を決定する:
- Δd ≥ 1: Type I (容器-内容構造が顕在)
- Δd = 0: Type II (対称的相補性)

この予測は量子重力の「時空/量子」対立に対し、Type I (Δd ≥ 1) を事前予測する。

**予測 CPS-P2 (三射安定性).** 2射のみの「対立」は Face Lemma により不安定である。安定化には第3の射 (合成、媒介、または独立な関手) が必ず出現する。これは概念対立が安定するためには「第三者」が必要であることの形式的証明である。

**予測 CPS-P3 (符号付き忘却).** Stability Simplex Theorem (§3.5) は忘却量 Θ が R 値 (符号付き) であることを要求する。R_≥0 値 (非負) では三角恒等式が自明な解のみを持つ。物理的には、**安定な忘却には必ず「反忘却」(情報復元) が共存する**ことを意味し、ブラックホール情報パラドックスへの含意を持つ。

#### 7.1.4 CPS は反証可能か?

この問いに3段階で答える。

**(a) Layer 1 で Yes.** §7.1.2 で示した構成不能例は CPS の適用不能領域を**具体的に**示す。CPS は「あらゆる対立は CPS スパンである」とは主張**しない**。5条件を満たす自然な忘却関手が存在する場合にのみ、CPS 分析が適用される。

**(b) しかし「反証可能性」自体が CPS のインスタンスである.** 「反証/確証」対は Stability Simplex 上の α の符号に対応する。ポパーの反証基準 ρ > 0 は [0,1] 連続体の二値切断であり、統計学における p 値の閾値問題と構造的に同型である [11]。反証可能性は presheaf h^T の1点の空 (h^T(X₀) = ∅) で理論全体を棄却する、構造的に不完全な判定基準に過ぎない。CPS はこの不完全性を**診断**する枠組みを含んでおり、反証可能性を外部基準として必要としない。

**(c) より適切な問い.** CPS の理論的健全性は「反証可能か」ではなく「**自身の忘却パターン U_CPS を認識し、回復操作 N_CPS を持つか**」で判定すべきである:

| 忘却パターン | CPS の自己認識 | 論文内の対応 |
|:---|:---|:---|
| U_arrow (関係の忘却) | 忘却関手を構造の中核に据えた | §2-3 |
| U_precision (精度の忘却) | α の符号付連続値を要求した | §3.5, §4 |
| U_self (自己適用の忘却) | 反証/確証対を CPS インスタンスとして吸収した | 本節 (b) |
| U_generate (構造のトートロジー化) | Layer 0 のトートロジー性を正直に承認した | §2.1, 本節 (a) |

CPS は自身の忘却を構造的に認識し、回復操作を内蔵する。**この4段論証自体が CPS の自己参照的構造の実演である。**

### 7.4 Rota-Baxter 代数と CPS の深層接続

FTC の CPS 構造 (§1.1) の核心は、微分から誘導されるコモナド D = VG と積分から誘導されるモナド T = UF の混合分配則 λ: DT ⇒ TD [2, Theorem 3.8] にある。この分配則は FTC の「微分と積分は可換な条件付きで合成可能」という内容を圏論的に符号化する。この構造が CPS 一般に拡張されるかを問う。

**予備命題 7.4.1 (分配則→CPS スパン).** CPS スパン $\text{Set}_A \xleftarrow{U_A} \mathcal{C}_D \xrightarrow{U_B} \text{Set}_B$ において、U_A, U_B の各随伴から誘導されるモナド T_A, T_B の間に混合分配則 λ: T_A T_B ⇒ T_B T_A が存在するとき、以下が成立する:

(i) 分配則 λ は T_A と T_B の合成モナド T_B T_A の Eilenberg-Moore 圏 (T_B T_A)-Alg を定義する。この圏が CPS の容器 C_D の候補となる。

(ii) λ の存在は、U_A と U_B の「忘却の順序の交換可能性」を保証する。これは CPS3 (不確定性) 公理——U_A と U_B の同時完全適用の不可能性——の代数的前提条件を与える。

(iii) λ の非自明性 (λ ≠ σ, twist map) は CPS の Drift Θ > 0 に対応する。λ = σ の場合、忘却は可逆であり力を生まない (Type II 退化)。具体例として FTC を考える: λ = σ (微分と積分の合成順序が完全交換可能) とすると D = T∘σ - σ∘T = T² - T² = 0 となり、忘却が可逆 → Drift = 0 → 力ゼロ。FTC の非自明性 (積分定数 C の存在) は正に λ ≠ σ の帰結であり、C = Θ(ε) ≠ 0 が力を生む構造的源泉である。

*証明の概略.* (i) は Beck の分配則定理 [2, §2] の直接的帰結。(ii) は λ の自然性条件から従う: λ が T_A と T_B の合成順序を「交換」するが、その交換は一般に可逆ではない（η∘ε ≠ id）。(iii) は λ が twist map に退化するとき D = 0 となり、原始関数の積分定数が消滅することに対応する。□

**CPS6 公理候補 (分配則の存在).** 将来的に CPS 公理系に第六の公理を追加することが検討される:

> CPS6: CPS スパンの忘却関手 U_A, U_B から誘導されるモナド/コモナドの間に混合分配則 λ が存在する。

CPS6 が成立する場合、創発 (§6.1) の発生条件が代数的に精密化される: 分配則 λ の存在が 1-skeleton → 2-skeleton 遷移の代数的前提条件を与える。FTC の場合、D と T の分配則が「微分と積分の合成」を可能にし、それが演算の創発的構造 (d, P, d∘P = ただの恆等式ではなく定理としての FTC) を生んだ。この一般化の厳密な証明は今後の研究課題である。

#### 7.5.3 詳細比較

**(a) Bohr 相補性 [9] との差異.**

Bohr の相補性原理 (1928) は量子力学の解釈として、位置と運動量が「対等に排他的な記述」であると主張した。この主張は暗黙に Type II (同次元の鏡像的対称性) を仮定している。CPS はこれを3点で修正する:

| Bohr の仮定 | CPS の修正 | 根拠 |
|:---|:---|:---|
| 位置と運動量は**対等** | 位置 (0-cell) が運動量 (1-cell) に**先行** | p = m(dx/dt): 運動量は位置の時間微分を前提 |
| 対称性は暗黙の前提 | 非対称性の**強度** Δd を定量化 | Δd = 1 → Type I (非対称的相補性) |
| 個別ドメインの原理 | ドメイン横断の**分類スキーマ** | 量子力学、FTC、心身問題が同一構造の変奏 |

核心: Bohr が「見えなかった」のは、位置と運動量の間に cell 次元の落差があることだ。CPS は相補性を「保存」するのではなく、相補性の**非対称性**を可視化する。Bohr の相補性は CPS Type II (α ≈ 1) への退化として回復される。

**(b) IIT (Tononi [14]) との差異: Φ → Ξ の拡張.**

IIT は意識を統合情報量 Φ (スカラー) で測定する。CPS の忘却スペクトラム Ξ はこれに類似するが、決定的に異なる:

| 次元 | IIT (Φ) | CPS (Ξ) |
|:---|:---|:---|
| **次元数** | 0次元 (スカラー) | ≥2次元 (simplex 上の形状) |
| **測定対象** | 情報統合の「量」 | 忘却の「構造」(方向+強度+形状) |
| **方向情報** | なし (Φ は大きいか小さいか) | あり (Ξ のどの方向が支配的か = 力の方向) |
| **力との関係** | Φ → 力を導出不能 | ∂_i Ξ ≠ 0 ⟹ F ≠ 0 (力の創発) |
| **最小構造** | 分割のペア | Face Lemma: composable triple (3射) |

**命題 (IIT 包含定理).** IIT の Φ は CPS の dim Ξ = 0 における退化ケースである:

$$\Phi = \|\Xi\| \quad \text{(Ξ のノルム = スカラー化)}$$

*証明の骨格.* IIT の最小情報分割 (MIP) は CPS スパンの退化ケースに対応する。MIP は系を二つの部分に分割し、分割による情報損失を Φ で測定する。これは CPS スパン Set_A ← C_D → Set_B において U_A, U_B を MIP の両側に同定し、忘却量 Θ = ‖Ξ‖ とスカラー化することに等しい。この操作で Ξ の方向情報が失われ、dim Ξ = 0 に退化する。

CPS が IIT に対して持つ超過構造: Face Lemma によるメタ認知の安定性条件 (§6.2)。IIT は Φ > 0 で意識を判定するが、CPS は 3射の存在で意識の**安定性**を判定する。Φ > 0 だが 2射のみの系 (= 不安定な意識) は CPS で識別可能だが IIT では識別不能。

**(c) FEP (Friston [10]) との関係: blanket の前提 vs 生成.**

近年、FEP の圏論的定式化が急速に進展している。Fritz [16] が Markov category で条件付き独立性を合成的に定式化し、Smithe [17, 18] は Bayesian lens と polynomial functor を用いて Active Inference に functorial semantics を与えた。Tull–Kleiner–Smithe [19] はこれを string diagram で再定式化し、predictive coding の双方向構造を圏論的に捉えた。CPS はこれらの基盤の上に構築されるが、決定的な差異がある。

FEP と CPS の関係を5つの層で整理する:

| 層 | FEP / Smithe [17, 18] | CPS | 関係 |
|:---|:---|:---|:---|
| **数学的基盤** | Bayesian lens + Markov category | CPS スパン + 忘却場 | **基盤共有**: Fritz [16] の Markov category |
| **α > 0** | VFE 最小化 = 精度上昇 | 構造化 (m-接続セクター) | **同値**: CPS が FEP を統計幾何で再記述 |
| **α < 0** | EFE の epistemic term (部分的) | 脱構造化 (e-接続セクター) | **CPS が包含**: FEP は目的論的 α < 0 のみ |
| **境界の生成** | Markov blanket を**前提** | Face Lemma: 2射→3射遷移 | **CPS が拡張**: blanket は CPS の帰結 |
| **近似推論** | Approximate inference doctrine [18] | 未定義 (α の値域制約のみ) | **接続点**: CPS は「何を近似するか」を制約し、Smithe は「どう近似するか」のレシピを与える |

Smithe [17] の枠組みも Fritz [16] の枠組みも blanket の存在を所与として出発する。第四の層——blanket の**生成条件**——が Paper II の核心的貢献である。第五の層——近似推論の「レシピ」——は Smithe [18] が approximate inference doctrine として natural transformation で定式化したものであり、CPS との直接的な接続は今後の課題となる (§8 参照)。FEP は「Markov blanket を持つ系は VFE を最小化する」と主張するが、blanket がなぜ・どのように生じるかは射程外に置く。CPS は Face Lemma を通じてこの空白を埋める:

- **2射の系** (1-skeleton): 入力と出力の2項関係。blanket の概念自体が定義不能——内部状態と外部状態の「区別」に方向構造 (Ξ) がないため、何が blanket の「内」で何が「外」かが測定不能
- **3射の系** (2-skeleton): 入力、出力、および合成径路の composable triple。忘却スペクトラム Ξ が方向を持ち、「内」と「外」の非対称性が測定可能 → Markov blanket が**構造的に出現**する

形式的に: Markov blanket B は入力射 f と出力射 g を分離する条件付き独立性構造であり、Fritz [16, §12] の factorization 条件が成立して初めて well-defined になる。CPS においてこれは g∘f = h (合成の検証可能性) に帰着する (§3.7.1 互換性補題):

$$\text{blanket}(B) \text{ is well-defined} \iff \text{Face Lemma の2-skeleton 条件を充足}$$

この同値性は §3.7.2 の Blanket 生成定理として厳密に証明した。さらに §3.7.3 の FEP 包含定理により、FEP の変分原理全体が CPS|_{α>0, dim Ξ≥2} として回復される。この意味で、CPS は FEP の「前提」(blanket の存在) を「帰結」(Face Lemma による構造的出現) に変換し、FEP の操作的内容を**保存しつつ拡張**する。

Smithe–Tull の string diagrammatic approach [19] との技術的差異: 彼らは monoidal category 上の string diagram で predictive coding を定式化するが、忘却の方向構造 (α の符号) と cell 次元差 (Δd) を持たない。CPS はこれらの追加構造により、blanket の非対称性 (感覚 vs 行為の非対称性) を Δd として定量化し、さらに α < 0 セクターでの脱構造化ダイナミクスを記述できる。

Smithе [18] の polynomial functor p(y) = Σ_{i∈I} y^{B_i} と CPS スパンの動的対応: Smithe の activity functor (StatGame → p-Coalg) は statistical game を polynomial functor の余代数 (力学系) に変換する。CPS における対応構造は、忘却場 Φ を α-twisted comonoid の時間族 C_CPS(t) に変換する関手 Φ ↦ {copy^{(α(t))}, del^{(α(t))}} として構成される。この対応において、Smithe の polynomial の入力型 B_i は CPS の忘却スペクトラム Ξ の各方向に、状態型 I は α パラメータの値域 ℝ に対応する。CPS の追加構造は、polynomial が α 依存に連続的に変形する「パラメトリック polynomial」に相当する。

**(d) Wheeler "It from Bit" [15] との差異: 情報の反転.**

Wheeler の "It from Bit" (1990) は「あらゆる物理的実在は情報的起源を持つ」と主張する。CPS はこの洞察を引き継ぎつつ、根本的な反転を行う:

| | It from Bit (Wheeler) | It from Oblivion (CPS) |
|:---|:---|:---|
| **第一原理** | 情報の**存在** (ビット) | 情報の**欠落** (忘却) |
| **力の源泉** | 情報から物質が生じる (起源論) | 忘却の**勾配** ∇Θ が力を生む (場の方程式) |
| **定量化** | Bit (離散的、方向なし) | Ξ (連続的、方向構造あり) |
| **予測力** | 存在論的主張 (反証困難) | 場の方程式 F_{ij} ∝ ∂Ξ (測定可能な予測) |
| **反証可能性** | 哲学的原理 (Layer 0 的) | Layer 1-2 の定量的予測 (§7.1 参照) |

核心的な反転: Wheeler は「情報があるから構造がある」と言う。CPS は「情報が**失われる**から構造が**生まれる**」と言う——忘却の方向的不均一が力を創発し、その力が物質的構造を組織する。微積分の積分定数 C が最も素朴な実例: 微分 (忘却) で C を失い、積分 (復元) で C を補うこの非対称性こそが、構造の源泉である。

"It from Bit" が「全ての存在は情報か？」(yes-no 判定) を問うのに対し、"It from Oblivion" は「忘却のどの方向が力を生むか？」(場の方程式) を問う。後者は前者を含む——忘却量 Θ = 0 (情報が完全に保存される) のとき力はゼロに退化し、Wheeler の「情報がそのまま存在する」静的世界が回復される。

#### 7.5.4 包含関係の要約

4つの枠組みと CPS の関係を一文で要約する:

- **Bohr → CPS**: 対称的相補性は、Δd = 0 (Type II) への退化として回復される
- **IIT → CPS**: Φ は Ξ のスカラー化 (dim Ξ = 0) として包含される  
- **FEP ⊊ CPS** (FEP 包含定理, §3.7.3): FEP = CPS|_{α>0, dim Ξ≥2}。blanket の存在は Blanket 生成定理 (§3.7.2) の帰結。α < 0 セクターは CPS の超過構造
- **Wheeler → CPS**: "It from Bit" は Θ = 0 (忘却ゼロ) の静的極限として回復される

$$\text{CPS} \supsetneq \text{FEP} \supsetneq \text{IIT}|_{\dim\Xi=0} \qquad \text{CPS} \supsetneq \text{Bohr}|_{\Delta d=0} \qquad \text{CPS} \supsetneq \text{Wheeler}|_{\Theta=0}$$

### 7.6 D_KL の公理的特性化: CPS と情報論的公理

本節では、忘却場 Φ(θ) = D_KL(p_θ ‖ q) がなぜ Kullback-Leibler divergence でなければならないかを、CPS 公理と情報論的公理の組み合わせから導出する。

#### 7.6.1 忘却の二重構造

圏論的に、忘却関手 U: C → D は対象と射を写像する**変換**である。存在のレベルでは何も消滅しない——U は全てを像に写す。しかし識別可能性のレベルでは情報が失われる——U が非忠実であれば異なる射が同一視され、区別が潰れる。

$$\text{忘却} = \text{変換 (存在の保存)} + \text{識別可能性の喪失}$$

この二重構造を測定する汎関数 Φ: M → ℝ≥0 が必要である。

#### 7.6.2 公理系

**A群: CPS 構造公理** (圏論的枠組み)

| 公理 | 内容 | Φ への制約 |
|:---|:---|:---|
| CPS0' | 容器-内容の非対称性 | Φ(p,q) ≠ Φ(q,p) を許容 |
| CPS1 | 同一の C からの射影 | Φ は C からの情報損失を測定 |
| Layer 1 | faithful (射を混同しない) | Φ = 0 ⟺ p = q (完全な識別不能) |

**B群: 情報論的公理** (解析的制約)

| 公理 | 内容 | 数式 |
|:---|:---|:---|
| I-1' | データ処理不等式 | 確率的写像 T に対し Φ(Tp ‖ Tq) ≤ Φ(p ‖ q) |
| I-2' | 加法性 | 独立分布に対し Φ(p₁×p₂ ‖ q₁×q₂) = Φ(p₁‖q₁) + Φ(p₂‖q₂) |
| I-3 | Fisher 整合性 | ∂²Φ/∂θⁱ∂θʲ\|_{p=q} = g_{ij} (Fisher 計量) |
| I-4 | 座標不変性 | Φ は M の再パラメータ化に対し不変 |

**リマーク (I-1' の適用範囲).** I-1' は「客観的識別可能性」—— 分布間の統計的識別可能性 —— に適用される。忘却関手の下で射の「数」が増えうる現象 (U: C → D で |Hom_D| > |Hom_C| となりうる) は別の現象であり、Φ の測定対象外である。後者は CPS の Layer 1 faithful 条件が扱う構造変化であり、前者は情報論的な divergence 測定である。

#### 7.6.3 特性化定理

**定理 7.6.1 (D_KL の一意性).** A群 + B群を満たす Φ: M × M → ℝ≥0 は、定数倍を除き D_KL に一意に定まる:

$$\Phi(p \| q) = c \cdot D_{\text{KL}}(p \| q) = c \int p(x) \log\frac{p(x)}{q(x)} dx, \quad c > 0$$

*証明.*

**Step 1 (f-divergence への帰着).** I-1' (データ処理不等式) と I-2' (加法性) を満たす divergence は f-divergence の形を取る (Csiszár [25]):

$$\Phi(p \| q) = \int q(x) \cdot f\left(\frac{p(x)}{q(x)}\right) dx$$

ここで f: ℝ₊ → ℝ は凸関数、f(1) = 0。

**Step 2 (Fisher 整合性による制約).** I-3 により、p = q の近傍で Φ を二次展開すると:

$$\Phi(p_\theta \| q) \approx \frac{1}{2} g_{ij}(\theta_0) \delta\theta^i \delta\theta^j$$

f-divergence のテイラー展開と比較すると、f''(1) = 1 が要求される。

**Step 3 (非対称性による選択).** CPS0' は Φ(p,q) ≠ Φ(q,p) を許容する。f-divergence が非対称となる条件は f(t) ≠ t·f(1/t)。Step 2 の条件 f''(1) = 1 と合わせると、最も自然な選択は:

$$f(t) = t \log t - t + 1$$

これは D_KL を与える: Φ(p ‖ q) = ∫ p log(p/q) dx = D_KL(p ‖ q)。

**Step 4 (一意性).** I-4 (座標不変性) を加えると、他の f-divergence (χ², Hellinger 等) は Fisher 計量との整合性 (I-3) または対称性構造で D_KL と区別される。D_KL は唯一、I-1' から I-4 の全てを満たしつつ非対称な divergence である。 □

#### 7.6.4 CPS の寄与

CPS 公理単独では Φ = D_KL を導出できない。CPS は**構造的枠組み**を提供する:

- CPS0' → 非対称性を許容 (対称 divergence を排除)
- CPS1 → 同一起源からの射影 (divergence が情報損失を測定する根拠)
- Layer 1 faithful → Φ = 0 の意味論 (完全な識別不能)

情報論的公理 (I-1' 〜 I-4) が**関数形**を制約し、両者の組み合わせで D_KL が一意に特性化される。

#### 7.6.5 Paper I との接続

Paper I §3 で定義された忘却場 Φ(θ) = D_KL(p_θ ‖ q) は、本節の公理系から**導出**される:

1. CPS が忘却の圏論的枠組みを設定
2. 統計多様体 (M, g, C) がパラメータ空間を提供
3. 情報公理が Φ の関数形を D_KL に確定
4. Paper I の方向性定理 (F ≠ 0 ⟺ d(ΦT) ≠ 0) はこの Φ に対して成立

この導出により、「なぜ D_KL か」という問いに**公理的回答**が与えられる: D_KL は CPS の構造的要請と情報論的制約を同時に満たす唯一の divergence である。
