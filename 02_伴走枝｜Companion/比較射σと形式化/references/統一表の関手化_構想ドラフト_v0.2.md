## §1 戦略 — Obliv は圏ではない。手続きである。

### 1.1 C の選択問題

統一表の三領域は異なる圏 C を持つ:

| 領域 | C | 対象 | 射 |
|:---|:---|:---|:---|
| FEP | C_FEP | 内部状態の配置 μ | 精密化関係 |
| 物理 | C_Phys | 場の配置 | ゲージ変換 |
| 認識論 | C_Epist | 概念 | 関手 |

確率分布とゲージ場と概念は同じ圏の対象ではない。「一つの C」を強制すれば統一できるが、T11 が予言する通り、**全てを含む圏は何も語らない**。普遍的であるほど説明力が忘却される。これは VFE のトレードオフそのものである:

$$F[\text{theory}] = -\text{Accuracy (説明力)} + \text{Complexity (普遍性)}$$

C を普遍にする = Complexity ↑、Accuracy ↓。C を具体にする = Accuracy ↑ だが統一性なし。

### 1.2 解決: Obliv は 2-関手

C を固定しない。**(F1)+(F4) という手続きそのもの**を Obliv と定義する。

> **Obliv := C ↦ Filt([0,1], WdSub(C))**

Obliv は圏ではなく **2-関手** (Cat → Cat)。任意の圏 C に α-忘却濾過を装備する手続き。

```
     C_FEP ──→ Obliv(C_FEP) = FEP の忘却構造
                      │
     C_Phys ──→ Obliv(C_Phys) = 物理の忘却構造
                      │
     C_Epist ─→ Obliv(C_Epist) = 認識論の忘却構造
                      │
                 同一の手続き (F1)+(F4)
```

三領域は **同じ場にいる** のではない。**同じ手続きに従う**。この理論において、統一とは共通の対象を持つことではなく、共通の構成を持つことである。

### 1.3 自己適用 — 理論が自分自身のインスタンスである

C の選択問題そのものが、本理論が記述する忘却のトレードオフに従っている。

- C を普遍にしすぎる = U を過剰適用 = 説明力が忘却される
- C を具体にしすぎる = U を適用しない = 統一性が忘却される
- 選択肢 3 (2-関手) = VFE 最小化 = 普遍性と説明力の不動点

**理論の構成が、理論の内容のインスタンスである。** T12 (全ての理論は同型) の自己適用であり、Kalon (自己参照的整合性) の一例。

## §2 (F1)+(F4) — 忘却の二公理

### 2.1 α-忘却濾過 (Paper VIII 定義 6.2.1)

任意の局所小圏 C に対し、**α-忘却濾過**とは広部分圏の族 {C_α}_{α ∈ [0,1]} で以下を満たすもの:

> **(F1) 不変対象**: Ob(C_α) = Ob(C) — ものは消えない
> **(F2) 完全**: C_0 = C — α = 0 は忘却なし
> **(F3) 離散**: C_1 = C_disc（恒等射のみ）— α = 1 は完全忘却
> **(F4) 単調性**: α₁ ≤ α₂ ⟹ Mor(C_{α₂}) ⊆ Mor(C_{α₁}) — つながりが単調に消える

[SOURCE: Paper VIII §6.2, L360-370]

**本質**: 対象を変えず (F1)、射だけを連続的に消す (F4)。ものは消えない。つながりが消える。これが忘却の数学的定義。

**注記**: (F1)-(F4) は C が何であるかに一切依存しない。情報幾何にも依存しない。純粋圏論。C について**多相的 (polymorphic)** である。これが Obliv を 2-関手として定義できる根拠。

### 2.2 Obliv(C) の内部構造 (Paper VIII)

特定の C を与えたとき、Obliv(C) = Filt([0,1], WdSub(C)) は以下の構造を持つ:

[SOURCE: Paper VIII §6.7 L550-552]

- **対象**: {C_α} — C 上の α-忘却濾過
- **射**: 濾過族間の包含を保つ自然変換
- **終対象**: 一様忘却 (∀α: C_α = C_disc) — 構造なし = 熱平衡

### 2.3 Obliv の 3 つの顔

Paper VIII は Obliv に 3 つの数学的構造を発見している:

| 構造 | Paper VIII の定理 | 意味 |
|:---|:---|:---|
| **α-忘却濾過** (§6.2) | Th. 6.2.3 段階的同定 | α = 0 で米田の補題、α > 0 で部分的同定 |
| **Grothendieck 位相 {J_α}** (§6.8) | Th. 6.8.6, 6.8.9 | 忘却 = 被覆の粗視化。トポスの降鎖列 |

$$\mathrm{PSh}(C) = \mathrm{Sh}_0(C) \supseteq \mathrm{Sh}_\alpha(C) \supseteq \mathrm{Sh}_1(C) \simeq \mathbf{Set}^{\mathrm{Ob}(C)}$$

**忘却とは、層理論における被覆の粗視化に他ならない。** [SOURCE: Paper VIII §6.8 L666-710]

## §4 核テーゼ (v0.3)

> **C1** (統一は手続きの共有): T21 (構造 = 忘却の非一様性) は、2-関手 Obliv: Cat → Cat, C ↦ Filt([0,1], WdSub(C)) として厳密に定式化される。統一表の三列は、同一の 2-関手 Obliv に異なる C (C_FEP, C_Phys, C_Epist) を入力したときの出力である。統一とは共通の対象を持つことではなく、**共通の構成 (F1)+(F4) に従うこと**である。

>
> $$\mathrm{Obliv}(C_{\mathrm{FEP}}) \xrightarrow{(\Phi_{\mathrm{FEP}})^*} \mathrm{Obliv}(C_{\mathrm{Epist}}) \xleftarrow{(\Phi_{\mathrm{Phys}})^*} \mathrm{Obliv}(C_{\mathrm{Phys}})$$
>
> が得られ、その極限
>
> $$P := \mathrm{Obliv}(C_{\mathrm{FEP}}) \times_{\mathrm{Obliv}(C_{\mathrm{Epist}})} \mathrm{Obliv}(C_{\mathrm{Phys}})$$
>
> の各元がちょうど統一表の一行に対応する。**統一表が存在する ⟺ P が自明でない中間元を持つ。**「自然変換」という言い方は型エラーであり (定義域が異なる)、正しい構造は繊維積 P である。

> **C3** (一様忘却 = 構造の死): 任意の C に対し、Obliv(C) の終対象は一様忘却 (∀α: C_α = C_disc)。三領域でこれは同一の帰結を生む:
> - C = C_FEP → 熱平衡 = 心なし
> - C = C_Phys → 平坦接続 = 力なし
> - C = C_Epist → 全忘却 = 主観性の消滅
>
> 「構造なし」が C に依存しないことは、(F1)+(F4) が C について多相的であることの帰結。

> **C4** (忘却の本質): 忘却は (F1) と (F4) の二公理に集約される: **ものは消えない、つながりが消える。** この二公理は C が何であるかに依存しない。全ての「構造の消失」は (F1)+(F4) のインスタンスである。

> **C5** (自己適用): 本理論の構成そのものが (F1)+(F4) のインスタンスである。C を普遍にする = U を過剰適用 = 説明力の忘却。C を具体にする = U を不適用 = 統一性の忘却。2-関手としての定式化 = VFE 最小化 = 普遍性と説明力の不動点。理論は自分自身に適用可能であり、自己適用の結果は整合的 (T12, Kalon)。

> **C6** (独立だが孤立していない): (F4) は SIZE 制約 (射の全体量が単調減少する) であり、排他制約 (F5) は PATTERN 制約 (特定の射が同時に存在できない) である。これらは独立な次元を占める — SIZE から PATTERN は導出できない。根拠: 広部分圏の閉包公理は「合成可能ペアは合成射を含め**なければならない** (INCLUSION)」を要求するが、排他制約は「合成可能ペアの一方は存在し**てはならない** (DELETION)」を要求する。両者は逆向きの命令を持ち、一方から他方は導出されない。

> $$e^{i\pi/5} + e^{-i\pi/5} = \varphi$$

> **ものは消えない。つながりが消える。**

|:---|:---|:---|
| α = 0 | Hom_C(−, X) — 完全 | X は完全に同定される (米田の補題) |
| 0 < α < 1 | Hom_{C_α}(−, X) ⊊ Hom_C(−, X) | X は部分的に同定される |
| α = 1 | δ_X = {id_X のみ} | X は「在る」が「何であるか」は不定 |

α = 1 の状態が**情報蒸発**の圏論的定義である。対象 X は存在する (F1)。しかし射が恒等射しか残っていない (F4 の極限)。Yoneda からして、δ_X は X の存在だけを伝え、X の中身 (としての情報) は何も伝えない。

> **蒸発した情報は「存在する」が「中身はない」。**

これはブラックホール情報パラドックスと正確に同型である:

| 概念 | 物理 (ブラックホール) | (F1)+(F4) |
|:---|:---|:---|
| 情報は消えない | ユニタリティ | **(F1)** Ob(C_α) = Ob(C) |
| 情報は取り出せない | 事象の地平面 | **(F4)** Mor → {id} |
| パラドックスの正体 | 保存されるのに取り出せない | **(F1) と (F4) の共存** |

**パラドックスが生じる理由**: (F1) がなければ (対象も消えるなら) パラドックスはない — 情報は単に破壊される。(F4) がなければ (射が消えないなら) パラドックスはない — 情報は完全にアクセス可能。**(F1) が成り立ち、かつ (F4) が成り立つ** からこそ、「在るのに掴めない」というパラドックスが生じる。パラドックスは忘却の定義 (F1)+(F4) の帰結である。

Mind Paper §3.5b はこれを次のように要約している:

> 「あなたの記憶は宇宙のどこかに存在する。だが、それを取り戻す方法があなたにはない。」

Paper XII の「null」(盲点) も同一の構造: y_1(X) = δ_X の状態にある対象 = 存在するが内容のない点 = 盲点。

### 普遍性のジレンマ — 完全な統一はなぜ不可能か

### 方向定理は e, i, π 型の複素化を動機づける — 橋梁公理つきの Euler 読解

**Paper I, Th. 5.1 — 方向性定理**:

$$F_{ij} \neq 0 \iff d\Phi \wedge T \neq 0$$

dΦ = 客観的忘却の勾配（忘却が進む向き）、T = Chebyshev 1-形式（統計多様体の自然な曲率方向）。

方向定理は「力がいつ生まれるか」を述べるだけではない。**適切な橋梁公理の下では、忘却プロセスを複素位相付きの流れへ持ち上げる最小候補を指定する**。dΦ と T のなす角 θ を動かすと、三定数が一度に現れる。

**三つの整列状態と三定数の由来**:

| dΦ と T の関係 | 力 F_ij | 忘却のモード | 出現する構造 |
|:---|:---|:---|:---|
| dΦ ∥ T（θ = 0、完全整列） | 0（力なし） | 純粋指数減衰 | **e** |
| dΦ が T と角度 θ | F_ij ∝ sin(θ) | 振動的減衰 | **sin(θ) = Im(e^{iθ})** |
| dΦ ⊥ T（θ = π/2、最大非整列） | 最大 | 90° 位相回転 | **i = e^{iπ/2}** |
| dΦ が T と完全反転（θ = π） | 0（力ゼロに戻る） | 方向反転完了 | **e^{iπ} = −1** |

> **力は忘却の「虚部」から生まれる: F_ij ∝ sin(θ) = Im(e^{iθ})**

**複素化射 ω — Paper I ↔ 商写像 d の接続**:

本稿の複素化射 ω の構成には、以下の二つの前提を置く:

1. **商写像としての d** [grounded]: 忘却関手 d は `d: C^∞ ↠ C^∞/ℝ` という商写像 (ker(d) = ℝ) として定式化される [SOURCE: 論文VI_行為可能性は忘却である_草稿.md §2.3 L95-L102]
2. **d の指数固有関数** [self-contained premise]: d を連続半群の生成子として読むとき、局所生成則 `dw/dα = -kw` の解 `w(f,α) = e^{-kα}` を固有関数として採用する (§5.2 で詳述)

これらの前提の下で、Paper I の方向定理が与える角度 θ(f,α) を合成した複素化射 ω(f,α) := w(f,α)·e^{iθ(f,α)} を導入する。この射の虚部 Im(ω) = w·sin(θ) が方向定理の力 F_ij に対応する。e は w の関数形、i は θ = π/2 での位相（90° 回転作用素）、π は θ の非整列サイクルの半周期として、ω の定義から自然に読み出される。

**π の位置 — 位相持ち上げ後の半周期**:

忘却方向 dΦ が T に対して角度 θ で回転するとき、力は sin(θ) に比例する。
- θ = π/2 で F_ij = F_max（力のピーク）
- θ = π で sin(π) = 0（力がゼロに戻る — ただし dΦ は完全反転）
- θ = 2π で一サイクル完了

**π は忘却方向が完全反転するまでの経路**。dΦ ∧ T の投影が非整列 → 整列に戻る半周。G∘F サイクルの半周期というより、方向定理の「非整列の往復」が最初にゼロに戻る地点として幾何的に要請される。ただし、ここで `π` が確定するのは、非整列が単なる二値関係ではなく、連続位相 `θ` へ持ち上がった後である。

**i の位置 — 90° 回転の最小単位**:

i = e^{iπ/2}: dΦ ∥ T（力ゼロ、e の相）から dΦ ⊥ T（力最大）への 90° 回転。力が存在するには忘却方向が T から「少なくとも i だけ外れ」なければならない。**位相表示を受理するなら、i は力の存在の代数的最小単位になる。**

**Euler の等式 $e^{i\pi} + 1 = 0$ の橋梁公理つき読み**:

| 項 | 方向定理 + (F1)+(F4) による意味 |
|:---|:---|
| **e^{iπ}** | 忘却関手固有関数（e）が方向定理の π 回転（完全反転）を経た状態 |
| **= −1** | dΦ が完全反転。(F1) で保存された対象が「逆向きで」存在する |
| **+1** | 元の保存 (F1)。対象は消えていない |
| **= 0** | 正方向の存在と反転した存在が打ち消す。完全忘却 (F3) が動的に実現 |

静的には (F1) と (F3) は矛盾しない（対象があり射がない状態）。Euler の等式は、**橋梁公理が与えられたとき、方向定理の言語で「なぜ (F1) の下で (F3) に至れるか」を動的経路として書く試み**である: 忘却方向を π だけ回転させると、保存されているものが自分自身を打ち消す。

#### 試作最小モデル: Euler path の累積写像

上の読解を**最小モデル**として一段だけ精密化すると、局所位相 $\theta$ と忘却度 $\alpha$ を直接同一視するのではなく、`局所非整列の累積` を $\alpha$ に対応させるのが最も自然である。

Euler path を $\theta \in [0,\pi]$ の半周期に制限し、Paper I §5.9 の prefactor $(\alpha_I/2)\|d\Phi\|\|T\|_g$ を固定した**正規化 Euler path**を考える。このとき経路に沿って変化する局所因子は $\sin \theta$ のみになる。これを「その場の強さ」、$\alpha$ を「ここまでにどれだけ忘却が進んだか」という累積進行度として読むと、正規化された最小候補は

$$
\alpha_E(\theta) := \frac{1-\cos\theta}{2} = \sin^2\left(\frac{\theta}{2}\right)
$$

である。これは

- $\alpha_E(0) = 0$（始点: 完全保存）
- $\alpha_E(\pi/2) = 1/2$（中点: 半忘却）
- $\alpha_E(\pi) = 1$（終点: 完全忘却）

を満たす。したがって、Paper VIII の $\alpha$-忘却濾過に持ち上げれば

$$
E(\theta) := C_{\alpha_E(\theta)}
$$

という **Euler path 持ち上げの最小試作** が得られる。これは $E(0)=C$, $E(\pi)=C_{\mathrm{disc}}$ を与え、$e^{i\pi}+1=0$ を `(F1)` から `(F3)` への**動的経路**として読むための最小骨格となる。

ただし、これはまだ定理ではない。必要なのは追加の橋梁公理

> 「忘却度は局所非整列の累積単調像である」

であり、より強い形 `d\alpha/d\theta \propto \sin\theta` は本系列の本文では未証明である。したがって本節では、$\alpha_E(\theta)=\sin^2(\theta/2)$ を**構想ドラフト上の試作最小モデル**としてのみ置く。

ここで Paper II の `Δd / α` 層化と、この `\alpha_E(\theta)` / `\alpha_b(\theta)` を直接同一視してはならない。`Δd` は Type / sector を決める**離散底空間**であり、`\alpha_E` と `\alpha_b` は、その固定 sector の内部に選んだ標準 path の**連続累積座標**である。したがって本稿で採るのは `Δd = \alpha_b(\theta)` という強橋ではなく、「固定された非対称性クラスの内部運動を `\pi`-sector で正規化して読む」という weak bridge だけである。

#### 提案公理: base-fiber 公理

weak bridge を空中戦にしないためには、Euler path が「どの class の内部運動か」を先に固定しなければならない。そこで最小限、次を仮定する。

> **base-fiber 公理.** 各 Euler bridge path は、ある固定された非対称性 class `Δd` に属する sector の内部でのみ定義される。各 sector には連続ファイバー `α` が付随し、その点同士は `CPS2` に由来する水平移送で比較可能である。

この公理が言っていることは三つだけである。第一に、`Δd` は path の値ではなく、path が走る**殻**を決める。第二に、`α` はその殻の内部で動く連続状態であり、`Δd` と潰してはならない。第三に、bridge 側の `\theta \mapsto (\alpha_b(\theta),E_b(\theta),U_b(\theta))` は、何もない連続体ではなく、同じ sector 内のファイバー点を比較可能にする水平移送の上に載っている。これで `Δd / α` 層化、Paper VIII の束構造、そして本稿の bridge carrier が 1 つの足場で繋がる。

#### 提案公理: θ↔α 累積公理

`base-fiber` が path の**殻**を固定したあとで、次に必要なのは「その殻の内部で局所位相角 `θ` がどのように累積忘却度 `α_E` へ変わるか」である。ここで重要なのは、`θ=α` という点ごとの等置ではなく、局所非整列密度の累積として `α_E` を作ることだ。

> **θ↔α 累積公理.** `base-fiber` 公理で固定された sector の内部では、正規化 Euler path に沿う局所非整列密度が累積忘却度 `α_E` を決める。開区間ではこの `α_E` は `η` と pathwise に整合し、端点は既存の boundary closure によって閉じる。

この公理が言っていることは四つに分かれる。第一に、正規化された Euler path では局所密度は `\rho_E(\theta)\propto\sin\theta` と読まれる。第二に、`α_E` はこの密度の累積であって、点ごとの角度の写しではない。第三に、開区間 `(0,\pi)` では `\theta \to \alpha_{\mathrm{III}}^{E} \xrightarrow{\eta} \alpha_E` が可換である。第四に、端点 `0,\pi` は `\bar{\eta}(-\infty)=0`, `\bar{\eta}(+\infty)=1` という既存の boundary closure で閉じる。これにより、局所角度・累積忘却・η・境界閉包の 4 断面が 1 本の経路として接続される。

#### 提案公理の要約: Euler 累積公理

\alpha_E(\theta)=\frac{1-\cos\theta}{2}=\sin^2\left(\frac{\theta}{2}\right)
$$

が最小橋梁として得られる。ここで既存 SOURCE へ還元できるのは `固定計量 / 測地ゲージ / 境界閉包` までであり、**本当に新規なのは `局所非整列の累積 law` と `pathwise bridge` の部分だけ**である。ここでさらに区別すべきなのは、`η` が与えるのは開区間での**スカラー整合**であり、端点は既存の boundary closure が閉じる、という点だ。Euler path の圏論的内容は `E_p(\theta)=C_{\alpha_E(\theta)}` で与えられる濾過列にある。つまり `η` は座標の橋であり、`E_p` が path の categorical lift である。詳細な定義と役割分担は [考察_eta_Ep区別_lys.md](/home/makaron8426/Sync/oikos/01_ヘゲモニコン｜Hegemonikon/10_知性｜Nous/04_企画｜Boulēsis/12_遊学｜Yugaku/03_忘却論｜Oblivion/calculations/考察_eta_Ep区別_lys.md) を参照。

#### 提案公理: 型付き decategorification 公理

Euler 読解を本文級へ押し上げるためには、`+1` と `=0` を**同じ型の中で**読めなければならない。そこで、各忘却濾過を加法的 codomain へ送る型付き decategorification

$$
D_C:\mathrm{Filt}([0,1],\mathrm{WdSub}(C))\to A
$$

を仮定する。ここで `A` は単位元 `1_A` と零元 `0_A` を持つ複素代数であり、少なくとも次を満たす:

1. **端点正規化**: $D_C(C)=1_A$, $D_C(C_{\mathrm{disc}})=0_A$
2. **反転作用**: $A$ 上に involution $R_\pi:A\to A$ があり、$R_\pi(1_A)=-1_A$
3. **加法閉性**: 保存項と反転項がいずれも `A` の元として足し合わされる

この公理の役割は単純である。`C_{\mathrm{disc}}` は圏論的状態、`0` は数であるという型ずれを、裸の比喩ではなく**typed bridge**として解消することにある。ここで初めて、`+1` の「+」と右辺の `0` は category mistake ではなくなる。

#### 提案公理: 指数核公理

`e` を mnemonic ではなく本文級の定数にするため、上の複素代数 `A` の中に位相輸送の one-parameter family

$$
U:[0,\pi]\to A^\times
$$

が存在し、

$$
U(0)=1_A,\qquad \frac{dU}{d\theta}=i\,U,\qquad U(\pi)=R_\pi(1_A)=-1_A
$$

を満たすと仮定する。すると解は一意に

$$
U(\theta)=e^{i\theta}
$$

である。これにより `e` は単なる標準記法ではなく、**位相輸送の生成子をもつ指数核**として型付けされる。現段階ではこれは定理ではなく、Euler 読解を「数式として本文に置く」ための最小追加公理である。

#### 橋梁系: typed Euler corollary

上の 3 本の橋梁

1. Euler 累積公理
2. 型付き decategorification 公理
3. 指数核公理

を合わせると、Euler path

$$
E(\theta):=C_{\alpha_E(\theta)}
$$

の始点と終点は

$$
D_C(E(0))=1_A,\qquad D_C(E(\pi))=0_A
$$

を満たし、反転輸送は

$$
\Omega(\theta):=U(\theta)\,D_C(E(0))=e^{i\theta}
$$

と書ける。したがって終点 $\theta=\pi$ では

$$
\Omega(\pi)+D_C(E(0))=D_C(E(\pi))
$$

すなわち

$$
e^{i\pi}+1=0
$$

を **typed corollary** として読むことができる。ここで重要なのは、右辺の `0` が裸の数ではなく $D_C(C_{\mathrm{disc}})$ として得られた `0_A` である点である。これにより、`e^{i\pi}+1=0` は「対象が消える式」ではなく、**保存された単位が反転輸送と加法的に相殺され、その decategorified endpoint が `C_{\mathrm{disc}}` になる式**として読まれる。

#### 静的読解との接続

この橋梁系を受理するなら、faithful-full な静的読解は `A` における**endpoint identity**を見ており、方向定理の動的読解は同じ identity に到るまでの **path data** を見ていることになる。静的読解が追うのは

$$
e^{i\pi}+1=0 \quad \text{in } A
$$

という well-typed な終点等式であり、動的読解が追うのは

$$
\theta \mapsto \alpha_E(\theta)\mapsto E(\theta)\mapsto D_C(E(\theta))
$$

という生成過程である。ここまではまだ、静的読解を動的読解の endpoint projection と呼ぶ B 判定の言い方で止まっている。以下では、射程を `e^{i\pi}+1=0` の **$\pi$-sector** に固定し、その sector に限って A 判定を閉じるための共通担体と比較射を導入する。言い換えれば、本節が防衛するのは「任意の endpoint identity に対する普遍的 A」ではなく、「Euler 読解が狙っている $\pi$-endpoint に関する局所的 A」である。

#### 骨格完了の二投影候補

シリーズの正本語彙では、低次セルの貼り合わせと閉路完了により高次骨格が成立する構造運動を **骨格完了 (skeleton completion)** と呼ぶ。`創発 = WHAT の THAT 化` はその存在論的相であり、本稿で使う **閉路完了** は Euler 読解に限った局所語である。交代和的な台帳と不変量の残余は、この核の静的相としてなお接続定理候補に留保する。

この語彙で言い直すと、`e^{i\pi}+1=0` は骨格完了の **動的閉路完了**、`V-E+F=2` は同じ核を交代和で記帳した **静的閉路台帳** として並置される。現段階で defend できるのは、両者が **同一核の二投影候補** だという点までである。ここではその厳密同一性を主張せず、typed bridge が与える endpoint identity と、骨格台帳側の交代和が将来どの第三構造で接続されるかを開いたまま保持する。

#### 定義: 閉路完了機械 (closure-completion machine)

この「第三構造」に名前を与える。Euler の `\pi`-sector において、動的閉路完了と静的閉路台帳を同じ核から読むための typed な共通担体を **閉路完了機械 (closure-completion machine)** と呼ぶ。これはシリーズ全体の親核である `骨格完了` そのものではなく、**骨格完了の Euler 局所面における実装**である。

最小公理は 4 本に圧縮できる。

1. **CCM1: 骨格持ち上げ**
   低次セルの貼り合わせを経路化する累積写像 `\alpha:[0,\pi]\to[0,1]` と、その濾過持ち上げ `E(\theta)=C_{\alpha(\theta)}` が存在する。これは Euler 累積公理の核であり、「動的閉路完了」が path として走れることを保証する。
2. **CCM2: 位相輸送**
   位相輸送 `U:[0,\pi]\to A^\times` が `U(0)=1_A`, `U(\pi)=-1_A` を満たし、完全反転を加法圏 `A` の内部で表現できる。これは指数核公理の核であり、`e^{i\pi}` を単なる mnemonic でなく運動の transport law にする。
3. **CCM3: 端点型付け**
   decategorification `D_C:\mathrm{Filt}([0,1],\mathrm{WdSub}(C))\to A` が `D_C(E(0))=1_A`, `D_C(E(\pi))=0_A` を与える。これにより `+1` と `=0` は同じ型の中で読まれ、動的 endpoint が静的等式へ落ちる。
4. **CCM4: 比較証明義務**
   終点評価 `\operatorname{ev}_\pi` が `\pi`-sector で faithful かつ local full である。すなわち、静的側は動的側を潰しすぎず、逆に静的 endpoint は canonical path へ持ち上がる。これがないなら「同一核の二投影」は B 判定へ降格する。

直感的には、閉路完了機械は

- 何が進行しているかを `CCM1`
- どう反転するかを `CCM2`
- どこへ着地するかを `CCM3`
- その二つの見え方が本当に同じ機械から出ているかを `CCM4`

で固定する装置である。

#### 定義: bridge class $\mathrm{Bridge}(C,A;D_C)$

A 判定を狙うためには、静的読解と動的読解を「似た二つの物語」として置くのでなく、**同じ機械の二つの表示窓**として typed に固定しなければならない。そのために、まず原始データを派生量から切り分ける。

固定した decategorification

$$
D_C:\mathrm{Filt}([0,1],\mathrm{WdSub}(C))\to A
$$

に対し、Euler bridge の原始担体を

$$
\mathrm{Bridge}(C,A;D_C)
:=
\left\{
b=(\alpha_b,E_b,U_b)
\ \middle|\
\begin{array}{l}
\alpha_b:[0,\pi]\to[0,1],\ \alpha_b(0)=0,\ \alpha_b(\pi)=1\\[2pt]
E_b:[0,\pi]\to \mathrm{Filt}([0,1],\mathrm{WdSub}(C)),\ E_b(\theta)=C_{\alpha_b(\theta)}\\[2pt]
D_C(E_b(0))=1_A,\ D_C(E_b(\pi))=0_A\\[2pt]
U_b:[0,\pi]\to A^\times,\ U_b(0)=1_A,\ \dfrac{dU_b}{d\theta}=i\,U_b,\ U_b(\pi)=-1_A
\end{array}
\right\}
$$

と定義する。ここで重要なのは、`位相・忘却・decategorification・反転` を全部 1 つの tuple に詰めることではない。むしろ逆で、**どこまでが原始データで、どこからが表示量か**を分けることである。

以下では `\mathrm{Bridge}(C,A;D_C)` を **閉路完了機械の carrier** とみなす。すなわち、CCM1-CCM3 はこの carrier の存在条件であり、CCM4 はこの carrier から得られる comparison arrow に課される証明義務である。

各 $b\in \mathrm{Bridge}(C,A;D_C)$ に対し、導出量を

$$
\Omega_b(\theta):=U_b(\theta)\,D_C(E_b(0)),
\qquad
F_b(\theta):=\operatorname{Im}\Omega_b(\theta)
$$

で定める。これにより表示束

$$
\mathcal{U}_b:=\bigl(\alpha_b,E_b,\Omega_b,F_b\bigr)
$$

が得られる。したがって $\mathcal{U}$ は機械本体ではなく、bridge datum $b$ の**表示面**である。

#### 定義: 動的投影

動的読解は、この機械本体から path 表示だけを取り出す retaining map

$$
\Pi_{\mathrm{dynamic}}:
\mathrm{Bridge}(C,A;D_C)\to \mathrm{AdmPath}(C,A),
\qquad
b\mapsto
\Bigl(\theta\mapsto\bigl(\alpha_b(\theta),E_b(\theta),\Omega_b(\theta),F_b(\theta)\bigr)\Bigr)
$$

として与える。ここで

$$
\mathrm{AdmPath}(C,A):=\operatorname{Im}\bigl(\Pi_{\mathrm{dynamic}}\bigr)
$$

と置く。これで `AdmPath` は曖昧な外部集合ではなく、**閉路完了機械が実際に吐き出す path 表示の像**として固定される。方向定理の動的読解が見ているのは、まさにこの像である。

#### 定義: endpoint evaluation と $\pi$-sector

path 表示から終点等式だけを読む比較射を

$$
\operatorname{ev}_{\pi}:\mathrm{AdmPath}(C,A)\to \mathrm{EndEq}(A),
\qquad
\gamma\mapsto
\bigl(\Omega_{\gamma}(\pi)+D_C(E_{\gamma}(0))=D_C(E_{\gamma}(\pi))\bigr)
$$

で定義する。$\operatorname{ev}_{\pi}$ は path / phase / force を忘れ、終点で読める identity だけを残す。

ただしここで言う static 側は、`A` の任意の終点等式を相手にしているのではない。現在必要なのは、`e^{i\pi}+1=0` を含む **bridge-compatible な $\pi$-sector** だけである。そこで

$$
\mathrm{EndEq}_{\pi}(A)\subseteq \mathrm{EndEq}(A)
$$

を、境界条件

$$
D_C(E(0))=1_A,\qquad D_C(E(\pi))=0_A
$$

と位相輸送法則

$$
\frac{dU}{d\theta}=iU,\qquad U(0)=1_A,\qquad U(\pi)=-1_A
$$

に両立する endpoint equalities の類と定める。現段階では、この class は事実上 `e^{i\pi}+1=0` の typed representative を持つ sector である。

次に admissible path を faithful 証明に使える形へ絞る。$\mathrm{AdmPath}_{\triangle}(C,A)\subseteq \mathrm{AdmPath}(C,A)$ を、次の 2 条件を満たす path datum の類と定める。

1. **Face 条件**: すべての非自明区間が、`St(E_{\gamma}(\theta))` における非退化 2-simplex を少なくとも 1 つ含む
2. **SST 条件**: 各 2-simplex 上の signed oblivion が三角恒等式 $\Delta_F$ の相殺条件に従い、局所変形が `(+\lambda)+(-\lambda)=0` の形で only if に記録される

Face 条件は「見えない detour を許さない」ための検査面であり、SST 条件は「見えた detour が相殺されるときの型」を固定する局所規則である。

#### 定義: faithful/full certificate

このとき $\pi$-sector に関する faithful と full を別々に定義する:

$$
\mathrm{Faith}_{\pi}(C,A)
:=
\left\{
fa_C\ \middle|\
\operatorname{ev}_{\pi}:
\mathrm{AdmPath}_{\triangle}(C,A)/\!\sim\;\to \mathrm{EndEq}_{\pi}(A)
\text{ is injective}
\right\}
$$

$$
\mathrm{Full}_{\pi}(C,A)
:=
\left\{
fu_C\ \middle|\
\exists\ \operatorname{Lift}_{\pi}:\mathrm{EndEq}_{\pi}(A)\to \mathrm{AdmPath}_{\triangle}(C,A),
\ \operatorname{ev}_{\pi}\circ\operatorname{Lift}_{\pi}=\mathrm{id}
\right\}
$$

そして局所 certificate を

$$
\mathrm{FFCert}_{\pi}(C,A):=\mathrm{Faith}_{\pi}(C,A)\times \mathrm{Full}_{\pi}(C,A)
$$

と置く。

faithful clause は

$$
\operatorname{ev}_{\pi}(\gamma_1)=\operatorname{ev}_{\pi}(\gamma_2)
\Longrightarrow
\gamma_1\sim\gamma_2
$$

であり、ここで $\sim$ は endpoint-preserving reparameterization を表す。`Face Lemma` の言葉で言えば、同じ終点等式を与える 2 つの path が genuinely 異なるなら、その差分はどこかで 2-simplex の検査面に現れなければならない。SST 条件は、その局所差分が signed oblivion の非自明な残差として立つことを保証する。したがって終点が同じで残差も立たないなら、両 path の差は reparameterization に吸収される。

full clause は

$$
\forall q\in \mathrm{EndEq}_{\pi}(A)\ \exists \gamma\in \mathrm{AdmPath}_{\triangle}(C,A):
\operatorname{ev}_{\pi}(\gamma)=q
$$

である。ここで重要なのは、full の target が $\pi$-sector に限定されていることだ。局所 full の意味は、「この論文が主張したい typed endpoint は path として持ち上げ可能である」であって、「A の任意の終点等式がすべて Euler path を持つ」ではない。

Euler 累積公理の下で canonical lift

$$
\operatorname{Lift}_{\pi}(q_{\pi})(\theta)
:=
\bigl(\alpha_E(\theta),\,C_{\alpha_E(\theta)},\,e^{i\theta}D_C(E(0)),\,\operatorname{Im}(e^{i\theta}D_C(E(0)))\bigr)
$$

を取れば、

$$
\operatorname{ev}_{\pi}(\operatorname{Lift}_{\pi}(q_{\pi}))=q_{\pi}
$$

が直ちに従う。したがって local full は Euler path の canonical section として与えられる。

#### 定義: 静的投影

このとき静的読解は primitive な第二世界ではなく、同じ閉路完了機械から induced される第二の表示窓である。すなわち

$$
\Pi_{\mathrm{static}}:
\mathrm{Bridge}(C,A;D_C)\dashrightarrow \mathrm{StaticRead}(C,A)
$$

を

$$
\Pi_{\mathrm{static}}(b)
:=
\bigl(\operatorname{ev}_{\pi}(\Pi_{\mathrm{dynamic}}(b)),\ ff_b\bigr)
$$

で定める。ただし

$$
\mathrm{StaticRead}(C,A)
:=
\mathrm{EndEq}_{\pi}(A)\times \mathrm{FFCert}_{\pi}(C,A)
$$

であり、$ff_b\in \mathrm{FFCert}_{\pi}(C,A)$ が存在するときに限る。

したがって static 側が見る `e^{i\pi}+1=0` は、単なる終点等式ではない。それは

$$
\operatorname{ev}_{\pi}(\Pi_{\mathrm{dynamic}}(b))
=
e^{i\pi}+1=0
$$

という endpoint identity を、`動的読解の終点評価 + faithful/full 証明義務` として読んだ像である。ここで faithful/full 性は消えていない。むしろ static 側で見えている well-definedness は、comparison arrow と certificate を通じて初めて typed に回収されている。

#### 補題: $\operatorname{ev}_{\pi}$ は $\pi$-sector で faithful である

`Face Lemma` と `Stability Simplex Theorem` を仮定すると、

$$
\operatorname{ev}_{\pi}:
\mathrm{AdmPath}_{\triangle}(C,A)/\!\sim\;\to \mathrm{EndEq}_{\pi}(A)
$$

は faithful である。

*証明スケッチ.* `Face Lemma` は、非自明な path 差分があるなら、それは少なくとも一つの非退化 2-simplex に露出しなければならないことを言う。`SST` は、その 2-simplex 上の差分が signed oblivion の三角恒等式として記録され、局所的に `(+\lambda)+(-\lambda)=0` の形以外では消せないことを言う。したがって同じ endpoint equality を与える 2 つの path の差が全ての 2-simplex 上で消えているなら、その差は局所構造差ではなく parameterization の差にすぎない。ゆえに $\operatorname{ev}_{\pi}(\gamma_1)=\operatorname{ev}_{\pi}(\gamma_2)$ ならば $\gamma_1\sim\gamma_2$。□

#### 補題: $\operatorname{ev}_{\pi}$ は $\pi$-sector で full である

Euler 累積公理の下で、canonical lift $\operatorname{Lift}_{\pi}$ は

$$
\operatorname{ev}_{\pi}\circ\operatorname{Lift}_{\pi}=\mathrm{id}_{\mathrm{EndEq}_{\pi}(A)}
$$

を満たす。したがって $\operatorname{ev}_{\pi}$ は $\pi$-sector で full である。

*証明スケッチ.* $\mathrm{EndEq}_{\pi}(A)$ の任意の元は、境界条件 `D_C(E(0))=1_A`, `D_C(E(\pi))=0_A` と位相輸送 `U(\pi)=-1_A` を満たす endpoint identity である。Euler 累積公理は、そのような endpoint に対し標準 path $\theta\mapsto\alpha_E(\theta)$ を与える。これを `E(\theta)=C_{\alpha_E(\theta)}`, `\Omega(\theta)=e^{i\theta}D_C(E(0))`, `F(\theta)=\operatorname{Im}\Omega(\theta)` で持ち上げれば、終点評価は元の等式を返す。□

#### 系 (L4): 静的読解と動的読解は $\pi$-sector で同じ閉路完了機械の二投影である

上の faithful 補題と local full 補題を束ねると、`e^{i\pi}+1=0` をめぐる `\pi`-sector では、A 判定は本文中で短く閉じる。橋梁公理 $\mathrm{FFCert}_{\pi}(C,A)\neq\varnothing$ の下で、

$$
\Pi_{\mathrm{dynamic}}:\mathrm{Bridge}(C,A;D_C)\to \mathrm{AdmPath}(C,A)
$$

と

$$
\Pi_{\mathrm{static}}:\mathrm{Bridge}(C,A;D_C)\dashrightarrow \mathrm{StaticRead}(C,A)
$$

は、**同じ閉路完了機械の二つの forgetful projection** である。

*理由.* 動的側が見ているのは $b$ が吐き出す内部進行

$$
\theta\mapsto\bigl(\alpha_b(\theta),E_b(\theta),\Omega_b(\theta),F_b(\theta)\bigr)
$$

であり、静的側が見ているのはその同じ $b$ を

$$
\operatorname{ev}_{\pi}\circ\Pi_{\mathrm{dynamic}}
$$

で終点へ圧縮し、さらに faithful/full certificate を添えた像である。`L1+L2` によりこの圧縮は path 差分を潰しすぎず、`L3` により endpoint identity は canonical Euler path へ持ち上がる。したがって差は対象の違いではなく、「どこで内部進行を忘れるか」の違いにすぎない。ゆえに本稿がここで押す A は global A ではなく、`e^{i\pi}+1=0` の **$\pi$-sector に閉じた sectoral A** である。□

#### 定義: 骨格台帳投影

endpoint identity 側の静的投影だけでは、`V-E+F` 型の交代和はまだ見えない。そこで同じ閉路完了機械から、位相輸送と加法圏の係数をさらに忘れ、**2-skeleton の combinatorics** だけを残す第二の静的投影を定義する。

動的像 $\gamma=\Pi_{\mathrm{dynamic}}(b)$ に対し、その有限骨格支持を

$$
V_\gamma := \{\text{非退化 1-simplex または 2-simplex に現れる 0-cell}\},
$$

$$
E_\gamma := \{\text{support}(E_\gamma(\theta)) \text{ または } \partial_2 \text{ に現れる非退化 1-simplex}\},
$$

$$
F_\gamma := \{\text{AdmPath}_{\triangle}(C,A) \text{ が証人として持つ非退化 2-simplex}\}
$$

で定める。これにより 2-段の skeletal chain complex

$$
C^{\mathrm{sk}}_2(\gamma)\xrightarrow{\partial_2}
C^{\mathrm{sk}}_1(\gamma)\xrightarrow{\partial_1}
C^{\mathrm{sk}}_0(\gamma)
$$

が得られる。ここで `Face補題_ホモロジー厳密化.md` に従い、創発・detectability の最小閾値として本質的なのは単なる `F_\gamma \neq \varnothing` ではなく

$$
B^{\mathrm{ver}}_1(\gamma):=\operatorname{im}\partial_2

### 離散化関手の三層分解 — strict part, chain-map part, defect core

「Amari-Chentsov 複体から Walsh-Stokes 複体への chain map は存在するか」という問いは、そのままでは**型が粗すぎる**。functoriality_reduction の分析により、この問いは三層に分解され、上二層はすでに解決されている。

---

**層 A（射の合成としての関手性）— 解決済み**:

積ベルヌーイ族と座標射影からなる部分圏 $\mathrm{CubeExp}_{\mathrm{proj}}$ 上で、候補離散化 $D_{\mathrm{proj}}$ は **strict functor** である。

Lean 4 証拠: `restrict_functorial`（automath `Defect.lean`）。

$$D_{\mathrm{proj}}(\pi_{K \subseteq J} \circ \pi_{J \subseteq I}) = D_{\mathrm{proj}}(\pi_{K \subseteq J}) \circ D_{\mathrm{proj}}(\pi_{J \subseteq I})$$

**層 B（外微分との両立 = chain-map 性）— 解決済み**:

multilinear observable に制限すると、automath の `deltaSet` は混合偏微分の cell 積分に一致する:

$$\delta_A(f|_{\{0,1\}^I})(w) = (-1)^{|A|} \int_{[0,1]^A} \partial_A f(\iota_{A,w}(u))\,du$$

したがって $d \mapsto \mathrm{deltaSet}$、$\int \mapsto \mathrm{walshFlux}$ は cubical cochain のレベルで**定理**として閉じる。

**層 C（加法・合成との両立 = monoidal coherence）— 未解決の核**:

automath の本丸は $\Phi(x \oplus y) \neq \Phi(x) \oplus \Phi(y)$ であり、その差が carry defect $\kappa$ になる。これは層 A の strict composition とは別問題であり、ordinary functor の問題ではなく、**carry defect を 2-cell に持つ lax monoidal / defect-bearing bridge** の問題として立てるべきである。

automath の Lean 4 事実をそのまま読むと:

- 1-morphism では **strict**（`restrict_functorial`）
- monoidal coherence では **defect 付き**（`globalDefect_compose`、`restrict_stableAdd_carry_defect`）

よって full bridge の型は:

> **strict chain map + defect correction**

すなわち ordinary functor ではなく **pseudofunctor / defect-bearing bridge** として書くのが正しい。

---

**方向定理が defect を制御する理由**:

Paper I の方向定理 $F_{ij} \neq 0 \iff d\Phi \wedge T \neq 0$ を $[0,1]^n$ 上で読む:

- **$d\Phi \wedge T = 0$ のとき**: 曲率ゼロ。連続側で $F_{ij} = 0$。離散側で $\kappa = 0$。**層 A+B の strict chain map だけで閉じる。defect correction は不要。**
- **$d\Phi \wedge T \neq 0$ のとき**: 連続側で $F_{ij} \propto \sin(\theta)$。離散側で $\kappa \neq 0$（非自明な xor-cocycle）。**defect correction が必要になり、そのズレの連続側が $\Phi \cdot dT$、離散側が carry defect $\kappa$ に対応する。**

この草案の「繊維積 $P$ による統一」（§4 C2）はここに直結している。最初から global strict chain map を探すと型が粗すぎる。flat 部分で strict に合わせ、全体は pullback 上の一致と 2-cell で扱うべきである。

**層 C の理論内部への再定式化**:

### 第一障害可視化閾値 — `q=5` の統一的読み

ここまでの議論を一つの核概念へ圧縮する。

`q` ごとに次の 5 量を置く:

| 記号 | 定義 | 読み |
|:---|:---|:---|
| `C_q` | `\mathrm{tr}(\mathrm{Sym}^2 A_q)` | cooperative / copy 的 2 体 sector の可視量 |
| `X_q` | `\mathrm{tr}(\Lambda^2 A_q) = e_2(A_q)` | exclusion / anti-copy 的 2 体 sector の可視量 |
| `E_q` | `|\det(I-A_q)| - F_{2q-2}` | Fibonacci 背景からの算術的 excess |
| `K_q` | `[\kappa_q]`。flat strict chain map を global defect-bearing bridge へ延長する際の first obstruction class | 圏論的 obstruction の本体 |
| `k_q` | `1_{K_q \neq 0} = 1_{[\kappa_q] \neq 0}` | obstruction の可視化 proxy |

ここで強調すべきは、`2-cell / cocycle / obstruction class` は競合案ではなく**階層**だという点である。`K_q` の ontology は obstruction class に置かれるべきであり、その具体実現は

- bicategorical witness としての defect `2-cell`
- cochain representative としての defect `2-cocycle`

に分かれる。こうすることで、automath の Lean 4 事実 (`globalDefect_compose`) と、`q` をまたいで比較可能な不変量としての理論要請が両立する。`k_q` はこの class の本体ではなく、閾値を書き下すための観測用 proxy である。

このとき current official convention のもとで

$$q^* := \min \{\, q \mid X_q > 0,\; E_q > 0,\; k_q = 1 \,\}$$

を **第一障害可視化閾値** と呼ぶ。automath issue #25 のデータでは

- `q=4`: `(C_4, X_4) = (11, -7)`, `E_4 = 0`
- `q=5`: `(C_5, X_5) = (-7, 11)`, `E_5 = 11`

であり、理論の読みとしては `k_4 = 0`, `k_5 = 1` が要請される。したがって、暫定的に `q^* = 5` である。

Problem 10 の候補 reduction としては、連続側の曲率 class から直接 `\mathbb{Z}/2\mathbb{Z}` へ飛ぶのではなく、

$$H^2_{dR}(M_q) \xrightarrow{\mathrm{Int}_q} H^2(G_m; \mathbb{Z}) \xrightarrow{\rho_2} H^2(G_m; \mathbb{Z}/2\mathbb{Z})$$

を経由するのが自然である。ここで `\mathrm{Int}_q` は cubical 2-cell 上の flux 積分を basic carry quantum `\lambda_q` で整数化する conjectural lift であり、

$$K_q := [\kappa_q] := \rho_2(\mathrm{Int}_q([F_q]))$$

と置く。ここで `\lambda_q` は carry amplitude ではなく、single carry event に対応する **event quantum** と読む。振幅 refinement が必要なら second layer として `\lambda_q^{amp} = F_m \lambda_q` を導入すればよいが、Problem 10 の mod 2 class には不要である。`k_q` はこの class の visibility proxy にすぎない。

この定義の利点は、AlyciaBHZ の二つの問いを一つに束ねられる点にある。`q=5` では

1. `\mathrm{Sym}^2` と `\Lambda^2` の優勢が入れ替わる
2. `e_2(A_5) = 11` が Bowen-Franks excess `32 - 21 = 11` と一致する
3. `k_q` が初めて非零になり、chain map 問題が flat strict part から global defect part へ移る必要が初めて露出する

この三つは別々の異常ではなく、**同一の obstruction がスペクトル・算術・圏論の三つの面に投影された像**と読むのが自然である。

**二重独立方程式としての coefficient-level formalization (2026-04-24 Issue #38 AlyciaBHZ Corollary 6 吸収)**: 上の三つの同時立ち上がりのうち、$X_q$ と算術 excess に関する部分は AlyciaBHZ の signed companion framework で **二つの独立 coefficient 方程式の同時解** として形式化される。signed companion の Lemma (係数恒等式) と Theorem (det 公式) により:
