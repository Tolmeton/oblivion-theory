## §I Paper I 顔 — wedge tangent と方向性定理

context 1。σ の第 1 顔として, 微分幾何側の実装を提示する。Paper I (忘却論シリーズ I, `論文I_力としての忘却_草稿`) の wedge 2-form $F_{ij}$ を pivotal 2-圏対象 $\mathrm{TangentCtx}(M, F)$ として構成し, $F_{ij}$ が §0 核定理の Yoneda 関手値であることを示す。

### §I.1 Paper I の舞台 — $(M, g, C, \Phi, T)$ 統計多様体

- **統計多様体** $(M, g, C)$: $M$ は滑らかな多様体, $g$ は Fisher 計量, $C$ は Amari-Chentsov テンソル
- **Chebyshev 1-形式** $T_i := g^{jk} C_{ijk}$: 統計多様体の方向的感受性
- **忘却場** $\Phi \in C^\infty(M)$: 各点での忘却強度
- **忘却曲率** $F_{ij}$ := 「忘却で重み付けされた Chebyshev 1-形式 $\Phi T$ の外微分」

[SOURCE: 論文I §3.3 (`論文I_力としての忘却_草稿.md` 行 113-121) — $F_{ij}$ 定義 / 論文I §5.1 (行 211-233) — 方向性定理]

### §I.2 忘却曲率 $F_{ij}$ と方向性定理

**定義:**

$$F_{ij} := \partial_i A_j - \partial_j A_i = \frac{\alpha}{2}[d(\Phi T)]_{ij} = \frac{\alpha}{2}\left[(\partial_i \Phi)T_j - (\partial_j \Phi)T_i + \Phi(dT)_{ij}\right]$$

特に指数型分布族では $dT = 0$ より $F_{ij} = (\alpha/2)(d\Phi \wedge T)_{ij}$ に簡約される。

**方向性定理 5.1 [Paper I §5.1]:** 統計多様体 $(M, g, C)$ 上で, 任意の $i, j$ に対し

$$F_{ij} = 0 \iff d(\Phi T) = 0.$$

**対偶 (力の条件):** $F_{ij} \neq 0 \iff d(\Phi T) \neq 0$ — 忘却が方向的に非均一。

**wedge 反対称性:** $F_{ij} = -F_{ji}$ は反対称 2-テンソルの Hodge 内積既約性 (Schur 補題 / Bleecker §3.3) により強制される [SOURCE: 論文I §3.3.4 (行 133)]。これが §I.3 pivotal 対象性の核。

### §I.3 $\mathrm{TangentCtx}(M, F)$ の pivotal 2-圏対象性 (C11 §4.8.bis.2.a)

§III.2 の weak spherical 2-圏 $\mathbf{SignTri}$ の対象として $\mathrm{TangentCtx}(M, F)$ を次で構成:

| level | 内容 |
|:--|:--|
| **対象** | $M$ の点 $p$ (または infinitesimal tangent frame) |
| **1-射** $f: p \to q$ | 接写像 (smooth map の derivative) |
| **2-射** $\sigma: g\circ f \Rightarrow h$ | triangle $\Delta$ 上の $F_{ij}\|_\Delta \in \Lambda^2 T^*_p M$ |
| **右双対** $f^*$ | 接ベクトルの orientation reversal $v \mapsto -v$ の lift |
| **pivotal data** | wedge 反対称 $F_{ij} = -F_{ji}$ が $\sigma^* = -\sigma$ を canonical に与える |

**pivotal axiom verification (§D.1.2 保存):**

1. **右双対の存在:** 接空間は線形空間, orientation reversal は自動
2. **合成逆転** $(g\circ f)^* = f^* \circ g^*$: 微分の連鎖律で標準

## §III Stasheff 顔 — associahedra と Face5 Lemma

context 3。Stasheff tower $\{K_n\}_{n \geq 2}$ を 5 context の **背骨** として構成する。本章は §0 核定理で宣言された 2-圏 $\mathbf{SignTower}$ と自由-忘却随伴 $F_\text{sph} \dashv U$ を具体化し, §VI 合流章の SOURCE 根拠を据える。

### §III.1 Stasheff tower の全景

Stasheff (1963) は homotopy associativity の普遍的機構として **associahedra $K_n$** を構成した:

- **次元定理:** $\dim K_n = n - 2$
- **具体:** $K_2$ = 点 (退化) / $K_3$ = 線分 (= walking triangle $\Delta^2$ の 2-cell 補填) / $K_4$ = 五角形 (pentagon, Face5 minimum) / $K_5$ = 3 次元 associahedron / 一般 $K_n$ = $(n-1)$-associator coherence の最小 habitat
- **face inclusion:** $K_n \hookrightarrow K_{n+1}$ が tower 構造を成す
- **operadic structure:** $\{K_n\}_{n \geq 2}$ は $A_\infty$-operad の cell complex として閉じる

Stasheff tower は **$(n-1)$-associator coherence の free pivotal generator** としての普遍的役割を持つ。これが 5 context 背骨として選ばれる根拠である。

[SOURCE: Stasheff (1963) "Homotopy associativity of H-spaces I, II" Trans. AMS 108 / Markl-Shnider-Stasheff (2002) "Operads in Algebra, Topology and Physics" AMS / Mac Lane "Categories for the Working Mathematician" Ch. VII (pentagon axiom と coherence theorem)]
